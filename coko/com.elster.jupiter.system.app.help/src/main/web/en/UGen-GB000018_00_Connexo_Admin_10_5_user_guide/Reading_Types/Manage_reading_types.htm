Manage reading types

Monitor reading types

You can monitor the reading types in Connexo in a structured way in order to get a grip on the data that you can collect.

View reading type sets and reading types

You can have a clear overview of all the reading types that are available in Connexo and of the reading type sets that are grouping them.

5

Steps

1.   Go to Administration > Reading types management > Reading types.

This page shows the reading type sets that are available in Connexo.

2.   Click a reading type set name (Alias) to see its details, i.e. the reading type set’s basic parameters.

3.   On the details page of the reading type set, click Reading types to see the associated reading types and the complete set of parameters for each reading type separately.

4.   Enter the needed criteria in the filter bar on top of the Reading types page to narrow down the shown reading types to those that are relevant to your needs. Don’t forget to click Apply to execute your filter.

5.   In the overview table, select the needed reading type to see its full CIM details in the details pane below.
