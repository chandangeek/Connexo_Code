Understand relative periods

Relative periods in Connexo

Relative periods define a time window, for example the previous week. The user can define the exact time window during the configuration of the feature that uses the relative period. Then you will select an exact time point and then the time window will be calculated relative to this date.

Connexo uses relative periods for different purposes. In MultiSense for example they are used for data estimation and data export.

Relative periods for gas days

In contrast to electricity, a gas day does not start at midnight. Depending on the country this start is early in the morning for example 05:00. The start of the gas day can be entered in the config.properties file upon installation.

Because the start of a gas day is different from electricity, relative periods are also different calculated based on the start of the gas day.
