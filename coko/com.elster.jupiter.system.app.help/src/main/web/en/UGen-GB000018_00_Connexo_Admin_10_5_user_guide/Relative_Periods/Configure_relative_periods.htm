Configure relative periods

Add relative period

5

Steps

1.   Go to Administration > Relative periods> Relative periods.

2.   Click Add relative period and complete the following fields:

Parameter

Description

Name

Unique name

Category

The category of task for which the relative period will be used. Select one or more categories.

Start of relative period

oStart

 

oOn

 

 

 

oAt

oPreview

 

oSelect the start date of the relative period, select a specific date or a relative date.

oWhen you select a relative date, select on what day the period should start. The days of the week are only available when you selected to start the period x weeks ago in Start.

oSelect the start time.

oShows a preview of the start of the relative period based on the values you specified.

End of relative period

oEnd

 

oOn

 

 

 

oAt

oPreview

 

oSelect the end date of the relative period, select the current day or a relative date.

oWhen you select a relative date, select on what day the period should end. The days of the week are only available when you selected to start the period x weeks ago in End.

oSelect the end time.

oShows a preview of the end of the relative period based on the values you specified.

Preview

Shows a preview of the relative period based on the current date. You can change this date and time to see a different preview of the relative period.

3.   Click Add.

3

Note

There are by default seven relative periods available when Connexo is installed. These are Today, Yesterday, This week, This month, Previous week, Previous month and the Last 7 days.

Edit relative periods

You can edit existing relative periods to add new categories and extend the availability of a relative period or you can change name, start and end time.

5

Steps

1.   Go to Administration > Relative periods> Relative periods.

2.   Select the relative period that you want to edit.

3.   Click the actions icon  or the actions button and select Edit. 

4.   Make the necessary changes.

5.   Click Save.

 

1

Important

You can’t edit the category of a relative period that is in use. First, select another relative period in the objects where it is used. You can see where a relative period is used by looking at its details. See “View relative period usage”.

Remove relative periods

5

Steps

1.   Go to Administration > Relative periods> Relative periods.

2.   Click the actions icon  or the actions button and select Remove. 

3.   Click Remove.

 

1

Important

You can’t remove relative periods that are in use. First, select another relative period in the  objects where it is used. You can see where a relative period is used by looking at its details. See “View relative period usage”.
