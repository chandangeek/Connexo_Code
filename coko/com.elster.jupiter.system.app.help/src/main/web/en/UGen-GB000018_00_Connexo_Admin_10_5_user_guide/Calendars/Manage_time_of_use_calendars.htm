Manage time of use calendars

View a time of use calendar

In Connexo Admin you have an overview of all the time of use calendars that exist in the system. This allows you to decide which one will fit the best for your needs or to see if there are calendars missing for certain periods.

5

Steps

1.   Go to Administration > Calendars.

The master table shows Name, Category and Description of the calendars.

The detail view shows the following fields.

  • Start of calculations

  • Periods with their start date.

  • Day types.

  • Event types.

For a description of the attributes for time of use calendars, see “Import time of use calendar”.

Preview a time of use calendar

You can view the contents of a time of use calendar, i.e. its entire setup with events, event types, etc., to verify if it fits your needs.

5

Steps

1.   Go to Administration > Calendars.

2.   Select a time of use calendar.

3.   Click the action icon  in the selected row or the click the Actions button in the details pane. Then click View preview.

4.   In the filter on top of the page, select the start day for the week you want to view. You will see six days of the calendar starting from the day in your filter. The first day of the current week is set by default.

5.   The time of use calendar is graphically represented with the week days on the horizontal axis and the hours of the day on the vertical axis. For each day, you can see the event type(s) used during that day. Hovering over each day segment shows a tooltip with the name, the start and the end hour of the event type.

6.   In the detail pane you see the periods, day types and event types that are used in the shown week.

Figure: Preview a time of use calendar.

 

Remove a time of use calendar

You can remove a time of use calendar if it is no longer required.

5

Steps

1.   Go to Administration > Calendars.

2.   Select a time of use calendar.

3.   Click the action icon  in the selected row or the click the Actions button in the details pane. Then click Remove.

3

Note

Removing a time of use calendar is only possible if it is not assigned to any device type.
