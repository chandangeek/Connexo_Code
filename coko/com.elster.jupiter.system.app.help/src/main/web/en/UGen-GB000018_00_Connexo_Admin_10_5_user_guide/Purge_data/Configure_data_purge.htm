Configure data purge

Set data purge settings

5

Steps

1.   Go to Administration > Data purge > Data purge settings.

The table shows the different types of data (category) that can be removed from the system together with the period the data is retained (retention period). The retention period is the number in the “Purge data older than” column multiplied by 30 days. The available categories contain the following data:

Category

Description

Interval data

All interval data.

Day/month profiles

All day/month profiles.

Register data

All data that has been collected from the registers.

Event data

All event data from the logbooks.

Logging data

All logging data in the system: connection log, communication log, data export log, etc.

Journal data

All data that has been removed is not permanently removed, but is maintained in the system.

 

2.   Click in the Purge data older than column to change the retention period. Write a number, the retention period will be this number multiplied by 30 days.

3.   Click Save.
