Understand web services

Web services in Connexo

A web service is defined as a software system used to share business logic, data and processes across a network using a specifically developed interface. More specifically these are clients and servers that communicate using XML messages. For the formatting of these XML messages we typically use SOAP (Simple Object Access Protocol) and REST (Representational State Transfer).

To add web services contact your system administrator, you need to add these in the software code. In the Connexo MultiSense interface you can only add and configure web service endpoints for the web services added by the system administrator.
