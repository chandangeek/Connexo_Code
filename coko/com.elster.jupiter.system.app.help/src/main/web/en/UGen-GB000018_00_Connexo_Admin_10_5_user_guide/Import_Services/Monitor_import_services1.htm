View all import service history

5

Steps

1.   Go to Administration > Data import > Import history.

The history table shows every started import together with the duration and status. When you select an item in the table, the details pane on the bottom of the page shows more specific details of this import.

2.   Use the filter on top of the page to only see the history of one import service or filter on other settings, click Apply. Click Clear all to remove the filter criteria.

3.   To sort the filter results click Add sort. Remove the sorting by clicking the remove icon  next to the sorting criteria or click Clear all.

4.   Click the actions icon  and select View log to see a detailed logging of the history item.

View the history of a single import service

5

Steps

1.   Go to Administration > Data import > Import services > click the actions icon  next to the import service and select View history or click an import service in the table and click History in the menu on the left.

The history shows every started import of this import service together with the duration and status. When you select an item in the table, the details pane on the bottom of the page shows more specific details of this import.

2.   Click the actions icon  and select View log to see a detailed logging of the history item.
