Manage custom attributes sets

View custom attribute sets

5

Steps

1.   Go to Administration > Custom attribute set management > Custom attribute sets.

2.   Use the filter on top to focus on the custom attribute sets that are relevant to you. Select an object type to see its custom attribute sets.

3.   Click Apply.

The attributes of the selected custom attribute set are shown in a details pane under the table. The column Identifier shows the unique identifier for each attribute of the selected custom attribute set. You can use this information to prepare an import file to update the attributes on a group of objects. Either copy and paste the unique identifiers or simply export the details pane into csv using the export button  on top of the table. For more information about importing values of custom attribute sets see “Import values of existing custom attribute sets”.

Edit privileges of custom attribute sets

The privileges for a specific custom attribute set can be changed to overrule the

5

Steps

1.   Go to Administration > Custom attribute set management > Custom attribute sets.

2.   Click the actions icon  next to the custom attribute set and select Edit levels.

3.   (De)Select one or more levels for the different actions to further detail access to the selected custom attribute set. For instance, you may want the meter expert to have access to all custom attribute sets except custom attribute set x.

The roles corresponding to the different levels are configured in User management. See “Add user role”. The configured privileges for custom attribute sets can be seen in the application section MultiSense and in the resource section Custom attribute sets privileges.

4.   Click Save.
