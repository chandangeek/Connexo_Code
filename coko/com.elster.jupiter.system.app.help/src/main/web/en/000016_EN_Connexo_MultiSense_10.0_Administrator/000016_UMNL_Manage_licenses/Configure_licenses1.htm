Configure licenses

View licenses

5

Steps

1.   Go to Administration > Licensing > Licenses

Different license specific features are shown:

Parameter

Description

Application

The application for which you acquired a license.

Type

Type of license

Description

Description of the acquired license

Status

The status of the license

Activation date

The date when the license was activated.

Expiration date

The date when the license will expire.

Grace period

Period after the expiration date when the license is still active. This provides you with the time to renew or change your license.

License coverage

Details of the different aspects that are covered by the license, e.g. number of licensed users, licensed protocols, etc
Figure: Licenses page

 

Upload licenses

5

Steps

1.   Go to Administration > Licensing > Licenses.

2.   Click Upload licenses.

3.   Click Browse. Select a license file and click Upload.

3

Note

All license files need to have the .lic extension.

Upgrade licenses

5

Steps

1.   Go to Administration > Licensing > Licenses.

2.   Click Upgrade licenses.

3.   Click Browse. Select the license file and click Upgrade.
