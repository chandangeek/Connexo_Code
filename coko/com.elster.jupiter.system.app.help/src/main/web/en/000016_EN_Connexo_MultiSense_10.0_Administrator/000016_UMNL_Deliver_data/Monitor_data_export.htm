View export task history

5

Steps

1.   Go to Administration > Data export > Data export tasks.

2.   Click the actions icon  and select View details or click the export task name.

3.   Click History in the menu on the left. A table is shown containing all history of the selected export task. Details of every line are given in the details pane below.

Filter data export history

5

Steps

1.   Go to Administration > Data export > Data export tasks.

2.   Click the actions icon  and select View details or click the export task name.

3.   Click History in the menu on the left.

4.   Select or type criteria in the filter fields to search for export tasks that started or finished in a specific time window or the export period contains a specific date.

5.   Click Apply.

The applied filters are shown on top of the page. Click the remove icon  to remove single filters or click Clear all to remove all applied filters.

View data export log

5

Steps

1.   Go to Administration > Data export > Data export tasks.

2.   Click the actions icon  and select View details or click the export task name.

3.   Click History in the menu on the left.

4.   Click the actions icon or the action button  and select View log. Details of the export task are shown on top of the page. All logs are shown in the table below with timestamp, log level and message.
