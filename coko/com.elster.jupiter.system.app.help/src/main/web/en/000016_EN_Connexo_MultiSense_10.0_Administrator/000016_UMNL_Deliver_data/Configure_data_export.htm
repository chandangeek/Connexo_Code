Configure data export

Manage data export tasks

View export task details

5

Steps

1.   Go to Administration > Data export.

2.   Click Data export tasks.

3.   Click an export task in the table, the details are shown on the bottom of the page. You can also see the export task details when you click the export task name or the actions icon  and select View details.

The data export task contains 5 subcategories:

Category

Description

Data sources

Describes what data (reading types) from which devices (device groups) will be exported.

Schedule

When is the task scheduled.

Also shows the status of the export task.

Data options

The period to export the data of

Formatter

The format of the exported data

Formatter properties

Properties of the exported format

4.   Click History in the menu on the to see the history of the selected export task.

Add export tasks

1

Important

First configure Device groups as described in “Manage device groups”.

5

Steps

1.   Go to Administration > Data export > Data export tasks.

2.   Click Add export task.

Complete the following fields:

Category

Parameter

Description

 

•   Name

•   Unique name

Data sources

•   Device group

 

 

•   Reading type

•   Select the device group to export data of. Click Add device group to add a new group.

•   Add one or more reading types as described in “Add reading type” to indicate which data will be exported.

Schedule

•   Recurrence

•   Start on

•   Indicate the recurrence of the task.

•   Define the start date of the task.

Data options

•   Export period

•   Select one of the predefined export periods or create a new one by clicking Add export period. For more information see “Add relative period”.

Formatter

•   Formatter

•   Select the formatter to define the format of the exported data.

File format

•   File name prefix

 

•   File extension

•   Export folder path

•   The prefix for the file name, this will be the same for all exported files.

•   Extension of the file

•   Specify the path to the folder where you want to save the exported data. When left empty the default folder of the application server is used.

Formatter properties

•   Separator

•   Select which separator to use to separate the values in the exported file.

3.   Click Add.

Add reading type

5

Steps

1.   Click Add reading types.

2.   Complete the filter fields to find the correct reading types.

3.   Click Apply.

Click the remove icon  to remove single filters or click Clear all to remove all applied filters.

4.   Select one or more reading types from the table.

You can see the reading type details by clicking the information icon .

5.   Click Add.
