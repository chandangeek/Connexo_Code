Deliver data

Want to export data, this chapter covers all features inside Admin such as export periods and tasks that need to be configured in order to be able to export your data. Defining device groups of which data needs to be exported is described in the Connexo MultiSense manual.

What would you like to do?

Export data

1. Add a static device group or Add a dynamic device group to group all devices that should be viewed together.

2. Add relative period to define the period to export data of.

3. Add export tasks to define when to export data and in what format.

4. Schedule data export in the data export task configurationor Export on request.
