View reading types

5

Steps

1.   Go to Administration > Reading types management > Reading types.

This page shows all reading types that are available in Connexo. When you select one of the reading types in the table, you can see the CIM code and the CIM code details in the details pane below.

Filter reading types

5

Steps

1.   Go to Administration > Reading types management > Reading types.

2.   Select or type criteria in the filter fields on top of the page to search for specific reading types.

3.   Click Apply.

4.   Deselect items in the drop down menu to remove single filters or click Clear all to remove all applied filters.

5.   Sort the results by adding a sorting criteria by clicking Add sort.

6.   Change the order of sorting by clicking the applied sorting criteria.

7.   Remove sorting criteria by clicking the remove icon next the criteria.
