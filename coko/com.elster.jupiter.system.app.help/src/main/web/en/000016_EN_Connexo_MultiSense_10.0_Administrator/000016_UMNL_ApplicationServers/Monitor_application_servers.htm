View application servers

5

Steps

1.   Go to Administration > Application server > Application servers.

2.   Select an application server in the table.

The application server details are shown on the bottom of the page. This shows the status of the server, import and export path but also provides a link to the message and import services running on this server.

3.   Click the application server name to open the application serverâ€™s landing page containing more detailed information.

View message services on application server

5

Steps

1.   Go to Administration > Application server > Application servers > click the application server name.

2.   Click Message services in the menu on the left or click the message services link available in the application server details.

The message services page shows all the services added to this application server together with their status and the number of threads.

View import services on application server

5

Steps

1.   Go to Administration > Application server > Application servers > click the application server name.

2.   Click Import services in the menu on the left or click the import services link available in the application server details.

The import services page shows all the services added to this application server together with their status.

View webservice endpoints on application server

5

Steps

1.   Go to Administration > Application server > Application servers > click application server name > select Webservice endpoints in the menu on the left.

2.   The table shows all webservice endpoint added to the application server together with their status.

3.   The details pane below all detailed information of the endpoint selected in the table. You can also see this information by clicking the endpoint name in the table.
