Manage processes

View system processes

5

Steps

1.   Go to Administration > Process management > Processes.

The overview table shows all processes that have been added and configured in Flow. For each process you can see which version it is, its status and the object it is associated with.

The associated object specifies the objects on which the process can run. If you select Device, then this process will be available to the user in Connexo MultiSense when starting a process on a device.

(De)Activate processes

5

Steps

1.   Go to Administration > Process management > Processes.

2.   Click the actions icon  next to the process and select Activate or Deactivate.

3

Note

When activating a process, you need to specify the object on which the process will run, the specific context of this object and the required user privileges. This is explained in “Edit processes”.

Edit processes

5

Steps

1.   Go to Administration > Process management > Processes.

2.   Click the actions icon  next to the process and select Edit process. You can edit the following fields:

Parameter

Description

Start on

Define on what object the process will run, e.g. Device.

Start when in

Specific categories the object must be part of to be able to start the process, e.g. device state for a device. The available options depend on the selection in Start on. Check only the applicable categories.

Start when allowed

Privileges of the users that are allowed to start processes. All available privileges are shown. Check only the applicable privileges.

3.   Click Save.
