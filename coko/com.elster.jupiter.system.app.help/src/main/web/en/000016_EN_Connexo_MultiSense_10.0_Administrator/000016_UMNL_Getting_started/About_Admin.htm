About Admin

What is Admin ?

Connexo Admin  is the administration application that is standard incorporated in every Connexo product. It allows you to configure the Connexo platform, such as users, user access, application servers and licenses.

For whom is this guide intended?

This user guide is specifically written for the system administrator and serves as a guide to configure all parameters such as users, roles, application servers and licenses.
