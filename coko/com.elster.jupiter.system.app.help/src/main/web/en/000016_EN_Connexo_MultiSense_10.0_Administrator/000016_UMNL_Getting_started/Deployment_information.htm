Deployment information

View components

5

Steps

1.   Go to Administration > Deployment information > Components list.

This page shows all deployed bundles together the application of the bundle, the version and its status.

2.   Use the filter on top of the page to only see specific bundles, click Apply.

3.   Remove filter by deselecting items in the drop down menu or remove all filters by clicking Clear all.

4.   Export the list of bundles by clicking the grid export button above the table.

View system information

5

Steps

1.   Go to Administration > Deployment information > System information.

The system information is split up in two parts:

Section

Description

Server information

Information concerning the server such as java properties, database location, server memory, etc.

Client information

Information concerning the browser that is used to run Connexo.

2.   Copy the information and send it to Connexo support.
