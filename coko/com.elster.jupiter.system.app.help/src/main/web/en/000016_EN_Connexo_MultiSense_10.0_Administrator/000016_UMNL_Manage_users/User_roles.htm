User roles

Configure user roles

Add and edit a user role

5

Steps

1.   Go to Administration > User management

2.   Click Roles.

3.   Click the actions icon to edit a user role.

4.   Click Add role. Fill in the following fields:

Parameter

Description

Name

Unique name

Description

General description of the role

Applications

Determine the access to the different licensed applications.

  •  For no access to all applications click No access.

  • To grant full access to all applications click Full control.

  • To change access right to individual applications click the actions icon next to the application in the table.

Resources

Each application has different resources for which privileges can be determined. Resources can be devices, users, domains, etc. The privileges to the individual resources can be adjusted by clicking the actions icon. Select the action and the level, where available.

5.   Click Save.

Assign user role(s) to users

5

Steps

1.   Go to Administration > User management.

2.   Click the actions icon  next to a user and select Edit.

3.   Select one or more roles.

4.   Click Save.
