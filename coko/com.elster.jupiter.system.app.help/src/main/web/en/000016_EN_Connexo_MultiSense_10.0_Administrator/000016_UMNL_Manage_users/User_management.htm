User management

In Connexo you can easily manage users and assign different roles to them. This ensures that only the correct people will be able to access certain data. This is achieved through privileges that are assigned to specific user roles. Every user will then have one or more roles assigned to him or her.

This chapter describes all user related administration that can be configured in Admin.

What would you like to do?

Grant privileges to meter operators with different expertises

By introducing different levels in privileges, Connexo MultiSense offers you the possibil­ity to assign roles that are specific for certain devices such as gas and electricity meters. To achieve this perform the following steps:

1. Add user role.

2. Determine the access to the different licensed applications. Add levels specifying access rights to the resource for device commands and the resource for device security. Select level 1 for one type of devices and create another role with level 2 privileges for another group of devices.

3. Assign user role(s) to users.
