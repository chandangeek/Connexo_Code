Configure users

Synchronize users from an LDAP user directory

5

Steps

1.   Go to Administration > User management > User directories.

2.   Select a user directory, click the actions icon  and select Synchronize.

3.   Click Select users.

4.   Select one or more users from the list and click Add.

5.   Click Save.

6.   Next activate the user (“Activate user”) and assign user roles (“Assign user role(s) to users”).

3

Note

If changes have been made to the users in the LDAP directory click the Synchronize button on the user selection page. If you remove a user, this user will no longer be able to login to the application.

Edit users

5

Steps

1.   Go to Administration > User management > Users.

2.   Click the actions icon  and select Edit.

You can change the description and the roles of the user and also set the language of the user.

3.   Click Save.

Activate user

5

Steps

1.   Go to Administration > User management > Users.

2.   Click the actions icon  and select Activate.

3

Note

If you inactivate a user, this user will no longer be able to login to the application.
