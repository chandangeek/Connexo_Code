Manage service call custom attribute sets

View service call custom attribute sets

View the custom attribute sets for service calls along with their properties and attributes. See “Custom attributes”.

5

Steps

1.   Go to Administration > Custom attribute sets management > Custom attribute sets.

2.   In the Custom attribute set type drop down, select Service call and click Apply.

Only the custom attribute sets for service calls are now shown in a paged table. The following columns are available:

Parameter

Description

Name

Name of the custom attribute set.

Type

Value for this field is always Service call.

Required

Indication if the custom attribute set is required or not. Value is Yes or No.

View levels

Not applicable for this type of custom attribute sets.

Edit levels

Not applicable for this type of custom attribute sets.

Time-sliced

Always No.

No actions are available for this type of custom attribute sets.

3.   In the details pane of the selected custom attribute set, the following information is shown for each of its attributes:

Parameter

Description

Order

Order of the attribute in the custom attribute set.

Attribute name

Name of the attribute.

Required

Indication if the attribute is required or not. Value is Yes or No.

Type

Type of attribute.

Default

Default value for this attribute.

Description

Free textual description.
