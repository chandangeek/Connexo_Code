Configure log level

Configure the log level for service call types to avoid that the database is flooded with unnecessary lines of logging information. The default log level is Warning.

5

Steps

1.   Go to Administration > Service calls > Service call types.

2.   Select the desired service call type.

3.   Click the actions icon  in the column Actions next to the selected service call type or click the button Actions in the details pane.

4.   Select Change log level.

5.   In the popup Select log level, select the desired log level.

The following log levels are available:

  • Severe

  • Warning

  • Info

  • Config

  • Fine

  • Finer

  • Finest

6.   Click Save.
