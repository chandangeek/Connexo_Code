Manage service call types

View service call types

See what service call types are installed and the life cycle that is associated with them. From here you are able to manage the service call types.

5

Steps

1.   Go to Administration > Service calls > Service call types.

All deployed service call types, including the deprecated ones, are shown in a paged table. The following fields are available:

Parameter

Description

Type

Name of the service call type.

Version

Version of the service call type.

Status

State of the service call type. It can be Active or Deprecated.

Log level

Log level for the service calls linked to the selected service call type. See “Configure log level”.

Life cycle

Indication of the service call status diagram that has been used.

2.   The details pane below the table shows the linked custom attribute sets and their corresponding fields. A complete overview of service call custom attribute sets can be seen in Custom attribute sets management. See “View service call custom attribute sets” and “Custom attributes”.
