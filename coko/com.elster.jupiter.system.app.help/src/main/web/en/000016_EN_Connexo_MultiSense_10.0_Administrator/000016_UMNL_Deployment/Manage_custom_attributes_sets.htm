Manage custom attributes sets

View custom attribute sets

5

Steps

1.   Go to Administration > Custom attribute set management > Custom attribute sets.

2.   Use the filter on top of the page to only see the custom attribute sets of the object type you are interested in.

3.   Click Apply.

Edit privileges of custom attribute sets

5

Steps

1.   Go to Administration > Custom attribute set management > Custom attribute sets.

2.   Click the actions icon  next to the custom attribute set and select Edit levels.

3.   Select one or more levels for the different actions. The roles corresponding to the different levels are configured here “Add user role”. The privileges for custom attribute sets can be found under the MultiSense application the resource Custom property sets privileges.

4.   Click Save.
