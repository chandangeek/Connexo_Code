Configure import services

View import services

5

Steps

1.   Go to Administration > Data import > Import services.

The overview page contains a table that shows al configured import services. In the table you can see the importer the service uses as well as the application that uses the importer. This table also shows the status of the import service and what type of file can be imported.

By selecting the import service in the table, you can see more details in the details pane on the bottom of the page.

2.   Click the import service name.

The page shows all details of the import service and also allows you to edit, remove and (de)activate the service.

Default importers

Connexo provides different default importers that allow the import of different types of data to perform different actions in the Connexo applications. These different importers are described here, the usage of the configured import services in MultiSense is described in the MultiSense user guide.

Importer

Description

Calendar importer

Import time of use (TOU) calendars.

Usage point importer

Import usage points using a CSV file.

Device connection attributes importer

Importer developed to import different connection attributes of different connection methods and devices.

Device security attributes importer

Importer developed to import different security attributes of different security settings and devices.

Devices commissioning importer

Importer to transition devices to the commissioning state.

Devices decommissioning importer

Importer to transition devices to the decommissioning state.

Devices installation importer

Importer to install devices.

Devices remove importer

Importer to remove selected devices from Connexo.

Devices shipment importer

Importer to add devices to Connexo in bulk.

Device activation/deactivation importer

Importer to transition devices to the active or inactive state.

Device readings importer

Importer designed to add meter readings to selected devices.

Add import service

5

Steps

1.   Go to Administration > Data import > Import services.

2.   Click Add import services. Complete the following fields:

Parameter

Description

Name

Unique name

File importer

Select the importer that will perform the import. Depending on the selected importer extra parameters may need to be defined.

Import folder

Define the location where files that need to be imported are located. This path is relative to the import location of the application server.

File pattern

Define which files can be imported.

Folder scan frequency

Select the frequency to check whether new files are available for import.

In progress folder

Define the location where in progress files will be stored. This path is relative to the import location of the application server.

Success folder

Define the location where successfully imported files will be stored. This path is relative to the import location of the application server.

Failure folder

Define the location where files that failed to be imported will be stored. This path is relative to the import location of the application server.

3.   Click Add.

4.   Click the actions icon  and select Activate.

3

Note

Before import services can be used in the other Connexo applications, this import service first has to be added to an application server.

Edit import services

5

Steps

1.   Go to Administration > Data import > Import services.

2.   Click the actions icon  and select Edit. All parameters except the file importer can be edited.

3.   Click Save.

Remove import services

5

Steps

1.   Go to Administration > Data import > Import services.

2.   Click the actions icon  and select Remove.

3.   Click Remove.

Configure import services on an application server

5

Steps

1.   Go to Administration > Application server > Application servers.

2.   Create a new one (“Add an application server”) or edit an existing application server (“Edit application server”).

3.   Click Add import services.

4.   Select one or more of the configured import services.

5.   Remove import services by clicking the actions icon  and selecting Remove.
