Import services

Understand import services

Import services are configured in Admin and used in the other Connexo applications. When creating new import services Connexo offers different default file importers that allow the import of different types of data to perform different actions. During configuration you need to specify the location where files that will be imported have to placed. Next to the import folder you will also specify the folder for in progress files and files of successful and failed import. All this information is also visible in the applications using the import services when you look at the import service details.

Before import services can be used in the different Connexo applications, they need to be added to an application server. This application server can then pick up the import task and execute it.
