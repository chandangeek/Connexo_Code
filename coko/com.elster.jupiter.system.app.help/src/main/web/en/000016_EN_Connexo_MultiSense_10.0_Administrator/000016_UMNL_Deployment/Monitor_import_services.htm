Monitor import services

View import service history

5

Steps

1.   Go to Administration > Data import > Import services > select an import service and click History in the menu on the left or click the actions icon  and select View history.

The history shows every started import together with the duration and status. When you select an item in the table, the details pane on the bottom of the page shows more specific details of this import.

2.   Click the actions icon  and select View log to see a detailed logging of the history item.
