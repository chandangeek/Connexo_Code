Manage certificates

Introduction

Using the trust store you can manage all general certificates that are used by different objects. The object specific certificates are managed in a different location in Admin .

You can add certificates or create an CSR (Certificate Signing Request) to later add the certificate.

Understand CSR in Admin

When you do not have a certificate already available you can create a CSR in Admin . This request usually contains the public key for which the certificate should be issued, identifying information (such as domain name) and integrity protection (e.g., a digital signature).

In Connexo you have to specify the key attributes and CSR attributes in the same step. For certificate generation the offline method is used. This means that once you completed the parameters, you need to download the CSR and then send it the CA (Certificate Agency).

The CA will generate the certificate which you can download and then import into Admin . Generating a CSR will get the certificate status Requested. Once you import the certificate the status will change to Available.

Add CSR

5

Steps

1.   Go to Administration > Certificate management > Certificates.

2.   Click the actions button, select Add CSR and complete the following fields:

Parameter

Description

Alias

Define the alias of the CSR. Must be unique.

Key attributes

Key encryption method

Select the method to encrypt the key.

CSR attributes

Certificate type

Shows all certificate types that are available in the system. Select one of these types.

Subject

This is automatically filled in once the other parameters are specified.

Common name (CN)

Name for which you are requesting the SSL Certificate.

Organisational unit (OU)

The department of the organisation.

Organisation (O)

The name of the company.

Locality (L)

Provide full name.

State or province (S)

Provide full name.

Country (C)

Country two digit code.

3.   Click Add.

The CSR has been created and the label of the certificate changes to requested.

4.   Click the actions icon and select Download CSR.

5.   Send CSR to CA to retrieve certificate.

Import certificate

When you receive a certificate from the CA after you sent a CSR, you can import this certificate to the CSR instance you created in Admin .

You can also import certificates to existing certificate instances. When there is already a certificate available, this will be overwritten by the newly imported certificate.

The certificate file format needs to be DER-encoded to be able to import it into Connexo.

5

Steps

1.   Go to Administration > Certificate management > Certificates.

2.   Click the actions icon next to the certificate instance and select Import certificate.

3.   Click Select file and select the certificate to upload.

4.   Click Import.

Add certificate

5

Steps

1.   Go to Administration > Certificate management > Certificates.

2.   Click the actions button, select Add certificate and complete the following fields:

Parameter

Description

Alias

Provide a unique alias for the certificate.

Certificate file

Click Select file and search and select the certificate file.

3.   Click Add.

Download certificate

5

Steps

1.   Go to Administration > Certificate management > Certificates.

2.   Click the actions icon next to the certificate instance and select Download certificate.

Remove certificate

5

Steps

1.   Go to Administration > Certificate management > Certificates.

2.   Click the actions icon next to the certificate instance and select Remove.

You cannot remove certificates that are still in use by an object in Connexo.

3.   Confirm by clicking Remove.
