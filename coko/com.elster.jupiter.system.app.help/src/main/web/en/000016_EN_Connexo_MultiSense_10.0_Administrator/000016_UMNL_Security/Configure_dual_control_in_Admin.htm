How to configure dual control in Admin

To be able to have dual control for different processes, default roles are available in Connexo. The user administrator just has to appoint the right role to the right user.

There are two roles needed for the dual control mechanism:

Role

Description

Dual control administrator

A user with this role is able to appoint which people need to approve alterations to specific parameters.

Approver

A user with this role needs to approve the alteration of the parameter. All users with this role for a specific parameter need to approve before the alteration is effective.

There is an approver role available per feature that requires dual control.

Only the dual control administrator can assign this role to a user.

The user administrator assigns the Dual control administrator role to specific users. These users can then assign the approver role to other specific users. For more information on how to configure and assign user and user roles see “User management”.
