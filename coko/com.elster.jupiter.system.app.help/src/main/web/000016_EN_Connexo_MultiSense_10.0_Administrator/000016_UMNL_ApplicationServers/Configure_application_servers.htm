Configure application servers

Manage application servers

Add an application server

1

Important

When you create an application server you need to stop and restart the Connexo instance that will execute that application server.

5

Steps

1.   Go to Administration > Application server > Application servers.

2.   Click Add application server and complete the following fields:

Parameter

Description

Name

Name of the server that the application runs on.

Export path

Default path to store exported files and data. The path can be overruled in the export task configuration.

Import path

Default path to import data from. The path can be overruled in the import service configuration.

Message services

Select the message services that need to be executed by the application server by removing those that do not need to be executed. This can be done by clicking the remove icon .

Change the number of threads or the status of the service by clicking the value.

Message services can be added when you click Add message services.

Import services

By default all configured import services are added to the application server. Remove import services by clicking the remove icon , add import services by clicking Add import services.

3.   Click Add.

4.   Click the actions icon  and select Activate.

3

Note

  • It is recommended that the name of the Application server should be the same as the name of the Connexo host server. If the hostname of the server is longer than 14 characters the user must add a custom application server with a different name than the hostname of the  server. In this case the system administrator must specify the custom Application server name in <connexo_install_dir>/conf/config.properties:com.elster.jupiter.server.name= <custom_appserver_name> and restart the service.

  • When you activate an application server this server becomes available in the application instead of being activated as such.

Edit application server

5

Steps

1.   Go to Administration > Application server > Application servers.

2.   Click the actions icon  next to the application server you want to edit and select Edit.

3.   Change the application server parameters

4.   Add, edit or remove message and import services.

5.   Click Save.

Remove application servers

5

Steps

1.   Go to Administration > Application server > Application servers.

2.   Click the actions icon  and select Remove to delete the application server. Click Remove in the dialogue that appears.
