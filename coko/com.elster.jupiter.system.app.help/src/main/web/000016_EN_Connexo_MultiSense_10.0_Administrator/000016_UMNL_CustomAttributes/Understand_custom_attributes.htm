Custom attributes in Connexo

Connexo offers the possibility to add different extra custom attributes to your system. This allows you to add additional information which is not covered by MultiSense â€™s basic CIM model, but which is relevant for the customer. This can for example be for integration with other applications, processes, operations, etc. This also allows you to tune the application to your specific regional needs

Custom attribute sets cannot be added using the interface. All custom attribute sets are coded and installed on the system. Default values and custom attribute set properties are hard coded, but can be seen in the interface.
