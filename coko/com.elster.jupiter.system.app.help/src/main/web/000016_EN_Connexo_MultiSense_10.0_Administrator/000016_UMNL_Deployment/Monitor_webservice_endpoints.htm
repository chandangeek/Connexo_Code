Monitor webservice endpoints

View webservice endpoints

5

Steps

1.   Go to Administration > Webservices > Webservice endpoints.

2.   The table shows all configured webservice endpoints together with their type, the webservice they use, status and log level. When a webservice is not available this is indicated by an exclamation icon next to the webservice name.

3.   The details pane shows more extensive details of the endpoint selected in the table. You can also see these details by clicking the webservice endpoint name in the table.

View webservice endpoint logging

5

Steps

1.   Go to Administration > Webservices > Webservice endpoints > click a webservice endpoint name > click Logging in the menu on the left.

2.   View all logging lines by scrolling down the table. The amount of log lines displayed can be altered by selecting another number in the Log lines per page field.

3.   Export the logging by using the export button on top of the table.
