Configure application servers

View application servers

5

Steps

1.   Go to Administration > Application server.

2.   Click Application servers.

The application server page shows all servers that have been configured together with their status. Application server details are shown in the details pane on the bottom of the page.

Add an application server

5

Steps

1.   Go to Administration > Application server.

2.   Click Application servers.

3.   Click Add application server and complete the following fields:

Parameter

Description

Name

Name of the server that the application runs on.

Export path

Default path to store exported files and data. The path can be overruled in the export task configuration.

Message services

Select the message services that need to be executed by the application server by removing those that do not need to be executed. This can be done by clicking the actions icon  and selecting Remove.

Change the number of threads by clicking the value.

Message services can be added when you click Add message services.

4.   Click Add.

5.   Click the actions icon  and select Activate.

3

Note

When you activate an application server this server becomes available in the application instead of being activated as such.

Edit and remove application servers

5

Steps

1.   Go to Administration > Application server.

2.   Click Application servers.

3.   Click the actions icon  and select Remove to delete the application server. Click Remove in the dialog that appears.
