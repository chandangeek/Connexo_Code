About Admin

What is Admin?

Connexo Admin is the administration application that is standard incorporated in every Connexo product. It allows you to configure the Connexo platform, such as users, user access and licenses.

For whom is guide intended?

This user guide is specifically written for the system administrator and serves as a guide to configure all parameters such as users, roles and licenses.
