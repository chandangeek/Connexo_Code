Understand the interface

General layout

All pages in Admin have a similar lay-out. To facilitate using the application the general layout of the app is given here.

Figure: Connexo layout.

 

A. Application center

B. Different menus: shortcut to the Administration page.

C. Collapse or expand navigation.

D. Return to homepage.

E. Online help

F. Logout: place in the application to logout.

G. Work area: space that changes for every page, it is the location where the configuration and analysis happens.

Overview page layout

When you click a specific feature within Admin, then in most cases an overview page of that feature will open. Throughout the application these overview pages have a similar layout.

The figure and text below show and explain the layout of the overview pages together with the actions that you can perform using the icons on the page.

Figure: Overview page layout.

 

A. Breadcrumb: represents where you are located in the application. Allows easy navigation to previous pages.

B. Overview table: this table shows all the items that have already been added to the system. Different actions can be performed on the items in the table, e.g. they can be edited, removed, activated, viewed, etc.

C. Details pane: shows some more details of an item that is selected in the table. By default the first item is selected.

D. Add button: button designated to add new items to the table, can also be another action depending on the feature you are viewing the overview page of.

E. Action icon: clicking this icons shows the different actions that can be performed on the selected item.

F. Action button: clicking this icons shows the different actions that can be performed on the selected item.
