Introduction to Admin

In Admin different system administration features are spread over different pages. A small introduction on the different pages is given here.

Administration

Via the administration page all system administration is grouped together. Here the system administrator can manage user directories, users and user roles and assign privileges. From the Administration page different licenses can be added or upgraded.

Using the Admin application you can set the parameters to purge your data, as well as adding and configuring application servers. Different parameters to export and import your data can be configured in Admin such as relative periods and import services.

On the Administration page you can also find deployment information to send to our support team in case of problems.
