Steps

1.   Go to Administration > Data import > Import services.

The overview page contains a table that shows al configured import services. In the table you can see the importer the service uses as well as the application that uses the importer. This table also shows the status of the import service and what type of file can be imported.

By selecting the import service in the table, you can see more details in the details pane on the bottom of the page.

2.   Click the import service name.

The page shows all details of the import service and also allows you to edit, remove and (de)activate the service.

Default importers

Connexo provides different default importers that allow the import of different types of data to perform different actions in the Connexo applications. These different importers are described here, the usage of the configured import services is described in the Connexo MultiSense and Connexo Insight user guides. MultiSense MultiSense

Importer

Description

Calendar importer

Import time of use (TOU) calendars.

Usage point importer

Device security attributes importer

Importer developed to import different security attributes of different security settings and devices.

Importer to install devices.

Devices shipment importer

Importer to add devices to Connexo in bulk.

Device activation/deactivation importer

Importer to transition devices to the active or inactive state.

Device readings importer

Importer designed to add meter readings to selected devices.
