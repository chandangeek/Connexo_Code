User roles

Configure user roles

Add user role

5

Steps

1.   Go to Administration > User management > Roles.

2.   Click Add role. Fill in the following fields:

Parameter

Description

Name

Unique name

Description

General description of the role

Applications

Determine the access to the different licensed applications.

  •  For no access to all applications click No access.

  • To grant full access to all applications click Full control.

  • To change access rights to individual applications click the actions icon next to the application in the table.

Resources

Each application has different resources for which privileges can be determined. Resources can be devices, users, domains, etc. The privileges to the individual resources can be adjusted by clicking the actions icon. Select the action and the level, where available.

3.   Click Add.

Edit user roles

5

Steps

1.   Go to Administration > User management > Roles.

2.   Click the actions icon  and select Edit.

3.   Edit the role and click Save.

Assign user role(s) to users

5

Steps

1.   Go to Administration > User management > Users.

2.   Click the actions icon  next to a user and select Edit.

3.   Select one or more roles.

4.   Click Save.

Remove a user role

It is possible to remove a user role that is not in use.

5

Steps

1.   Go to Administration > User management > Users.

2.   Click the actions icon  next to a user and select Remove.

3

Note

For safety reasons the Administrators role cannot be removed.
