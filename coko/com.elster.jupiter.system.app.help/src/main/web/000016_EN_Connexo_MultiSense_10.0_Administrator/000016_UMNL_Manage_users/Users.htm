Understand user directories

A user directory is a place where you store information about users. User information includes a person's full name, user name, password, email address and other personal information.

Using the Admin application you can connect to external user directories such as for example your companyâ€™s LDAP. The local directory stores internal user and group information (e.g. admin, batch executor) in the Connexo database, but should not be used to add external users.

Add user directory

5

Steps

1.   Go to Administration > User management > User directories.

2.   Click Add user directory and complete the following fields:

Parameter

Description

Name

Unique name for the user directory.

URL

The URL of the user directory.

Back-up URL

The back-up URL, if available.

Security protocol

Defines the security of the connection between Connexo and the active directory.

Type

Type of directory can be LDAP or Active directory.

User

An LDAP username with sufficient privileges to view the sections of the directory that contain the information for LDAP users.

Password

The password of the user entered above.

User base DN

The location of the user database on the user directory.

3.   Click Add.

4.   Click Set as default if you want this user directory to be the default one upon login. For the not default user directory you first have to specify the directory name before the username e.g. Local/admin.

Edit user directories

5

Steps

1.   Go to Administration > User management > User directories.

2.   Click the actions icon  and select Edit.

Name and type can no longer be edited.

3.   Click Save.

Remove user directory

5

Steps

1.   Go to Administration > User management > User directories.

2.   Click the actions icon  and select Remove. This option is only possible when the user directory is not the default directory.

3.   Click Remove.
