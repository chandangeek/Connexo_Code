View data purge history

5

Steps

1.   Go to Administration > Data purge.

2.   Click Data purge history.

The data purge history shows the different data purge tasks that have been performed. In the top table the different data purge tasks are shown together with the date the purge started, the duration and the status.

3.   Click a data purge task to see the data purge settings of that task. Note that these settings can be different than the current data purge settings because these were the settings when the viewed task was performed.

View data purge log

5

Steps

1.   Go to Administration > Data purge > Data purge history.

2.   Click the actions icon next to the data purge task and select View log.

On top of the page the data purge task details are shown: when the task started and the status of the task. Below the data purge task details there is a table containing all logs together with the timestamp, log level and message. By default the log lines are sorted based on the timestamp.
