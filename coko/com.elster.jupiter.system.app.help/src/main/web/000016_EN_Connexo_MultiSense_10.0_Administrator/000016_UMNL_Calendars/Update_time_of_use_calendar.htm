Update by re-importing the time of use calendar

You can update a time of use calendar by re-importing it into Connexo. Updating the time of use calendar is mostly done to add specific events such as variable holidays and other special periods that need an adjusted rate.

To import the calendar, use the existing import framework and exactly the same name as the one that was used for the earlier import. Any changes will simply overwrite the existing information in the time of use calendar.

If data had already been calculated with the previous version of the calendar, then, recalculate the data as from the start year. If no data had been calculated yet or if the previous version was Ghost, then no calculation is required.

3

Note

The status of the time of use calendar is Ghost if it originates from the device and does not exist in Connexo.

1

Important

  • If you change the start year, all data before that date won’t be calculated.

  • Some delay may occur when first using the updated calendar because time series for an entire year will be calculated. Afterwards, new series will be calculated via background process.
