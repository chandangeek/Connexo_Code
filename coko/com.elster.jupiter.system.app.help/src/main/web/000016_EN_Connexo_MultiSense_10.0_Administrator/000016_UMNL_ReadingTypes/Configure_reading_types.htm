Configure reading types

Add reading types

It is possible to add one or more reading types at the same time.

5

Steps

1.   Go to Administration > Reading types management > Reading types.

2.   Click Add reading types and complete the following fields:

Parameter

Description

Alias

Part of the name of the reading type that is common for all reading types added in this step.

Specify by

Select how to specify the reading type:

o   Enter the complete CIM code.

  • Use the form which will generate the CIM code for you.

CIM code form

With the CIM code form you can select each of the 18 CIM attributes from a drop down menu containing all possible values. Next to every value the number of the CIM attribute is shown.

You can select or multiple values from the drop down menu. Below the form you can see the total amount of reading types that will be added. A maximum of 1000 reading types can be added at the same time.

3.   Click Add.

4.   Click Add in the pop-up window.

5.   Click the actions icon  and select Activate.

Edit single reading types

To limit the amount of reading types visible in the other applications, deactivate the reading types that will not be used.

5

Steps

1.   Go to Administration > Reading types management > Reading types.

2.   Click the actions icon  and select Edit.

3.   Change the reading type alias.

4.   Click Save.

Edit reading types in bulk

5

Steps

1.   Go to Administration > Reading types management > Reading types.

2.   You can use the filters as described in “Filter reading types”.

3.   Click Bulk action.

4.   Select to apply the change on all (filtered) reading types or select specific reading types.

5.    Click Next.

6.   Select Edit. This action will only be successful if all selected reading types have the same alias.

7.   Click Next.

8.   Change the reading type alias.

9.   Click Next.

10.   Click Confirm. The change is now queued to be executed.

11.   Click Finish.

Activate/deactivate reading types in bulk

5

Steps

1.   Go to Administration > Reading types management > Reading types.

2.   You can use the filters as described in “Filter reading types”.

3.   Click Bulk action.

4.   Select to apply the change on all (filtered) reading types or select specific reading types.

5.    Click Next.

6.   Select Activate or Deactivate. When you deactivate the reading types, they will no longer be available for selection in the configuration of different features in the other Connexo applications. You are still able to collect and save data from reading types that you inactivate.

7.   Click Next.

8.   Click Confirm. The change is now queued to be executed.

9.   Click Finish.
