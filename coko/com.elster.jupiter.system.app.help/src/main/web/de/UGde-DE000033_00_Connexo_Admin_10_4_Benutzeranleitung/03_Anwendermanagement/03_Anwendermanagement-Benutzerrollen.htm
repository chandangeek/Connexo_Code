Anwenderrollen konfigurieren

Anwenderrolle hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendermanagement > Rollen.

2.   Klicken Sie auf Rolle hinzufügen. Füllen Sie die folgenden Felder aus:

Parameter

Beschreibung

Name

Eindeutiger Name

Beschreibung

Allgemeine Beschreibung der Rolle

Anwendungen

Bestimmen Sie den Zugang zu den verschiedenen lizenzieren Anwendungen.

  •  Für keinen Zugang zu allen Anwendungen klicken Sie auf Kein Zugang.

  • Um vollständigen Zugang zu allen Anwendungen zu gewähren, klicken Sie auf Vollständige Kontrolle.

  • Um die Zugangsrechte zu einzelnen Anwendungen zu ändern, klicken Sie auf das Aktionssymbol  neben der Anwendung in der Tabelle.

Ressourcen

Jede Anwendung hat verschiedene Ressourcen für die Rechte bestimmt werden können. Ressourcen können Geräte, Anwender, Domänen usw. sein. Die Rechte für die einzelnen Ressourcen können durch Anklicken des Aktionssymbols eingestellt werden. Selektieren Sie die Handlung und die Ebene, wo verfügbar.

3.   Klicken Sie auf Einfügen.

Anwenderrollen bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendermanagement > Rollen.

2.   Klicken Sie auf das Aktionssymbol

 und selektieren Sie Bearbeiten.

3.   Bearbeiten Sie die Rolle und klicken Sie Speichern.

Anwendern Anwenderrollen zuweisen

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendermanagement > Anwender.

2.   Klicken Sie auf das Aktionssymbol

 neben einem Anwender und selektieren Sie Bearbeiten.

3.   Selektieren Sie eine oder mehrere Rollen.

4.   Klicken Sie auf Speichern.

Eine Anwenderrolle löschen

Man kann eine Anwenderrolle löschen, die nicht verwendet wird.

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendermanagement > Anwender.

2.   Klicken Sie auf das Aktionssymbol

 neben einem Anwender und selektieren Sie Löschen.

3

Anmerkung

Aus Sicherheitsgründen kann die Rolle Administratoren nicht gelöscht werden.
