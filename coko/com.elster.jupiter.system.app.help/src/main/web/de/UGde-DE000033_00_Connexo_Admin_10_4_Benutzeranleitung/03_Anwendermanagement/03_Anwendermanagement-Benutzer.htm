Anwenderverzeichnisse verstehen

Ein Anwenderverzeichnis ist ein Ort, wo Sie Informationen über Anwender speichern können. Zu den Anwenderinformationen gehören der vollständige Name der Person, der Nutzername, das Passwort, die E-Mail-Adresse und andere persönliche Informationen.

Unter Verwendung der Connexo Admin Anwendung, können Sie externe Anwenderverzeichnisse wie beispielsweise Ihre Unternehmens LDAP verbinden. Das lokale Verzeichnis speichert interne Anwender und Gruppeninformation (z.B. Admin, Batchvollstrecker) in der Connexo-Datenbank, sollte aber nicht zum Hinzufügen externer Anwender verwendet werden.

Anwenderverzeichnis hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendermanagement > Anwenderverzeichnisse.

2.   Klicken Sie auf Anwenderverzeichnis hinzufügen und füllen Sie die nachfolgenden Felder aus:

Parameter

Beschreibung

Name

Eindeutiger Name für das Anwenderverzeichnis.

URL

Die URL des Anwenderverzeichnisses.

Back-up URL

Die Back-up URL, wenn verfügbar.

Sicherheitsprotokoll

Definiert das Sicherheitsverschlüsselungsprotokoll für die Verbindung zwischen Connexo und dem aktiven Verzeichnis. Abhängig von der Auswahl in diesem Feld, erscheinen weitere Konfigurationsfelder.

Quelle

Dieses Feld erscheint, wenn im Sicherheitsprotokoll das TLS oder SSL Verschlüsselungsprotokoll selektiert wird. Die Quelle kann ein Zertifikat oder ein Trust Store sein.

Zertifikat-Alias

Dieses Feld erscheint, wenn im Sicherheitsprotokoll und Zertifikatedas TLS oder SSL Verschlüsselungsprotokoll als Quelle selektiert wird. Selektieren Sie das notwendige Zertifikat aus der Liste der verfügbaren Zertifikate.

Trust Store

Dieses Feld erscheint, wenn im Sicherheitsprotokoll und Trust Storesdas TLS oder SSL Verschlüsselungsprotokoll als Quelle selektiert wird. Selektieren Sie den notwendigen Trust Store aus der Liste der verfügbaren Trust Stores.

Typ

Der Verzeichnistyp kann LDAP oder Aktives Verzeichnis sein.

Anwender

Ein LDAP Benutzername mit ausreichenden Rechten, zum Anzeigen der Abschnitte des Verzeichnisses, das die Informationen für LDAP Anwender enthält.

Passwort

Das Passwort des Anwenders, wie oben eingegeben.

Anwenderbasierte DN

Der Speicherort der Anwenderdatenbank im Anwenderverzeichnis.

3.   Klicken Sie auf Einfügen.

4.   Klicken Sie Als Standard einstellen, wenn Sie möchten, dass dieses Anwenderverzeichnis bei Anmeldung das Standardverzeichnis ist. Für das Anwenderverzeichnis, das nicht Standard ist, müssen Sie erst den Verzeichnisnamen vor dem Nutzernamen angeben, z.B. Lokal/Admin.

Anwenderverzeichnisse bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendermanagement > Anwenderverzeichnisse.

2.   Klicken Sie auf das Aktionssymbol

 und selektieren Sie Bearbeiten.

Der Name und der Typ können nicht mehr bearbeitet werden.

3.   Klicken Sie auf Speichern.

Anwenderverzeichnis löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendermanagement > Anwenderverzeichnisse.

2.   Klicken Sie auf das Aktionssymbol

 und selektieren Sie Löschen. Diese Option ist nur möglich, wenn das Anwenderverzeichnis nicht das Standardverzeichnis ist.

3.   Klicken Sie auf Löschen.
