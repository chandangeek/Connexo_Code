Arbeitsgruppen verwalten

Arbeitsgruppen anzeigen

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendermanagement > Arbeitsgruppen.

Diese Tabelle zeigt alle konfigurierten Arbeitsgruppen gemeinsam mit ihren Konfigurationsdetails.

Arbeitsgruppe hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendermanagement > Arbeitsgruppen.

2.   Klicken Sie auf Arbeitsgruppe hinzufügen und füllen Sie die nachfolgenden Felder aus:

Parameter

Beschreibung

Name

Geben Sie der Arbeitsgruppe einen eindeutigen Namen.

Beschreibung

Geben Sie eine Beschreibung der Arbeitsgruppe an.

Benutzer

Klicken Sie auf Anwender selektieren um Anwender zur Arbeitsgruppe zu fügen. Selektieren Sie die Anwender und klicken Sie auf Hinzufügen.

3.   Klicken Sie auf Einfügen.

Arbeitsgruppe bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendermanagement > Arbeitsgruppen.

2.   Klicken Sie auf das Aktionssymbol

 neben der Arbeitsgruppe, die Sie bearbeiten möchten und selektieren Sie Bearbeiten.

3.   Ändern Sie die Parameter der Arbeitsgruppe, Sie können nur Beschreibung und Anwender bearbeiten.

4.   Klicken Sie auf Speichern.

Arbeitsgruppe löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendermanagement > Arbeitsgruppen.

2.   Klicken Sie auf das Aktionssymbol

 neben der Arbeitsgruppe, die Sie löschen möchten und selektieren Sie Löschen.
