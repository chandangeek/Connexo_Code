Anwendermanagement

In Connexo können Sie Anwender einfach verwalten und ihnen verschiedene Rollen zuweisen. Das gewährleistet, dass nur die richtigen Personen Zugang zu bestimmten Daten haben. Das wird über Rechte erreicht, die bestimmten Anwenderrollen zugewiesen werden. Jeder Anwender hat dann eine oder mehrere Rollen, die ihm zugewiesen werden.

Dieses Kapitel beschreibt die anwenderbezogene Verwaltung, die in Connexo Admin konfiguriert werden kann.

Was möchten Sie tun?

Messtellenbetreibern mit verschiedenen Kompetenzen Rechte zuweis­en

Die Einführung verschiedener Berechtigungsebenen, bietet Ihnen Connexo MultiSense die Möglichkeit Rollen zuzuweisen, die für bestimmte Geräte wie Gas- und Stromzähler spezi­fisch sind. Dafür gehen Sie wie folgt vor:

1. Anwenderrolle hinzufügen.

2. Bestimmen Sie den Zugang zu den verschiedenen lizenzieren Anwendungen. Fügen Sie Ebenen ein, welche die Zugangsrechte zu den Ressourcen der Gerätekommandos und den Ressourcen für die Gerätesicherheit präzisieren. Selektieren Sie Ebene 1 für einen Gerätetypen und erstellen Sie eine weitere Rolle mit Rechten der Ebene 2 für eine weitere Gerätegruppe.

3. Anwendern Anwenderrollen zuweisen.
