Lizenzmanagement

Ihre Lizenzen definieren, welche Eigenschaften verfügbar sind, aber auch die Anzahl der Nutzer, die Sie hinzufügen können. Lizenzen können einfach aktualisiert oder erneuert werden. Alle lizenzrelevanten Informationen und Handlungen werden in Connexo Admin auf den Lizenzseiten, die in diesem Kapitel beschrieben werden, zentralisiert.
