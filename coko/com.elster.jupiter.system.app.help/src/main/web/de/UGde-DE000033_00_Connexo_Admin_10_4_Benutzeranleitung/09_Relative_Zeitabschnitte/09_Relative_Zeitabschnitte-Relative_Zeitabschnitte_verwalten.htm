Details des relativen Zeitabschnittes anzeigen

Sie erhalten einfach einen Überblick über alle relativen Zeitabschnitte, die an verschiedenen Stellen in Connexo verfügbar sind.

5

Schritte

1.   Öffnen Sie Verwaltung > Relative Zeitabschnitte > Relative Zeitabschnitte..

2.   Selektieren Sie den benötigten relativen Zeitabschnitt, um dessen Details im nachfolgenden Detailfenster anzuzeigen.

3.   Klicken Sie auf den Namen des relativen Zeitabschnitts, um dessen Detailseite zu öffnen.

Bei allen Ereignissen haben Sie eine Vorschau der selektierten relativen Zeitabschnitte, die deren Start- und Endzeit aufgrund eines Referenzdatums und Zeit angibt.

Im Anzeigemodus können Sie die Vorschau-Planung ändern, indem Sie das Datum und die Uhrzeit der Referenzzeit ändern.

Verwendung des relativen Zeitabschnittes anzeigen

Sie können sehen, wo ein einzelner relativer Zeitabschnitt in Connexo verwendet wird. Das ist wichtig, wenn Sie einen relativen Zeitabschnitt, der aktuell verwendet wird, bearbeiten oder löschen möchten. Sie müssen zuerst andere relative Zeitabschnitte in diesen Instanzen selektieren.

5

Schritte

1.   Öffnen Sie Verwaltung > Relative Zeitabschnitte > Relative Zeitabschnitte > selektieren Sie einen relativen Zeitabschnitt in der Tabelle.

2.   Klicken Sie auf Verwendung im Menü an der linken Seite.

3.   Selektieren Sie eine Kategorie aus dem Menü, um zu sehen, welche Aufgaben im relativen Zeitabschnitt verwendet werden.

4.   Klicken Sie auf Anwenden.

In der Tabelle sehen Sie die Namen der Aufgaben, wo die aktuellen relativen Zeitabschnitte verwendet werden. Die Tabelle gibt auch die Art der Aufgabe, die Anwendung, in der sie läuft und wann die Aufgabe erneut laufen wird, an.
