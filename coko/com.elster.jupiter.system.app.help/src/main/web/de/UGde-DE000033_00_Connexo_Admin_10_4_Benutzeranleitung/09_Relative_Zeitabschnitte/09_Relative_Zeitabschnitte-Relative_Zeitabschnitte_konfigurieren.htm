Relative Zeitabschnitte konfigurieren

Einen relativen Zeitabschnitt hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Relative Zeitabschnitte  > Relative Zeitabschnitte .

2.   Klicken Sie auf Einen relativen Zeitabschnitt hinzufügen und füllen Sie die nachfolgenden Felder aus:

Parameter

Beschreibung

Name

Eindeutiger Name

Kategorie

Die Kategorie der Aufgabe, für die der relative Zeitabschnitt verwendet wird. Selektieren Sie eine oder mehrere Kategorien.

Anfang des relativen Zeitabschnitts

oAnfang

 

oEin

 

 

 

oZum

oVorschau

 

oSelektieren Sie das Anfangsdatum des relativen Zeitabschnitts, selektieren Sie ein bestimmtes Datum oder ein relatives Datum.

oWenn Sie ein relatives Datum selektieren, selektieren Sie, an welchem Tag der Zeitabschnitt beginnt. Die Tage der Woche sind nur verfügbar, wenn Sie in Start wählen, dass der Zeitabschnitt vor x Wochen beginnt.

oSelektieren Sie die Anfangszeit.

oZeigt eine Vorschau des Anfangs des relativen Zeitabschnitts auf der Grundlage der Werte, die Sie angeben.

Ende des relativen Zeitabschnitts

oEnde

 

oEin

 

 

 

oZum

oVorschau

 

oSelektieren Sie das Enddatum des relativen Zeitabschnitts, selektieren Sie den aktuellen Tag oder ein relatives Datum.

oWenn Sie ein relatives Datum selektieren, selektieren Sie, an welchem Tag der Zeitabschnitt endet. Die Tage der Woche sind nur verfügbar, wenn Sie in Ende wählen, dass der Zeitabschnitt vor x Wochen beginnt.

oSelektieren Sie die Endzeit.

oZeigt eine Vorschau des Endes des relativen Zeitabschnitts auf der Grundlage der Werte, die Sie angeben.

Vorschau

Zeigt eine Vorschau des relativen Zeitabschnittes auf der Grundlage des aktuellen Datums. Sie können dieses Datum und die Zeit ändern, um eine andere Vorschau des relativen Zeitabschnitts zu sehen.

3.   Klicken Sie auf Einfügen.

3

Anmerkung

Es stehen standardmäßig sieben relative Zeitabschnitte zur Verfügung, wenn Connexo installiert wird. Diese sind Heute, Gestern, Diese Wochen, Dieser Monat, Vorige Woche, Voriger Monat und die letzten 7 Tage.

Relative Zeitabschnitte bearbeiten

Sie können bestehende relative Zeitabschnitte bearbeiten, um neue Kategorien einzufügen und die Verfügbarkeit eines relativen Zeitabschnitts zu erweitern oder Sie können den Namen, die Anfangs- und Endzeit ändern.

5

Schritte

1.   Öffnen Sie Verwaltung > Relative Zeitabschnitte > Relative Zeitabschnitte..

2.   Selektieren Sie den relativen Zeitabschnitt, den Sie bearbeiten möchten.

3.   Klicken Sie auf das Aktionssymbol oder die Aktionstaste und selektieren Sie Bearbeiten.

4.   Nehmen Sie die notwendigen Änderungen vor.

5.   Klicken Sie Speichern.

 

1

Wichtig

Sie können die Kategorie eines relativen Zeitabschnitts, der verwendet wird, nicht bearbeiten. Selektieren Sie zuerst einen anderen relativen Zeitabschnitt in den Objekten, wo er verwendet wird. Sie können sehen, wo ein relativer Zeitabschnitt verwendet wird, wenn Sie dessen Details betrachten. Siehe “Verwendung des relativen Zeitabschnittes anzeigen”.

Relative Zeitabschnitte löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Relative Zeitabschnitte  > Relative Zeitabschnitte .

2.   Klicken Sie auf das Aktionssymbol  oder die Aktionstaste und selektieren Sie Löschen.

3.   Klicken Sie auf Löschen.

 

1

Wichtig

Sie können relative Zeitabschnitte, die verwendet werden, nicht löschen. Selektieren Sie zuerst einen anderen relativen Zeitabschnitt in den Objekten, wo er verwendet wird. Sie können sehen, wo ein relativer Zeitabschnitt verwendet wird, wenn Sie dessen Details betrachten. Siehe “Verwendung des relativen Zeitabschnittes anzeigen”.
