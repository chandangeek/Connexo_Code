Endpunkte der Web Services konfigurieren

Web Service Endpunkte verwalten

Endpunkte der Web Services hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Web Services > Web Services-Endpunkte.

2.   Klicken Sie auf Web Service Endpunkt hinzufügen. Füllen Sie die folgenden Felder aus:

Parameter

Beschreibung

Name

Geben Sie einen eindeutigen Namen ein.

Web Service

Selektieren Sie einen Web Service. Abhängig von dieser Auswahl müssen verschiedene Parameter definiert werden.

URL Pfad

Erforderlich für eingehende Web Services. Bietet einen eindeutigen Pfad für die URL des nachfolgenden Formats https://<host>:<port>/<path>.

URL

Erforderlich für ausgehende Web Services. Bietet eine eindeutige URL im folgenden Format  https://<host>:<port>/<path>.

Log Level

Selektieren Sie die Log Ebene des Endpunkts. Wenn Sie eine hohe Log Ebene selektiert haben, führt das zu einem detaillierten Protokoll mit hoher Datenbanknutzung.

Suchanfrage

Wenn Sie diese Option selektieren, erzeugt Connexo eine Datei mit dem Namen, die Sie angegeben haben, mit allen Suchdetails.

HTTP Kompression

Komprimiert ausgehende http-Daten und akzeptiert eingehende komprimierte http-Daten.

Validierung des Schemas

Validieren Sie die XML-Schemata.

Bestätigung

Fügen Sie eine Bestätigungsebene ein. Wenn Sie die Basisbestätigung selektieren, müssen Sie den Nutzernamen und das Passwort angeben. Bei eingehenden Services können Sie eine Nutzerrolle angeben.

3.   Klicken Sie auf Einfügen.

Web Service Endpunkte aktivieren/deaktivieren

5

Schritte

1.   Öffnen Sie Verwaltung > Web Services > Web Services-Endpunkte.

2.   Klicken Sie auf das Aktionssymbol neben dem Endpunkt, den Sie aktivieren/deaktivieren möchten.

3.   Selektieren Sie Aktivieren/Deaktivieren.

Endpunkte der Web Services bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Web Services > Web Services-Endpunkte.

2.   Klicken Sie auf das Aktionssymbol  neben dem Endpunkt, den Sie bearbeiten möchten und selektieren Sie Bearbeiten.

3.   Die Parameter des Web Service Endpunkts ändern.  Beachten Sie, dass nicht alle Parameter bearbeitet werden können, wenn der Endpunkt aktiv ist.

4.   Klicken Sie auf Speichern.

Endpunkte der Web Services löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Web Services > Web Services-Endpunkte.

2.   Klicken Sie auf das Aktionssymbol  neben dem Endpunkt, den Sie löschen möchten und selektieren Sie Löschen.

3.   Bestätigen Sie und klicken Sie auf Löschen.

3

Anmerkung

Anfragen, die während des Löschens des Endpunkts ausgeführt werden, werden mit den Einstellungen des Endpunkts zum Zeitpunkt der Anfrage finalisiert. Anfragen nach Löschung des Endpunkts, ergeben eine Fehlermeldung.
