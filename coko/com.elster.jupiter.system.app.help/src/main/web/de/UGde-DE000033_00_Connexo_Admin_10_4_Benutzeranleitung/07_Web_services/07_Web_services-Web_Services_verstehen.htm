Web Services verstehen

Web Services in Connexo

Ein Webservice wird als ein Softwaresystem definiert, welches unter Verwendung einer speziell entwickelten Schnittstelle Geschäftslogik, -daten und Prozesse in einem Netzwerk teilt. Insbesondere sind dieses Kunden und Server, die über XML-Nachrichten kommunizieren. Zum Formatieren dieser XML-Nachrichten verwenden wir normalerweise SOAP (Simple Object Access Protocol) und REST (Representational State Transfer).

Um Web Services hinzuzufügen, kontaktieren Sie Ihren Systemadministrator. Sie müssen diese in den Softwarecode einfügen. In der Connexo MultiSense Schnittstelle können Sie nur Web Service-Endpunkte für die vom Systemadministrator hinzugefügten Web Services hinzufügen und konfigurieren.
