Endpunkte der Web Services überwachen

Endpunkte der Web Services anzeigen

5

Schritte

1.   Öffnen Sie Verwaltung > Web Services > Web Services-Endpunkte.

2.   Die Tabelle zeigt alle konfigurierten Web Service Endpunkte gemeinsam mit ihrem Typen, dem verwendeten Web Service, Status und Log-Ebene. Wenn ein Web Service nicht verfügbar ist, wird das mit einem Ausrufezeichen neben dem Namen des Web Service angegeben.

3.   Das Detailfenster zeigt umfangreichere Details des Endpunkts, der in der Tabelle selektiert wurde. Sie können diese Details auch sehen, wenn Sie den Namen des Web Service Endpunkts in der Tabelle anklicken.

Protokollierung der Web Services anzeigen

5

Schritte

1.   Öffnen Sie Verwaltung > Web Services > Web Service Endpunkte > klicken Sie auf den Namen eines Web Service Endpunkts> klicken Sie Protokollieren im Menü auf der linken Seite.

2.   Zeigen Sie alle Protokollzeilen, indem Sie in der Tabelle nach unten scrollen. Die Anzahl der angezeigten Protokollzeilen kann durch Auswahl einer anderen Zahl im Feld Protokollzeilen pro Seite geändert werden.

3.   Exportieren Sie das Protokoll unter Verwendung der Export-Taste  im oberen Teil der Tabelle
