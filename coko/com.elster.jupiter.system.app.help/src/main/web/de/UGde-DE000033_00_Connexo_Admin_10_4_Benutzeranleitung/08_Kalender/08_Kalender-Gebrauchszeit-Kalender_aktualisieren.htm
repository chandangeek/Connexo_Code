Aktualisieren mit Reimport des Gebrauchszeit-Kalenders

Sie können einen Gebrauchszeit-Kalender durch Reimport in Connexo aktualisieren. Aktualisierung des Gebrauchszeit-Kalenders erfolgt meistens durch Hinzufügen besonderer Ereignisse wie variabler Ferientage und anderer besonderer Perioden, die einen festgelegten Tarif benötigen.

Zum Import des Kalenders verwenden Sie den bestehenden Importrahmen und genau denselben Namen, wie den, der für den früheren Import verwendet wurde. Alle Änderungen überschreiben einfach die bestehenden Informationen im Gebrauchszeit-Kalender.

Wenn Daten bereits mit der vorigen Version des Kalenders berechnet wurden, berechnen Sie die Daten ab Anfangsjahr erneut. Wenn bisher keine Daten berechnet wurden oder wenn die vorige Version Ghost war, ist keine Berechnung notwendig.

3

Anmerkung

Der Status des Gebrauchszeit-Kalenders ist Ghost, wenn er vom Gerät stammt und nicht in Connexo existiert.

1

Wichtig

  • Wenn Sie das Anfangsjahr ändern, werden alle Daten vor diesem Datum nicht berechnet.

  • Es kann etwas Verspätung auftreten, wenn Sie den aktualisierten Kalender zum ersten Mal verwenden, da Zeitserien für ein ganzes Jahr berechnet werden. Anschließend werden neue Serien über einen Hintergrundprozess berechnet.
