Anwendungsserver konfigurieren

Anwendungsserver verwalten

Einen Anwendungsserver hinzufügen

1

Wichtig

Wenn Sie einen Anwendungsserver erstellen, müssen Sie das Connexo-Gerät, das diesen Anwendungsserver ausführt, unterbrechen und neustarten.

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendungsserver > Anwendungsserver.

2.   Klicken Sie auf Anwendungsserver hinzufügen und füllen Sie die nachfolgenden Felder aus:

Parameter

Beschreibung

Name

Name des Servers, auf dem die Anwendung läuft.

Exportpfad

Standardpfad zum Speichern der exportierten Dateien und Daten. Der Pfad kann in der Exportdatenkonfiguration übergangen werden.

Importpfad

Standardpfad für den Import von Daten. Der Pfad kann in der Exportdatenkonfiguration übergangen werden.

Nachrichtendienste

Selektieren Sie die Nachrichtendienste, die vom Anwendungsserver ausgeführt werden müssen, indem diese entfernt werden, die nicht ausgeführt werden müssen. Dazu kann das Löschsymbol angeklickt werden .

Ändern Sie die Anzahl der Threads oder den Status des Service, indem der Wert angeklickt wird.

Nachrichtendienste können eingefügt werden, wenn Sie  Nachrichtendienste einfügen anklicken.

Importdienste

Standardmäßig werden alle konfigurierten Importdienste zum Anwendungsserver gefügt. Löschen Sie Importdienste, indem Sie das Löschsymbol anklicken, fügen Sie Importdienste ein, indem Sie auf Importdienste hinzufügen klicken.

3.   Klicken Sie auf Einfügen.

4.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Aktivieren.

3

Anmerkung

Wenn Sie einen Anwendungsserver aktivieren, wird dieser Server in der Anwendung verfügbar, anstatt als solcher aktiviert zu werden.

Anwendungsserver bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendungsserver > Anwendungsserver.

2.   Klicken Sie auf das Aktionssymbol  neben dem Anwendungsserver, den Sie bearbeiten möchten und selektieren Sie Bearbeiten.

3.   Ändern Sie die Parameter des Anwendungsservers.

4.   Nachrichten und Importdienste einfügen, bearbeiten oder löschen.

5.   Klicken Sie auf Speichern.

Anwendungsserver löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendungsserver > Anwendungsserver.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Löschen um den Anwendungsserver zu löschen. Klicken Sie Löschen im erscheinenden Dialogfenster.
