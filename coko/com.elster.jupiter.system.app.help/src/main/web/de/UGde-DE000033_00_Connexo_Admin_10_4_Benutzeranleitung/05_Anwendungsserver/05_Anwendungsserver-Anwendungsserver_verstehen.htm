Anwendungsserver verstehen

Was sind Anwendungsserver

Ein Anwendungsserver ist ein Server, der verschiedene Dienste im Hintergrund laufen lässt, die Prozessen ermöglichen, die einen längeren Zeitraum laufen, ohne den Anwender zu behindern. Beispiele für solche Prozesse sind Datenexport und -import und die Berechnung von KPIs.

Die Dienste, die am Anwendungsserver laufen, werden Nachrichtendienste genannt, da der Server Nachrichten behandelt. Diese Nachrichtenservices können konfiguriert werden, um die Belastung des Systems zu reduzieren. Das kann erreicht werden, indem verschiedene Anwendungsserver für verschiedene Dienste erstellt werden. Beispielsweise können zwei Anwendungsserver für Datenexport erstellt und konfiguriert werden. Das verteilt die Exportnachrichten über die zwei Server, sodass sie gleichzeitig ausgeführt werden können.

Sie können auch die Threads eines Anwendungsservers anpassen, indem Sie die Anzahl der Threads des Nachrichtendienstes erhöhen. Das bestimmt die Anzahl der Dienste wie Exportaufgaben, die parallel an einem einzigen Server ausgeführt werden können.

Die Verwendung mehrerer Server sorgt auch für ein stabileres System, da der zweite Server übernehmen kann, wenn ein Server aufgrund von Fehler oder Wartung ausfällt.

Anwendungsserver in Connexo

Anwendungsserver in Connexo können unter Verwendung der Admin:-Anwendung einfach hinzugefügt und verwaltet werden. Wenn Sie einen neuen Anwendungsserver hinzufügen, müssen drei Parameter definiert werden: Ein eindeutiger Name, der Export- und Importpfad und die Nachrichten- und Importdienste.

Der Exportpfad stellt den Standardpfad ein, in den Dateien und Daten exportiert werden. Dieser Pfad kann jedoch überschrieben werden, wenn Sie eine Exportaufgabe konfigurieren. Achten Sie darauf, wenn Sie mehrere Anwendungsserver für Datenexportaufgaben erstellen, die zusammen gehören, dann ist der Exportpfad der verschiedenen Anwendungsserver derselbe. Ansonsten werden Ihre Daten über verschiedene Standorte verstreut. Dasselbe gilt für Importdienste.

Standardmäßig werden alle Nachrichtendienste und Importdienste zum Anwendungsserver gefügt, aber diese können einfach gelöscht werden, sodass der Anwendungsserver Ihren Anforderungen entspricht.
