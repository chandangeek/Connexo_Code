Trust Stores verwalten

Trust Stores anzeigen

Sie können alle Trust Stores, die in Connexo verfügbar sind, ansehen und filtern.

5

Schritte

1.   Öffnen Sie Verwaltung > Zertifikatmanagement > Trust Stores.

2.   Geben Sie in den Filterbalken Kriterien ein, um die Liste der Trust Stores einzugrenzen. Vergessen Sie nicht auf Anwenden zu klicken, um den Filter anzuwenden.

3.   Selektieren Sie einen Trust Store und zeigen Sie seine Details im Detailfenster unten.

4.   Klicken Sie auf den Namen eines Trust Stores in der Tabelle, um die Detailseite des Trust Stores zu öffnen.

3

Anmerkung

In der Tabelle, die die Zertifikate für den selektierten Trust Store aufführt, zeigt Ihnen die Spalte Status die Zertifikate, die den ungültigen Status haben.  Die Nutzung dieser Zertifikate ist auf Sicherheitszugriffsberechtigte, Geräte und Anwenderverzeichnisse beschränkt. Sie können auch den Filter Status nutzen, um die Zertifikate mit dem benötigten Status zu sehen.

Trust Store hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Zertifikatmanagement > Trust Store.

2.   Klicken Sie auf Einen Trust Store hinzufügen und füllen Sie die nachfolgenden Parameter aus:

Parameter

Beschreibung

Name

Geben Sie dem Trust Store einen eindeutigen Namen.

Beschreibung

Wenn Sie möchten, können Sie eine zusätzliche Beschreibung hinzufügen.

3.   Klicken Sie auf Einfügen.

Trust Store bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Zertifikatmanagement > Trust Store.

2.   Klicken Sie auf das Aktionssymbol  neben dem Trust Store und selektieren Sie Bearbeiten.

3.   Die Trust Store Parameter einstellen.

4.   Klicken Sie auf Speichern.

Trust Store löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Zertifikatmanagement > Trust Store.

2.   Klicken Sie auf das Aktionssymbol  neben dem Trust Store und selektieren Sie Löschen.

Trust Stores und ihre Zertifikate, die noch immer von einem Objekt in Connexo verwendet werden, können nicht gelöscht werden.

3.   Bestätigen Sie indem Sie auf Löschen klicken.

Vertrauenswürdige Zertifikate in Trust Stores importieren

Vertrauenswürdige Zertifikate müssen das Format .JKS haben, sodass sie in Connexo importiert werden können.

5

Schritte

1.   Öffnen Sie Verwaltung > Zertifikatmanagement > Trust Store.

2.   Klicken Sie auf das Aktionssymbol  neben dem Trust Store und selektieren Sie Vertrauenswürdige Zertifikate importieren oder klicken Sie auf einen Trust Store Namen und klicken Sie auf die Taste Vertrauenswürdige Zertifikate importieren auf der Übersichtsseite.

3.   Selektieren sie die Hauptstore-Datei.

Wenn der Alias der neuen Datei derselbe ist, wie eine bestehende Datei, wird diese von der neuen Datei überschrieben.

4.   Geben Sie das richtige Passwort ein.

5.   Klicken Sie auf Import.

Der Status des Zertifikats ist Verfügbar solange das Ablaufdatum nicht erreicht wurde.

Vertrauenswürdige Zertifikate herunterladen

In einige Fällen möchten Sie eventuell die vertrauenswürdigen Zertifikate herunterladen um Fehler zu korrigieren oder eine Prüfung des Zertifikats durchzuführen.

5

Schritte

1.   Öffnen Sie Verwaltung > Zertifikatmanagement > Trust Store.

2.   Klicken Sie auf den Trust Store Namen, um zur Übersichtsseite zu gelangen.

3.   Klicken Sie auf das Aktionssymbol  neben dem vertrauenswürdigen Zertifikat und selektieren Sie Herunterladen.

Vertrauenswürdige Zertifikate löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Zertifikatmanagement > Trust Store.

2.   Klicken Sie auf den Trust Store Namen, um zur Übersichtsseite zu gelangen.

3.   Klicken Sie auf das Aktionssymbol  neben dem vertrauenswürdigen Zertifikat und selektieren Sie Löschen.

Vertrauenswürdige Zertifikate, die noch immer von einem Objekt in Connexo verwendet werden, können nicht gelöscht werden.

4.   Bestätigen Sie, indem Sie auf Löschen klicken.
