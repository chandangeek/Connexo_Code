Benutzerdefinierte Attribute in Connexo

Connexo bietet die Möglichkeit verschiedene zusätzliche benutzerdefinierte Attribute zu Ihrem System zu fügen. Das ermöglicht Ihnen zusätzliche Informationen hinzuzufügen, die nicht von Connexo MultiSenses Basis CIM-Modell abgedeckt werden, aber wichtig für den Kunden sind. Das könnte beispielsweise für die Integration mit anderen Anwendungen, Prozessen, Operationen usw. sein. So können Sie die Anwendung an Ihre besonderen regionalen Bedürfnisse anpassen.

Benutzerdefinierte Attribute können nicht über die Schnittstelle eingefügt werden. Alle benutzerdefinierten Attribute sind codiert und im System installiert. Die Eigenschaften der Standardwerte und benutzerdefinierten Attribute sind fest programmiert, aber können in der Schnittstelle angezeigt werden.
