Sätze benutzerdefinierter Attribute verwalten

Sätze benutzerdefinierter Attribute anzeigen

5

Schritte

1.   Öffnen Sie Verwaltung > Benutzerdefiniertes Attributemanagement > Sätze benutzerdefinierter Attribute.

2.   Verwenden Sie den Filter ganz oben, um sich auf die benutzerdefinierten Attribute zu konzentrieren, die Ihnen wichtig sind. Selektieren Sie einen Objekttypen, um dessen benutzerdefinierte Attribute anzuzeigen.

3.   Klicken Sie auf Anwenden.

Die Attribute der selektierten benutzerdefinierten Attribute werden in einem Detailfenster unter der Tabelle angezeigt. Die Spalte Identifizierer zeigt den eindeutigen Identifizierer für jedes Attribut des ausgewählten Satzes benutzerdefinierter Attribute. Sie können diese Informationen nutzen, um eine Importdatei zum Aktualisieren der Attribute in einer Objektgruppe vorzubereiten. Entweder kopieren Sie die eindeutigen Identifizierer und fügen sie ein oder Sie exportieren einfach das Detailfenster mit der Exporttaste  im oberen Teil der Tabelle in csv. Weitere Informationen über den Import von Werten benutzerdefinierter Attributsätze entnehmen Sie bitte “Werte aus bestehenden benutzerdefinierten Attributen importieren”.

Rechte der benutzerdefinierten Attributesätze bearbeiten

Die Rechte für einen bestimmten benutzerdefinierten Attributsatz können Sie ändern durch Übergehen der

5

Schritte

1.   Öffnen Sie Verwaltung > Management der benutzerdefinierten Einstellungen > Benutzerdefinierte Attributsätze..

2.   Klicken Sie auf das Aktionssymbol neben dem benutzerdefinierten Attributsatz und selektieren Sie Ebenen bearbeiten.

3.   (De)Selektieren Sie eine oder mehrere Ebenen für die verschiedenen Handlungen um weitere Details zum selektierten benutzerdefinierten Attributsatz zu öffnen. Eventuell möchten Sie, dass der Messgerätspezialist Zugang zu allen benutzerdefinierten Attributsätzen außer dem benutzerdefinierten Attributsatz x hat.

Die entsprechenden Rollen zu den verschiedenen Ebenen sind in Anwendermanagement definiert. Siehe “Add user role”. Die konfigurierten Rechte für benutzerdefinierte Attributsätze können im Anwendungsabschnitt MultiSense und im Ressourcenabschnitt Rechte für benutzerdefinierte Attributsätze eingesehen werden.

4.   Klicken Sie Speichern.
