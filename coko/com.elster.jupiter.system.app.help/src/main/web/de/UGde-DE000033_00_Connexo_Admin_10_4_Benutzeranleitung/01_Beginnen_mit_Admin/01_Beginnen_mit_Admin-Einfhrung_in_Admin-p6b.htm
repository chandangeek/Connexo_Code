Einführung in Admin

In Admin sind verschiedene Systemverwaltungsfunktionen auf verschiedene Seiten verteilt. Hier erhalten Sie eine kurze Einführung in die verschiedenen Seiten.

Verwaltung

Auf der Verwaltungsseite ist die gesamte Systemverwaltung zusammen gruppiert. Hier kann der Systemverwalter Anwenderverzeichnisse, Anwender und Anwenderfunktionen verwalten und Rechte zuweisen. Von der Verwaltungsseite können verschiedene Lizenzen hinzugefügt oder aktualisiert werden.

Unter Verwendung der Admin Anwendung können Sie die Parameter zum Bereinigen Ihrer Daten einstellen, sowie Anwendungsserver hinzufügen und konfigurieren. In Admin können verschiedene Parameter für den Export und Import Ihrer Daten konfiguriert werden, wie relative Zeiträume und Importdienste.

Auf der Verwaltungsseite finden Sie auch Deployment-Angaben, die Sie bei Problemen an unserem Supportteam schicken können.
