Alle Seiten in Admin haben ein gleiches Layout. Um die Verwendung der Anwendung zu erleichtern, wird hier das allgemeine Layout erklärt.

Abbildung: Connexo-Layout.

 

A. Anwendungszentrum

B. Vertikaler Navigationsbalken: Shortcut zur Verwaltungsseite.

C. Komprimieren oder vergrößern Sie die Navigation.

D. Zurückkehren zur Startseite.

E. Online-Hilfe

F. Abmelden: Platz in der Anwendung zum Abmelden.

G. Arbeitsbereich: Änderungen hängen von der Funktion ab, die Sie anzeigen.

Übersichtsseite Layout

Wenn Sie auf eine spezielle Funktion in Admin klicken, öffnet sich in den meisten Fällen eine Übersichtsseite dieser Funktion. In der Anwendung haben diese Übersichtsseiten ein gleichartiges Aussehen.

Die nachfolgend gezeigte Abbildung und der Text zeigen und erklären das Layout der Übersichtsseiten gemeinsam mit den Handlungen, die Sie unter Verwendung der Symbole auf der Seite ausführen können.

Abbildung: Übersichtsseite Layout.

 

A. Brotkrumen: zeigt, wo Sie sich in der Anwendung befinden. Ermöglicht einfache Navigation zu Seiten, die höher in der Struktur sind.

B. Übersichtstabelle: Diese Tabelle zeigt alle Objekte, die bereits zum System gefügt wurden. Es können verschiedene Handlungen mit den Items in der Tabelle ausgeführt werden, z.B. können sie bearbeitet, gelöscht, aktiviert, angezeigt usw. werden.

C. Detailfenster: Zeigt mehr Details eines Items, das in der Tabelle selektiert wurde. Standardmäßig ist das erste Item selektiert.

D. Hinzufügen-Taste: Taste, mit der neue Items zur Tabelle gefügt werden können. Kann auch eine andere Handlung sein, je nachdem in welcher Funktion auf der Übersichtsseite Sie sich befinden.

E. Aktionssymbol: Wenn Sie auf dieses Symbol klicken, erscheinen die verschiedenen Handlungen, die beim selektierten Item ausgeführt werden können.

F. Aktionstaste: Wenn Sie auf dieses Symbol klicken, erscheinen die verschiedenen Handlungen, die beim selektierten Item ausgeführt werden können.
