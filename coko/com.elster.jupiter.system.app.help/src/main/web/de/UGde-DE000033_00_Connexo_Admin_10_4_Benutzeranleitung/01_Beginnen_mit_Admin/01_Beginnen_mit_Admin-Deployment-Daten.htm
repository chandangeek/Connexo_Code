Deployment-Daten

Komponenten anzeigen

5

Schritte

1.   Öffnen Sie Verwaltung > Deployment-Angaben > Liste der Komponenten.

Diese Seite zeigt alle verfügbaren Bündel mit der Anwendung des Bündels, der Version und seines Status.

2.   Verwenden Sie den Filter im oberen Bereich der Seite, wenn Sie nur bestimmte Bündel sehen möchten. Klicken Sie auf Anwenden.

3.   Entfernen Sie die Filter, indem Sie die Auswahl der Objekte im Drop-down-Menü aufheben oder löschen Sie alle Filter, indem Sie auf Alle löschen klicken.

4.   Exportieren Sie die Liste der Bündel, indem Sie auf die Rasterexporttaste  über der Tabelle klicken.

Systeminformation anzeigen

5

Schritte

1.   Öffnen Sie Verwaltung > Deployment-Angaben > Systeminformation.

Die Systeminformation ist in zwei Teile aufgeteilt:

Abschnitt

Beschreibung

Serverdaten

Informationen zum Server, wie Javaeigenschaften, Datenbank-Standort, Serverspeicher usw.

Client-Daten

Informationen zum Browser, der verwendet wurde, um Connexo verwenden.

2.   Kopieren Sie die Informationen und schicken Sie sie an Connexo Support.
