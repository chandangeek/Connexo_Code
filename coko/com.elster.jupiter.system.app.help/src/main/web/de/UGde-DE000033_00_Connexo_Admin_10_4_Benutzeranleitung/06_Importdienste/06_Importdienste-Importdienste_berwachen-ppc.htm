Importdienste anzeigen

5

Schritte

1.   Öffnen Sie Verwaltung > Datenimport > Importdienste.

Die Übersichtsseite enthält eine Tabelle, die alle konfigurierten Importdienste anzeigt. In der Tabelle sehen Sie den Importer, den die Dienste nutzen, sowie die Anwendung, die den Importer nutzt. Diese Tabelle zeigt auch den Status der Importdienste und welcher Dateityp importiert werden kann.

Indem Sie die Importdienste in der Tabelle selektieren, sehen Sie weitere Details im Detailfenster im unteren Teil der Seite.

2.   Klicken Sie auf den Importdienstnamen.

Die Seite zeigt alle Details des Importdienstes und ermöglicht Ihnen auch den Dienst zu bearbeiten, zu  löschen und zu (de)aktivieren.

StandardImporter

Connexo bietet verschiedene Standardimporter, die den Import verschiedener Datentypen gestattet, um verschiedene Handlungen in den Connexo-Anwendungen auszuführen. Diese verschiedenen Importer werden hier beschrieben. Die Nutzung der konfigurierten Importdienste wird in den Gebrauchsanweisungen von Connexo MultiSense und Connexo Insight beschrieben.

Importer

Beschreibung

Kalenderimporter

Import von Gebrauchszeit (TOU) Kalendern.

CSR Importer

Einen oder mehr CSRs (Zertifikatsregistrierungsanforderung) von einem Gerätehersteller, der sie dann zum Gerät hochladen kann. Außerdem können Sie auch (in derselben Datei) eine Anzahl Trust Anchors (Zertifikate von einem Trust Store) hochladen, um sie bei der Herstellung in das Gerät zu integrieren.

Geräte-Verbindungsattribut-Importer

Importer, der zum Import verschiedener Verbindungsattribute unterschiedlicher Verbindungsmethoden und Geräte entwickelt wurde.

Geräte-Sicherheitsattribut-Importer

Importer, der zum Import verschiedener Sicherheitsattribute unterschiedlicher Sicherheitseinstellungen und Geräte entwickelt wurde.

Geräteinbetriebnahme Importer

Importer für den Übergang von Geräten in den Inbetriebnahmestatus.

Außerbetriebnahme Importer für Geräte

Importer für den Übergang von Geräten in den Außerbetriebnahmestatus.

Geräte-Installations-Importer

Importer für die Installation von Geräten.

Geräte-Entfernungs-Importer

Importer um selektierte Geräte aus Connexo zu löschen.

Geräte-Lieferungs-Importer

Importer für die kombinierte Einfügung von Geräten zu Connexo.

Geräteaktivierung / Deaktivierung Importer

Importer für den Übergang von Geräten in den aktiven oder inaktiven Zustand.

Geräte-Messdaten-Importer

Importer, entwickelt zum Einfügen von Messgeräteständen zu selektierten Geräten.

Verwendungsstellen-Importer

Import von Verwendungsstellen unter Verwendung einer CSV-Datei.

3

Anmerkung

In den Importdateien gibt es nur eine Spalte für und Name. Importer suchen zuerst nach. Wenn es einen Match gibt, aktiviert der Importer. Wenn es keinen Match gibt, sucht der Importer nach Namen. Wenn es einen Match gibt, aktiviert der Importer. Wenn es keinen Match gibt, wird das Objekt eingefügt und der Wert als Namen verwendet.    wird von Connexo erzeugt.
