Importdienste konfigurieren

Den Importdienst einrichten

Importdienste hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Datenimport > Importdienste.

2.   Klicken Sie auf Importdienste einfügen. Füllen Sie die folgenden Felder aus:

Parameter

Beschreibung

Name

Eindeutiger Name

Datei-Importer

Selektieren Sie den Importer, der den Import ausführt. Je nach selektiertem Importer müssen eventuell weitere Parameter definiert werden.

Importordner

Definieren des Standorts, wo Dateien, die importiert werden müssen, sich befinden. Dieser Pfad hängt mit dem Importstandort des Anwendungsservers zusammen.

Hochladevorgänge in Anwendung zulassen

Selektieren Sie diese Option, wenn Sie möchten, dass der Anwender in der Lage ist, Dateien mithilfe der Anwenderschnittstelle auf der Seite “Datei für Import”

Dateimuster

Definieren, welche Dateien importiert werden können.

Ordner-Scan-Häufigkeit

Selektiert die Häufigkeit mit der überprüft wird, ob neue Dateien für den Import verfügbar sind.

Ordner In Bearbeitung

Definiert den Standort, wo während des Prozesses Dateien gespeichert werden. Dieser Pfad ist relativ für den Importstandort des Anwendungsservers.

Ordner Erfolg

Definiert den Standort, wo erfolgreich importierte Dateien gespeichert werden. Dieser Pfad hängt mit dem Importstandort des Anwendungsservers zusammen.

Fehlerordner

Definieren des Standorts, wo Dateien, die nicht importiert werden konnten, gespeichert werden. Dieser Pfad ist relativ für den Importstandort des Anwendungsservers.

Wenn Sie eine Datei vom fehlerhaften Verzeichnis in das Importverzeichnis verschieben, wird der Import dieser Datei erneut versucht.

3.   Klicken Sie auf Einfügen.

4.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Aktivieren.

3

Anmerkung

Bevor Importdienste in den anderen Connexo-Anwendungen verwendet werden können, muss dieser Importdienst zu einem Anwendungsserver gefügt werden.

Importdienste bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Datenimport > Importdienste.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Bearbeiten. Alle Parameter außer dem Dateiimporter können bearbeitet werden.

3.   Klicken Sie auf Speichern.

Importdienste löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Datenimport > Importdienste.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Löschen.

3.   Klicken Sie auf Löschen.
