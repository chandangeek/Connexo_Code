Importdienste am Anwendungsserver einrichten

Importdienste am Anwendungsserver konfigurieren

5

Schritte

1.   Öffnen Sie Verwaltung > Anwendungsserver > Anwendungsserver.

2.   Erstellen eines neuen (“Add an application server”) oder bearbeiten eines bestehenden Anwendungsservers (“Edit application server”).

3.   Klicken Sie auf Importdienste einfügen.

4.   Selektieren Sie einen oder mehrere der konfigurierten Importdienste.

5.   Löschen Sie Importdienste, indem Sie auf das Aktionssymbol  klicken und Löschen selektieren.
