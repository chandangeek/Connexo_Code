Log Level konfigurieren

Konfigurieren Sie das Log Level für Serviceanruf-Typen, um zu vermeiden, dass die Datenbank mit unnötigen Zeilen mit Logging-Information überspült wird. Das Standard Log Level ist Achtung.

5

Schritte

1.   Öffnen Sie Verwaltung > Serviceanrufe > Serviceanruf-Typen.

2.   Selektieren Sie den gewünschten Serviceanruf-Typen.

3.   Klicken Sie auf das Aktionssymbol  in der Spalte Aktionen neben dem ausgewählten Serviceanruf-Typen oder klicken Sie auf die Taste Aktionen in dem Detailfenster.

4.   Selektieren Sie Log Level ändern.

5.   Selektieren Sie im Popup Log Level selektieren das gewünschte Log Level. 

Die folgenden Log Levels sind verfügbar:

  • Schwer

  • Warnmeldung

  • Information

  • Konfig

  • Fein

  • Feiner

  • Feinste

6.   Klicken Sie Speichern.
