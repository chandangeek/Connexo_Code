Benutzerdefinierte Einstellungen von Serviceanrufen verwalten

Benutzerdefinierte Einstellungen von Serviceanrufen anzeigen

Anzeigen der benutzerdefinierten Eigenschaften für Serviceanrufe gemeinsam mit ihren Merkmalen und Eigenschaften. Siehe “Custom attributes”.

5

Schritte

1.   Öffnen Sie Verwaltung > Management der benutzerdefinierten Einstellungen > Benutzerdefinierte Einstellungen..

2.   Selektieren Sie im Drop-Down-Menü Typ der benutzerdefinierten Einstellungen Serviceanruf und klicken Sie Anwenden.

Nun werden in einer nummerierten Tabelle nur die benutzerdefinierten Einstellungen für Serviceanrufe angezeigt. Die folgenden Spalten sind verfügbar:

Parameter

Beschreibung

Name

Name der benutzerdefinierten Einstellung.

Typ

Der Wert für dieses Feld ist immer Serviceanruf.

Erforderlich

Anzeigen, ob die benutzerdefinierte Einstellung erforderlich ist oder nicht. Der Wert ist Ja oder Nein.

Levels anzeigen

Nicht anwendbar für diesen Typ benutzerdefinierter Einstellungen.

Levels bearbeiten

Nicht anwendbar für diesen Typ benutzerdefinierter Einstellungen.

Zeitscheibe

Immer Nein.

Für diesen Typ benutzerdefinierter Einstellungen sind keine Handlungen verfügbar.

3.   Im Detailfenster der ausgewählten benutzerdefinierten Einstellungen werden die folgenden Informationen für jede dieser Eigenschaften angezeigt:

Parameter

Beschreibung

Bestellung

Bestellen der Eigenschaften in den benutzerdefinierten Eigenschaften.

Name der Eigenschaft

Name der Eigenschaft.

Erforderlich

Anzeigen, ob die Einstellung erforderlich ist oder nicht. Der Wert ist Ja oder Nein.

Typ

Typ der Eigenschaft.

Standard

Standardwert für diese Eigenschaft.

Beschreibung

Freie Textbeschreibung.
