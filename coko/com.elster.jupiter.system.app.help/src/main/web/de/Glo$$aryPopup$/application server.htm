application server

Ist ein Server, auf dem verschiedene Prozesse laufen können. Mit diesen Servern kann die Belastung ausgeglichen werden, indem verschiedene Aufgaben auf verschiedene Anwendungsserver verteilt werden. Im Anwendungsserver können Nachrichtendienste hinzufügt werden und die Anzahl der Threads kann geändert werden, um die Anzahl der parallelen Dienste zu definieren.
