import service

Wird verwendet, um Daten in Connexo zu importieren. Der Importservice nimmt die zu importierende Datei, verschiebt sie in das korrekte Verzeichnis und setzt sie anschließend in die Warteschlange. Der aktuelle Import wird von einem Nachrichtendienst bearbeitet. Das bedeutet, dass für den Erfolg des Imports, sowohl die Importservices als auch die Nachrichtendienste konfiguriert werden müssen und zu einem Anwendungsserver hinzugefügt werden müssen.
