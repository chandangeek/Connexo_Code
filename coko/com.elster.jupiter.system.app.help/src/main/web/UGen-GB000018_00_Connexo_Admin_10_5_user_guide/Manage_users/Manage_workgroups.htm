Manage workgroups

View workgroups

5

Steps

1.   Go to Administration > User management > Workgroups.

This table shows all configured workgroups together with their configuration details.

Add workgroup

5

Steps

1.   Go to Administration > User management > Workgroups.

2.   Click Add workgroup and complete the following fields:

Parameter

Description

Name

Give a unique name to the workgroup.

Description

Provide a description for the workgroup.

Users

Click Select users to add users to the workgroup. Select the users and click Add.

3.   Click Add.

Edit workgroup

5

Steps

1.   Go to Administration > User management > Workgroups.

2.   Click the actions icon  next to the workgroup you want to edit and select Edit.

3.   Change the workgroup parameters, you can only edit description and users.

4.   Click Save.

Remove workgroup

5

Steps

1.   Go to Administration > User management > Workgroups.

2.   Click the actions icon  next to the workgroup you want to remove and select Remove.
