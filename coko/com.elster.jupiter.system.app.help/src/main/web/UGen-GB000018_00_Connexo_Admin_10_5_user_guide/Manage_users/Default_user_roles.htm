Default user roles

Introduction

Connexo offers different default user roles to improve the every day usage of Connexo. These consist out of the roles needed to perform all tasks to set up, maintain and operate Connexo. Because these are available upon installation, you can immediately start using Connexo. You can edit some of the roles but not all. A detailed description of all default user roles are given below

Installer

A user with the installer role is the person who will do the initial installation of the system. For this reason this role has full license privileges as well as user management privileges to add the first person to the system that will set up the LDAP and all users.

System administrator

The system administrator role gives you complete access to the Admin application.

User administrator

The user administrator role is intended for users who will define which users can do what. This user role has full user administrator privileges in the Admin application.

We advise not to combine this role with an LDAP administration role separate the creation of user from the definition of what users can do. We also advise not to combine the user administrator role with the dual control administrator role due to a conflict of interest.

Dual control administrator

With the dual control administrator role you can define who can accept or reject dual control managed features such as command limitation rules in the Connexo Admin application. You cannot edit this role.

It is impossible to combine this role with the approver role of dual control managed features such as the command limitation rule approver role. Due to a conflict of interest we advise not to combine this role with other user management roles.

Command limitation rule approver

This role you can assign to users that should approve or reject the changes made to the command limitation rule parameters in MultiSense.

You cannot combine this role with the dual control administrator role. It is not possible to edit this role.

Meter expert

With this role you will have full meter management privileges inside MultiSense. With this role you can configure all device features such as device types and configurations as well as the communication infrastructure and VEE setup.

Meter operator

This role contains all meter operations privileges inside MultiSense. Users with this role are able view and administrate device data, but are not allowed to configure most device features only view them.

Business process designer

This role contains all privileges to design processes in Flow.

Report designer

This role contains all privileges to design reports in Facts.

Report viewer

This role contains all privileges to view reports created in Facts in MultiSense. Combine this role with the meter operator role to allow your user to operator MultiSense and view relevant reports.
