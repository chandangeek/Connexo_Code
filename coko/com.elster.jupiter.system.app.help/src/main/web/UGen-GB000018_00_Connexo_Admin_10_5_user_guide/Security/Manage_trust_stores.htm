Manage trust stores

View trust stores

You can view and filter all the trust stores that are available in Connexo.

5

Steps

1.   Go to Administration > Certificate management > Trust stores.

2.   Enter criteria in the filter bar on top to narrow down the list of trust stores. Don’t forget to click Apply to execute your filter.

3.   Select a trust store and view its details in the details pane below.

4.   Click the name of a trust store in the table to open the trust store’s details page.

3

Note

In the table that lists the certificates for the selected trust store, the column Status shows you the certificates that have the revoked status. These certificates are restricted from use on security accessors, devices and user directories. You can also use the filter Status to see the certificates with the needed status.

Add trust store

5

Steps

1.   Go to Administration > Certificate management > Trust stores.

2.   Click Add trust store and complete the following parameters:

Parameter

Description

Name

Give a unique name to the trust store.

Description

If wanted you can add an extra description.

3.   Click Add.

Edit trust store

5

Steps

1.   Go to Administration > Certificate management > Trust stores.

2.   Click the actions icon  next to the trust store and select Edit.

3.   Adjust the trust store parameters.

4.   Click Save.

Remove trust store

5

Steps

1.   Go to Administration > Certificate management > Trust stores.

2.   Click the actions icon  next to the trust store and select Remove.

Trust stores and its certificates that are still used by an object in Connexo cannot be removed.

3.   Confirm by clicking Remove.

Import trusted certificates to trust store

Trusted certificates must have the.JKS format to be able to import them into Connexo.

5

Steps

1.   Go to Administration > Certificate management > Trust stores.

2.   Click the actions icon  next to the trust store and select Import trusted certificates or click a trust store name and click the Import trusted certificates button on the overview page.

3.   Select the Key store file.

If the alias of the new file is the same as an existing file then this one will be overwritten by the new file.

4.   Provide the correct password.

5.   Click Import.

The status of the certificate will be Available as long as the expiration date is not reached.

Download trusted certificates

In some cases you may want to download the trusted certificates to debug or perform a check of the certificate.

5

Steps

1.   Go to Administration > Certificate management > Trust stores.

2.   Click the trust store name to go the overview page.

3.   Click the actions icon  next to the trusted certificate and select Download.

Remove trusted certificate

5

Steps

1.   Go to Administration > Certificate management > Trust stores.

2.   Click the trust store name to go the overview page.

3.   Click the actions icon  next to the trusted certificate and select Remove.

Trusted certificates that are still used by an object in Connexo cannot be removed.

4.   Confirm by clicking Remove.
