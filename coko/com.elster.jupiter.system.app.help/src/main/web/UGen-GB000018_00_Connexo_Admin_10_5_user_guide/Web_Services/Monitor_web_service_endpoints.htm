Monitor web service endpoints

View web service endpoints

5

Steps

1.   Go to Administration > Web services > Web service endpoints.

2.   The table shows all configured web service endpoints together with their type, the web service they use, status and log level. When a web service is not available this is indicated by an exclamation icon next to the web service name.

3.   The details pane shows more extensive details of the endpoint selected in the table. You can also see these details by clicking the web service endpoint name in the table.

View web service endpoint logging

5

Steps

1.   Go to Administration > Web services > Web service endpoints > click a web service endpoint name > click Logging in the menu on the left.

2.   View all logging lines by scrolling down the table. The amount of log lines displayed can be altered by selecting another number in the Log lines per page field.

3.   Export the logging by using the export button on top of the table.
