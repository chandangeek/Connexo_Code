View relative period details

You can easily obtain an overview of all the relative periods that are available on different places in Connexo.

5

Steps

1.   Go to Administration > Relative periods> Relative periods.

2.   Select the needed relative period to see its details in the details pane below.

3.   Click the name of the relative period to open its details page.

On all occurrences you have a preview of the selected relative period indicating its start and end time based on a reference date and time.

In view modus you can change the previewed schedule by changing the date and hour of the reference time.

View relative period usage

You can see where a particular relative period is used in Connexo. This is important when you want to edit or remove a relative period that is currently used. You need to select other relative periods in these instances first.

5

Steps

1.   Go to Administration > Relative periods> Relative periods > select a relative period in the table.

2.   Click Usage in the menu on the left.

3.   Select a category from the menu to see which tasks are using the relative period.

4.   Click Apply.

In the table you see the names of the tasks where the current relative period is used. The table also indicates the kind of task, the application in which it runs and when the task will run again.
