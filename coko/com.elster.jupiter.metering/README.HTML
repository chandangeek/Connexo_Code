com.elster.jupiter.metering

This bundle manages meter data. It registers a dataModel with the MTR acronym. It closely follows the object model defined in the CIM Metering standard (IEC 61968-9: Application integration at electric utilities. System interfaces for distribution management. Part 9: Interfaces for meter reading and control)
