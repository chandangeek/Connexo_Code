com.elster.jupiter.security.thread

This bundle provides support for associating data with a thread. It currently support principal, module, action and locale. The principal is a java interface to identify a user. Locale captures the language and country of the thread user, and is used for internatiolization of the application Module and action are used for performance and troubleshooting.
