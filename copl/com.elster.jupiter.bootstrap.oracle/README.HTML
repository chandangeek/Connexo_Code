com.elster.jupiter.bootstrap.oracle

This bundle immplements the bootstrap api for creating an Oracle datasource. It also redirect log messages from java.util.logging to the Osgi Log Service. It does not export any api package. Bundle properties:
com.elster.jupiter.datasource.jdbcurl mandatory
com.elster.jupiter.datasource.jdbcuser mandatory
com.elster.jupiter.datasource.jdbcpassword mandatory
com.elster.jupiter.datasource.pool.maxlimit optional, default 50
com.elster.jupiter.datasource.pool.maxstatements optional, default 50
com.elster.jupiter.datasource.pool.oracle.ons.nodes ons nodes information. The "nodes=" will be automatically added.
