com.elster.jupiter.fsm

This bundle provides support for finate state machines. The main interface is FinateStateMachine, a named wrapper for a set of States and StateTransitions. Every State can be configured to trigger an external process on entry and on exit. As events are produced, the finate state machine will respond to the event and change the state accordingly. Events are globally defined and can therefore be reused across different state machines. Currently, the external processes are defined in jBPM. States can be "standard" and linked to the device life cycle management. Finate state machine that contain these standard device life cycle management states are in fact DeviceLifeCycleFinateStateMachines.
