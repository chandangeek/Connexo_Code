com.elster.jupiter.pubsub

Pubsub allows publish-subscribe-style communication between components without requiring the compenents to explicitly register with one another. Conceptually very similar to guava's com.google.common.Eventbus. Main difference is that exceptions thrown by the subscriber are propagated to the publisher
