Change the order of estimation rules in an estimation rule set

Within the rule set you can define the order of the rules to determine what estimation rule will be applied first. Typically, the estimation rules that yield the most realistic values are on top in the estimation rule set.

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets.

2.   Select the needed estimation rule set and open its detail page.

3.   Click Estimation rules in the vertical navigation menu on the left.

4.   Click the button Edit order.

5.   The column Actions changes into Ordering and it contains the reordering icon .

6.   Drag the needed estimation rule to the right position in the table and drop it there.

A green indicator  indicates when it is possible to drop the estimation rule on that position.

7.   Click Undo to cancel changing the order of estimation rules or click Save order to save your changes.
