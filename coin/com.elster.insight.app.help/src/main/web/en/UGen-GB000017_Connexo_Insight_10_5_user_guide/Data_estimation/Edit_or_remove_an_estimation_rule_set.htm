Edit or remove an estimation rule set

It is possible to edit or remove an estimation rule set to correct mistakes.

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets.

2.   Click the action icon  next to an estimation rule set in the table or click the actions button on its detail page.

3.   Click Edit or Remove.

4.   Make the required changes and click Save or confirm the removal message.
