Mark a usage point group as favourite

You can mark usage point groups as favourite so that they appear on the dashboard in My favourite usage point groups. Using this feature, usage point groups are a lot easier to access.

1

Important

The list of favourite usage point groups is specific for the logged on user. Consequently, each user may have a different list of favourite usage point groups.

Mark a usage point group as favourite and add a comment

You can mark a usage point group to make it better accessible and add a comment explaining the reason why this usage point group needs to be marked as a favourite.

5

Steps

1.   Go to Usage points > Usage point groups.

2.   Click a value in Name to open the selected usage point group details page.

3.   Click the star icon  next to the page title Overview to mark the current usage point group as favourite.

The Add comment dialogue appears and the star icon becomes solid black to indicate that the usage point group has been flagged. It will now appear on the dashboard in My usage point groups.

4.   If needed, type a comment in the field Comment.

5.   Click Add comment if you have entered a comment. Click Cancel if you don’t need to enter a comment.

6.   On the Dashboard, hover over a favourite usage point group in the widget My favourite usage point groups.

The number of favourite usage point groups is indicated between brackets next to the widget title.

7.   Hover over a favourite usage point group to see a tooltip containing some basic information about the usage point group as well as the flagging date and the comment that was added when marking the usage point group.

Remove a usage point group from the list of favourite usage point groups

You can remove a usage point group from the list of favourite usage point groups on the dashboard. The unmarked usage point group remains available in Connexo.

5

Steps

1.   Go to Dashboard.

2.   In the widget My favourite usage point groups, click the star next to the usage point group that should no longer appear in the list of favourite usage point groups.

The star becomes white (). After refreshing the page it will no longer appear on the dashboard. The usage point group can still be retrieved in Usage points > Usage point groups.sing the search page. See “Find usage points”.

Mark and unmark multiple usage point groups as favourite simultaneously

You can mark or unmark multiple usage point groups as favourite simultaneously to speed up the process. The impact will be visible in the My favourite usage point groups widget on the dashboard.

5

Steps

1.   Go to Dashboard.

2.   In the widget My favourite usage point groups, click Select.

3.   On the page Select favourite usage point groups, select the usage point groups that you want to add to the list of favourite usage point groups or deselect the usage point groups that should no longer be marked as favourite.

4.   Click Save.

The list of favourite usage point groups on the dashboard is updated accordingly.
