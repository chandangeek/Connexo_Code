Flag a usage point

You can flag usage points so that they appear on the dashboard in My flagged usage points. Using this feature, usage points that temporarily need extra attention are a lot easier to access.

1

Important

The list of flagged usage points is specific for the logged on user. Consequently, each user may have a different list of flagged usage points.

Flag a usage point and add a flagging comment

You can flag a usage point to make it better accessible and add a comment explaining the reason why this usage point needs to be flagged.

5

Steps

1.   Open a usage point landing page using the search page.

2.   Click the star icon  next to the page title Overview to mark the current usage point as favourite.

The Add comment dialogue appears and the star icon becomes solid black to indicate that the usage point has been flagged. It will now appear on the dashboard in My flagged usage points.

3.   If needed, type a comment in the field Comment.

4.   Click Add comment if you have entered a comment. Click Cancel if you don’t need to enter a comment.

5.   On the Dashboard, hover over a flagged usage point in the widget My flagged usage points.

   The number of flagged usage points is indicated between brackets next to the widget title.

6.   Hover over a flagged usage point to see a tooltip containing some basic information about the usage point as well as the flagging date and the comment that was added when flagging the usage point.

Unflag a usage point

You can remove the flag on a usage point to remove it from the list of flagged usage points on the dashboard. The ‘unflagged’ usage point remains available in Connexo.

5

Steps

1.   Go to Dashboard.

2.   In the widget My flagged usage points, click the star next to the usage point that should no longer appear in the list of flagged usage points.

The star becomes white (). After refreshing the page it will no longer appear on the dashboard. The usage point can still be retrieved using the search page. See “Advanced search”.
