Estimate data on request for a purpose

Run the estimation process manually to estimate data for a purpose on the metrology configuration that is associated with the current usage point. The estimation is based on the rules that are configured on the metrology configuration.

5

Steps

1.   Open a usage point with configured validation rules using the search page.

2.   Click the desired purpose in the group Purposes in the vertical navigation bar to the left.

3.   Select Estimate now from the action button.

4.   Choose Re-validate estimated data if you want Connexo to automatically revalidate the estimated data.

5.   Click Estimate.

A progress bar appears while the data is estimated. After successful estimation a message is shown.

1

Important

Estimation is done immediately. If the estimation process takes more than 60 seconds, then estimation will continue in the background and you can proceed with other tasks. Note that no logging is created. So, if estimation fails, it won’t be possible to check why it failed. If you run estimation from the estimation task, then logging is created. See “Run estimation on data for all usage points associated with a particular metrology configuration and purpose”.
