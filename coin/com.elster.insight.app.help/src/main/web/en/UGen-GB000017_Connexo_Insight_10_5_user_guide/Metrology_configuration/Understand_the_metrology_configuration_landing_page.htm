Understand the metrology configuration landing page

The metrology configuration landing page shows the entire setup information that can be applied to a set of usage points all in one place.

You must have sufficient privileges to view the metrology configuration.

View the metrology configuration landing page

5

Steps

1.   Go to Administration > Metrology configurations.

2.   Click the link on the name of applicable metrology configuration to open its landing page.

Under General information you find Name, Description, Status and Allow gaps.

Allow gaps indicates whether gaps between meters on a meter role and between metrology configuration start/stop date and meter activations are allowed on usage points.

The Usage point requirements group the attributes that are available and specific for this metrology configuration. They depend on the service category, on the type and on the attributes that were deemed necessary for the selected metrology configuration. The Meter specifications list the meter roles that have been made available for this metrology configuration. They depend on the service category and on what was deemed necessary for the selected metrology configuration.

Purposes of the metrology configuration are listed in a table. Each purpose can be collapsed or expanded to hide or show the outputs of a purpose. For each purpose output the following information is shown in the table:

Field

Description

Output name

An alias that serves as a reference in formulas for output calculations. Name can be Billing, Information, Voltage monitoring. Others may be available depending on what your Connexo delivery team installed.

Reading type

Fully specified reading type. Reading types are unique per metrology configuration purpose.

Formula

Textual representation of a formula provided by the data aggregation framework. The formula is used to calculate the output.

3.   Select one of the outputs in the Purposes table to see its details in the details pane below.

In addition to what is already available in the table, you get more detailed reading type information and a detailed list of the formula components. They contain a list of inputs that are grouped by meter role as well as a list of attributes. Both inputs and attributes are part of the formula for the selected output.

View and manage custom attributes of a metrology configuration

You can always view the custom attribute sets that have been made available for a metrology configuration. Only if the metrology configuration hasn’t yet been activated, you can remove or add custom attribute sets to the selected metrology configuration. The custom attribute sets are used to capture the specific information that you require about usage points linked to a specific metrology configuration.

5

Steps

1.   Go to Administration > Metrology configurations.

2.   Click Custom attributes in the vertical navigation bar on the left.

On the Custom attribute sets page the custom attributes for the selected metrology configuration are listed in a table.

3.   Select a custom attribute set to see a list of its attributes in the details pane below.

4.   Click the remove icon  at the bottom of a row in the custom attribute sets table to remove it from the metrology configuration.

5.   Click Remove to confirm removal.

6.   Click the button Add custom attribute sets to add on of the custom attribute sets that are available depending on the service category of the metrology configuration. Each set can be added only once.

View and manage the validation rule sets on a metrology configuration

Understand how the validation is configured on the selected metrology configuration. Verify what validation rules are applicable for a specific purpose.

5

Steps

1.   Go to Administration > Metrology configurations.

2.   Click Validation configuration in the vertical navigation bar to the left.

On the Validation configuration page the validation rule sets are listed in a table and they are grouped by purpose. For each validation rule set you can see the start date of its active version. You can click the link on each validation rule set’s name to open its landing page.

3.   Select a validation rule set to see the validation rules of the active validation rule set version in the table below.

In the table you can see what validation rules are active and which are inactive. You can click the link on each validation rule to open its landing page.

4.   Select a validation rule to see its details in the details pane below or click it to see its details on a separate page.

5.   Click the remove icon  at the bottom of a row in the validation rule set table to remove it from the metrology configuration.

6.   Click Remove to confirm the removal.

7.   Click the button Add validation rule set to add a new validation rule set.

8.   Select one or more applicable validation rule sets.

You only see the validation rule sets that have reading types that match those of the current purpose and that haven’t been added to the metrology configuration’s purpose yet.

If there are no suitable validation rules available, then click the button Add validation rule set in the No validation rule sets found panel to create one. See “Data validation”.

9.   In Usage point states, specify if the validation rule set must be applied to the data when the usage point is in one or more specific states or if it can be applied to the data no matter what state the usage point is in at that moment.

10.   Click Add.

 

1

Important

From the validation rule set page it is possible to link the same validation rule set to multiple metrology configuration simultaneously. See “Manage linked metrology configuration purposes on a validation rule set”.

View and manage the estimation rule sets on a metrology configuration

Understand how estimation is configured on the selected metrology configuration. Verify what estimation rules are applicable for a specific purpose.

5

Steps

1.   Go to Administration > Metrology configurations.

2.   Click Estimation configuration in the left navigation menu.

3.   In the first table you can see the list of estimation rule sets that are associated with the selected metrology configuration. They are grouped by the metrology configuration purpose. For each estimation rule set, there is a numeric overview of the active and inactive rules that each estimation rule set contains.

4.   The purpose groups in the estimation rule sets table can be collapsed by clicking the minus button (-).

5.   You can click the estimation rule set name to see its details on a separate page.

Under the estimation rule set table the active and inactive rules that are associated with the selected estimation rule set are shown.

Under the estimation rules table, the details of the selected estimation rule are shown in a details pane.

6.   Select an estimation rule to see its details in the details pane below or click it to see its details on a separate page.

7.   Click the remove icon at the bottom of a row in the estimation rule set table to remove it from the metrology configuration.

8.   Click Remove to confirm the removal.

9.   Click the button Edit order just above the table to change the order of estimation rule sets.

10.   The column Actions changes into Ordering and it contains the reordering icon .

11.   Drag the needed estimation rule set to the right position in the table and drop it there.

A green indicator  indicates when it is possible to drop the estimation rule on that position.

12.   Click Undo to cancel changing the order of estimation rules or click Save order to save your changes.

3

Note

Only the order of the estimation rule sets can be changed. The order of the estimation rules is shown but can’t be edited.

 

13.   Click the button Add estimation rule set to add a new estimation rule set.

14.   Select one or more applicable estimation rule sets.

You only see the estimation rule sets that have reading types that match those of the current purpose and that haven’t been added to the metrology configuration’s purpose yet.

If there are no suitable validation rules available, then click the button Add estimation rule set in the No estimation rule sets found panel to create one. See “Data estimation”.

15.   Click Add.

 

1

Important

From the estimation rule set page it is possible to link the same estimation rule set to multiple metrology configuration simultaneously. See “Manage linked metrology configuration purposes on an estimation rule set”.
