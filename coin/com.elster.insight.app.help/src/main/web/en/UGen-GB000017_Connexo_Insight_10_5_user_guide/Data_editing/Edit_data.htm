Edit data

You can edit data that is marked as suspect and that you expect is false due to certain technical issues. You can edit both channel and register data.

Edit channel data

You can edit suspect channel data to correct false values caused by a technical issue.

5

Steps

1.   Open a usage point using the search page.

2.   Click the needed purpose to see its outputs.

3.   Click a channel to see its readings.

4.   In the table with readings under the graph, select the row with a suspect value.

 Suspect values are marked with a red flag .

5.   Click the action icon  at the end of the selected row and select Edit.

The corresponding cell holding the value will become editable allowing you to edit it inline.

Alternatively, you can simply click the needed value and start editing it inline

6.   Edit the value.

Click outside the edited cell to finish the inline editing.

The value is updated in the table. The red flag has disappeared and the edit icon  indicates that the value has been edited. On the graph the value is shown as an edited value.

7.   Click Save changes.

The value is no longer suspect.

3

Note

instead of editing value per value, you can also simply paste a row of values from an external source - for instance from MS Excel. Just click the value in the first row of the readings table that needs to be edited. This puts the cell in edit mode. Then press CTRL-V. The copied values are pasted in the consecutive rows of the table.

Edit register data

You can edit suspect register data to correct false values caused by a technical issue.

5

Steps

1.   Open a usage point using the search page.

2.   Click the needed purpose to see its outputs.

3.   Click a register to see its readings.

4.   In the table with readings, select the row with a suspect value.

 Suspect values are marked with a red flag .

5.   Click the action icon  at the end of the selected row and select Edit to open the edit page.

6.   Edit the value.

7.   Click Save.

The value is updated in the table. The red flag has disappeared and the edit icon  indicates that the value has been edited. The value is no longer suspect.

Add register data

In case of a manual meter read, you can add register data by entering a value yourself.

5

Steps

1.   Open a usage point using the search page.

2.   Click the needed purpose to see its outputs.

3.   Click a register to see its readings.

4.   Click the button Add reading just above the table to open the Add reading page.

5.   Enter the read value and the corresponding measurement time.

6.   Click Add.

The reading is now listed in the overview table of the register readings.

3

Note

Depending on the kind of register, the behaviour of the Add reading page will be slightly different.

Register

Characteristics of the Add reading page

Index value register

Measurement time and value must be entered.

Value shouldn’t exceed the configured overflow value.

Billing register

Measurement time, start of period, end of period and value must be entered. The measurement time should be within the specified period, i.e. the interval of the billing register. Also, the value shouldn’t exceed the configured overflow value.

Flag register

Measurement time and value must be entered.

Text register

Measurement and value must be entered. The value is text.
