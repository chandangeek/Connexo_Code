Estimate channel readings

Instead of manually editing suspect channel data that you know to be incorrect you can let Connexo estimate a particular value using an available estimator.

Estimate a channel reading using a specific estimator

You can estimate a suspect channel reading using a specific estimator to calculate the most appropriate value.

5

Steps

1.   Open a usage point using the search page.

2.   Click the needed purpose to see its outputs.

3.   Click a channel to see its readings.

4.   In the table with readings under the graph, select the row with a suspect value.

 Suspect values are marked with a red flag .

5.   Click the action icon  at the end of the selected row and select Edit with estimator to open the Edit with estimator popup.

6.   Select an estimator to show its properties.

For estimation rules with attributes that are configured on the rule and on the reading type, all attributes are shown. The attributes that are configured on the reading type level come first.

7.   Review the estimator’s properties and make the needed adjustments.

8.   Click Estimate.

The data is updated in the table. The red flag has disappeared and the estimation icon  indicates that the value has been estimated. On the graph, the estimated value is shown as an edited value.

9.   Click Save changes.

The value is no longer suspect.

The reading quality for this value will be Manually edited - generic (2.7.0).

Estimate a channel reading using an estimation rule

You can estimate a suspect channel reading using a specific estimation rule to calculate the most appropriate value. The estimation rule has a specific estimator and reading types associated with it. See “Add an estimation rule to an estimation rule set and activate it”.

5

Steps

1.   Open a usage point using the search page.

2.   Click the needed purpose to see its outputs.

3.   Click a channel to see its readings.

4.   In the table with readings under the graph, select the row with a suspect value.

 Suspect values are marked with a red flag .

5.   Click the action icon  at the end of the selected row and select Estimate with rule to open the Estimate with rule popup.

6.   Select an estimation rule to show its properties.

Estimation rule contains only the estimation rules (active or inactive) from the estimation configuration that have reading types that match the channel. If several estimation rule sets contain matching estimation rules with the same name, then they are shown in the dropdown with the name of the estimation rule set between brackets.

If attributes have been changed on the reading type level for this estimation rule, then the changed values are shown.

7.   Review the estimation rule’s properties. The estimation rule’s properties can’t be adjusted on this page. See “Edit or remove an estimation rule”.

8.   Click Estimate.

The data is updated in the table. The red flag has disappeared and the estimation icon  indicates that the value has been estimated. On the graph, the estimated value is shown as an edited value.

9.   Click Save changes.

The values are no longer suspect.

The reading value for this value will be Estimated with a specific rule (2.8.y) where y is the estimation method ID.

Estimate multiple channel readings using a specific estimator

You can estimate multiple suspect channel readings using an estimator simultaneously to work faster.

5

Steps

1.   Open a usage point using the search page.

2.   Click the needed purpose to see its outputs.

3.   Click a channel to see its readings.

4.   In the table with readings under the graph, select the rows with a suspect value.

 Suspect values are marked with a red flag .

You can select multiple rows by holding the ctrl key while selecting.

5.   Click Bulk action on top of the table and select Edit with estimator to open the Edit with estimator popup.

6.   Select an estimator to show its properties.

7.   Review the estimator’s properties and make the needed adjustments.

8.   Click Estimate.

The data is updated in the table. The red flag has disappeared and the estimation icon  indicates that the value has been estimated. On the graph, the estimated value is shown as an edited value.

9.   Click Save changes.

The value is no longer suspect.

The reading quality for the edited values will be Manually edited - generic (2.7.0).
