Confirm data

You can confirm data that is marked as suspect if you know that the data is as it is expected. Both channel and register data can be confirmed.

Confirm a single channel reading

Suspect channel readings can be confirmed one by one.

5

Steps

1.   Open a usage point using the search page.

2.   Click the needed purpose to see its outputs.

3.   Click a channel to see its readings.

4.   In the table with readings under the graph, select the row with a suspect value.

 Suspect values are marked with a red flag .

5.   Click the action icon  at the end of the selected row and select Confirm.

In the table, the red flag has disappeared and the confirm icon  indicates that the value has been confirmed. On the graph the value is shown as an edited value.

6.   Click the button Save changes on top of the table.

The value is no longer suspect.

Confirm multiple channel readings

You can confirm multiple suspect channel readings simultaneously to work faster.

5

Steps

1.   Open a usage point using the search page.

2.   Click the needed purpose to see its outputs.

3.   Click a channel to see its readings.

4.   In the table with readings under the graph, select all the rows with suspect values that you need to confirm.

 Suspect values are marked with a red flag .

You can select multiple rows by holding the ctrl key while selecting.

5.   Click Bulk action on top of the table and select Confirm.

In the table, the red flag has disappeared and the confirm icon  indicates that the values have been confirmed. On the graph the values are shown as edited values.

6.   Click the button Save changes on top of the table.

The value of the selected values is no longer suspect.
