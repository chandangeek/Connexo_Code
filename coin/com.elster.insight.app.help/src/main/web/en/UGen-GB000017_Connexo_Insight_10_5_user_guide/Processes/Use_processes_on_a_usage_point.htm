View running processes on a usage point

You can view the processes that are currently running on a usage point to better understand what Connexo is doing with the usage point in the background.

5

Steps

1.   Open a usage point using the search page.

2.   Click Processes in the vertical navigation bar on the left to open the Processes page.

3.   Click the tab Running processes.

The processes that are currently running on the selected usage point are shown in a table. In the details pane under the table you can see the related open tasks. The assignment information is added between brackets. Each task is a clickable link that refers to its detail page. See “Task management”.

4.   Click the tab Status overview in the details pane under the table to see the steps that the process has already gone through. The variables that were used during these steps are also shown. This information is important when troubleshooting a process.

View process history on a usage point

Once a process has completed it is no longer shown under the Running processes tab. Completed processes are shown under the tab History.

5

Steps

1.   Open a usage point using the search page.

2.   Click Processes in the vertical navigation bar on the left to open the Processes page.

3.   Use the filter bar on top to narrow down the list of shown processes. Don’t forget to click Apply to execute your filter.

4.   Click the tab History.

All processes that have completed on the selected usage point are shown in a table including their duration and information on how they have ended. In the details pane under the table the end date and version information are added.

5.   Click the tab Status overview in the details pane under the table to see the steps that the process has already gone through. The variables that were used during these steps are also shown. This information is important when troubleshooting a process.
