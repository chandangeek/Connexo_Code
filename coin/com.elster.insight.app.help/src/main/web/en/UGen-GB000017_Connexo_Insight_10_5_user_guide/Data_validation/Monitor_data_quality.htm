Monitor data quality

Connexo offers you a central view on usage points with their reading qualities. On these pages you can quickly evaluate the data quality of a usage point. For instance, you can see the number and type of suspect readings, edited and estimated values.

View data quality

This page shows a filtered list of usage points with an overview of the related data quality like the number of suspect, confirmed, estimated, edited and informative values as well as a more detailed breakdown of these numbers.

5

Steps

1.   Go to Workspace > Data validation > Data quality.

2.   Enter the needed filter values in the filter bar on top to narrow down the data quality results. Don’t forget to click Apply to execute your filter.

By default, the filter is set to display today’s data.

3.   The table shows a list of usage points with for each of their purposes the number of suspects, confirmed values, estimated values, informatives, edited and projected values. If used, the number of project values is also shown. The list is sorted on usage point name, then on metrology configuration activation time and lastly on the purpose.

You can click on the usage point, metrology configuration and purpose names to go to their landing pages.

Metrology configurations from the past are indicated with an icon .

4.   The details pane shows more detailed information about the usage point data quality in different sections.

  • The first section identifies the usage point, metrology configuration and purpose for which the information is shown.

  • The second section shows the number of suspect readings for registers and channels as well as the date for the last suspect reading.

  • The third section shows the number of suspects based on their type. The informatives aren’t included in this list.

  • The fourth section shows the number of edited values broken down in added, edited and removed values.

  • The fifth section shows the number of suspects on the purpose of a usage point per estimator.

View data quality KPIs

The data quality KPIs allow you to monitor the data quality in a usage point group. They can be viewed on the Connexo Workspace. See “View data quality”. The KPIs will calculate the fixed metrics as described in “View data quality” but for different usage point groups, purposes and with different calculation frequencies.

5

Steps

1.   Go to Administration > Data validation > Data quality KPIs

The available data quality KPIs are shown in a table. For each KPI you can see for what usage points and associated purposes the metrics are calculated with the frequency of their calculation cycle. The last calculation date is also shown.

 

1

Important

KPIs for inactive purposes of usage points aren’t included in the KPI calculations.

KPIs are each time calculated for a period of three months and in line with their respective frequencies. For this reason,

  • Meter readings with reading qualities that are updated after the last calculated KPI will exist in Connexo but won’t be considered until the next KPI calculation cycle has taken place. For instance, a daily KPI that is calculated yesterday doesn’t contain reading qualities that have been changed today.

  • KPIs shown for periods that are larger than three months may be inaccurate for that part of the chosen period that is beyond the month limit.

Add a data quality KPI

You can add a data quality KPI that will be shown on the Data quality page on the Connexo Workspace.

5

Steps

1.   Go to Administration > Data validation > Data quality KPIs.

2.   Click the button Add data quality KPIs.

3.   Select a usage point group.

The new data quality KPI will be calculated for all the usage points in this usage point group.

If the needed usage point group doesn’t exist yet, then first create it. See “Add, edit and remove usage point groups”.

After selection of the usage point group the relevant purposes appear on screen.

4.   In Purpose select the needed purposes.

5.   In Calculation frequency select the needed frequency to calculate the predefined metrics on the selected usage points.

6.   Click Add.

The new data quality KPI is now listed in the overview table. It will be calculated on the specified time.

Remove a data quality KPI

You can remove data quality KPIs to correct an error or to avoid calculations that are no longer needed.

5

Steps

1.   Go to Administration > Data validation > Data quality KPIs.

2.   Select the needed data quality KPI.

3.   Click the remove icon in the column Actions .or select Remove from the Actions button in the details pane.

4.   Click Remove to confirm.

The data quality KPI is no longer listed in the overview table.
