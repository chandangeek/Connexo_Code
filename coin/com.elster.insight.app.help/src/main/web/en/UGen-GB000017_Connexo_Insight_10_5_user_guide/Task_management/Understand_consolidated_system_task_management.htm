Understand consolidated system task management

Throughout Connexo Insight a number of system tasks can be added and monitored. For instance data estimation and data validation tasks. You can use the consolidated system task management page to see and manage all of these system tasks together on one page. Note that data estimation and validation tasks can also be monitored in “View validation tasks” and “View estimation tasks”. However, there are also system tasks that can only be seen in this consolidated task management.
