Switching apps

Understand the application centre

The Connexo platform is built up out of different applications. The applications Admin and MultiSense are standard. Depending on the activities you want to perform, other applications are available and can be purchased. For example Facts is the application to design your own reports.

Switch between applications

You can easily switch between applications without having to login every time, simply go to the application centre and select a different application.

5

Steps

1.   Click the application centre button in the upper left corner.

2.   Select the application you want to use.

No login is required when changing applications. The active application is shown in white.
