Overview page layout listing items of a particular feature

When opening a feature in Connexo Insight, in most cases you get a feature landing page which is an overview page listing the items for that feature. These items can be configurations, instances or results related to the feature. This kind of overview pages has a similar layout in Connexo.

The image below explains how to use the overview layout.

Figure: Overview page layout.

 

A. Breadcrumb: Indicates your current location in the application and allows easy navigation to pages that are higher in the navigation structure.

B. Feature navigation menu: Displays the available components that are linked to the main feature. Some items may be grouped for a better overview of related information. These groups can be collapsed or expanded for more efficient use of the available space on the navigation menu. Click the down arrow next to the grouping item  to expand the group and see further links leading to more information about the shown object. Click the up arrow  to collapse the group and make the navigation menu more compact. Your settings will be remembered the next time you visit the page.

C. Overview table: Shows a list containing the available records for the current feature. In most cases, the value in Name is a link referring to the item’s detail page. Depending on your permissions multiple actions can be performed on the items in the table. For example, they can be edited, removed, activated, viewed, etc.

D. Details pane: Shows all details of the item that is selected in the table above. The first table item is selected by default.

E. Feature action button: Button to do an action that doesn’t relate to one specific item in the table. For example, add a new item or a bulk action on more than one item at the same time. If multiple actions are available, then the button can be expanded to show the different options. The actions that are available to you depend on your privileges and on the feature.

F. Item action icon: Click the action icon in the column Actions to show the different actions that can be done on the item that is selected in the table. For example view, edit, remove, etc. The actions that are available to you depend on your privileges and on the feature.

G. Item action button: Click the action button in the details pane to show the different actions that can be done on the item that is currently viewed. The actions that are available to you depend on your privileges and on the feature.

H. Grid export button: Export the table contents to a CSV file. Only the data that is visible on screen will be exported
