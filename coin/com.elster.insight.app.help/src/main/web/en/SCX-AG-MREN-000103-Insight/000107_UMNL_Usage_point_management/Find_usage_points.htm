Find usage points

Use Connexo’s search function to look for the desired usage points.

5

Steps

1.   Click Search in the upper right corner of any page in Connexo Insight .

2.   Specify the object that you are searching for in Search for. By default, Usage point is already selected.

3.   Specify the criteria in Criteria.

4.   Specify any additional criteria by clicking the button Add criteria.

5.   Click Search.

6.   The search results are displayed in a paged table under the criteria.

7.   Click the MRID of any item to open it.
