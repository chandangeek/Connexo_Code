Action buttons

On most pages in Connexo where objects are shown you have the possibility to do one or more actions on the object that is shown or selected on the page. These actions are always offered on the same locations on the page and they are the same on each location.

Actions on single objects are usually available on the following locations:

  • At the end of each row in the last column in tables that are listing objects. This column is called Actions and the available actions are represented by the actions icon . Click the actions icon to show the list of available actions for the selected object.

  • On the title bar of a details pane to the right under a table that is listing objects. The available actions are represented by the actions button. Click the actions button to show the list of available actions for the selected object.

  • In the upper right corner of the details page of an object that is typically reached by clicking an object in a table listing the available objects of that type. The available actions are represented by the actions button . Click the actions button to show the list of available actions for the shown object.

Note

The available actions on an action button are always grouped in four blocks:

  • Actions starting a service on the object, e.g. (de)activation, run, etc.

  • Actions to update or edit the object.

  • Actions to view more detailed information about the object such as history or logging information.

  • Removal action.

Within each block the items are sorted alphabetically. If there isn’t any action available in a block, then this block isn’t shown in the list.

Actions done on multiple objects simultaneously are called Bulk action. They are represented by the bulk action button. Click the bulk action button to start the wizard that will guide you through the process of selecting the needed objects, the action and entering the needed parameters to do the selected action.
