Monitor import services

See what import services are available and how they are doing.

View the available import services

You can view the available import services to get an idea of the import capabilities in Connexo Insight .

5

Steps

1.   Go to Administration > Data exchange > Import services.

The paged overview table shows all configured import services. For each import service the following information is shown:

  • Name of the import service.

  • File importer determining the type of data that is imported.

  • Status of the import service.

  • Import folder from where files are picked up for import.

  • File pattern of the files that are imported. Only files in the import folder with this pattern are picked up for import.

In the details pane under the table, further details of the selected import service are shown.

View details of an import service

For each import service you can open a detail page with all the information about the selected import service.

5

Steps

1.   Go to Administration > Data exchange > Import services.

2.   Click the name of the needed import service.

All available fields are displayed on a full page. In vertical navigation bar on the left, you find links to related information such as the history of occurrences of the selected import service.

Parameter

Description

Name

Free text name of the import service.

File importer

Technical identifier of the importer.

Status

The import service can be active or inactive. The import service must be active before it can be used.

Import folder

Path to the location where files meant for import must be put. This location is relative to the root location that is defined on the application server and must be on the same machine as the application server itself.

Allow uploads in application

If selected, then users can select a file and upload it using the selected importer without needing access to the defined import folder.

File pattern

Pattern to filter which files will be imported, based on file name and/or extension. Click the info icon for some suggestions.

Folder scan frequency

The frequency with which Connexo looks in the import folder to find new files that are waiting to be imported.

In progress folder

Folder where import files are moved to from the Import folder after they have been picked up by the selected importer.

Success folder

Folder where import files are moved to from the In progress folder after the import has ended successfully.

Failure folder

Folder where import files are moved to from the In progress folder after the import has ended with unsuccessfully.

Delimiter

Symbol used to separate values in the import file.

Date format

Format for date and time values in the import file.

Time zone

Considered time zone for the data in the importer.

Number format

Format for number values in the import file.

1

Important

  • The parameters of import services can’t be edited in Connexo Insight . They are managed in Connexo Admin.

  • Don’t forget that the import service must be added to an active application server before it can be used.

  • The imported files remain available after import for your reference.

Refer to the Connexo Admin user guide for more detailed information.

View history of an import service

You can view all occurrences of a selected import service. This comes in handy if you need to do some troubleshooting.

5

Steps

1.   Go to Administration > Data exchange > Import services.

2.   Click the name of the needed import service to open its detail page.

3.   Click History in the vertical navigation bar on the left.

Alternatively, select View history from the actions icon at the bottom of the selected row in the import services overview table.

4.   Filter the import service occurrences by specifying a particular time window or status.

Don’t forget to click Apply to execute your filter.

The occurrences (history) of the selected import service are shown in a paged table. For each occurrence, the following information is shown:

  • Start date and time.

  • Duration.

  • Status.

In the details pane, further details are shown about the selected occurrence.

View logging information of an import service occurrence

For each import service occurrence you can view detailed logging information allowing you to investigate what steps were made to come to the shown status. This is the most detailed track for troubleshooting purposes.

5

Steps

1.   Go to Administration > Data exchange > Import services.

2.   Click the name of the needed import service to open its detail page.

3.   In the vertical navigation menu on the left, click History to open the History page.

4.   Click the import service occurrence for which you want to see the logging information to open the Log page. 

On top there is summarised information about the selected import service. The logging information is shown in a paged table that you can sort. For each logged step the following information is shown:

  • Time.

  • Log level.

  • Message.

 

To sort data:

5

Steps

1.   Select the items you want to sort the table on from the button Add sort.

The items are added to the sort bar.

2.   Click the arrow to the left of a sort button on the sort bar to change the sort order of that column.

3.   Click the cross to the right of a sort button on the sort bar to remove it.

The data is no longer sorted after this column.

4.   Click the button Clear all to remove all sort buttons.

3

Note

The default sort order is errors first, then the oldest items.

View history of combined import services

For advanced troubleshooting you can view the history of combined import services.

5

Steps

1.   Go to Administration > Data exchange > Import history > Data exchange > Import history.

2.   Filter the occurrences by selecting one or more import services, a time window and one or more statuses on the filter bar.

3.   Don’t forget to click Apply to execute your filter.

The occurrences of the selected filters are shown in a paged table that behaves exactly the same as the history page of a single import service. Similarly, by clicking the name of the occurrence you can view detailed logging information for the selected occurrence.

See also “View history of an import service” and “View logging information of an import service occurrence”.

3

Note

Depending on your privileges the import history may be available to you from the workspace.
