Reset data

You can restore edited data from a register or channel to revert to the initial aggregated data from the meters.

Reset channel data to initial aggregated value

You can restore the initial aggregated value of an edited channel reading to correct eventual mistakes.

5

Steps

1.   Open a usage point using the search page.

2.   Click the needed purpose to see its outputs.

3.   Click a channel to see its readings.

4.   In the table with readings under the graph, select the row with an edited channel reading.

Edited channel readings are marked with an edit icon  or estimation icon .

5.   Click the action icon  at the end of the selected row and select Reset.

The value in the table is replaced by the initial value and it is marked by the reset icon . On the graph, the restored value is shown as an edited value.

6.   Click Save changes.

Reset register data to initial aggregated value

You can restore the initial aggregated value of an edited register reading to correct eventual mistakes.

5

Steps

2.   Click the needed purpose to see its outputs.

3.   Click a register to see its readings.

4.   In the table with readings, select the row with an edited register reading.

 Edited register readings are marked with an edit icon  or estimation icon .

5.   Click the action icon  at the end of the selected row and select Reset.

The value in the table is replaced by the initial value and it is marked by the reset icon . On the graph, the restored value is shown as an edited value.

1

Important

In case there is no underlying data, i.e. in the case of a manually added register reading, you aren’t able to restore to the initial aggregated value. Instead of a reset you can select Remove. The selected register reading and its measurement time will no longer be available after removal.

Reset multiple channel readings to their initial aggregated values

You can restore the initial aggregated value of multiple edited channel readings simultaneously to work faster.

5

Steps

1.   Open a usage point using the search page.

2.   Click the needed purpose to see its outputs.

3.   Click a channel to see its readings.

4.   In the table with readings under the graph, select the rows with an edited channel reading.

Edited channel readings are marked with an edit icon  or estimation icon .

You can select multiple rows by holding the ctrl key while selecting.

5.   Click Bulk action and on top of the table and select Reset.

The values in the table are replaced by their initial values and they are marked by the reset icon . On the graph, the restored values are shown as edited values.

6.   Click Save changes.
