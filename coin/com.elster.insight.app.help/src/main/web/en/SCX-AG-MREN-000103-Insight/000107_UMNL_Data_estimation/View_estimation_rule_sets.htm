View estimation rule sets

Get a complete overview of all estimation rule sets that are available in Connexo Insight  and drill down to the needed detail level.

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets.

The complete list of estimation rule sets with a numeric overview of the active and inactive rules each rule set contains is shown in the first table.

Details of each estimation rule set can be viewed by clicking the link in its name.

The second table shows the estimation rules that are associated with the selected estimation rule set. For each rule, its status is indicated. Click the estimation rule’s name to its details on a separate page.

At the bottom of the page is a pane showing the details of the selected estimation rule. See “Add an estimation rule to an estimation rule set and activate it”.
