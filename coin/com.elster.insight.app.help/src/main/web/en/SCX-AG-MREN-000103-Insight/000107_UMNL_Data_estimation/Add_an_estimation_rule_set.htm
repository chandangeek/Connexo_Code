Add an estimation rule set

Estimation rule sets must first be created before they can be associated with a metrology configuration in a separate step.

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets.

2.   Click the button Add estimation rule set.

3.   On the Add estimation rule set page specify a Name and a Description for future reference.

4.   Click Add.

The new estimation rule set is listed in the first table with estimation rule sets.

After adding an estimation rule set, the next task is to add estimation rules. “Add an estimation rule to an estimation rule set and activate it”.
