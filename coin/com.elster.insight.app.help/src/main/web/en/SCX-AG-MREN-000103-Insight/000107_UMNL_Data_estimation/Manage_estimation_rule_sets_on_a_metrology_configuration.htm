Manage estimation rule sets on a metrology configuration

Estimation rule sets are associated with a specific purpose of a metrology configuration. Because a usage point is also linked with a metrology configuration, it becomes clear what estimation rules apply to the data of each usage point.

View associated estimation rule sets on a metrology configuration

For each metrology configuration an orderly overview of its estimation configuration is available on one page.

5

Steps

1.   Go to Administration > Metrology configurations.

2.   Click Estimation configuration in the left navigation menu.

3.   In the first table you can see the list of estimation rule sets that are associated with the selected metrology configuration. They are grouped by the metrology configuration purpose. For each estimation rule set, there is a numeric overview of the active and inactive rules that each estimation rule set contains.

4.   The purpose groups in the estimation rule sets table can be collapsed by clicking the minus button (-).

5.   You can click the estimation rule set name to see its details on a separate page.

Under the estimation rule set table the active and inactive estimation rules that are associated with the selected estimation rule set are shown.

Under the estimation rules table, the details of the selected estimation rule are shown in a details pane.

Add an estimation rule set to a metrology configuration

An estimation rule set can be added to a metrology configuration for one of its purposes. The estimation rule set must be created and configured beforehand in Administration > Data Estimation > Estimation rule sets. See “Add an estimation rule set”. Only estimation rule sets with reading types that match those of the selected metrology configuration purpose are shown. An estimation rule can be added only once to the same metrology configuration purpose.

5

Steps

1.   Go to Administration > Metrology configurations.

2.   Click Estimation configuration in the left navigation menu.

3.   Click Add estimation rule set.

4.   In Purpose, select one of the purposes of the current metrology configuration.

5.   Select on or more of the available estimation rule sets in the table by selecting the checkbox to the left of each line. If there is no valid estimation rule set available, click Add estimation rule set to add one as described in “Add an estimation rule set”.

For your information the selected estimation rule set’s estimation rules are shown in the table at the bottom. Details of the selected estimation rule are shown in the details pane under the table.

6.   Click Add.

The selected estimation rule set(s) are now added to the estimation rule sets table. If the estimation rule set was added for another purpose, then you will notice that an additional group has been added to the table.

1

Important

Data estimation is always active on the metrology configuration. It isn’t possible to deactivate it.

Remove an estimation rule set from a metrology configuration

An estimation rule set can be removed from a metrology configuration to correct mistakes. The rule set will remain available to add it to other metrology configurations or to the same one at a later time.

5

Steps

1.   Go to Administration > Metrology configurations.

2.   Click Estimation configuration in the left navigation menu.

3.   Select the needed estimation rule set.

4.   Click the cross in the column Actions.

5.   Confirm the removal.

1

Important

Removing estimation rule sets is only possible on active and inactive metrology configurations.
