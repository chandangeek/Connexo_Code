Understand the field Estimation comment

The field Estimation comment is a dropdown containing predefined comments. The available estimation comments have been created by your Connexo delivery team. You can specify one or more estimation comments during manual editing/estimation of channel data to understand later on how the readings were modified. If there are multiple reading types, e.g. collected, calculated, then the estimation comment can be shown for both the collected and calculated value. This field is shown on all the details panels for channel readings. In Connexo Insight, you will find it on the channel data page and on the reading history page.

The estimation comments are stored in specific reading qualities. You can add estimation comments to the following reading qualities:

  • x.8.y - Estimated by a specific rule.

  • x.7.0 - Manually edited.

  • x.7.1 - Manually added.

  • x.7.3 - Manually removed.

 

So, only estimated and edited values can have an estimation comment.
