Edit or remove an estimation rule

It is possible to edit or remove an estimation rule to correct mistakes.

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets.

2.   Select the needed estimation rule set and open its detail page.

3.   Click Estimation rules in the vertical navigation menu on the left.

4.   Click the action icon  next to the needed estimation rule in the table or click the actions button on its detail page.

5.   Click Edit or Remove.

6.   Make the required changes and click Save or confirm the removal message.

 

1

Important

Changing the name of an estimation rule can potentially remove estimation attributes that were changed on the usage point output. See “View/Edit/Restore estimation attributes on reading type level”.
