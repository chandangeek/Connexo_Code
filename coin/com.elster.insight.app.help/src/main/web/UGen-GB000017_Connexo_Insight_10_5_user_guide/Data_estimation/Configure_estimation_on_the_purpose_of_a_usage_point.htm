Configure estimation on the purpose of a usage point

On the purpose of a usage point you can do some actions on how the estimation is set up via the metrology configuration that is linked with the concerned usage point. For instance, you can have a look at the details of the rule sets and rules or (de)activate estimation for the purpose.

View the estimation configuration on a usage point purpose

You can see how estimation is configured on a usage point purpose - if you have the needed privileges. The shown estimation rule sets and rules are managed on the metrology configuration that is linked with the concerned usage point.

5

Steps

1.   Open a usage point that is linked to a metrology configuration with configured estimation. You can use the “Search” feature for this.

2.   Select a purpose under Purposes in the navigation bar on the left.

3.   Click the Estimation configuration tab on top of the purpose page.

On this page you can see all the details of the selected purpose’s estimation configuration, i.e.

  • Status.

  • Estimation rule sets.

  • Estimation rules.

4.   Click on the name of a rule set or rule to see its detail page.

(De)activate estimation for a usage point purpose

You can temporarily activate or deactivate the estimation for a specific purpose of a usage point.

5

Steps

1.   Open a usage point that is linked to a metrology configuration with configured estimation. You can use the “Search” feature for this.

2.   Select a purpose under Purposes in the navigation bar on the left.

3.   Click the Estimation configuration tab on top of the purpose page.

4.   Click the button Deactivate data estimation to deactivate the validation on the selected usage point purpose.

If estimation was currently inactive on the purpose, then the button will read Activate data estimation. So, you can easily toggle the estimation status of the usage point purpose.

(De)activate a validation rule set on a usage point purpose

You can activate or deactivate an estimation rule set on the purpose of a specific usage point.

5

Steps

1.   Open a usage point that is linked to a metrology configuration with configured estimation. You can use the “Search” feature for this.

2.   Select a purpose under Purposes in the navigation bar on the left.

3.   Click the Estimation configuration tab on top of the purpose page.

4.   Select the needed estimation rule set.

5.   Click the action icon  at the bottom of the selected line and select Deactivate.

If the estimation rule was currently inactive, then the option will read Deactivate. So, you can easily toggle the status of an estimation rule set on the usage point purpose.
