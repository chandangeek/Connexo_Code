Manage linked metrology configuration purposes on an estimation rule set

You can add a metrology configuration purpose to an estimation rule set to make sure that the usage point data will be estimated using the estimation rules of the selected estimation rule set.

View linked metrology configuration purposes on an estimation rule set

On the validation rule set page you can see the metrology configuration purposes that are linked to it.

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets.

2.   Open the needed estimation rule set by clicking its name.

3.   Click Metrology configuration purposes in the vertical navigation bar on the left.

You can see all linked metrology configuration purposes with their status in a paged table. If you have the right privileges, you can click the metrology configuration’s name to see its details.

The details (outputs) of the selected metrology configuration purpose are shown in a details pane under the table. An icon next to each output indicates if it has reading types that match estimation rules in the active version of the selected rule set.

Add one or more metrology configuration purposes to an estimation rule set

You can add multiple metrology configuration purposes to the same estimation rule set simultaneously to work faster.

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets.

2.   Open the needed estimation rule set by clicking its name.

3.   Click Metrology configuration purposes in the left navigation bar on the left.

4.   Click the button Add metrology configuration purposes.

5.   In the table, select the metrology configuration purposes that you want to link.

If needed, you can click the metrology configuration purpose’s name to see its details.

6.   Click Add.

 

1

Important

Only metrology configuration purposes with at least one matching reading type are shown.

Remove a metrology configuration purpose from an estimation rule set

You can remove a metrology configuration purpose from an estimation rule set if it isn’t relevant any longer or to correct a mistake.

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets.

2.   Open the needed estimation rule set by clicking its name.

3.   Click Metrology configuration purposes in the left navigation bar on the left.

4.   Click the remove icon .

5.   Click Remove to confirm the removal.

 

1

Important

Removing estimation rule sets is only possible on active and inactive metrology configurations.
