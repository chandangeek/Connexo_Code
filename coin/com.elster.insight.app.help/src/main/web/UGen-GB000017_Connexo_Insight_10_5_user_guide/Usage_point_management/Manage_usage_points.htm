Understand usage points

In the CIM model, usage points represent the point of delivery of a single commodity to a customer. On the usage point, it is possible to view the customerâ€™s usage of the commodity over time regardless of the specific meter or device that is installed at any point in time.

Each usage point gets data from the devices that are associated with it and it is associated with a metrology configuration that determines how this data is used.
