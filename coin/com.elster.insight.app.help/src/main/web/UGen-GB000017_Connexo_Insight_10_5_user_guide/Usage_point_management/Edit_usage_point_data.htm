Edit usage point data

You can edit different groups of data of a usage point to correct mistakes or to align it with a changed situation. You can either make changes to multiple usage points at once or to one single usage point at a time.

Edit usage points with a csv file

You can edit multiple usage points by importing a csv file with existing usage points containing adjusted attributes. This is done with the Connexo import framework in the Connexo Admin application. For detailed information on import services, refer to the Connexo Admin user guide. Remember that you need specific privileges to access the Connexo Admin application and to edit usage points in bulk.

See also “Import usage points with a csv file”.

Edit usage point attributes

Edit the attributes of a single usage point to keep the usage point’s information up to date. You must have sufficient privileges to edit the different (custom) attribute sets.

5

Steps

1.   Open a usage point using the search page.

2.   Click More attributes in Usage point summary or click Usage point attributes in the vertical navigation bar to the left to open the Usage point attributes page.

3.   There are two procedures to edit the usage point’s attributes.

a. Click the Actions button and select the section you want to edit. First two items are fixed. They are General information and Technical information. The other items represent the custom attribute sets in the second column of the page. They are sorted alphabetically by name.

b. Click the pencil icon .

The widget switches to edit mode. You can edit only one group of attributes at the same time.

In General information you can’t edit MRID, Service category, Created and Type of usage point.

4.   Edit the applicable fields.

5.   Click Save to confirm your changes.

The panel switches back to view mode and the page reloads to show the updated information. In case of errors, the panel remains in edit mode and you are asked to make the necessary corrections before saving again.

Click Cancel to discard your changes.

Link a metrology configuration to a usage point

Link an existing metrology configuration to a usage point to define how data is calculated, including validation and estimation and also how many meters must be activated. A metrology configuration can only be linked to a usage point that is in a pre-operational stage in its life cycle.

It is possible to link a metrology configuration during creation of the usage point. See “Add a single usage point”. Afterwards, you can unlink and link another metrology configuration to the usage point.

5

Steps

1.   Open a usage point without linked metrology configuration using the search page.

2.   Click the link Link metrology configuration in the Metrology configuration widget on the usage point overview page or click the button Link metrology configuration on the Metrology configuration page.

3.   In the first step of the Link metrology configuration wizard, select a metrology configuration from the drop-down.

The drop-down contains only the active metrology configurations that have no custom attribute sets or custom attribute sets for which you have view or edit privileges. In addition to these constraints, the metrology configurations in the drop-down list must be compatible with the completed attributes for the current usage point.

If there are no metrology configurations available to you, then you can’t proceed with this procedure.

4.   In Start date, specify an activation date and time for the metrology configuration.

If gaps are allowed:

  • Metrology configuration activation date is later than or equal to the usage point creation date.

  • In case of linking to a usage point with meters, the metrology configuration activation date must be later than or equal to the meter activation date in the meter data collection system.

If gaps aren’t allowed, then the metrology configuration will only be linked to a usage point with meters:

  • Metrology configuration activation date is later than or equal to the usage point creation date.

  • Metrology configuration activation date is later than or equal to the meter activation date in the meter data collection system.

  • Meters must be specified for all the meter roles that are needed for the calculation of the active purposes of the current metrology configuration.

  • All meters must be in operational stage starting from the metrology configuration activate date until an infinite date in the future.

5.   in Purposes, you see the purposes that are available for the metrology configuration appear on the page. The required ones are selected and can’t be edited. The optional ones can be selected if needed.

6.   Click Add.

If the selected metrology configuration has custom attribute sets then the corresponding additional steps appear in the wizard menu on the left. There is one extra step per custom attribute set. They are sorted alphabetically by name.

The custom attributes that are marked as required are required in the custom attribute set or they are required for calculation by required purposes of the metrology configuration.

7.   Click Next to go to the next custom attribute set to complete its attributes.

8.   Click Add after completing the last step of the wizard.

The selected metrology configuration is now associated with the current usage point.

Unlink a linked metrology configuration and link another one

When the usage point is in a pre-operational stage of its life cycle, it is possible to unlink the linked metrology configuration from the usage point.

You can then link another metrology configuration. You will only be able to choose from metrology configurations that have the same meter roles as the meter roles that have been specified on the usage point. So, if a meter is linked to the production meter role on a usage point, then the list of available metrology configurations will contain only metrology configurations with the production meter role. The match must be exact. For instance, if you have a usage point with two meters and two meter roles (production and consumption) and you want to add a metrology configuration with only the consumption meter role, then you will need to remove the production meters first.

5

Steps

1.   Open a usage point with a linked metrology configuration.

2.   Click Metrology configuration in the vertical navigation bar on the left.

3.   Click the button Unlink metrology configuration.

You are now ready to link another metrology configuration. See “Link a metrology configuration to a usage point”.

1

Important

Be aware that data won’t be available on the current usage point until another metrology configuration is linked.

View history of linked metrology configurations

You can see current and historical metrology configurations of a usage point to understand what reading types have been collected and will continue to be collected.

5

Steps

1.   Open a usage point landing page.

2.   Click History in the vertical navigation bar on the left.

3.   Click the tab Metrology configurations.

The table shows all metrology configurations that have been linked to the current usage point. The currently linked meter is indicated in the column Current. In Period you can see during what period the metrology configuration was linked. You can click the metrology configuration name to see its landing page.

In the details pane, you can click the process ID’s - if there are any - to see their details on the running processes page. The details pane also shows the active purposes of the current metrology configuration and their reading types. Click the info icon next to each item to see more details about the reading type.

Link meters on a usage point with a defined metrology configuration

Add meters to a usage point with a defined metrology configuration so that the usage point can be activated and enter the operational stage of its life cycle. In this case, the usage point must be in the pre-operational stage of its life cycle and its linked metrology configuration can have a date in the past or in the future.

5

Steps

1.   Open a usage point with a defined metrology configuration but without set meters.

2.   Click the link Link meters in the Metrology configuration widget on the usage point overview page or click the button Link meters on the Metrology configuration page of the current usage point.

For each of the meter roles that are defined on the metrology configuration of the usage point you can select a meter.

If meters have been set earlier for this usage point’s metrology configuration, then you can make modifications.

3.   Start typing the meter name in Meter next to the corresponding meter role.

The list is populated as you are typing. You can even type the middle part of an meter name. A maximum of 50 items is shown in the list. Keep typing to narrow down your filter if you don’t see the needed meter yet. Wildcards are not needed.

Meters with the Decommissioned or Deleted status aren’t included in the list.

4.   Select the needed meter.

5.   Specify the needed activation date and time in Activation date.

The meter data will be available on the selected usage point starting from the specified date and time.

By default, the activation date equals the date when the metrology configuration has been linked to the usage point. When a meter is selected, the activation date is updated to the meter’s activation date in the meter data collection system. You can specify another activation date taking into account the following restrictions:

If gaps are allowed (See “View the metrology configuration landing page”.),

  • Meter activation date on usage point must be later than or equal to the creation date of the usage point.

  • The selected meters must be in the operational stage starting from the activation of the metrology configuration.

If gaps aren’t allowed,

  • Meter activation date on usage point must be later than or equal to the creation date of the usage point.

  • The selected meters must be in the operational stage starting from the activation of the metrology configuration until an infinite date in the future.

  • Meters must be specified for all of the meter roles that are needed for the calculation of the active purposes of the selected metrology configuration.

6.   Click Save.

The linked meters are now shown in the Meter roles table and the list of meters will remain visible even if there isn’t any metrology configuration linked to the usage point anymore.

If the selected meter is already linked to another usage point, then you need to confirm this before the meters can be saved. The confirmation message indicates with what usage point(s) and in what role(s) the meter is already linked.

1

Important

  • The same meter can’t be set for different meter roles of the same metrology configuration.

  • Reading types provided by the meter must match the reading type patterns of the inputs of the meter role.

  • Additional custom validation checks may be in place. Custom validation checks are implemented by your Connexo delivery team.

Link meters on a usage point without a defined metrology configuration

Add meters to a usage point without a defined metrology configuration so that the usage point can be prepared to enter the operational stage of its life cycle. In this case, the usage point can be in the pre-operational or operational stage of its life cycle.

5

Steps

1.   Open a usage point without a linked metrology configuration.

2.   Click the button Link meters on the Metrology configuration page of the current usage point.

If meters have been set earlier for this usage point’s metrology configuration, then you can make modifications.

3.   Start typing the meter role name in Meter role.

You can select any meter role that is available in the system.

The list is populated as you are typing. You can even type the middle part of an meter name. A maximum of 50 items is shown in the list. Keep typing to narrow down your filter if you don’t see the needed meter yet. Wildcards are not needed.

4.   Start typing the meter name in Meter.

The list is populated as you are typing.

Meters with the Decommissioned or Deleted status aren’t included in the list.

5.   Select the needed meter.

Specify a meter for each meter role.

6.   Specify an activation date and time in Activation date.

This simply indicates the date of linking the meter to the usage point.

The meter activation date must be later than or equal to the creation date of the usage point.

7.   If needed, click the plus icon in the column Actions to add a new row and add an extra meter role, meter and activation date.

8.   Click Save.

The linked meters are now shown in the Meter roles table.

If the selected meter is already linked to another usage point, then you need to confirm this before the meters can be saved. The confirmation message indicates with what usage point(s) and in what role(s) the meter is already linked.

1

Important

  • The same meter can’t be set for different meter roles of the same metrology configuration.

  • Reading types provided by the meter must match the reading type patterns of the inputs of the meter role.

  • Additional custom validation checks may be in place. Custom validation checks are implemented by your Connexo delivery team.

  • When linking a metrology configuration, the list of available metrology configurations will be limited to those that have all of the roles that have already been configured on the usage point.

Unlink a meter on a usage point with a defined metrology configuration

You can unlink a meter from a usage point with defined metrology configuration to replace it with another meter - if needed.

5

Steps

1.   Open a usage point with a defined metrology configuration with linked meters.

2.   Click the button Link meters on the Metrology configuration page of the current usage point.

3.   Click the remove icon to unlink the meter.

4.   Click Save.

The meter is no longer shown in the table and another meter can be entered for the selected meter role.

View history of linked meters on a usage point

You can see what meters have been linked to the current usage point over time.

5

Steps

1.   Open a usage point landing page.

2.   Click History in the vertical navigation bar on the left.

3.   Click the tab Meters.

The table shows all meters that have been linked to the current usage point. The currently linked meter is indicated in the column Current. Add a calendar to a usage point manually

If you have sufficient privileges, you can add a time of use calendar to a usage point in order to use the calendar for calculating data on the usage point. The available calendar categories and calendars are determined by the metrology configuration.

5

Steps

1.   Open a usage point using Search.

2.   Click Calendars in the vertical navigation bar on the left.

3.   Click the button Add calendar.

4.   Select a category in Category.

The selected category determines what calendars are available from the combobox in Calendar. Categories that have no calendars aren’t shown in the list.

5.   Select a calendar in Calendar.

Only active calendars are shown.

If calendar events are used in the formula of the metrology configuration, then it is verified that the calendar events are matching. Calendars with non-matching event sets aren’t shown in the list.

Multiple calendars from different categories can be linked at the same time to the same usage point.

6.   Specify the calendar’s activation date in Activate calendar.

You can’t pick a date in the past. So, calculated results won’t change anymore.

7.   Click Add.

If the calendar has all events that are mentioned in the formulas of the metrology configuration, it is duly added to the list of linked calendars.

If a calendar of the same category was active and linked with the current usage point, then the ‘old’ calendar gets an end date which is identical to the activation date of the newly added calendar. The newly added calendar will become active on the specified date.

Add calendars to multiple usage points in bulk

If you have sufficient privileges, you can add a time of use calendar to multiple usage points at the same time to quickly set up a new batch of usage points.

5

Steps

1.   Open Search and specify the needed search criteria.

2.   Execute the search.

3.   Click the button Bulk actions.

4.   In step one, choose if you want to do your bulk action on all of the usage points that are included in your search results or on a limited list of usage points that are selected from that list.

5.   Click Next.

6.   In step two, select the action that you want to make. Select Add calendar.

7.   Select a category in Category.

The selected category determines what calendars are available from the combobox in Calendar. Categories that have no calendars aren’t shown in the list.

8.   Select a calendar in Calendar.

Only active calendars are shown.

If calendar events are used in the formula of the metrology configuration, then it is verified that the calendar events are matching. Calendars with non-matching event sets aren’t shown in the list.

Multiple calendars with different categories can be linked at the same time to the same usage point.

9.   Specify the calendar’s activation date in Activate calendar.

You can’t pick a date in the past. So, calculated results won’t change anymore.

10.   Click Next.

11.   In step four, click Confirm to execute the action. Click Cancel to make no changes to the selected usage points.

12.   In step five, the result of the executed action is shown.

13.   Click Finish to finish the bulk action wizard.
