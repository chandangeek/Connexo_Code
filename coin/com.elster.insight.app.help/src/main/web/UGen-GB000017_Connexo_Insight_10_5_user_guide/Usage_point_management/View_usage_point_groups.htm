View usage point groups

You can view all the usage point groups that are available in Connexo Insight and consult detailed information.

5

Steps

1.   Go to Usage points > Usage point groups.

The available usage point groups are shown in a paged table sorted by Name. In the details pane under the table you see further details of the selected usage point group.

2.   Click a value in Name to open the selected usage point group details page.

On the details page you see the usage point groupâ€™s name and type and the included usage points are shown in a paged table sorted by name.

3.   Click the value in Name to open the corresponding usage point landing page.

4.   Click the button Count above the table to get an exact count of the usage points that are included in the usage point group.

5.   Click the button Columns to select what columns you want to see in the table.
