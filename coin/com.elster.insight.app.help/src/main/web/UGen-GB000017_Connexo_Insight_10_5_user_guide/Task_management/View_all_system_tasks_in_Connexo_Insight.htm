View all system tasks in Connexo Insight

You can make an overview of system tasks in Connexo Insight that are relevant to you at a particular point in time.

5

Steps

1.   Go to Administration > Task management > Tasks.

2.   In the filter bar on top, enter the necessary criteria to narrow down the shown system tasks to what is relevant for your needs. You can select the queue that holds the task or the time when the task was started. Donâ€™t forget to click Apply to show the filtered data.

The table shows all system tasks that are currently available in Connexo Insight. You can see the name of the queue they are on and when their next execution is scheduled.

3.   Click the name of a task to see its details.

The details page will depend on the type of task that is selected.
