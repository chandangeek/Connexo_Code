Run a task from the consolidated overview

You can queue a system task for immediate execution if this is available for the selected task type. For instance, an estimation task can be run right from the overview page.

3

Note

You can’t run export tasks on this page. For this type of tasks you still need to go to the export task page. See “Run an export task manually”

5

Steps

1.   Go to Administration > Task management > Tasks.

2.   Select Run from the actions icon in the line of needed task.

3.   Click Run to put the selected task on the queue.
