Add a system task from the consolidated overview

In the consolidated overview of Connexo Insight system tasks you can add system tasks.

3

Note

You can’t add export tasks on this page. For this type of tasks you still need to go to the export task page. See “Add an export task”.

5

Steps

1.   Go to Administration > Task management > Tasks.

2.   Click the button Add task.

3.   Select the type of task that you want to add.

Depending on the entered task type, the remaining fields are added to the screen.

4.   Complete all needed fields.

5.   If needed, select a follow up task that must be triggered after the selected task has finished. For this purpose, use the field Followed by.

6.   Click Add.

The new system task is now shown in the overview table. You can see the added task also on the corresponding overview page for that particular task.
