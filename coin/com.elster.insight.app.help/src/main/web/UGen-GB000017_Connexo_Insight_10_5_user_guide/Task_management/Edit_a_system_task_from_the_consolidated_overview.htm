Edit a system task from the consolidated overview

In the consolidated overview of Connexo Insight system tasks you can edit system tasks to correct eventual mistakes.

3

Note

You can’t edit export tasks on this page. For this type of tasks you still need to go to the export task page. See “Edit export tasks”.

5

Steps

1.   Go to Administration > Task management > Tasks.

2.   Select Edit from the actions icon in the line of needed task.

3.   The edit page shows the fields that are relevant for the corresponding task type.

4.   Click Save.
