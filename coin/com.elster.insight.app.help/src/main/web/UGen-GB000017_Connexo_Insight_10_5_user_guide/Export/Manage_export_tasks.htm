Manage export tasks

Edit export tasks

You can edit existing export tasks to keep them up to date.

5

Steps

1.   Go to Administration > Data exchange > Export tasks.

2.   Select the needed export task in the overview table.

3.   Select Edit from the actions icon at the bottom of the selected row.

Alternatively, select Edit from the Actions button on the details pane or detail page.

4.   If you have sufficient privileges, you can edit:

  • Name.

  • Schedule.

  • Data selection, except for the data selector.

  • Destinations.

5.   Click Save to keep the changed export task.

Remove export tasks

You can remove obsolete export tasks to keep the data export configuration up to date.

1.   Go to Administration > Data exchange > Export tasks.

2.   Select the needed export task in the overview table.

3.   Select Remove from the actions icon at the bottom of the selected row.

Alternatively, select Remove from the Actions button on the details pane or detail page.

4.   Click Remove to confirm the removal of the selected export task.

1

Important

The removal is irreversible.
