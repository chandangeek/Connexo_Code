Tables

On many screens, data is presented in a table format. Depending on the amount of involved data the table will be paged or have infinite scrolling. On top of each table you can see the number of shown records and an indication of the total number of available records. If the table is paged, then there is a navigation bar under the table where you can specify the number of records shown per page. On the right, you have several options to scroll through the different pages of the table. Under the table there is often a details pane that shows more detailed information about the record that is currently selected in the table. You can easily select items in the tables by clicking them with the mouse or by simply scrolling through the list using the up and down cursor keys. Finally, you can export tables to a csv file by clicking the export button on top of the table. Note that in case of a paged table, only the data in the current page will be exported.
