Understand import services

Import services are configured in Connexo Admin. Refer to the Connexo Admin user guide for detailed information on managing import services in Connexo Admin. In Connexo Insight you can view the available import services and monitor their progress and status.

When creating a new import service in Connexo Admin a predefined file importer will be selected that is specific for the type of data that needs to be imported. Different file locations will be specified for the files that need to be picked up for import, the files that are being imported, the files for which import failed and for the files for which the import was successful. It is possible to define a frequency to organise regular imports and a specific file pattern to filter the files that will be imported based on file name and/or extension. All of this information is visible in Connexo Insight when viewing the import services.
