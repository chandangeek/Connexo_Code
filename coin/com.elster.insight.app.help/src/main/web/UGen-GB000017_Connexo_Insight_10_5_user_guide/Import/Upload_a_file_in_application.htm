Upload a file in application

In order to use the Connexo import framework, you need to have access to the import folder that is specified on the application server. This procedure is explained in the Admin user guide. In some cases it may be useful for users in Connexo Insight to use the import tool - even if they don’t have access to this import folder location. Importing data into Connexo using a file may be handy to do some basic testing.

5

Steps

1.   Go to Administration > Data exchange > Upload file for import.

2.   In Import service select the needed import service.

Only the import services where the field Allow uploads in application has been selected appear in this dropdown. This is done in Connexo Admin. Refer to the Connexo Admin user guide for more detailed information.

After you select the import service its details are shown for your information.

See “View details of an import service”.

3.   Check the shown import service parameters and verify that your import file complies to the specifications.

4.   In File click Select file... to select your import file.

The maximum size for an import file is 100 MB.

5.   Click Upload.

The selected file will be picked up by the selected importer according to the folder scan frequency that is defined on the import service.

6.   Follow up progress of the import in the history section of the concerned import service.

Important

To make the import service work it must have the active status and it must be added to an active application server.
