Manage validation rule sets

The validation rule set is linked to a purpose of a particular metrology configuration. It contains active and inactive versions. Inactive versions are validation rule sets that are not yet or no longer valid. The validation rules in each validation rule set’s version are mapped to the reading types matching the inputs (collected data) and outputs (computed data) of the metrology configuration. Depending on the reading types in the metrology configuration the relevant rules can be selected and their options restricted. So, it isn’t possible to add validation rules to a metrology configuration that has no data for it to validate.

The criteria for adding validation rule sets to usage points via the metrology configurations can be different based on market, country or other elements.

View validation rule sets

Get a complete overview of all validation rule sets that are available in Connexo Insight and drill down to the needed detail level.

5

Steps

1.   Go to Administration > Data validation > Validation rule sets.

The complete list of validation rule sets with their status is shown in the first table.

  • Details of each validation rule set can be viewed by clicking the link in its name. On the Details page you can see the active version for the selected validation rule set.

  • On the Versions page you can see the versions for the selected validation rule set as well as the related validation rules. From this page, you can add additional versions and validation rules.

  • On the Metrology configuration purposes page you can see the metrology configuration that this validation rule set is linked to. From this page it is possible to link validation rule sets to multiple metrology configurations simultaneously.

The second table shows the versions of the selected validation rule set and the number of active and inactive rules that are associated with them. See “Add a version to a validation rule set”.

  • Details of each version can be viewed by clicking the link in the version name. In the left column of the version details page you can click Validation rules to see the active and inactive validation rules that are associated with this validation rule set’s version. For each validation rule details are shown.

The third table shows the validation rules that are associated with the selected version of the selected validation rule set. For each rule, its status is indicated. At the bottom of the page is a pane showing the details of the selected validation rule. See “Add a validation rule to a validation rule set version and activate it”

  • Click the validation rule’s name to see its details on a separate page.

Add a validation rule set

Validation rule sets must first be created before they can be associated with a metrology configuration in a separate step.

5

Steps

1.   Go to Administration > Data validation > Validation rule sets.

2.   Click the button Add validation rule set.

3.   On the Add validation rule set page specify a Name and a Description for future reference.

4.   Click Add.

The new validation rule set is listed in the first table with validation rule sets.

After adding a validation rule set, the next task is to add a validation rule set version (See “Add a version to a validation rule set”) and validation rules (See “Add a validation rule to a validation rule set version and activate it”).

Edit or remove a validation rule set

It is possible to edit or remove a validation rule set to correct mistakes.

5

Steps

1.   Go to Administration > Data validation > Validation rule sets.

2.   Click the actions icon  next to a validation rule set in the table or click the actions button on its detail page.

3.   Click Edit or Remove.

4.   Make the required changes and click Save or confirm the removal message.

3

Note

It isn’t possible to remove a validation rule set that is still associated with one or more metrology configurations.
