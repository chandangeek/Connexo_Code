Start a process on a usage point

You can start processes on a usage point to do complex tasks on a usage point that may include input from several users and/or systems. The processes that are available in Connexo Insight must have been configured and activated in Connexo Admin first. Refer to the Connexo Admin user guide for more detailed information.

5

Steps

1.   Open a usage point using the search page.

2.   Select Start process from Actions button on the usage point overview page.

3.   Select the needed process in Process on the Start process page.

Depending on the selected process a set of parameters appear on the page.

4.   Complete the process parameters.

5.   Click Start.

The Processes page is opened showing the running processes in the tab Running processes. The just started process appears on this list.

As soon as the process is complete, it will disappear from the tab Running processes and it will be listed under the History tab.

See “View running processes on a usage point”.

1

Important

Permission to start a process is restricted by privileges that are set in Connexo Admin. Refer to the Connexo Admin user guide for detailed information on processes.

There are a number of processes available out-of-the-box in Connexo Insight. Custom processes can be added if necessary. Your Connexo delivery team can develop these or - if you have a Connexo Flow Designer license and privileges - then you can draw the processes yourself. Your Connexo delivery team will assist you in implementing the processes that you created.
