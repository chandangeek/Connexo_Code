Search Connexo

Understand the search feature

In Connexo it is possible to search for different types of objects using an extensive set of parameters. The search feature can be accessed by clicking Search in the upper right corner of the page. The search function is available on every page in the application.

A powerful set of search criteria is found on top of the search page. The number of selected values per criterion is indicated by a number on the button that represents the criterion. Criteria can be edited or removed by removing the specified or selected values inside it. To do this, you simply click the criterion button to expand it and make its values visible.

The default search criteria depend on the type of objects that you are searching. On top of the default search criteria you can add more criteria that are relevant to the selected object type.

When all criteria have been defined you can apply them and see the search results in a table under the criteria. Click the button Columns to change the shown columns. From the search results you can navigate to a specific item by clicking the link in its row. Mostly, the name of the item is hyperlinked. In addition, you can do actions on one or on multiple items at once using the action icon  at the end of each row or the Bulk action button on top of the table. The available actions depend on the selected object type. Finally, you can calculate the total number of found items with the button Count.
