Advanced search

The advanced search feature allows you to make very powerful searches using complex combinations of search criteria. Depending on the object that you are searching for a different set of criteria and actions is available and the search results are presented differently in the table. However, the search concept remains the same.

Understand the common functionality

5

Steps

1.   Click Search in the upper right corner of any page in Connexo.

2.   In Search for, select the type of object you are searching for.

3.   In Criteria, specify the desired default search criteria. Click the criterion button to expand it and enter your search criteria.

4.   Click Add criteria to add extra criteria for better fine-tuning your search.

5.   Remove search criteria or change their impact by deleting the values that define them. For added criteria you can click the remove icon  next to the criterion button to completely remove it. The default criteria buttons canâ€™t be removed and you can only delete their values.

6.   Click Search to show the search results in a table.

7.   To change the default columns in the search result table,

a. Click the Columns button above the search results table.

b. Select the columns you want to see.

c. Uncheck the columns you want to hide.

d. Click Done to apply your changes.

8.   Click the Count button above the search results table to get an exact count of the items in the search result. The number of found items is shown on the button.

The calculation of found items may take some time depending on the size of the data set. A time-out is possible in case of very large data sets.

9.   Click Clear all to remove all criteria and reset the search page.

Important 

Be aware that some search criteria depend on others. They become available after you have specified one or more other criteria. The required criteria are listed in a tooltip on the master criterion.

Note

The following wildcards are accepted when specifying your search criteria:

Wildcard

Function

*

Represents none, one or more characters.

?

Represents exactly one character.

If no search criteria were specified, then no search results are returned. If - for any reason - you would like to show all items, type a star (*) in one of the free text search criteria to see all instances of the searched object type.

Enter search criteria

 When you expand a criterion button on the search bar the functionality to enter your search criteria depends on the type of data you are searching for.

Search for free text

If you are searching on fields that are free text values or if the list of possible values is simply too long to be presented in a list for you to select from, then you can enter a search value in a text box and select an available operator to adjust your filter. Wildcards are allowed in the text box.

Figure: Search for free text.

 

Search for multiple values from a limited list

If you are searching on fields that can have only a limited number of values, then you can simply select the needed values from a list. To narrow down the list of available values, start typing in the text box on top. You have extra options to customise the list of values that are shown in the list for selection. Select an available operator to adjust your filter. Wildcards are allowed in the text box.

Figure: Search for multiple values from a limited list.

 

Search for a date range

If you are searching on fields that contain date and time information, then you can type or select dates and times and select an available operator to specify the range.

Figure: Search for a date range.

 

Search on yes/no fields

If you are searching on fields that have a yes/no value, then you can only choose between two radio buttons, i.e. Yes and No. Yes is selected by default.

Figure: Search on yes/no fields.

 

Search for numbers

If you are searching for numbers, then you can type the values you are searching for and specify an available operator to specify different types of ranges.

Search for usage points

When searching for usage points a specific set of criteria and actions is available and the search results are presented with the most important columns for this object type.

5

Steps

1.   In Search for select Usage point.

The following primary criteria are available by default when searching for usage points.

Parameter

Description

Name

Type the name of the usage point or use wildcards.

Note: Click Add criteria and select MRID to search on the Master Resource Identifier. The MRID is the objectâ€™s unique identifier.

Service category

Select one or more service categories from the list or search for one by typing its name. The list starts building as you start typing. You can enter any part of the name to find the needed items.

Metrology configuration

Select one or more metrology configurations from the list or search for one by typing its name. The list starts building as you start typing. You can enter any part of the name to find the needed items.

Purpose

Select one or more purposes from the list or search for one by typing its name. The list starts building as you start typing. You can enter any part of the name to find the needed items.

State

Select one or more states from the available usage point life cycles or search for one by typing its name. The list starts building as you start typing. You can enter any part of the name to find the needed items.

Connection state

Select one or more connection states from the list or search for one by typing its name. The list starts building as you start typing. You can enter any part of the name to find the needed items.

Location

Select one or more locations from the list or search for a location by selecting Advanced search and completing the known information in the fields that appear.

Usage point group

Select one or more usage point groups from the list or search for one by typing its name. The list starts building a you start typing. You can enter any part of the name to find the needed items.

2.   Click Add criteria to see the other criteria that you can add when searching for usage points.

3.   Click the name of the item to open its overview page in Connexo Insight.
