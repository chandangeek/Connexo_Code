Manage validation rule sets on a metrology configuration

Validation rule sets are associated with a specific purpose of a metrology configuration. Because a usage point is also linked with a metrology configuration, it becomes clear what validation rules apply to the data of each usage point.

View associated validation rule sets on a metrology configuration

For each metrology configuration an orderly overview of its validation configuration is available on one page.

5

Steps

1.   Go to Administration > Metrology configurations.

2.   Click Validation configuration in the left navigation menu.

3.   In the first table you can see the list of validation rule sets that are associated with the selected metrology configuration. They are grouped by the metrology configuration purpose. For each validation rule set, the start date of its current version is shown.

4.   The purpose groups in the Rule sets table can be collapsed by clicking the minus button (-).

5.   You can click the validation rule set name to see its details on a separate page.

Under the validation rule set table the active and inactive validation rules that are associated with the active version of the selected validation rule set are shown.

Under the validation rules table, the details of the selected validation rule are shown in a details pane.

Add a validation rule set to a metrology configuration

A validation rule set can be added to a metrology configuration for one of its purposes. The validation rule set must be created and configured beforehand in Administration > Data validation > Validation rule sets. See “Add a validation rule set”. Only validation rule sets with reading types that match those of the selected metrology configuration purpose are shown. A validation rule set can be added only once to the same metrology configuration purpose.

5

Steps

1.   Go to Administration > Metrology configurations.

2.   Click Validation configuration in the left navigation menu.

3.   Click Add validation rule set.

4.   In Purpose, select one of the purposes of the current metrology configuration.

5.   Select one or more of the available validation rule sets in the table by selecting the checkbox to the left of each line. If there is no valid validation rule set available, click Add validation rule set to add one as described in “Add a validation rule set”.

For your information the selected validation rule set’s validation rules are shown in the table at the bottom. Details of the selected validation rule are shown in the details pane under the table.

6.   Click Add.

The selected validation rule set(s) are now added to the validation rule sets table. If the validation rule set was added for another purpose, then you will notice that an additional group has been added to the table.

1

Important

Data validation is always active on the metrology configuration. It isn’t possible to deactivate it.

Remove a validation rule set from a metrology configuration

A validation rule set can be removed from a metrology configuration to correct mistakes. The rule set will remain available to add it to other metrology configurations or to the same one at a later time.

5

Steps

1.   Go to Administration > Metrology configurations.

2.   Click Validation configuration in the left navigation menu.

3.   Select the needed validation rule set.

4.   Click the cross in the column Actions.

5.   Confirm the removal.

 

1

Important

Removing validation rule sets is only possible on active and inactive metrology configurations.
