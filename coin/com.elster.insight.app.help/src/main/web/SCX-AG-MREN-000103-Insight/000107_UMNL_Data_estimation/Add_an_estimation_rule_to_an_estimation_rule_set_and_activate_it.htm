Add an estimation rule to an estimation rule set and activate it

Estimation rules must be added to an estimation rule set. Don’t forget to activate the added estimation rules. In case you don’t want to use certain estimation rules anymore, you can deactivate them.

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets.

2.   Select the needed estimation rule set and open its detail page.

3.   Click Estimation rules in the vertical navigation menu on the left.

4.   Click Add estimation rule to open the Add estimation rule page.

5.   Complete the following fields:

Field

Description

Estimation rule

Type a meaningful name for the estimation rule.

Estimator

Th estimator defines how estimation will be done. Depending on the selected estimator, additional parameters may appear on screen. See “Add reading types”.

Reading types

Select the reading types to define the type of data that will be estimated. See “Add reading types”.

Estimation comment

Add a predefined estimation comment as a reference to better understand the rule’s content later on.

6.   Click Add.

7.   On the Estimation rules page, click the action icon  and select Activate.

8.   Verify the status of the estimation rule in the column Status.
