Add usage points

Add a single usage point

Add a usage point to link to one or more devices and get meter data to work with.

5

Steps

1.   Click Usage points in the quick access bar on the left side on any page.

2.   In Usage point life cycle management, click Add usage point.

You must have sufficient privileges to see this link.

The usage point information is entered in several steps that are specified by a wizard. Initially, only the first four steps are shown: General information, Technical information, Link metrology configuration and Life cycle transition. The next steps contain custom attribute sets and these depend on the service category that is selected in the first step. Only the service categories for which you have sufficient privileges are available for you to choose from. After you select the service category the remaining steps are shown and the technical information form is filled with the appropriate fields.

To enter a location in General information you can just start typing an address. Connexo will suggest a value. You can also open the combobox and select one of the addresses. If the address is not in the list, select Input location to show the address fields and the address details yourself.

When selecting the fields Limiter or Bypass in the step Technical information, additional fields are added to the form for completion.

For more information with regard to the step Link metrology configuration, see “Link a metrology configuration to a usage point”. It is possible to skip this step.

For more information with regard to the step Life cycle transition, see “Manage usage point life cycles”. It is possible to skip this step.

3.   After you have gone through all of the steps and completed the required information, click Add.

Import usage points with a csv file

You can import multiple usage points using a csv file and the Connexo import framework. This is done in the Connexo Admin application. For detailed information on import services, refer to the Connexo Admin user guide. Remember that you need specific privileges to access the Connexo Admin application and to import usage points.

In Connexo Admin the following steps will need to be executed.

5

Steps

1.   Click the hamburger icon in the upper left corner of any Connexo screen and select Admin to open the Connexo Admin app.

2.   In Administration, click Import services and make sure that an import service of the type Usage Point Importer exists. Verify the other information such as folder scan frequency and the folders for input file, output file, etc. Make sure that the configured folders exist and that they are accessible.

3.   Make sure that the import service is activated. If not, select Activate from the action icon at the end of the corresponding row.

4.   Make sure that the import service is added to an active application server. Go to Administration > Application servers and open an active application server. Click Import services on the left vertical navigation bar. If the import service is not in the list, then add it by clicking Add import services.

5.   Add the import file to the configured import folder and wait for the application server to pick it up.

6.   Monitor the import progress by checking the In progress, Success and Failure folders. In Import history you can review the results of the finished import and see where errors have occurred.

7.   Verify the added usage points using the search page.

 

Characteristics of the usage point import file

The usage point import file must be a csv file with specific column headers. The order of the columns is not fixed. A limited number of columns is required by the importer. If these columns don’t contain any value, the usage point importer gives warnings and the concerned usage points won’t be imported.

If values in columns with the boolean data type are left empty, then they are imported as ‘false’.

Custom attribute sets are supported by the importer. Usage points can be imported without custom attribute sets as long as there are no required fields in those custom attribute sets. It is even possible to import usage points with time-sliced custom attribute sets.

You can add multiple import files.

View import services in Connexo Insight

An overview of the available import services along with their status and history information can also be seen in Connexo Insight . This feature is for viewing only. In this environment, you can’t edit any information. Import services in Connexo Insight  is especially meant for troubleshooting and monitoring.

For more detailed information about how to view import services in Connexo Insight , see “Import”.
