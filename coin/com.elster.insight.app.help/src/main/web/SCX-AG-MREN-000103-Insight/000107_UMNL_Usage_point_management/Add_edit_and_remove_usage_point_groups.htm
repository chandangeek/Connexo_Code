Add, edit and remove usage point groups

Add a dynamic usage point group

You can gather usage points in a usage point group yourself for use in bulk actions and to avoid repetitive actions. In the case of dynamic usage point groups, only the used search criteria are saved. Consequently, the contents of this group may be different each time you open it. Some usage points may no longer fit the saved criteria of the usage point group and others may have been added or adjusted matching them.

5

Steps

1.   Go to Usage points > Usage point groups to see the overview of the available usage point groups.

2.   Click the button Add usage point group.

Adding a usage point group happens with a wizard in four steps:

  • General attributes.

  • Select usage points.

  • Confirmation.

  • Status.

 

Step one

3.   Type a name for the usage point group. This name must be unique.

4.   Specify the type of usage point group that you want to add. By default, Dynamic usage point group is selected.

5.   Click Next to go to step two, i.e. Select usage points.

In step two, you are selecting the usage points that need to be included in your new usage point group.

Step two

6.   In Criteria, specify the needed criteria to find the usage points for your usage point group.

Adding criteria works the same way as in the search functionality. See “Search”.

7.   Click the button Add criteria to include additional criteria. Click the button Clear all to remove all criteria and reset your entire filter.

8.   Click the button Preview to see in a paged table what usage points currently answer the specified criteria for your new usage point group.

9.   Click the button Columns to choose the columns for showing the usage points in the table.

10.   Click Next to go to step three, i.e. Confirmation.

Step three

The number of specified search criteria and included usage points are counted and you are asked to confirm the creation of the new usage point group.

11.   Click Confirm to create the usage point group and go to step four, i.e. Status.

Step four

You can see a confirmation that the new usage point group has been created.

12.   Click Finish to complete the Add usage point group wizard and open the overview of available usage point groups.

The new usage point group appears in the overview on the page Usage point groups.

Add a static usage point group

You can gather usage points in a usage point group yourself for use in bulk actions and to avoid repetitive actions. In case of static usage point groups, the list of usage points itself is saved. This list will be the same each time you open it.

5

Steps

1.   Go to Usage points > Usage point groups to see the overview of the available usage point groups.

2.   Click the button Add usage point group.

Adding a usage point group happens with a wizard in four steps:

  • General attributes.

  • Select usage points.

  • Confirmation.

  • Status.

 

Step one

3.   Type a name for the usage point group. This name must be unique.

4.   Specify the type of usage point group that you want to add. Select Static.

5.   Click Next to go to step two, i.e. Select usage points.

In step two, you are selecting the usage points that need to be included in your new usage point group.

Step two

6.   In Criteria, specify the needed criteria to find the usage points for your usage point group.

Adding criteria works the same way as in the search functionality. See “Search”.

7.   Click the button Add criteria to include additional criteria. Click the button Clear all to remove all criteria and reset your entire filter.

8.   Click Apply to decide which of the found usage points to include in your new usage point group.

9.   Select All usage points to include all found usage points or Selected usage points to show the found usage points and select only those that you want to retain.

If you choose to select usage points yourself, you can see the number of selected usage points on top of the table.

10.   Click Next to go to the next step, i.e. Confirmation.

Step three

The number of included usage points is counted and you are asked to confirm the creation of the new usage point group.

11.   If needed, you can click Back to go back to step two and adjust your search criteria. Click Apply to show the new list of usage points and select the needed usage points. Click Next to add them to your selection. In step four you can see that the number of usage points in your new usage point group has increased.

12.   Click Confirm to create the usage point group and go to step four, i.e. Status.

Step four

You can see a confirmation that the new usage point group has been created.

13.   Click Finish to complete the Add usage point group wizard and open the overview of available usage point groups.

The new usage point group appears in the overview on the page Usage point groups.

Edit a usage point group

You can edit a usage point group to correct eventual mistakes or to align it with new requirements.

5

Steps

1.   Go to Usage points > Usage point groups to open the overview of available usage point groups.

2.   Select the needed usage point group.

3.   Select Edit from the action icon at the bottom of the selected row.

Alternatively, you can select Edit from the action button in the details pane under the overview table or from the action button on the usage point details page.

1

Important

You can’t edit the usage point group type.

Remove a usage point group

You can remove a usage point group if it isn’t relevant any longer.

5

Steps

1.   Go to Usage points > Usage point groups to open the overview of available usage point groups.

2.   Select the needed usage point group.

3.   Select Remove from the action icon at the bottom of the selected row.

 

1

Important

  • The removal is irreversible.

  • You can’t remove a usage point group if it is still in use.
