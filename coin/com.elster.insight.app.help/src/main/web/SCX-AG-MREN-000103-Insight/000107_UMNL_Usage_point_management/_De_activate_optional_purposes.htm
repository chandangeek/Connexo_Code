(De)activate optional purposes

Under Data on the vertical navigation bar on the left of a usage point landing page you can see all the active purposes. Check the linked metrology configuration to see what other optional purposes are available and activate them if needed.

5

Steps

1.   Open a usage point using the search page.

2.   Click Metrology configuration in the left navigation bar.

In the table under the section Purposes you can see all the available purposes. The optional purposes are indicated by a No in the column Required. Only the optional purposes have an action icon  in the column Actions.

3.   Select the needed optional purpose.

The value in the column Active shows you the purpose’s activation status.

4.   Click the action icon  at the bottom of the row and select Activate.

If a purpose is activated, then a validation is done to check if the meter is providing all the inputs that are defined in the reading types of the purpose. If one of the outputs of the purpose doesn’t match the reading types of the meter, then the purpose can’t be activated. If the activation is successful, then the Active status changes into Yes and the purpose is listed in the section Data on the left navigation bar of the usage point’s landing page and also in the combobox Purpose in the section Data completion under Overview. The action Activate changes into Deactivate.

If needed, active, optional purposes can be deactivated in the same fashion.

3

Note

Required purposes can’t be deactivated.

There is no versioning applied to (de)activating purposes. So, you won’t be able to track (de)activations.
