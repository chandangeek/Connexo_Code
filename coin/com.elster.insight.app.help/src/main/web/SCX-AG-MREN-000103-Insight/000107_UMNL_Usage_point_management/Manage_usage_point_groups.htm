Understand usage point groups

Usage point groups collect usage points based on certain criteria. These groups can be either dynamic or static. In case of dynamic usage point groups, only the criteria are saved and the results can be different each time you open the usage point group. Static usage point groups contain the actual usage points. Therefore, the output of a static usage point group is always identical.

A distinguished set of actions can be done on a usage point group impacting all usage points in this group. This avoids repetitive work and allows for a more user-friendly handling of usage points.
