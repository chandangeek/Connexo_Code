Monitor export tasks

You can view all the export tasks that are available in Connexo Insight  all in one place. From here, you are able to actually manage these export tasks.

View list of export tasks

This list provides an overview of all the export tasks that are available in Connexo Insight . From this list you have access to the details pages of each export task to see more detailed information.

5

Steps

1.   Go to Administration > Data exchange > Export tasks.

The available export tasks are shown in a paged table. For each export task its Status and the date when it runs next time are shown.

2.   Select the needed export task to see more details in the details pane under the table.

The export task details are grouped in four sections:

Category

Description

Schedule

When is the export task scheduled to run?

When did it last run and what was the status of that run? If Status was failed, then the Reason why is shown.

When is the next scheduled run?

Data selection

Describes the exported data.

Output format

Describes the format of the exported data.

Destinations

Configuration data describing how and where the data is exported to.

View the export task details page

You can see the export details on a dedicated details page where you have access to further related information such as a history of occurrences of the selected export task, the related logging information and data sources.

5

Steps

1.   Go to Administration > Data exchange > Export tasks.

2.   In the table click the correct export task to open its details page.

On the details page the details of the selected export task are grouped in four sections as explained in “View list of export tasks”.

View the export task history

View the export task history to understand how the export task is performing over time.

5

Steps

1.   Go to Administration > Data exchange > Export tasks.

2.   In the table click the correct export task to open its details page.

3.   In the vertical navigation menu on the left, click History to open the Export task history page.

Alternatively, just select View history after clicking the action icon  in the row of the export task for which you want to see history information.

On this page you see the historical occurrences of the selected export task. They are arranged in a paged table with the most recent one on top. The overview table shows for each occurrence when it started, how long it took, its status and the export period.

4.   Use the filter parameters in the filter bar on top of the page to narrow down the list of historical occurrences of the selected export task.

Don’t forget to click Apply to execute the filter.

5.   Select on the occurrences in the table to see its details in the details pane under the table.

In the details pane the details of the selected export task are grouped in four sections as explained in “View list of export tasks”.

View logging information for the export task

For each occurrence of an export task you can view logging information to use for troubleshooting and tracing.

5

Steps

1.   Go to Administration > Data exchange > Export tasks.

2.   In the table click the correct export task to open its details page

3.   In the vertical navigation menu on the left, click History to open the History page.

4.   Click the export task occurrence for which you want to see the logging information to open the Log page.

On top of the page summary information is shown. This information depends on the applied data selector and data format.

The logging information is presented in a paged table. It is sorted after Timestamp with the most recent on top. The log levels can be Info, Warning or Severe. They are determined by the specified data selector and data format. If the data was successfully exported to its destination(s), then there is an Info log item for each of the destinations.

1

Important

If the log level is Severe, then the export task stops and fails.

View history of combined export services

For advanced troubleshooting you can view the history of combined export tasks.

5

Steps

1.   Go to Administration > Data exchange > Export history > Data exchange > Export history.

2.   Filter the occurrences by selecting one or more export services, a time window and one or more statuses on the filter bar.

3.   Don’t forget to click Apply to execute your filter.

The occurrences of the selected filters are shown in a paged table that behaves exactly the same as the history page of a single export task. Similarly, by clicking the name of the occurrence you can view detailed logging information for the selected occurrence.

See also “View the export task history” and “View logging information for the export task”.

3

Note

Depending on your privileges the export history may be available to you from the workspace.

View an export task’s data sources

You can see the data sources of an export task to understand what reading types have been exported and when the last interval was exported. The corresponding usage points with their connection status are equally shown.

Steps

1.   Go to Administration > Data exchange > Export tasks.

2.   In the table click the correct export task to open its details page

3.   In the vertical navigation menu on the left, click Data sources.

The data sources are shown in a paged table.
