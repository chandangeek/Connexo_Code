The quick access bar in Connexo Insight

In Connexo Insight  the different features are spread over multiple pages that can be easily accessed via the blue quick access bar that you find to the left of each screen in Connexo. This section provides a brief overview of the navigation options.

Dashboard

Figure: Dashboard icon

 

The dashboard provides a specific overview of Connexoâ€™s status using visual representations and lists including direct links to the most important items on your personal worklist.

Workspace

Figure: Workspace page icon

 

The workspace provides a view on issues, tasks and processes. It shows links to the (pre-filtered) work that needs to be picked up and what to give attention first. The workspace landing page allows you to look at the work as a general list of things to do or from the perspective of the utility services that the specific work items belong to. There are pre-filtered work lists and additional information to better understand the context. You can click any link to see the actual content. Depending on the privileges assigned to a specific user role, the view of the workspace may vary for different users.

Usage points

Figure: Usage points page icon

 

On the Usage points page you can manage all usage points. For example, you can add individual usage points. Use the search feature to get an overview of usage points.

Administration

Figure: Administration page icon

 

Via the Administration page all meter data management features can be configured. For example: Data validation, metrology configurations management, service categories.
