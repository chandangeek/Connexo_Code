Manage metrology configurations

The metrology configuration centralises a set of configurations for usage points that can be used as a configuration template for the usage points that are associated with it. It identifies the usage points and determines how it behaves within Connexo. The metrology configuration defines:

  • What devices are linked.

  • The purpose(s) of the usage point data.

  • What data can be expected as input and output

  • How this data should be validated and estimated

  • The applicable aggregation rules

  • Available custom attribute sets and other aspects that are relevant to the associated usage points.

The entire setup can be quite complex and specific. By organising the physical setup and its virtual restrictions in metrology configuration templates with which multiple usage points can be associated, the management of the usage point as an object in the Connexo data model is efficiently facilitated.

View the list of metrology configurations in Connexo

View the list of available metrology configurations all in one place. You can drill down to see more detailed information by opening one of the metrology configuration’s landing pages.

5

Steps

1.   Go to Administration > Metrology configurations.

The available metrology configurations are shown in a table sorted alphabetically by name. The table summarises each metrology configuration by showing its Name, Description and Status.

2.   Select one of the metrology configurations in the table to see its details in the details pane under the table.

Apart from Name, Description and Status, the details include also the Usage point requirements that determine the remaining attributes and the Meter specifications with Meter roles and Purposes.

Activate a metrology configuration

As long as the metrology configuration is inactive, the metrology configuration is considered to be under construction. It is prepared by an expert and can be tested. An inactive metrology configuration can’t be used in the Connexo Insight  system. When a metrology configuration is properly configured it must be activated to make it available for associating usage points with it.

You must have sufficient privileges to activate a metrology configuration.

5

Steps

1.   Go to Administration > Metrology configurations.

2.   Select the applicable metrology configuration.

3.   Select Activate from the actions icon  at the and of the row or from the Actions button in the details pane.

4.   Click Activate to confirm the metrology configuration activation.

1

Important

After activating the metrology configuration you can’t modify it anymore.

Deprecate a metrology configuration

When a metrology configuration is no longer valid in the market it can be deprecated. Then, usage points can no longer be associated with it. All usage points that were already associated with that metrology configuration aren’t modified and it is still possible to monitor data on them.

You must have sufficient privileges to deprecate a metrology configuration.

5

Steps

1.   Go to Administration > Metrology configurations.

2.   Select the applicable metrology configuration.

3.   Select Deprecate from the actions icon  at the end of the row or from the Actions button in the details pane.

4.   Click Deprecate to confirm that you want to deprecate the selected metrology configuration.

1

Important

After deprecating the metrology configuration you can reactivate it afterwards.
