Approve command limitation rule changes

View command limitation rule pending changes

5

Steps

1.   Go to Administration > Device communication > Command limitation rules.

2.   Click the actions icon  next to the command limitation rule and select View pending changes.

If there are changes available requiring approval then this page shows a table showing which attribute was changed for example the status or the day limit. Next to the attribute you can see what was the original value and what the new value is.

Approve/reject pending command limitation rule change

5

Steps

1.   Go to Administration > Device communication > Command limitation rules.

2.   Click the actions icon  next to the command limitation rule and select View pending changes.

3.   Click Approve or Reject.
