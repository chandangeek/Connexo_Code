Configure security on device

Understand security on the device

On the device, all the security accessors defined on device type are shown. Here you can specify the values of the keys and attributes, if you have the correct privileges. It is impossible to edit the security accessors of the security sets. It is possible the change the security accessors in the general attributes and connection methods.

When you specify the values of the security accessors, these values will be added to connection methods, general attributes and security sets that use these security accessors.

View security accessors on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   In the menu on the left go to Communication > Security accessors.

 

Edit security accessors on device

1

Important

You are only able to edit and/or view the security accessors on a device if you have the correct privileges.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   In the menu on the left go to Communication > Security accessors.

3.   Click the actions icon  next to the security accessor and select Edit.

4.   Add the key value.

5.   Click Save.

View security sets on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   In the menu on the left go to Communication > Security sets.

This page shows all security sets that were configured on the device configuration used by the device. To see the actual values of the attributes go to the security accessor page on the device. These values will only be visible if you have the correct privileges see “Understand privileges” for more information.
