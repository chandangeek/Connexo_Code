Key renewal process

Understand key renewal

When a key has reached its expiration date, it needs to be renewed. In MultiSense  you can manage by using the Active and the Passive key. The current key is stored in the Active key while the new key is stored in the Passive key.

The key renewal process consists out of the following steps:

1.   Generate a new key -> Passive key.

2.   Send the new key to the device using a command. For this communication the old key will be used.

3.   Replace the current key by the new key -> active Passive key. The Passive key will now be the Active key and the Active key is kept as the Passive key. If communication fails with the new key then the old key can be restored.

4.   Perform communication tests to check if no key is working.

5.   If communication is successful clear the Passive key, if not reactivate the Passive key.

You can perform this process manually or use the standard process which is available by default How to start a process on a device see “Start processes on devices”.

Generate a passive key

This option is only possible when the active key value is defined.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   In the menu on the left go to Communication > Security accessors.

3.   Click the actions icon  next to the security accessor and select Generate passive key.

A new passive key is automatically generated.

Activate passive key

Activating the passive key in MultiSense  has no effect on the key on the device in the field. To be able to change the effective key in the device itself you need to send a device command to this device.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   In the menu on the left go to Communication > Security accessors.

3.   Click the actions icon  next to the security accessor and select Activate passive key.

When you activate the passive key, the current active key will become the passive key. This key is not removed unless you specifically perform the clear action. This will allow you to restore the key in case communication fails with the new key value.

Clear passive key

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   In the menu on the left go to Communication > Security accessors.

3.   Click the actions icon  next to the security accessor and select Clear passive key.

4.   Confirm the action by clicking Clear.
