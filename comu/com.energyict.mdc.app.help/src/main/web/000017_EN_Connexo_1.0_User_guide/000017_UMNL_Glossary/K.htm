Load profile

A load profile is an object containing one or more channels with the same reading interval. Each channel will contain a different time series. The load profile indicates the variation in the electrical load versus time. Load profiles will vary according to customer type, temperature and season. Power producers typically use this information to plan how much electricity will have to be made available at any given time.

Load profile configuration

A load profile configuration contains further specification of different load profile parameters that are specific for a certain device configuration.

Load profile type

A load profile type contains all essential load profile parameters so that this load profile type can be reused for different device types and device configurations, but it doesn’t contain values.

Logbook

A logbook categorizes events that have occurred on a device. A single logbook can contain different types of events. A device can have more than one logbook. Each logbook has its own unique identifier, which is needed to retrieve its contents from the device. The content of the logbook is recorded during periodic communication to a device. Typical events that are recorded are power outages, connection status, firmware logging, etc.

Logbook configuration

A logbook configuration contains further specification of different logbook parameters that are specific for a certain device configuration.

Logbook type

A logbook type contains all essential logbook parameters so that this logbook type can be reused for different device types and device configurations, but it doesn’t contain values.
