Filter issues using the filter menu

5

Steps

1.   Go to Workspace > Data collection > Issues or Workspace > Data validation > Issues.

2.   By default the issues are sorted by ascending due date.

3.   Filter using the provided criteria in the menu on top of the page.

4.   Click Apply.

5.   Click the remove icon  to remove single filters or click Clear all to remove all applied filters

3

Note

If you click My open issues in the Workspace all open issues that are assigned to you will be shown.

Filter issues using the filter icon

5

Steps

1.   Go to Workspace > Data collection > Issues.

2.   Click the filter icon next to a parameter in the detail pane of an issue to use as filter.
