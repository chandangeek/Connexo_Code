Monitor meter activations

MultiSense creates meter activations when a device undergoes different changes such as a modification of the multiplier, changing the device configuration, performing an install active/inactive transition or linking the device to a usage point. On the device its history page you are able to track these changes.

View meter activation history of a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click History in the menu on the left.

3.   Select the Meter activation tab.

This table shows the current status of the device together all previous meter activations. The active meter activation is marked with a check icon. For every entry you can see the period for which the meter activation was active, together with the meter activation, the multiplier and the linked usage point.

The table also provides links to device configurations and usage point listed.
