Understand device life cycles

A device life cycle is a collection of statuses a device goes through during its lifespan. In MultiSense part of this life cycle is automated which reduces the amount of user interaction and the risk of errors. This is achieved through transitions that use pretransition checks and auto-actions to move devices from one status to another.

Device life cycle management in MultiSense

Most of the configuration is done on the Administration page. Here all device life cycles are added as well as all states and transitions. MultiSense includes a default device life cycle template with default states and transitions.

On the device type you indicate which device life cycle all devices will follow. It is still possible to change this life cycle after the device type has been created.
