Connections

Retrigger connections in bulk

5

Steps

1.   Go to Workspace > Communication > Connections or go to Workspace > Communication > Connection overview > select a group of connections for example the broken connections.

2.   Click Bulk action.

3.   Select to perform the action on all results in the table or make a custom selection.

4.   Click Next.

5.   Select the action Run now.

6.   Click Next.

7.   In the next step you are asked to confirm the action, click Confirm.

8.   Click Finish.

Adjust connection attributes in bulk

5

Steps

1.   Go to Workspace > Communication > Connections or go to Workspace > Communication > Connection overview > select a group of connections for example the connections belonging to a certain device type.

2.   Click Bulk action.

3.   Select to perform the action on all results in the table or make a custom selection.

4.   Click Next.

5.   Select the action Adjust connection attributes. Beware all connections need to have the same connection type or else connection attributes cannot be changed.

6.   Click Next.

7.   Click the edit button  to change the attribute value, click the cancel button  to undo your change and leave the original value.

8.   Click Next.

9.   In the next step you need to confirm the adjustments. These will be queued to be executed. When edited attributes are empty, MultiSense will erase the current connection values. Click Confirm.

10.   The next step shows the result of the action, click Finish.
