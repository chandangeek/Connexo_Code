Understand communication schedules

Certain communication tasks need to be performed at a regular base such as reading registers, load profiles and logbooks. To avoid having to schedule the task every single time, Connexo MultiSense offers the possibility to configure communication schedules.

Shared communication schedules are used to schedule recurring communication tasks that can be used by different devices. When a device needs to perform a non-recurring communication task, it is possible to schedule this task on the device itself. All communication schedules and task linked to a certain devices can be seen on the Communication planning page of that device.
