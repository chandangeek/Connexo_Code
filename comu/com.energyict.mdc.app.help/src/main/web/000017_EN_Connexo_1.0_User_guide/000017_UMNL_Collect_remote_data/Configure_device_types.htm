Configure device types

Configure a device type

5

Steps

1.   Go to Administration > Device management.

2.   Click Device types.

3.   Edit or remove device types via the actions icon .

4.   Click Add device type. Complete the following fields:

Parameter

Description

Name

A unique name.

Communication protocol

Protocol to allow communication with the server.

5.   Click Add.

Add master data to a device type

1

Important

To be able to add register types, load profile types and logbook types to a device type, these first need to be configured. See “Configure master data” for more information.

5

Steps

1.   Go to Administration > Device management > select a device type.

2.   Click Register types > Add register type.

3.   Click Load profiles > Add load profile type.

4.   Click Logbook types > Add logbook type.
