Change urgency of a communication task

1

Important

Only the urgency of scheduled communication tasks can be changed.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

3.   Click the actions icon  and select Change urgency.

4.   Give in a new value and click Save.

Run a communication task on request from the communication task overview

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

3.   Click the actions icon  and select Run or Run now.

  • Select Run now if you want the task to be performed immediately independent of the connection strategy the communication task uses. In this case a new connection will be set up.

  • Select Run when the task can be performed the next time a connection a set up according to the connection strategy.

Run a communication task on request using the communication planning

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication planning in the menu on the left.

3.   Click the actions icon  next to the communication task in the Non recurring communication table. Click Run.
