Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Load profiles in de menu on the left.

3.   Select a load profile, details are shown in the details pane below.

You can also see a load profile’s specifications when you click the load profile name and select the Specifications tab.

View load profile data in a table

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Load profiles in de menu on the left.

3.   Click the load profile name.

4.   Click Table view in the upper right corner of the Readings tab.

All channels are displayed in the columns which allows to see the values of each interval of the different channel on the load profile next to each other.

Details of each value of the channels together with some general information are shown in the details pane on the bottom of the page when you select the interval in the table.

View load profile data in a graph

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Load profiles in de menu on the left.

3.   Click the load profile name.

4.   Click Graph view in the upper right corner of the Readings tab.

Channels of the load profile are displayed under each other, indicated by A in the screenshot below. The unit of the channel is displayed on the left side of the graph. Details of all channels for one time point are shown when hovering over one value.

You can zoom in or out by selecting a standard interval of 4 hours, 1 day, 2days or 1 week (B in the screenshot) or give in a time window in the field indicated by C in the screenshot.

On the bottom of the page (D in the screenshot) there is a bar that shows where the interval that is displayed positions in a larger time frame. The displayed interval can be shifted by sliding the bar to the left or the right. You can also make the interval larger or smaller by making the scroll bar wider or smaller respectively.
