Configure communication protocols

Understand communication protocols

In order to be able to communicate with a device, a communication protocol is required. This protocol allows communication with a specific device from a specific manufacturer. Which communication protocols are available in your system depends on the license your company acquired. Consequently it is not possible to add new protocols as such. Only licensed protocols can be edited.

Edit communication protocols

When you change the attributes of a communication protocol via Administration, these changes will be applied to all device types and configurations that use this protocol.

5

Steps

1.   Go to Administration > Device communication.

2.   Click Communication protocols.

3.   Click the actions icon  to edit. Depending on the protocol different parameters can be defined, e.g.:

  • DeviceTimeZone

  • NodeAddress

  • DevideId

  • CallHomeId

  • ForceToReadToCache

4.   Click Save.

Edit general attributes on a device configuration

General attributes are the attributes that defined by the communication protocol. When you change these general attributes on a device configuration, these changes will be applied to all devices that use this configurations. Unless a different attribute value has been defined on an individual device

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration.

2.   Click General attributes in the menu on the left.

3.   Click Edit.

4.   Change or add values and click Save. Click Restore to default settings to undo changes and set the values back to the default settings of the used protocol.

Edit general attributes on a device

General attributes are the attributes that defined by the communication protocol. When you change these attributes on a device, these changes will only influence this device.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click General attributes in the menu on the left under Communication.

3.   Click Edit.

4.   Change or add values and click Save. Click Restore to default settings to undo changes and set the values back to the default settings of the device configuration.
