Remove devices

Remove devices using a CSV file

1

Important

A device cannot be removed when data has already been collected. To be able to remove devices, you need a CSV file, the first line of this file has to be the header and fields are separated using  "," or ";". The following information has to be included in this file:.

Columns

1. Device arrival date (Format: dd/MM/yyyy)

2. mRID

3. Serial number

4. Year of certification (Format: dd/MM/yyyy)

5. Batch

5

Steps

1.   Go to Devices > Device lifecycle management.

2.   Click Revoke devices.

3.   In step 1 of the wizard select the CSV file containing the devices that need to be revoked.

4.   Click Revoke.

5.   Step 2 of the wizard shows the status of the revoke process.

6.   Click Devices to return to the devices page

3

Note

Click Cancel to stop the import process in step 1 of the wizard. No devices will be removed.
