Understand register groups

Registers can be grouped together. Register groups are used to collect different register readings that belong together. A group will contain all necessary information needed for a certain process. E.g. status registers are used to collect all relevant information for when there is a problem with a device. This group contains registers for firmware versions, battery status, etc.

Manage register groups

5

Steps

1.   Go to Administration > Device management.

2.   Click Register groups.

3.   Click the actions icon to edit or remove register groups.

4.   Click Add register group. Give a unique name to the group and select register types.

5.   Click Save.

Note
