Configure communication tasks on device configurations and devices

Configure communication tasks on the device configuration

1

Important

To be able to add communication tasks to device configurations, first a shared communication task has to be created. See “Configure communication tasks”.

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration.

2.   Click Communication task in the menu on the left.

3.   Edit or remove communication tasks via the actions icon .

4.   Click Add communication task. Complete the following fields:

Parameter

Description

Communication task

List of tasks that were defined in “Add/edit communication tasks”. Only communication tasks that have not already been used for the device configuration are visible.

Security set

List of security settings that have been added to the device configuration.

Connection method

List of connection methods that have been added to the device configuration.

Protocol dialect

Dialects configured for that device configuration.

Urgency

Presents a hierarchy to prioritise tasks.

Always execute for inbound

Indicates that all inbound connections will be accepted.

5.   Click Add. By default the new communication task is set on active.

6.   Click the actions icon to (de)activate the task.

View communication tasks on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

All communication task that the device needs to perform are shown in the table. Details of each communication task are shown in the details pane below.

View communication tasks in the communication planning

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication planning in the menu on the left.

Communication tasks that are scheduled using a communication schedule that is also used by other tasks are shown the top table.

Communication tasks using a schedule that is unique for that task is shown in the table in the middle of the page.

Unscheduled communication tasks are shown in the bottom table.

For more information see “Configure communication on a device”.

Change protocol dialects of a communication task

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

3.   Click the actions icon  next to the communication task and select Change protocol dialect.

4.   Select a new protocol dialect

5.   Click Save.

This change will be executed for all communication tasks that are part of the same communication schedule.
