Reporting

Different standard reports are available within MultiSense.

Create a communication report

5

Steps

1.   Go to Workspace > Data communication > Communications.

2.   Click Generate report.

3.   The report category Device communication is selected. Select one report subcategory out of the list of standard reports.

Report

Description

Device communication task

Lists all communication tasks for the devices in the selected device group(s).

Gateway performance

Lists all gateways together with their performance statistics calculated over the period of time specified when creating the report.

Meter performance

Displays all devices of a device group and the gateway they connect through together with their performance statistics. All statistics are calculated over the period of time specified when creating the report.

Point to point performance

This report combines communication and connection statistics over the period of time specified when creating the report.

4.   Click Next.

5.   Complete the Mandatory filters. With these filters you can select the devices you want to view the data of. Set the In report filters to filter out the data from the devices you selected with the mandatory filter. The in report filters can also be set after the report was created.

6.   Click Next.

7.   If all filters are correct, click Generate report.

The report will open in a new page. The records in the report can be filtered by setting the In report filters and clicking Apply filters.

Navigate to the next page of the report by clicking . You view the results in a table when you click  or click  to view the results in a graph. Click  to export the report as a CSV, Word, Excel, PDF, RTF or text document.

Create a connections report

5

Steps

1.   Go to Workspace > Data communication > Connections.

2.   Click Generate report.

3.   The report category Device connections is selected. Select one report subcategory out of the list of standard reports.

Report

Description

Device connections

List of all connections and connection details of all devices in the selected device group(s).

4.   Click Next.

5.   Complete the Mandatory filters. With these filters you can select the devices you want to view the data of. Set the In report filters to filter out the data from the devices you selected with the mandatory filter. The in report filters can also be set after the report was created.

6.   Click Next.

7.   If all filters are correct, click Generate report.

The report will open in a new page. The records in the report can be filtered by using the In report filters and clicking Apply filters.

Navigate to the next page of the report by clicking . You view the results in a table when you click  or click  to view the results in a graph. Click  to export the report as a CSV, Word, Excel, PDF, RTF or text document.
