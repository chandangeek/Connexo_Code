Device attributes

View device attributes

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   On the landing page of the device you can see a summary of multiple device features in the general information pane. The information that is you see depends on the state of the device.

3.   To see all device attributes click Device attributes in the menu on the left.

Edit device attributes on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   On the meter landing page click the actions button  and select Edit attributes or click Device attributes in the menu on the left and click Edit.

3.   Add or edit device attributes and click Save.

Import connection attributes

1

Important

To be able to import connection attributes, you need a CSV file. the first line of this file has to be the header and fields are separated using  "," or ";". The following information has to be included in this file:

Field

Required

Device MRID

Yes

Connection method name

Yes

Connection attribute 1

Depends on the configuration of the attribute

...

Depends on the configuration of the attribute

Connection attribute N

Depends on the configuration of the attribute

If the connection method specified in the CSV file is not yet available on the device, but is configured on the device configuration, this method will be added to the device. All connection attribute names are case sensitive.

5

Steps

1.   Put the CSV file containing the connection attributes in the import folder.

2.   Once import is finished successfully the original CSV is moved to the success folder. Connection attributes are now added to the devices in MultiSense.
