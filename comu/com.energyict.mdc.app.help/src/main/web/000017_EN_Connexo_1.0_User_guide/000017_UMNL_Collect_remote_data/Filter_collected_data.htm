Filter collected data

Filter data

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Register/Load profile/Logbook/Channels/Events in the menu on the left.

3.   Click the name of an item.

4.   Complete the desired filters in the filter pane on top of the page.

5.   Click Apply. Applied filters are shown on the top of the page.

6.   Click Clear all in the filter menu or in the upper right corner to remove all filters.

Deselect items in the drop down menu to remove single filters.

Filter load profile and channel data on interval

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   View the collected data as described in “Monitor load profile data” and “Monitor channel data”.

3.   Select the from date and hour in the filter pane on top of the page. This is the start date of the interval you want to view the data of.

4.   Select the length of the interval by selecting a value for the field Duration.

5.   Click Apply

6.   Click Clear all, to reset all filters.
