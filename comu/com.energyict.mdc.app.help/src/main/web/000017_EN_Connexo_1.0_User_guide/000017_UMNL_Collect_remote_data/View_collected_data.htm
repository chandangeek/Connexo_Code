View register data

1

Important

Add registers to devices via device configurations, see “Configure register configurations”.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Registers in the menu on the left.

3.   Click the register name.

On the Readings tab the overview table shows all values of that register together with the measurement time. On the bottom of the page more details of the selected register value are provided. This is general information of the measurement together with validation details.

The following fields are shown:

Parameter

Description

Measurement time

The timestamp when the register data was measured by the meter.

Reading time

The timestamp when the data was collected from the meter by for example the data collection system.

Value

The value collected by the device.

Delta value

The difference between the current value and the previous value.

Validation status

Status of the validation rules (active or inactive).

Data validated

Indicates whether the values has been validated or not.

Validation result

Indicates if value is suspect or not and for suspect values the violated rule is shown.
