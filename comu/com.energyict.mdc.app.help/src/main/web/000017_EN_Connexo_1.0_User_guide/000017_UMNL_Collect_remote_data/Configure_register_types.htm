Configure register types

Add/remove register types

5

Steps

1.   Go to Administration < Device management.

2.   Click Register types.

3.   Click the actions icon  to edit or remove a register type.

4.   Click Add register type. Complete the following fields:

Parameter

Description

OBIS code

Object Identification System, provides standard identifiers for all data originating from the metering devices.

Reading type

Select reading type by typing the reading type name.

5.   Click Add.

3

Note

To see the reading type details of a register click the information icon  next to the reading type in the overview table.

When a register type is being used by a certain device type then only the name of the register type can be edited. This is to prevent any harmful effects on the data collection process.
