Configure shared communication schedule(s) on device(s)

Add/remove shared communication schedules to multiple devices

With Connexo MultiSense you have the option to schedule different communication tasks on multiple devices in a single operation.

5

Steps

1.   Go to Search and use the search criteria to find the devices you want to add communication schedules to as described in “Advanced search”.

2.   Click Bulk actions.

3.   Select the devices in step 1 of the wizard.

4.   Click Next.

5.   Select Add shared communication schedules or Remove shared communication schedules in step 2 of the wizard.

6.   Click Next.

7.   Select the shared communication schedule(s) you want to add or remove from the devices in step 3.

8.   Click Next.

9.   Confirm the action by clicking Confirm in step 4.

10.   In step 5 the status is shown.

If successful, a message a shown with a summary of the performed action. If unsuccessful, an error message is shown with the details of why the action failed.

11.   Click Finish.

3

Note

A communication task on a device can only be scheduled in maximum one shared communication schedule.

Click Cancel to end the wizard, no communication schedules will be added or removed.
