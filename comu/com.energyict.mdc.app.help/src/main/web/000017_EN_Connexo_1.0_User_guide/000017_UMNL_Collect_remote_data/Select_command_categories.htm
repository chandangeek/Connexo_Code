Select command categories

Add device command categories to a communication task

5

Steps

1.   Go to Administration > Device communication > Communication tasks.

2.   Click the action icon  of a communication task and select Edit or add a new communication task as described in “Add/edit communication tasks”.

3.   Select in the Command categories one command or multiple using the CTRL or SHIFT key.

4.   Press the add button , the add all button or drag and drop.

5.   Select in the Selected command categories one command or multiple using the CTRL or SHIFT key.

6.   Click the remove button , the remove all button  or drag and drop to remove commands.

7.   Click Save when editing an existing communication task or click Add when creating a new one.
