Configure device configurations

Add/remove device configurations

5

Steps

1.   Go to Administration > Device management > Device type.

2.   Select a device type by clicking the device type name or create a new one as described in “Configure a device type”.

3.   Click Configuration in the menu on the left.

4.   Click the actions icon  to edit or remove configurations. To remove a configuration, you must first deactivate it.

5.   Click Add device configuration. Complete the following fields:

Parameter

Description

Name

A unique name.

Description of the configuration

Non mandatory description of the device configuration.

Directly addressable

If the device cannot communicate directly with the communication server, e.g. in the case of a slave device, this option is disabled

Device is as a gateway

If the device doesn’t have gateway functionalities this option is disabled.

Type of gateway

Select the type of gateway:

•   Home Area Network gateway: connects to a small number of slave devices in a home environment.

•   Local Area Network gateway: connects to a large number of slave device in a local (larger) environment.

Dependent of the type of gateway that you select the look & feel of the landing page will be different.

6.   Click Add.

7.   Click the actions icon  to activate the device configuration. Or click the actions button on the device configuration overview page.

8.   Add and configure register, load profile and logbook configurations.

3

Note

When the device configuration is activated it can be used to create devices. Note however that once a device uses a specific device configuration, this configuration can no longer be deactivated.
