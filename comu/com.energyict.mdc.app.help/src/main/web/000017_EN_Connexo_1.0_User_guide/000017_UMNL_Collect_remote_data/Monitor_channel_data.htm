View channel data

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channel in de menu on the left.

3.   Click the channel name.

4.   The Readings tab shows all channel data. The graphical representation and table are combined in one view.

Channel data is displayed in a graph indicated by A in the screenshot below. The unit of the channel is displayed on the left side of the graph. Details of the channel for one time point are shown when hovering over the value. The validation status of the values are indicated using different colours:

Colour

Meaning

Blue

Data is not validated.

Green

Data is validated and not suspect. That is not suspect because values were valid upon collection or value was confirmed or edited.

Red

Data is validated and suspect.

Yellow

Data is validated and has the label informative.

Green border + transparent green

Data is validated and edited but changes have not been saved yet.

Dark green

Data is validated and estimated.

You can zoom in or out by selecting a standard value, this can be 4 hours, 1 day, 2days or 1 week (B. in the screenshot), but these are dependent of the configured interval. A second option to zoom in or out is to change the time window in the field indicated by C in the screenshot.

Under the graph (D in the screenshot) there is a bar that shows where the interval that is displayed positions in a larger time frame. The displayed interval can be shifted by sliding the bar to the left or the right. You can also make the interval larger or smaller by making the scroll bar wider or smaller respectively.

Figure: Graphical representation of channel data

 

On the bottom of the page there is a table containing all values. End of interval, Calculated (delta) (for non cumulative data), Collected and Last update are shown in the columns. When you select a value in this table then this value is also shown in the graph.

On the bottom of the page more details of the selected value are provided. Three tabs show different information.

a. General: shows information of the measured value.

b. Reading values: shows the actual values of the channel.

c. Reading quality: important observation concerning the value, for example a power outage occurred.

The following parameters are displayed:

Parameter

Description

Interval

The consumption interval that was measured.

Last update

Shows when the reading was last handled. This can be due to validation, estimation or editing.

Data validated

Indicates whether the values has been validated or not.

Calculated value

The effective consumption measured in the defined interval together with the result of the validation.

Collected value

The integral consumption measured on the device by the end of the defined interval together with the result of the validation.

Reading quality

Indicates that there is something remarkable about this value. Hover over the information icon for more details.

View channel history of a single value

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Got to Data sources > Channel in de menu on the left.

3.   Click a channel name.

4.   Click the actions icon  next to the value you want to see the history of and select View history.

This page shows a table containing all edits made to this value together with the date of the change and who changed the value. A new history item is created when a value is suspect, edited, estimated or confirmed.

When you select a value you can see all details in the tables on the bottom of the page divided over three tabs: general info, reading value and reading quality.

View channel history of multiple values

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Got to Data sources > Channel in de menu on the left.

3.   Click a channel name.

4.   The Bulk action button and select View history.

The history page shows all the history of all values that comply with the defined filters, which are the same as on the previous page.

5.   Use the filters on top to see specific values. You can define the start date as well time period you want to visualise. Select Changed data to only show values that have been changed over time.
