Import connection attributes

1

Important

To be able to import connection attributes, you need a CSV file, the first line of this file has to be the header and fields are separated using  "," or ";". To see what information has to be included in this file click the information icon .

5

Steps

1.   Go to Devices > Device attributes import > Import connection attributes.

Connection attributes are added using a wizard.

2.   Select the connection type to add the attributes to in the first step of the wizard.

3.   Click Next.

4.   In step 2 of the wizard select the CSV file that contains the attributes.

5.   Click Import.

In step 3 the status of the import is shown together with a link to the devices page.

Import security attributes

1

Important

To be able to import security attributes, you need a CSV file, the first line of this file has to be the header and fields are separated using  "," or ";". To see what information has to be included in this file click the information icon .

5

Steps

1.   Go to Devices > Device attributes import > Import security attributes.

Security attributes are added using a wizard.

2.   In step 1 of the wizard select the security set you want to add attributes to.

3.   Click Next.

4.   In step 2 select the CSV file containing all attributes.

5.   Click Import.

In step 3 the status of the import is shown together with a link to the devices page.
