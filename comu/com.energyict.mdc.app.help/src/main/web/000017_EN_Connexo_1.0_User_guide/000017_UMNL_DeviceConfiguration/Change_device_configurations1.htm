Change device configurations

Change device configuration on a single device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click the action button  on the landing page and select Change device configuration.

3.   Select the new device configuration.

4.   Click Save.

Change device configurations on multiple devices

5

Steps

1.   Go to Search and use the search criteria to find the devices with the same device configuration as described in “Advanced search”.

If you do not search for devices with the same device configuration, the action Change device configuration will not be available in the bulk action menu.

2.   Click Bulk actions.

3.   In step 1 select to change the device configuration of all devices of the search result or make a custom selection.

4.   Click Next.

5.   In step 2 select Change device configuration.

6.   Click Next.

7.   Select the new configuration.

8.   Click Next.

9.   In the next step a summary is given of the actions that will be taken and the consequences this could have. Click Confirm. If it is not possible to change the device configuration of the selected device, this will be shown here. Click Finish and try to solve the problem.

10.   If the action was successful the last step will show that the task is put on the queue.

11.   Click Finish.
