Custom attributes

Understand custom attributes

Connexo offers the possibility to add different extra custom attributes to your system. This allows to add additional information which is not covered in by MultiSense’s basic CIM model, but which is relevant for the user, for example for integration with other applications, processes, operations, etc. This also allows you to tune the application to your specific regional needs

In MultiSense it is possible to add custom attributes to devices, channels and registers. Every attribute set is defined by a number of privileges.

In MultiSense it is possible to version some custom attribute sets for more information see “Manage custom attribute sets over time”.
