Manage custom attributes on type objects

View custom attributes on a device type

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Custom attributes > Custom attribute sets.

The page shows all custom attribute sets that have been added to this device type. You can see the details of every attribute set by clicking the information icon . This table also shows the privileges configured on the attribute set.

Add custom attributes sets to device type

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Custom attributes > Custom attribute sets.

2.   Click Add custom attribute sets.

3.   Select one or more attribute sets, click the informations icon  to see the attribute set details.

4.   Click Add.

Remove custom attributes from device type

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Custom attributes > Custom attribute sets.

2.   Click the actions icon  next to the attribute set you wish to remove and select Remove.

3.   Click Remove.

Removing a custom attribute set from a device means that all values of the attribute set will be lost.

View custom attributes on register type

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Data sources > Register types.

2.   Click a register type in the table.

The custom attributes configured on this register type are shown in the details pane on the bottom of the page. Click the information icon to see the attribute set details.

Add (edit) custom attributes on register type

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Data sources > Register types.

2.   Click the actions icon  next to the register type and select Edit.

3.   Select one of the custom attributes from the drop down menu next to the field Custom attribute set.

4.   Click Save.

3

Note

Only one custom attribute set per register type can be added.
