Configure data sources on a device

Edit register specifications

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Data sources < Registers in the menu on the left.

3.   Select a register in the table.

4.   Select the Specifications tab.

5.   Click the actions button  and select Edit.

6.   Change the OBIS code, overflow value or number of fraction digits.

7.   Click Save.

Edit channel specifications

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Data sources < Channels in the menu on the left.

3.   Select a channel in the table.

4.   Select the Specifications tab.

5.   Click the actions button  and select Edit.

6.   Change the OBIS code, overflow value or number of fraction digits.

7.   Click Save.
