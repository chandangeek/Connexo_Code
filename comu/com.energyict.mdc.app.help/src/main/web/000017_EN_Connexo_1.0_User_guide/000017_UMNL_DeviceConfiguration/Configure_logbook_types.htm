Configure logbook types

Add logbook types

5

Steps

1.   Go to Administration < Device management > Logbook types.

2.   Click Add logbook type. Complete the following fields:

Parameter

Description

Name

A unique name.

OBIS code

Provide the six attributes of the OBIS code, separated by “.”.

3.   Click Add.

3

Note

Only the name of the logbook type can be edited when a logbook type is used by a device type. This is to prevent that a device might no longer be functional as consequence of the change.

Edit logbook type

5

Steps

1.   Go to Administration < Device management > Logbook types.

2.   Click the actions icon  and select Edit.

3.   Change the logbook parameters.

4.   Click Save.

Remove logbook types

5

Steps

1.   Go to Administration < Device management > Logbook types.

2.   Click the actions icon  and select Remove.

3.   Click Remove.
