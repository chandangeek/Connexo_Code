Firmware campaigns

View firmware campaigns

5

Steps

1.   Go to Workspace > Firmware management > Firmware campaigns.

In the overview table you can see all configured firmware campaigns and their details. The table shows the name of the firmware campaign, the device type on which the firmware will be uploaded, the type of firmware, the firmware version and when the campaign was started.

In the table you can also see the status of the different devices in the firmware campaign. Here you see the amount of devices with a successful upload, the amount of failed uploads and the amount of uploads failed due to a configuration error. Next to these options you can also see the total amount of ongoing, pending and cancelled uploads.

The data is in this table can be exported to a CSV using the grid export button .

2.   Click the firmware campaign name to see the overview page of the firmware campaign. This page shows the same details as the table as well as the firmware management options defined by communication protocol and device type.

3.   Click Devices in the menu on the left to see which devices are included in the campaign and what their status is.

Add firmware campaign

5

Steps

1.   Go to Workspace > Firmware management > Firmware campaigns.

2.   Click Add firmware campaign. Complete the following fields:

Parameter

Description

Name

Name of the firmware campaign. A default name is provided, but can be overwritten.

Device type

Select the device type to define which device will be upload with the firmware.

Device group

Select the device group that contains all devices that need the firmware upload. Only the devices with the device type defined in the device type parameter will be subject to the firmware campaign.

Firmware management option

Select when to upload and activate the firmware on the device of the firmware campaign. See “Understand firmware management options” for more information.

Firmware type

Select the type of firmware to upload. When there is only one type of firmware available on this device type this will be selected by default.

Firmware version

Select the firmware version to upload on all devices.

Activation date

Define the activation date of the firmware version when you select to upload the firmware with activation date.

3.   Click Add.

Cancel firmware campaign

5

Steps

1.   Go to Workspace > Firmware management > Firmware campaigns.

2.   Click the actions icon  and select Cancel.

It is only possible to cancel firmware campaigns that have the status ongoing.

3.   Click Cancel campaign.

When you cancel the firmware campaign the upload all devices that still need to be executed will be cancelled, ongoing uploads will be finished and successful uploads are not reversed.
