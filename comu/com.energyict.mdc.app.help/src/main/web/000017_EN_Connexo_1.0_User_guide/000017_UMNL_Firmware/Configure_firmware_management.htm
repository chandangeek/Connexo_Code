Manage firmware versions

Understand firmware versions

A firmware version can have one of the following four statuses: Test, Final, Ghost or Deprecated. A version can have the status Test when you need to test the new firmware on a small batch of devices to check whether everything is running without errors. Once the test version has been successfully tested, you can promote it to Final. You can then install this version on a large amount of devices.

Ghost versions are created by Connexo when a device has a firmware version installed that does not exist in MultiSense yet. This firmware version can be promoted to Test or Final if you upload the corresponding firmware file.

Once a version is no longer needed or there is an error in the firmware version, you can give this version the status Deprecated. This version can no longer be uploaded to devices. A Deprecated firmware version is shown in the overview table, but the firmware file of this version is removed.

View firmware versions

5

Steps

1.   Go to Administration > Device management > Device types > select a device type > Firmware > Firmware versions.

The table shows all firmware versions that have been added to the device type and are available to upload on a device. For every version the type of firmware as well as the status of the firmware is shown.

Filter firmware versions

5

Steps

1.   Go to Administration > Device management > Device types > select a device type > Firmware > Firmware versions.

You can filter firmware version using different parameters. When the communication protocol only allows one type of firmware the type filter will not be shown.

2.   Specify the desired filters.

3.   Click Apply. The applied filters are shown on top of the page.

4.   Deselect items in the drop down menu to remove single filters or click Clear all to remove all filters. Donâ€™t forget to click Apply to execute the changes in your filter.

Add firmware version

5

Steps

1.   Go to Administration > Device management > Device types > select a device type > Firmware > Firmware versions.

2.   Click Add firmware version. Complete the following fields:

Parameter

Description

Firmware file

Upload the file containing the firmware version.

Version

The version indication of the firmware should be unique for the device.

Firmware type

When the communication protocol only allows one type of firmware only that firmware type will be shown.

o   Meter firmware: contains all metrics

  • Communication firmware: contains are features needed to allow communication between the device and the meter data collection system.

Firmware status

o   Final: use this status when the firmware version has been tested and is ready to upload on a large number of devices.

o   Test: use this status when testing new firmware versions. This will prevent you to upload a firmware version still containing errors on devices.

3.   Click Add.

Edit firmware versions

5

Steps

1.   Go to Administration > Device management > Device types > select a device type > Firmware > Firmware versions.

2.   Click the actions icon  next to the firmware version and select Edit.

When you edit a firmware version you are able to upload a new firmware file, the new firmware file will then replace the existing one.

If the current status of the firmware version is Ghost, you are able to change the status of the version to Test or Final, but not the name of the version. If the status is Test or Final you can change the name of the version, but not the status. Deprecated version can no longer be edited.

Set firmware version as Final

A firmware version with status Final can be used to upgrade devices in the field.

5

Steps

1.   Go to Administration > Device management > Device types > select a device type > Firmware > Firmware versions.

2.   Click the actions icon  next to a test firmware and select Set as final.

Deprecate firmware versions

5

Steps

1.   Go to Administration > Device management > Device types > select a device type > Firmware > Firmware versions.

2.   Click the actions icon  and select Deprecate.

Once the firmware version has been deprecated it is still shown in the table, but it can no longer be used to upload on devices.
