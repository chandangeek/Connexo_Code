Manage firmware options

Understand firmware management options

Firmware management options determine if it is possible to upload firmware and how it will be activated. What options are available are defined by the communication protocol used by the device type. You can edit these firmware settings further on the device type.

There are three firmware management options possible. Once you uploaded the firmware version you have the option to activate the firmware later, immediately or use an activation date. With the option activate later you need to activate the firmware later. You can do this manually or using a command. When you select to activate the firmware with an activation date, you need to specify the date on which the firmware needs to be activated.

However, uploading a firmware version uses a communication task and as described previously when this communication task will be executed is determined by the connection strategy of the communication task’s connection method, see “How connections and communication work in Connexo” for more information.

View firmware management options

5

Steps

1.   Go to Administration > Device management > Device types > select a device type > Firmware > Firmware management options.

2.   This page shows the following information:

Parameter

Description

Firmware management allowed

Indicates whether a remote upload is possible.

Firmware management options

Shows which options are possible to upload firmware on this device type.

Edit firmware management options

5

Steps

1.   Go to Administration > Device management > Device types > select a device type > Firmware > Firmware management options.

2.   Click Edit. Select the following fields:

Parameter

Description

Firmware management allowed

Indicates whether it should be possible to upload firmware remotely.

Firmware management options

Select which options are possible for this device type. The available options are determined by the communication protocol used by the device type.

3.   Click Save.

3

Note

The communication protocol used by the device type defines whether it is possible to edit the upgrade options. When this is not possible the Edit button is disabled.
