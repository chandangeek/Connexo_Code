Filter connections

Filter connections from the overview page

5

Steps

1.   Go to Workspace > Data communication > Connections overview.

2.   Click one of the links provided on the overview page.

You will go to the connection page where a filter has been applied, showing only those connections that correspond to the link you selected.

3

Note

When you click one of the parameters in Connection summary, then multiple filters are combined.

Parameter

Filter

Success

Waiting

Ongoing

Pending + Busy + Retrying

Failed

Never completed + Failed

Filter connections using the filter option

5

Steps

1.   Go to Workspace > Data communication > Connections.

2.   Select one or more values for one or more filter items from the filter menu on top of the page.

3.   Click Apply.

The applied filters are shown on top of the page.

4.   Click Clear all in the filter pane or in the upper right corner, to remove all applied filters.

Single filters can be removed by deselecting the item in the drop down menu.
