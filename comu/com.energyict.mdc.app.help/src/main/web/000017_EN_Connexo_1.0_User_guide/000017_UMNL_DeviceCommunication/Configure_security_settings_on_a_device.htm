Configure security settings on a device

Edit security settings

1

Important

Security settings have to be added to the device configuration before security settings can be edited on a device. See “Add security settings” for more information.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Security settings in the menu on the left.

3.   Click the actions icon  and select Edit. Security settings that are missing attributes are indicated by the status Incomplete.

4.   Complete the attributes and click Save. Select Show values to see the values (only available if you have edit and view rights).

3

Note

 If you have view and edit rights, you will be able to see and edit passwords and encryption keys. When only have edit rights, you will be able to edit passwords and encryption keys, but you will not see the values.

You can also add security settings to multiple devices at once by importing security attributes as described in “Import security attributes”.
