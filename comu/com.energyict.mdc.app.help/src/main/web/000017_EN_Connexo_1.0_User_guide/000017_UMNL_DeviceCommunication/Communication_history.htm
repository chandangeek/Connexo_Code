Communication history

View communication history

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

3.   Click the actions icon  or the actions button  in the details pane and select View history.

This page shows the history of the selected communication task. For every communication the duration, status, result and used connection are shown.

4.   Click an item in the table. Below the table more information is shown.

Two details panes show further details of the run of the communication task and the used connection. Here you can find the communication and connection summary showing the errors and warning that occurred during execution.

Click Show communication details or Show connection details to see the configuration details of the communication task and connection method together with the execution details.

View communication log

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

3.   Click the actions icon  or the actions button  and select View history.

4.   Click the actions icon or the actions button in the details pane of one of the communications and select View communication log.

5.   You can filter the logging and log level (Debug or Trace). Click Apply.

Below the filter you can find the table containing all communication logs with the message. An icon next to the log entry shows whether it is an error  or a warning . More details are given in the details pane below the table.

Filter communication log

5

Steps

1.   View the communication logs as described in “View communication log”.

2.   Select the filters in the filter pane on top of the page.

3.   Click Apply. The applied filters are shown on top of the page below the connection details.

4.   Deselect items in the drop down menu to remove the filters.

5.   Click Clear all to remove all filters.
