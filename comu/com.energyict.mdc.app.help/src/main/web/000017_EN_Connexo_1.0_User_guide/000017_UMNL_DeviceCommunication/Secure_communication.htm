Secure communication

Understand secure communication

To guarantee that the data to and from the devices remains secure, certain measures have to be taken. In Connexo MultiSense this is accomplished via the use of security settings. Security settings consist out of all security properties to communicate with a device such as authentication and encryption. Authentication ensures that both ends of the connection are secure while encryption ensures that the information within a session is not compromised.

In Connexo Admin the system administrator adds privileges to different user roles. These define what a user is allowed to do within the different parts of the applications. These privileges are further divided in levels. With these levels you can group user roles. For example level 1 can be used for people working with electricity meters while level 2 is used for people working with gas meters.

In MultiSense you can then link these privileges to the security settings configured on the device configuration. This will define what different users can do on a device. For security settings the following scenarios are possible:

  • No privileges:

  • A user that has no privileges will not see the different security attributes of the selected security set.

  • All action buttons are disabled.

  • Only view privileges:

  • The names of the different security attributes of the selected security set are shown. Values are encrypted and shown as ********, if the value is not empty.

  • All action buttons are disabled.

  • Only edit privileges:

  • The names of the different security attributes of the selected security set are shown. Values are encrypted and shown as ********, if the value is not empty.

  • All action buttons are enabled and it is possible to edit values.

  • View and edit privileges:

  • The names of the different security attributes of the selected security set are shown. Values are encrypted and shown as ********, if the value is not empty.

  • All action buttons are enabled and it is possible to edit values and show or hide the value.
