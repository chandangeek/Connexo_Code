Configure security setting configurations

Add security settings

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Configurations > select a configuration.

2.   Click Security settings in the menu on the left.

3.   Click Add security setting. Complete the following fields:

Parameter

Description

Name

A unique name.

Authentication level

The options are dependent of the device protocol used.

Encryption level

The options are dependent of the device protocol used.

4.   Click Add.

Edit security settings

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Configurations > select a configuration > click Security settings in the menu on the left.

2.   Click the actions icon  and select Edit.

3.   Change the security setting parameters.

4.   Click Save.

Remove security settings

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Configurations > select a configuration > click Security settings in the menu on the left.

2.   Click the actions icon  and select Remove.

3.   Click Remove.

Add/remove privileges to security settings

Privileges indicate which actions you are allowed to do inside MultiSense. Different levels exist for different privileges (edit, view, execute, etc.). Levels are groups of user roles that are linked to privileges and are configured in Admin. Privileges prevent unauthorised access to specific features thereby preventing errors.

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Configurations > select a configuration.

2.   Click Security settings in the menu on the left.

All configured security privileges are shown on the bottom of the page.

3.   Click Add privileges.

4.   Select one or more privileges and levels and click Add.

5.   Click the actions icon  next to the privilege and click Remove to remove privileges.

3

Note

The levels of the privileges are configured in Connexo Admin by the system administrator. If you have view and edit rights, you will be able to see and edit passwords and encryption keys. When only have edit right, you will be able to edit passwords and encryption keys.
