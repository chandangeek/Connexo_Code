Configure communication tasks on devices

View communication tasks on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

All communication task that the device needs to and could execute are shown in the table. Details of each communication task are shown in the details pane below.

Collect data on request using a communication task

  • For more information how to change the urgency of a communication task see “Change urgency of a communication task”.

  • To run a communication task on request see “Run a communication task on request from the communication task overview” and “Run a communication task on request using the communication planning”.

  • To change the connection method of a communication task see“Change connection method of a communication task on a device”

Change connection method of a communication task on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

3.   Click the actions icon  and select Change connection method.

4.   Select another connection method and click Save.

3

Note

If you change the connection method of a communication task that is part of a shared schedule, then the connection method of all tasks in this schedule will be changed.

Change protocol dialect of a communication task on a device

1

Important

You can only change the protocol dialect of unscheduled communication tasks or tasks with an individual communication schedule.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

3.   Click the actions icon  and select Change protocol dialect.

4.   Select another protocol dialect and click Save.

3

Note

This change will be executed for all communication tasks that are part of the same communication schedule.
