Communication

Retrigger communications in bulk

5

Steps

1.   Go to Workspace > Communication > Communications or go to Workspace > Communication > Communication overview > select a group of communications for example the broken communications.

2.   Click Bulk action.

3.   Select to perform the action on all results in table or make a custom selection.

4.   Click Next.

5.   Select the action Run or Run now.

  • Select Run when the task can be performed the next time a connection is set up according to the connection strategy and schedule.

  • Select Run now if you want the task to be performed immediately, only possible when the connection strategy of the connection method is As soon as possible. In this case a new connection will be set up.

6.   Click Next.

7.   In the next step you are asked to confirm the action, click Confirm.

Click Finish.
