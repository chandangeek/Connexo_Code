Search Connexo

Understand the search feature

In Connexo MultiSense it is possible to search for devices based on different parameters. The search feature can be accessed by pressing Search in the upper right corner of the page. The search function is available on every page throughout the application.

On the Search page the search criteria are located on the left side of the page. Above the search results the applied search criteria are shown as separate buttons that can be removed. Below the search criteria, the applied sort parameters are also shown as separate buttons that can be removed.

From the search results you can navigate to a specific device by clicking the name of a device. There is also an option to perform actions on multiple devices at once using the feature Bulk action.
