Advanced search

Perform a device search

5

1.   Click Search in the upper right corner of the page.

2.   Use the following criteria to search for specific device(s):

Parameter

Description

MRID

Type the complete Master Resource IDentifier (MRID) of a device or use wildcards.

Serial number

Type the complete serial number of a device or use wildcards.

Type

Select one or more device types.

Configuration

The option is only shown when a device type is selected. Select one or more configurations linked to the selected device type.

3.   Click Search or press enter.

4.   Remove search criteria by clicking the remove icon  on the search criteria button.

5.   Click Clear all to remove all criteria.

6.   Go directly to a device page by clicking the device name.

3

Note

The following wildcards can be used:

Wildcard

Function

*

Matches zero or more characters

%

Matches zero or more characters

?

Matches exactly one character

When all fields are empty and you click Search, then no result will be shown. You need to type “*” in one of the search criteria to see all possible search results.
