Add sort criteria

5

Steps

1.   Search as described in “Search for devices”.

2.   Click Add sort. Select one or more of the following parameters:

Parameter

Description

MRID

Sort based on the unique device identifier

Serial number

Sort based on the serial number of the devices

Type

Sort based on device type

Configuration

Sort based on configuration

3.   Click the sort criteria button to change the order of sorting.

Remove sort criteria

5

Steps

1.   Search as described in “Search for devices”.

2.   Click the remove icon on the sort criteria button to remove a single sort criteria.

3.   Click Clear all to remove all sort criteria.

Change sort order

5

Steps

1.   Search described in “Search for devices”

2.   Add sort criteria.

3.   Click the sort criteria button to change the order.
