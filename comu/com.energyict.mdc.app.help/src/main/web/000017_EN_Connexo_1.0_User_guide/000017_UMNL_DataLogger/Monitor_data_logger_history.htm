Monitor data logger history

View history of linked data logger slaves on data logger channel

5

Steps

1.   Select a data logger device using Search, see “Advanced search”.

2.   Go to Data sources > Channels in the menu on the left.

3.   Select the Specifications tab.

On the bottom of the page you can find Data logger slave history. Here you can find which data logger slave is linked to this data logger channel and when. A link to the data logger slave is available.
