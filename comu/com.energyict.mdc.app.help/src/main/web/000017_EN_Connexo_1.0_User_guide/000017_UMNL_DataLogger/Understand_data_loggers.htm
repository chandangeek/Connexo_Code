Understand data loggers

Data loggers in MultiSense

A data logger is a device that stores data from other devices in its own channels. You do not know upfront which data these slave devices will have. The data logger does not save information of its data logger slave devices. You do not have to specify the slave devices before you can collect data, you can link the slave devices later.

Validation of data logger data

Typically you will validate the channel and register data of the data logger and data logger slaves on the slave device. However you are still able to configure validation on the data logger because this device can contain other data logger independent data.

For more information in validation see “Data validation”.
