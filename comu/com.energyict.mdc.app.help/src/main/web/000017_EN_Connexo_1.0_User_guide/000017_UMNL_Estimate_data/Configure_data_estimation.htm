View data estimation rule set

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets.

On top of the page you see all configured estimation rule sets. Per estimation rule set you can see the amount of active and inactive rules.

In the middle of the page you can see all estimation rules that are part of the estimation rule set. When you select one of the estimation rules, detailed information of this rule will be shown on the bottom of the page

2.   Click an estimation rule set name. On the estimation rule set home page there are three different men items:

Menu item

Description

Overview

Shows all estimation rule set details with the amount of active and inactive rules.

Estimation rules

Shows all estimation rule configured in this set and allows you to add new ones or edit and remove existing ones.

Device configurations

Here you can all device configurations that have been linked to this estimation rule set. Here you can also add other device configurations to this rule set.

View estimation rules on a device configuration

5

Steps

1.   Go to Administration > Device management > Device types > select a device type > click Configurations > select a configuration > Reading quality > Estimation rule sets.

This page shows all the estimation rules and rule sets that are available on this device configuration.

View estimation rules on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Data estimation under Reading quality to see the estimation rules on this device.

On top of the page, the status of the estimation is shown and can be changed by clicking the button on the upper right corner (only if the device is in a state that allows estimation see “Device options and the device life cycle state” for more information).

Below the status there is an overview table with all estimation rule sets configured on this device. When you select a rule set in this table, you will see the included estimation rules and estimation rule details.
