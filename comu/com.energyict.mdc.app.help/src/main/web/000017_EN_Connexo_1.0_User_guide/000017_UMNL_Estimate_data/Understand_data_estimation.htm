Understand data estimation

What is data estimation

Data estimation corrects suspect values, detected by the validation process, using  ‘predefined’ algorithms. With data estimation you can estimate the data based on valid collected data or using custom values.

In some cases data cannot be estimated and needs to be edited, see “Edit data” for more information.

Data estimation in MultiSense

The configuration of data estimation is similar to that of data validation. Estimation rules are grouped in estimation rule sets. Within the rule set you define the order of the rules to select which estimation rule will be used first. Typically the rules that result in the most realistic values are put on top of the list.

In MultiSense different default estimation rules already available upon installation. These estimation rules cover all basic needs. You can find a more detailed description of these estimation rules in the topic.
