Manage data estimation on a device

Add estimation rule set to a device configuration

5

Steps

1.   Go to Administration > Device management > Device types > select a device type > Configurations > select a device configuration > Reading quality > Estimation rule sets.

2.   Click Add estimation rule sets.

3.   Select to add All estimation rule sets or Selected estimation rule sets.

If you select Selected estimation rule sets you need to select one or more of the estimation rule sets shown in the table.

4.   Click Add.

3

Note

Only estimation rule sets that have matching reading types are shown.

Add device configuration to a estimation rule set

5

Steps

1.   Go to Data estimation > Estimation rule sets > select an estimation rule sets > Device configurations.

2.   Click Add device configurations.

3.   Select to add All device configurations or Selected device configurations.

If you select Selected device configurations you need to select one or more of the device configurations shown in the table.

4.   Click Add.

3

Note

Only device configurations that have matching reading types are shown.

Change order of estimation rule sets on a device configuration

5

Steps

1.   Go to Administration > Device management > Device types > select a device type > Configurations > select a device configuration > Reading quality > Estimation rule sets.

2.   Click Edit order.

3.   Drag and drop the estimation rule set up or down and release when the green check appears.

4.   Click Undo to cancel changing the order or click Save order to save the changes.

Manage activation status of data estimation on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Data estimation under Reading quality.

The status of data estimation is shown on top of the page.

3.   Click Activate data estimation or Deactivate data estimation to change the status.

4.   Confirm the action by clicking Activate or Deactivate.

Manage activation status of estimation rule sets and rules on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Data estimation under Reading quality.

3.   Click the actions icon  next to estimation rule set and select Activate or Deactivate.

To change the activation status of the estimation rules of this rule set, click the estimation rule set to go to its homepage.
