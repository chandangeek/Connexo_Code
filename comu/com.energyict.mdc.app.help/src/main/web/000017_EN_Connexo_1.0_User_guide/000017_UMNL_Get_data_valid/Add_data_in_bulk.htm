Import readings

1

Important

This importer only allows the import of daily readings.

To be able to import device readings, you need a CSV file, the first line of this file has to be the header and fields are separated using  "," or ";". The following information has to be included in this file:.

Field

Required

Device MRID

Yes

Reading date

Yes

Reading type MRID

Yes

Value

Yes

It is possible to add different readings from the same device, create different rows with the same device MRID, but with different values. All values should match the configuration settings, being the overflow defined as well as the number of digits and number of fraction digits.

5

Steps

1.   Put the CSV file containing the readings in the import folder.

2.   Once import is finished successfully the original CSV is moved to the success folder. The defined values are now add to the devices in MultiSense.
