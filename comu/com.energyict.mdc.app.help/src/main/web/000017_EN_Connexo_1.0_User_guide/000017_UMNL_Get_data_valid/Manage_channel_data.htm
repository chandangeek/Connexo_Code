Manage channel data

Edit channel readings

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channels in de menu on the left.

3.   Click the name of the channel you want to edit and select Table view in the upper right corner.

4.   Click Edit readings 

5.   Click the actions icon next to the value you want to edit and select Edit value or click the value.

6.   Press enter. A red tag is displayed, indicating that the value has been edited.

7.   Click Save. The edited value will be marked by the edit icon . When you hover over this icon the date of editing is shown.

Remove channel readings

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channels in de menu on the left.

3.   Click the name of a channel you want to remove and select Table view in the upper right corner.

4.   Click Edit readings.

5.   Click the actions icon next to the value you want to remove and select Remove reading. A red tag is displayed, indicating that the value has been edited.

6.   Click Save. The edited value will be marked by the remove icon . When you hover over this icon the date of removal is shown.

3

Note

Removed values are not deleted from the system, but are retained to allow monitoring of changes.
