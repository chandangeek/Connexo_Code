Remove register data

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Registers in the menu on the left.

3.   Click the name of the register of which you want to remove data or click View suspects to only see suspect values.

4.   Click the actions icon next to the register value you want to remove and select Remove.

5.   A warning message appears to confirm the removal, click Ok.

3

Note

Removed values are not removed from the system, but are retained to allow monitoring of changes.

Remove channel readings

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channels in de menu on the left.

3.   Click the name of a channel you want to remove data of.

4.   Click the actions icon  next to the value you want to remove and select Remove reading. A red tag is displayed, indicating that the value has been edited.

5.   Click Save changes. The removed value will be marked by the remove icon . When you hover over this icon the date of removal is shown.

3

Note

Removed values are not removed from the system, but are retained to allow monitoring of changes.
