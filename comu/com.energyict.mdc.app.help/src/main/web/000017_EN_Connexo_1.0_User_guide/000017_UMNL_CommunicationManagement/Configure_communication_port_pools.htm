Configure communication port pools

Add outbound communication port pools

5

Steps

1.   Go to Administration > Device communication > Communication port pools.

2.   Click Add communication port pool > Outbound. Complete the following fields:

Parameter

Description

Name

A unique name

Type

Select a communication type.

3.   Click Add.

4.   Click the actions icon  and select Activate.

Add inbound communication port pools

5

Steps

1.   Go to Administration > Device communication. > Communication port pools.

2.   Click Add communication port pool > Inbound. Complete the following fields:

Parameter

Description

Name

A unique name

Type

Select a communication type.

Protocol detection

Discovery protocol used to detect devices that contact or send data to MultiSense.

3.   Click Add.

4.   Click the actions icon  and select Activate.

Edit communication port pools

5

Steps

1.   Go to Administration > Device communication > Communication port pools.

2.   Click the actions icon  and select Edit.

3.   Change the communication port pool parameters.

4.   Click Save.

3

Note

Once a communication port pool has been configured and saved, the parameter Type can no longer be edited. This is to prevent potential harmful effects on devices that use this port pool.

Remove communication port pools

5

Steps

1.   Go to Administration > Device communication > Communication port pools.

2.   Click the actions icon  and select Remove.

3.   Click Remove.
