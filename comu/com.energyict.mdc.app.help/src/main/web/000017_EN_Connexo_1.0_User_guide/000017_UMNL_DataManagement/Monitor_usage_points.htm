Monitor usage points

View usage point details

Get an overview of the main information about a usage point such as its service category, metrology configuration, linked devices, etc.

5

Steps

1.   Search for a usage point as described in “Search for usage points”.

2.   Click the desired usage point name.

The usage point landing page opens in the tab Overview. It shows the usage point’s active metrology configuration and the currently linked devices in the panel Configuration. The general attributes are shown in the panel Usage point summary.

3.   If you have the necessary privileges to view devices, you can click the linked device(s) to open their landing page.

4.   Click Versions next to a linked device to navigate to the Device tab on the History page of the current usage point. See “View history of linked devices on a usage point”.

5.   Click Versions next to the metrology configuration to navigate to the Metrology configuration tab on the History page of the current usage point. See “View metrology configuration versions of a usage point”.

View history of linked devices on a usage point

Get an overview of all devices that are or have been linked to the usage point during its lifetime. This information allows you to understand from which devices data came to the usage point at different periods in time.

5

Steps

1.   On the desired usage point, click History in the vertical navigation menu.

2.   Click the tab Device (if it is not yet shown).

The history of linked devices is shown in a table sorted by Period. Devices that are currently linked to the usage point are marked in the column Active.

3

Note

Each record in the table represents aggregated meter activations. One device can have multiple meter activations if the multiplier was changed. These meter activations are combined into one single record.

3.   Select a line in the table to view further details in the details pane under the table.

MRID and Device type can be clicked to open their landing pages if you have the needed privileges.

4.   Click View meter activation history to see the Meter activation tab on the device History page. Here, you can see an overview of all meter activations of a device that were created after a change of multiplier, device configuration, linked usage point or active/inactive transition of the device.

View metrology configuration versions of a usage point

A metrology configuration defines the setup of a usage point, for more information see “Understand metrology configuration”.

Get an overview of the metrology configuration versions that are linked with a usage point and manage them. See “Manage metrology configuration versions on a usage point”.

5

Steps

1.   Search for a usage point as described in “Search for usage points”.

2.   Click History in the menu on the left.

3.   Click the tab Metrology configuration.

The available metrology configuration versions are shown in a table sorted by Period.

4.   Select a line in the table to view further details in the details pane under the table such as the list of reading types that are associated with the metrology configuration version on the usage point.

View channels on a usage point

Get an overview of channels that are available on the usage point as determined by the associated metrology configuration. By navigating to the corresponding channel data page you can view the channel’s details and its data.

5

Steps

1.   Search for a usage point as described in “Search for usage points”.

2.   Click Channels in the menu on the left.

The overview table shows all channels linked to this usage point. Channels that were linked to a metrology configuration version of the past are indicated by an icon.

3.   Select a channel in the table, more information is available in the details pane below.

The Device channels parameters shows all channels and their corresponding device that are and were linked to this usage point. It is possible that the channel is no longer available if the device configuration was changed. For each device channel you can see for which period the device was active on the usage point.

4.   Click a channel to view its data and reading qualities.

5.   Specify the desired period for your channel data output in the filter bar on top of the page. You can enter start date and time as well as duration. Don’t forget to click Apply to see the filtered data. Click Clear all to remove your filter(s).

In case there are multiple meter activations with different devices during the specified period, then the data from the different devices is concatenated.

The data is shown in a graph with the interval information (date and time) on the X-axis and the units on the Y-axis. A column chart is used to display volume units such as kWh and m³. A line chart is used in case of flow units such as kW or m³/h. Hovering over a point on the graph shows a tooltip with interval boundaries and measured values. The bars in the graph can have different colours:

Color

Description

Blue

Not validated data

Green

Valid data

Red

Suspect data

Yellow

Informative

Transparent red

Missing data

Grey

Gap between meter activations with different devices during the specified period.

 

Below the graph, the channel data is shown in a table. Suspect values are marked by a red flag. Not validated values are marked by a white flag. Gaps are marked by a grey flag.

Below the reading data table, a reading details panel is shown. It contains two tabs: General and Reading value. General shows information about the interval, reading time, validation status and result. Reading value shows the actual value followed by the validation flag. In case the value is suspect or informative, reading qualities are shown.

You can move back and forth in time or extend and reduce the period for the channel data you are looking at by dragging the handles on the navigator under the graph. Note that the minimum number of shown intervals is seven.

You can easily cycle through channels by clicking the arrows in the upper right corner of the page.

Navigate to a usage point from a device

While looking on a device landing page at the linked usage point, you can click that usage point to see its landing page.

5

Steps

1.   Find a device using the search functionality. See “Search for usage points”.

2.   Click the device to see its landing page.

3.   In the General information, click the usage point value to navigate to its landing page.

3

Note

If you don’t have privileges to view usage points, the usage point will be shown as a label and not as a link.

If Connexo Insight is installed, the link navigates to the usage point landing page in Connexo Insight. If Connexo is not installed, the link navigates to the basic usage point landing page in Connexo MultiSense.
