Usage points

Understand usage points

In the CIM model, usage points represent the point of delivery of a single commodity to a customer. At the usage point, it is possible to view the customer's usage of the commodity over time regardless of the specific meter or device installed at any one point in time.

Usage points are normally managed in Connexo Insight. This chapter describes the basic actions that can be done on usage points in Connexo MultiSense in case there is no valid license available for Connexo Insight.
