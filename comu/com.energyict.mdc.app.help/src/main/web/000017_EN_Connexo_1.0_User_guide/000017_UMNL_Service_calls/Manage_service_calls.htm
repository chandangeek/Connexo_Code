Manage service calls

Cancel a service call

It is possible to cancel a service call to interrupt a long running task. Example: A dispatched read action and 10000 devices are waiting to be read out.

5

Steps

1.   Go to Workspace > Service calls.

2.   Select the service call that you want to cancel.

3.   Click the action icon  in the column Actions next to the selected service call or click the button Actions in the details pane.

4.   Click Cancel to transition the selected service call to the cancel state.

3

Note

The cancel option will only be available if the applied service call type life cycle allows a transition to the cancel state.

Cancel one service call or all service calls on an object

It is possible to cancel one or all running service calls on an object to interrupt long running tasks.

5

Steps

1.   Open an object using Search, e.g. a device.

2.   Click Service calls on the left vertical menu bar.

3.   Select the service call on this object that you want to cancel.

4.   Click the actions icon  in the column Actions next to the desired service call, then click Cancel. The selected service call will be transitioned to the cancel state if the service call type life cycle allows it.

5.   Click the Actions button to the right above the table displaying the service calls, then click Cancel all to transition all ongoing service calls to the cancel state if the service call type life cycle allows it.
