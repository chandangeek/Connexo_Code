Monitor communication topology

View communication topology in a table

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication topology in the menu on the left. The displayed table shows all slave devices of the device you selected together with the device type, configuration and when the device was added.

3

Note

The menu option topology is only available on devices that can act as a gateway.

Filter communication topology

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication topology in the menu on the left.

3.   Complete or select values for one or more filter fields.

4.   Click Filter. Applied filters are shown on top of the results table.

5.   Click the remove icon  to remove single filters or click Clear all to remove all applied filters.
