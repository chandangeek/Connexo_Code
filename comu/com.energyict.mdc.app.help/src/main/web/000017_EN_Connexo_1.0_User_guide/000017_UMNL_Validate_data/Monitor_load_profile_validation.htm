Monitor load profile validation

View validation status of load profile

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Load profiles in the menu on the left.

3.   Select a load profile in the table. The validation details of this load profile are shown in the second part of the details pane of the or on the specification page when you click the load profile name and select the Specifications tab.

The following information is shown:

Parameter

Description

Device validation status

Indicates whether validation on the device is active.

Load profile validation status

Indicates that there are active validation rules available on the device that can validate the load profile’s channel values (matching reading types).

All data validated

Shows if all data is validated.

Suspects

Shows the amount of suspect values per violated validation rule and parameters.

Lasted checked

Moment when the last interval was checked in the validation process.

View validation status of load profile data

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Load profiles in the menu on the left.

3.   Click the load profile name to see the validation status of all values.

4.   Click Table view in the upper right corner to see all values in a table.

Suspect values are marked by a red flag, unvalidated values are marked by a black flag and valid values are not marked. Values with a reading quality are marked with the following icon: .

5.   Select a value in the table, the following validation details are shown in details pane below: validation status of the validation rules, validation status of the data and the result of the validation. If the validated data was found suspect, then you can also see the validation rule and parameters that were violated.
