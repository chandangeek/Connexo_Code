View suspect data

View suspect values from the overview page

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Go to the register, load profile or channel overview page as described in “View register details”, “View load profile details” and “View channel details” respectively.

3.   Click the actions icon  and select View suspects. Suspect values are marked by red flags in the table or using red in the graph.

Filter suspect values

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Go to the register, load profile or channel overview page as described in “View register details”, “View load profile details” and “View channel details” respectively.

3.   Click the actions icon  and select View data.

4.   Click Suspects in the filter menu on top of the page.

5.   Click Apply.
