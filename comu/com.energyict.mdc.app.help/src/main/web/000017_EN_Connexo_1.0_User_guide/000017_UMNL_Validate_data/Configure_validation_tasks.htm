Configure validation tasks

Add validation tasks

5

Steps

1.   Go to Administration > Validation > Validation tasks.

2.   Click Add validation task. Fill in the following fields:

 

Parameter

Description

 

Name

Unique name without special characters and no longer than 256 characters.

 

Log level

Define the log level needed for the execution of the validation tasks. Options are Error, Warning, Information, Debug and Trace. Where the level error only creates loggings if errors occur and trace gives a detailed description of all executions.

Warning is the default log level.

Data source

oDevice group

oSelect the device group of which you want to schedule the validation.

Schedule

oRecurrence

Define the schedule of the validation task.

oStart on

Define the date and time when the validation task should become active.

3.   Click Add.

Edit validation task

5

Steps

1.   Go to Administration > Validation > Validation tasks.

2.   Click the actions icon  and select Edit.

3.   Edit the validation task parameters and click Save.

Remove validation task

5

Steps

1.   Go to Administration > Validation > Validation tasks.

2.   Click the actions icon  and select Remove.

3.   Click Remove.
