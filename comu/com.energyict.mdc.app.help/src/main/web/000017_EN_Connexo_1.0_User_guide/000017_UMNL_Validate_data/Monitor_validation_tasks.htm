Monitor validation tasks

View validation tasks history

5

Steps

1.   Go to Administration > Validation > Validation tasks.

2.   Click the actions icon  next to the task of which you want to see the history and select View history. Another way to view the history is to select the task in the overview table and then select History in the menu on the left.

The history screen lists all the times the validation task was executed, the result and the duration of the task.

View validation task logging

5

Steps

1.   Go to Administration > Validation > Validation tasks > select the validation task > click actions icon and select View history.

2.   Click the actions icon  next to the history item of which you want to see the logging and select View log.

The log page shows the details of the selected validation task on top of the page. Below a table is shown containing all task loggings with some details. More details are given in the details pane below the table.
