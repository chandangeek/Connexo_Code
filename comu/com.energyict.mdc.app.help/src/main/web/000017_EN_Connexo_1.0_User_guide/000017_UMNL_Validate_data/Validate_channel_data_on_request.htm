Validate channel data on request

Validate channel data on request from the overview page

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channels in the menu on the left

3.   Click the actions icon  of a channel and select Validate now.

4.   Select the start date for validation of the channel data and click Validate.

3

Note

The option Validate now is only available when validation on the device is active and validation rules have been added. See “Manage validation status on a device” for more information.

Validate channel data on request from the details page

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channels in the menu on the left

3.   Click the channel name.

4.   Select the Specifications tab.

5.   Click the actions button  and select Validate now.

6.   Select the start date for validation of the channel data and click Validate.

3

Note

The option Validate now is only available when validation on the device is active and validation rules have been added. See “Manage validation status on a device” for more information.
