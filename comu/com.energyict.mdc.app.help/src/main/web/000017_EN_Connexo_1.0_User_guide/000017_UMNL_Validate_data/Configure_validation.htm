View all configured validation rules

5

Steps

1.   Go to Administration > Validation.

2.   Click Validation rule sets.

On top of the page you see all configured validation rule sets.

In the middle you the validation rules that belong to the validation rule set that is selected in the first table.

Under the validation rules table you can see the details of the validation rule you selected in the table.

3.   Click a validation rule set.

On the validation rule sets page three items are shown in the menu on the left:

Menu item

Description

Overview

Shows the name, description and the amount of active and inactive rule in this set.

Validation rules

Shows a list of all validation rules included in this set and the possibility to add new rules.

Device configurations

Shows a list of all device configurations linked to this rule set and the possibility to add device configurations.
Figure: Validation rule sets homepage.

 

View validation rules on a device configuration

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration.

2.   Click Validation rule sets in the menu on the left to see all rule sets assigned to this device configuration.

These validation rule sets are used to validate all data that is generated from devices with this configuration.

3

Note

When you go to the register, and load profile configuration pages the used validation rules are also shown here.

View validation rules on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Data validation to see the validation rules on this device.

On top op the page, the status of the validation is shown and can be changed by clicking the button on the upper right corner.

On the bottom of the page the used validation rule sets and their validation rules are shown.
