Task management

Monitor tasks

View all tasks

5

Steps

1.   Go to Workspace > Task management > Tasks, this view shows all tasks that are available in the system.

2.   Click a task to more information in the details pane below.

Filter tasks

5

Steps

1.   Go to Workspace > Task management > Tasks or click My open tasks, Overdue tasks or Unassigned tasks to see views where already some filtering was applied.

2.   Create a custom filter by selecting one or more filter criteria.

3.   Click Apply.

4.   Remove single filters by deselecting items in the drop down menu or remove all filters by click Clear all.

View tasks on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Processes in the menu on the left.

3.   Select a process in the table, process details are shown in the details pane below. In these details a list of all tasks is shown. Click the task link to see the task details.
