Perform tasks

Perform an individual task

5

Steps

1.   Go to Workspace > Task management > Tasks or select one of the other pre-filtered views.

2.   Select a task.

3.   Click the action icon  and select Perform task.

4.   Complete the required parameters and click Start to start the task. When started, you have the option to complete or save the task.

3

Note

When clicking a task from the details pane of a process on an object, you are directed to the task performance window. There, you can perform the task.

Perform bulk actions on tasks

5

Steps

1.   Go to Workspace > Task management > Tasks or select one of the other pre-filtered views.

2.   Filter tasks if needed.

3.   Click Bulk actions.

4.   In step one select the tasks for the bulk action.

All tasks means all the tasks from the task list on the previous screen. If you had filtered this list, then this filter still applies. Select Selected tasks to make a custom selection of tasks in a table.

5.   Click Next.

6.   In step two you need to select the type of action. Select Task management to manage the selected tasks and change assignees, due date and priority or select Task execution to execute the selected tasks.

7.   Click Next.

8.   In step three specify the parameters for the action selected in step two. This can for example be the assignee for task management or specific task details that are needed to execute the task.

9.   Click Next.

10.   Confirm the bulk action in step four by clicking Confirm.

11.   Step five shows the result of the action.

12.   Click Finish.
