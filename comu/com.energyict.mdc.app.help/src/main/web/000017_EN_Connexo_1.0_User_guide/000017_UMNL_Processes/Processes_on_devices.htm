Processes on devices

Monitor processes on a device

View running processes on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Processes in the menu on the left.

3.   Make sure that the tab Running processes is selected.

This overview shows all the processes that are currently running on the device. In the Details you can see when and who started the process, status, version and the tasks linked to this process. You can click the task to perform it. See “Perform tasks”.

4.   Click the tab Status overview to see all the steps the process went through along with any applicable variables used during these steps.

View process history on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Processes in the menu on the left.

3.   Select the tab History.

This overview shows all the processes that have been executed and ended on the device. In the Details you can see when and who started the process, status, version and end date.

4.   Click Status overview to see all the steps the process went through along with any applicable variables used during these steps.

5.   Use the filter on top of the screen to see only specific processes. Click Apply to execute the filter.

6.   Remove filters by deselecting items in the drop down menu or by clicking Clear all. Don’t forget to click Apply to execute the changes in your filter.
