Monitor multi-element history

View history of linked multi-element slaves on multi-element master channel

5

Steps

1.   Select a multi-element device using Search, see “Advanced search”.

2.   Go to Data sources > Channels in the menu on the left. Click a channel name.

3.   Select the Specifications tab.

On the bottom of the page you can find Multi-element slave history. Here you can find which multi-element slave is linked to this channel and when. A link to the slave is available.

View history of linked multi-element slaves on multi-element master register

5

Steps

1.   Select a multi-element device using Search, see “Advanced search”.

2.   Go to Data sources > Registers in the menu on the left. Click a register name.

3.   Select the Specifications tab.

On the bottom of the page you can find Multi-element slave history. Here you can find which slave is linked to this register and when. A link to the slave is available.
