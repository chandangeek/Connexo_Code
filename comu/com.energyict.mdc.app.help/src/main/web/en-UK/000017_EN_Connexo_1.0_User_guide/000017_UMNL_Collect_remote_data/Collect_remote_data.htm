Collect remote data

With Connexo MultiSense you can easily add and configure multiple devices while limiting the amount of parameters that need configuration.

What would you like to do?

SET up communication infrastructure

1. Follow the Quick guide to collect data.

2. Understand communication in Connexo MultiSense.

3. Edit communication protocols if necessary.

4. Configure communication servers

5. Configure communication ports of the server.

6. Configure communication port pools.

7. Configure communication tasks to define the tasks that need be executed.

Solve connection failures

1. Go to the The dashboard to monitor connection failures.

2. Go to Connection overview to see more details.

3. Filter connections to only see the failed connections.

4. Retry connection to solve the failure.

5. Go back to the Connection overview to see the result.
