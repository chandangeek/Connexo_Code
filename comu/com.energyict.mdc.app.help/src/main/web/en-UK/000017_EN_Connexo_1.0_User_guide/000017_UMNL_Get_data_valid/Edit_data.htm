Add register data

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Registers in the menu on the left.

3.   Click the action icon  of the register you want to add data to and select View data.

4.   Click Add reading above the table. Complete the following fields (depending on the type of register):

Parameter

Description

Measurement time

Time when data was recorded.

Start of period

Start of the billing period (for billing registers only)

End of period

End of the billing period (for billing registers only)

Value

Value of the reading, numerical or text depending on the register.

5.   Click Add. The value will be added to the register and marked by the edit icon . When you hover over this icon the date of editing is shown.

Edit register data

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Registers in the menu on the left.

3.   Click the action icon  of the register you want to add data to and select View data or View suspects to only see suspect values.

4.   Click the actions icon next to the register value you want to edit and select Edit.

5.   Write the new value and click Save. The edited value will be marked by the edit icon . When you hover over this icon the date of editing is shown.

Remove register data

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Registers in the menu on the left.

3.   Click the action icon  of the register you want to add data to and select View data or View suspects to only see suspect values.

4.   Click the actions icon next to the register value you want to edit and select Remove.

5.   A warning message appears to confirm the removal, click Ok.

3

Note

Removed values are not deleted from the system, but are retained to allow monitoring of changes.
