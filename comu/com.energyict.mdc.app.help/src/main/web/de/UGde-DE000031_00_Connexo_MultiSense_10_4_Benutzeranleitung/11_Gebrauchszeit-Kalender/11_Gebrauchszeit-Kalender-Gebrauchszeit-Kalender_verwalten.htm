Gebrauchszeit-Kalender verwalten

Gebrauchszeitkalender an Gerätetypen konfigurieren

Einen Gebrauchszeitkalender zum selektierten Gerätetypen hinzufügen

Bevor Sie damit beginnen, die Gebrauchszeitkalender an Geräten zu verwenden, müssen sie einem Gerätetypen zugewiesen sein.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätetypen > Spezifikationen> Gebrauchszeitkalender.

2.   Klicken Sie im Karteireiter ToU Kalender auf die Schaltfläche ToU-Kalender hinzufügen.

3.   Selektieren Sie auf der Seite Gebrauchszeitkalender hinzufügen die gewünschten Gebrauchszeitkalender.

4.   Klicken Sie auf Hinzufügen um die selektierten Gebrauchszeitkalender zum aktuellen Gerätetypen hinzuzufügen.

Löschen eines Gebrauchszeitkalenders aus einem Gerätetypen

Sie können einen Gebrauchszeit-Kalender löschen, wenn er nicht mehr nötig ist.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätetypen > Spezifikationen> Gebrauchszeitkalender.

2.   Selektieren Sie im Karteireiter ToU Kalender einen Gebrauchszeitkalender.

3.   Klicken Sie auf das Aktionssymbol  in der selektierten Zeile oder klicken Sie auf die Aktionstaste im Detailfenster. Klicken Sie dann auf Löschen.

3

Anmerkung

Der Gebrauchszeitkalendar kann bei Bedarf erneut hinzugefügt werden. Wenn er keinem Gerätetypen mehr zugewiesen ist, kann der Gebrauchszeitkalender aus Connexo in Connexo Admin gelöscht werden.

Gebrauchszeitspezifikationen bearbeiten

Sie können die Spezifikationen für die Gebrauchszeitkalender mit dem aktuellen Gerätetypen anzeigen und bearbeiten.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätetypen > Spezifikationen> Gebrauchszeitkalender.

2.   Klicken Sie auf den Karteireiter Spezifikationen.

3.   Klicken Sie auf die AktionsTaste und selektieren Sie Bearbeiten, wenn Sie die Gebrauchszeitspezifikationen bearbeiten möchten.

Parameter

Beschreibung

Zugelassene Gebrauchszeit

Der Standardwert ist Nein.

 

Wenn die Gebrauchszeit NICHT gestattet ist:

  • Der Karteireiter Spezifikationen ist der Standardkarteireiter.

  • Die Taste ToU Kalender einfügen ist deaktiviert.

  • Handlungen für Gebrauchszeitkalender wie Vorschau anzeigen und Löschen sind deaktiviert.

  • An Geräten des aktuellen Gerätetypen gibt es kein Gebrauchszeit-Menüobjekt.

 

Wenn die Gebrauchszeit gestattet ist:

  • Der Karteireiter ToU-Kalender ist der Standardkarteireiter.

  • Gebrauchszeitkalender können hinzugefügt, vorher angesehen und gelöscht werden.

  • Es gibt ein Gebrauchszeit-Menüobjekt an Geräten des aktuellen Gerätetypen.

Gebrauchszeit-Optionen

Die verfügbaren Gebrauchszeitoptionen werden im Protokoll definiert, das vom Gerätetypen verwendet wird. Sie können mehrere Gebrauchszeitoptionen auswählen, indem Sie die entsprechenden Kästchen ankreuzen.
