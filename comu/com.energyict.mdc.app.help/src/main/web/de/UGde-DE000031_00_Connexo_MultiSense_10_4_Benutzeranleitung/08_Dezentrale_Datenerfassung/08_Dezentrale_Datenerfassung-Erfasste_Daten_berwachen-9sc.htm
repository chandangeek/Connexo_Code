Ablesequalität verstehen

Die Ablesequalität bedeutet, dass es etwas Bemerkenswertes mit dem Wert gibt. Es könnte beispielsweise sein, dass ein Netzausfall aufgetreten ist oder dass Daten geändert oder manipuliert wurden. Diese Ablesequalitäten basieren auf CIM-Normen und bestehen aus drei verschiedenen Feldern.

  • Das erste Feld ist der Systemidentifikator und gibt an, welches System das Problem gemeldet hat. Es gibt die folgenden Möglichkeiten:

    Code

    Beschreibung

    Name in MultiSense

    0

    Nicht anwendbar

    Nicht verfügbar

    1

    Endgerät

    Gerätequalität

    2

    Messsystem (Datenerfassung)

    MDC-Qualität

    3

    Datenmanagementsystem

    MDM-Qualität

    4

    Anderes System (nicht aufgeführt)

    Qualität der Drittanbieter

    4

    Extern angegeben (siehe begleitende Daten)

    Qualität der Drittanbieter

  • Das zweite Feld ist die Kategorisierung und gibt die Kategorie des Problems an.

  • Das dritte Feld ist der Index und jede Kategorie bietet den Kontext, in dem ein bestimmter Index zu verstehen ist.

Sie sehen die Beschreibung des Ablesequalitätscodes, wenn Sie die Maus über das Informationssymbol halten.

Ein mutmaßlicher Wert kann die Ablesequalität 2.5.258 haben:

  • 2: Es betrifft ein Endgerät.

  • 5: fehlerhafter Zumutbarkeitstest, Daten erscheinen auf den ersten Blick ungültig.

  • 258: Daten sollten als suspekt betrachtet werden.
