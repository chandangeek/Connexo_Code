Kommunikation an einem Gerät konfigurieren

Kommunikationspläne an einem Gerät anzeigen

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kommunikationsplanung im Menü auf der linken Seite.

Die Übersicht zeigt die folgenden Informationen:

Parameter

Beschreibung

Übertragungsaufgabe

Kommunikationsaufgabe, verfügbar an diesem Gerät.

Planen

Welches ist der Plan dieser Kommunikationsaufgabe. Kann spezifisch für dieses Gerät oder Teil eines geteilten Kommunikationsplans sein.

Gemeinsamer Übertragungszeitplan

Dieser Kommunikationsaufgabe zugewiesener geteilter Kommunikationsplan.

Geplantes Datum

Wenn die Kommunikationsaufgabe als nächstes ausgeführt wird.

Einen geteilten Kommunikationsplan zu einem Gerät fügen

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kommunikationsplanung im Menü auf der linken Seite.

3.   Klicken Sie auf Geteilten Kommunikationsplan hinzufügen.

Diese Liste enthält nur geteilte Kommunikationspläne, die keine Kommunikationsaufgaben enthalten, die bereits unter Verwendung eines anderen geteilten Kommunikationsplans an diesem Gerät geplant sind.

4.   Selektieren Sie einen oder mehrere Kommunikationspläne.

5.   Klicken Sie auf Einfügen.

6.   Klicken Sie das Aktionssymbol , um einen geteilten Kommunikationsplan zu löschen.

3

Anmerkung

Alle Aufgaben in einem Plan müssen dieselbe Verbindungsmethode, Protokolldialekt, Sicherheitseinstellung und Dringlichkeit  haben.

Einen geteilten Kommunikationsplan von einem Gerät löschen

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kommunikationsplanung im Menü auf der linken Seite.

3.   Klicken Sie auf Geteilten Kommunikationsplan löschen.

4.   Selektieren Sie einen oder mehrere geteilte Kommunikationspläne.

5.   Klicken Sie auf Löschen.

Fügen Sie einen Plan zu einer Kommunikationsaufgabe in der Kommunikationsplanung

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kommunikationsplanung im Menü auf der linken Seite.

3.   Klicken Sie auf das Aktionssymbol neben der Kommunikationsaufgabe

4.   Klicken Sie auf Plan hinzufügen.

5.   Definieren Sie den Plan und klicken Sie auf Hinzufügen.

Einen Plan von einer Kommunikationsaufgabe in der Kommunikationsplanung ändern

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kommunikationsplanung im Menü auf der linken Seite.

3.   Klicken Sie auf das Aktionssymbol neben dem Kommunikationsplan

4.   Klicken Sie auf Plan ändern.

5.   Definieren Sie den Plan und klicken Sie auf Hinzufügen.

Einen Plan von einer Kommunikationsaufgabe in der Kommunikationsaufgabe löschen

Mit dieser Handlung können Sie einen einzigen Plan löschen, der einer bestimmten Kommunikationsaufgabe an einem Gerät zugewiesen war.

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kommunikationsplanung im Menü auf der linken Seite.

3.   Klicken Sie auf das Aktionssymbol neben dem Kommunikationsplan

4.   Klicken Sie auf Plan löschen.

5.   Bestätigen Sie und klicken Sie auf Löschen.
