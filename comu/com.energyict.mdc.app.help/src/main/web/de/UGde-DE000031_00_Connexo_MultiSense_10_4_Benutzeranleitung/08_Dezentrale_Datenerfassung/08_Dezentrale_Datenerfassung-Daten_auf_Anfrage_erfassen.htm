Dringlichkeit einer Kommunikationsaufgabe ändern

1

Important

Nur die Dringlichkeit geplanter Kommunikationsaufgaben kann geändert werden.

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kommunikationsaufgaben im Menü auf der linken Seite.

3.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Dringlichkeit ändern. Wenn der Kommunikationsaufgabe ein geteilter Kommunikationsplan zugewiesen ist, wird die Dringlichkeit aller Kommunikationsaufgaben, die diesen Plan nutzen, geändert.

4.   Geben Sie einen neuen Wert ein und klicken Sie auf Speichern.

Ausführung einer Kommunikationsaufgabe auf Anfrage aus der Kommunikationsaufgabenübersicht

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kommunikationsaufgaben im Menü auf der linken Seite.

3.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Start oder Jetzt starten.

  • Selektieren Sie Jetzt starten, wenn Sie die Aufgabe sofort ausführen möchten. Das ist nur möglich, wenn die Verbindungsstrategie der Verbindungsmethode schnellstmöglich ist. In diesem Fall wird eine neue Verbindung eingerichtet.

  • Selektieren Sie Starten, wenn die Aufgabe beim nächsten Mal, wenn eine Verbindung hergestellt wird, entsprechend der Verbindungsstrategie und –planung ausgeführt werden kann.

Start einer Kommunikationsaufgabe auf Anfrage aus der Kommunikationsplanung

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kommunikationsplanung im Menü auf der linken Seite.

3.   Klicken Sie auf das Aktionssymbol  neben der Kommunikationsaufgabe Klicken Sie Start oder Jetzt starten.

Vergangenheit der Kommunikationsaufgabe anzeigen

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kommunikationsaufgaben im Menü auf der linken Seite.

3.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Vergangenheit anzeigen.

Die Vergangenheitsseite zeigt eine Übersicht aller ausgeführten Instanzen der Kommunikationsaufgabe, gemeinsam mit dem Ergebnis. Im unteren Teil der Seite finden Sie weitere Details der Kommunikationsaufgabe und eine detaillierte Beschreibung der verwendeten Verbindungsmethode.

4.   Klicken Sie auf das Aktionssymbol  oder die Aktionstaste, um das Kommunikations- oder Verbindungslog anzuzeigen. Weitere Informationen finden Sie in “Monitor connections” und “Monitor communication”.
