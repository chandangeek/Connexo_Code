Dezentrale Datenerfassung

Übersicht

Mit Connexo MultiSense können Sie einfach mehrere Geräte hinzufügen und konfigurieren, während die Anzahl der Parameter, die Konfiguration benötigen, beschränkt ist. Dieses Kapitel deckt die Funktionen für die Planung von Datenerfassung und Erfassung auf Anfrage ab. Wir erklären auch, wie Sie Ihre erfassten Daten überwachen können.

Was möchten Sie tun?

Kommunikationsinfrastruktur einstellen

1. Folgen Sie den Quick guide to collect data.

2. Understand communication in Connexo MultiSense.

3. Edit communication protocols wenn nötig.

4. Configure communication servers

5. Configure communication ports des Servers.

6. Configure communication port pools.

7. Configure communication tasks (Administration) zur Definition der Aufgaben, die aus­geführt werden müssen.

Verbindungsfehler beseitigen

In MultiSense gibt es verschiedene Möglichkeiten, Verbindungsfehler zu erfassen und zu überwachen. Die erste Möglichkeit ist es das Instrumentenbrett zu verwenden.

1. Öffnen Sie Alle Daten werden in eine CSV-Datei exportiert., um die Verbindungsfehler zu überwachen.

2. Öffnen Sie Connection overview, um weitere Details zu sehen.

3. Filter connections um nur die Verbindungsfehler anzuzeigen.

4. Versuchen Sie die Verbindung erneut, um den Fehler zu beseitigen.

5. Öffnen Sie erneut Connection overview, um das Ergebnis zu sehen.

 

Eine zweite Möglichkeit für die Ermittlung von Problemen mit Verbindungen ist es Prob­leme zu überwachen, die entstehen, wenn Verbindungsfehler auftreten. Für weitere Infor­mationen, siehe Issue management.

 

Die dritte Möglichkeit ist es die Verbindung eines bestimmen Geräts über die Zielseite des Messgeräts/Gateways dieses Geräts zu überwachen

1. Search for devices um das Gerät zu finden.

2. Im oberen Teil der Übersichtsseite dieses Geräts werden alle Verbindungen angezeigt. Es gibt eine Möglichkeit hier verschiedene Handlungen auszuführen: jetzt starten, (De)Aktivieren und Vergangenheit anzeigen.

Kommunikationsplanung Nachsorge

1. Add device und spezifizieren Sie Attribute gemeinsam mit Gerätetyp und Konfiguration. Das Gerät übernimmt alle Konfigurationsparameter des Geräts.

2. View communication tasks on a device und prüfen, ob die Verbindungsmethode kor­rekt ist, wenn nicht Edit connection method on a device.

3. Ausführung einer Kommunikationsaufgabe auf Anfrage aus der Kommunikationsauf­gabenübersichtoder Start einer Kommunikationsaufgabe auf Anfrage aus der Kommu­nikationsplanung.

4. Oder fügen Sie Kommunikationsaufgaben hinzu, die Sie planen, wenn Sie Geteilte Kom­munikationsplan/pläne konfigurieren und dann Geteilte Kommunikationspläne an mehreren Geräten hinzufügen/löschen.

5. Kommunikationspläne an einem Gerät anzeigen.

Ein Messgerät verbinden/trennen

Zuerst müssen Sie das Gerätekommando konfigurieren:

1. Add communication tasks und die Kommandokategorie des Vertragspartners zur Kom­munikationsaufgabe fügen.

2. Add communication task configuration on device configuration um Kommandos hin­zufügen zu können.

3. Change privileges of commands und (De)Activate commands, damit diese für ein Gerät verwendet werden können.

 

In den nächsten Schritten müssen Sie das Kommando zum Gerät fügen und die Kommu­nikation starten:

1. Search for devices um das Gerät zu finden, dass getrennt/verbunden werden muss.

2. Add commands to a device und selektieren Sie das Öffnen- oder Schließen-Kommando in der Vertragspartner-Kategorie.

3. Trigger communication for a command oder warten Sie, bis die Kommunikationsauf­gabe von einer Verbindung aufgenommen wird.

Rechte für bestimmte Sicherheitseinstellungen und Kommandos an­zeigen/bearbeiten

Rechte anzeigen und bearbeiten wird von Privilegien bestimmt. Alle Privilegien in Connexo MultiSense werden in der Systemverwaltungsanwendung konfiguriert Admin:.

In der MultiSense Anwendung können Sie sicherstellen, dass nur bestimmte Anwender Zu­gang zu empfindlichen Daten haben oder gefährliche Handlungen ausführen. Das wird er­reicht, wenn Sie Change key security ac­cessor privileges und Change privileges of commands.
