Firmwareversionen verwalten

Firmwareversionen verstehen

Eine Firmwareversion kann einen der folgenden vier Status haben: Test, Definitiv, Ghost oder Veraltet. Eine Version kann den Status Test haben, wenn Sie die neue Firmware bei einer geringen Menge an Geräten testen müssen, um zu prüfen, ob alles fehlerlos läuft. Wenn die Testversion erfolgreich getestet wurde, können Sie sie in Definitiv erhöhen. Dann können Sie diese Version bei einer größeren Anzahl an Geräten installieren.

Ghost Versionen werden von Connexo kreiert, wenn bei einem Gerät eine Firmwareversion installiert wurde, die noch nicht in MultiSense besteht. Diese Firmware-Version kann zu Test oder Definitiv erhöht werden, wenn Sie die entsprechende Firmware-Datei hochladen.

Wenn eine Version nicht mehr benötigt wird oder es einen Fehler in der Firmware-Version gibt, können Sie dieser Version den Status  Veraltet geben. Diese Version kann nicht mehr bei Geräten geladen werden. Eine veraltete Firmwareversion wird in der Übersichtstabelle angezeigt, aber die Firmware-Datei dieser Version wird gelöscht.

3

Anmerkung

Ghost oder veraltete Firmwareversionen können nicht installiert werden.

Firmware-Versionen anzeigen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetypen > und selektieren Sie einen Gerätetyp > Firmware > Firmwareversionen.

Die Tabelle zeigt alle Firmwareversionen, die zum Gerätetypen gefügt wurden und zum Hochladen verfügbar sind. Für jede Version wird der Firmwaretyp und der Status der Firmware angezeigt.

Firmware-Versionen filtern

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetypen > und selektieren Sie einen Gerätetyp > Firmware > Firmwareversionen.

Sie können Firmwareversionen unter Verwendung verschiedener Parameter filtern. Wenn das Kommunikationsprotokoll nur einen der Firmwaretypen gestattet, wird der Typenfilter nicht angezeigt.

2.   Geben Sie die gewünschten Filter an.

3.   Klicken Sie auf Anwenden. Die angewendeten Filter werden im oberen Teil der Seite angezeigt.

4.   Heben Sie die Auswahl von Items im Drop Down Menü auf, um einzelne Filter zu löschen oder klicken Sie auf Alle löschen , um alle Filter zu löschen. Vergessen Sie nicht auf Anwenden zu klicken, um die Änderungen in Ihrem Filter auszuführen.

Firmware-Version hinzufügen

Sie können eine neue Kommunikations-Firmwareversion oder Messgeräte-Firmwareversion an einem Gerätetypen hinzufügen, um Geräte, die eine veraltete Firmwareversion haben, vor Ort zu aktualisieren. Sie können auch die neue Firmware-Version als Testversion kennzeichnen, um sie in einem Messlabor zu verwenden und nicht zum Aktualisieren von Geräten vor Ort.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetypen> selektieren Sie einen Gerätetypen > Firmware > Firmwareversionen.

2.   Klicken Sie auf Firmwareversion hinzufügen. Füllen Sie die folgenden Felder aus:

Parameter

Beschreibung

Firmware-Datei

Hochladen der Datei mit der Firmwareversion.

Version

Die Versionsangabe der Firmware sollte für das Gerät einzigartig sein.

Abbildungsidentifikator

Je nach Protokoll muss ein Abbildungsidentifikator angegeben werden. Diesen Abbildungsidentifikator gibt es auch am Messgerät. Nur wenn beide Abbildungsidentifikatoren übereinstimmen, wird die Firmwareversion zur Installation verfügbar.

Firmware-Typ

Wenn das Kommunikationsprotokoll nur einen der Firmwaretypen gestattet, wird nur der Firmwaretyp angezeigt.

o Messgerätfirmware: enthält alle Messwerte

  • Kommunikations-Firmware: enthält alle Funktionen, die benötigt werden, um die Kommunikation zwischen dem Gerät und dem Messwert-Datenerfassungssystem.

Firmware-Status

o Definitiv: Verwenden Sie diesen Status, wenn die Firmwareversion getestet wurde und bereit ist, bei vielen Geräten hochgeladen zu werden.

o Test: Verwenden Sie diesen Status, wenn Sie neue Firmwareversionen testen. Das verhindert, dass Sie eine Firmwareversion hochladen, die noch immer Fehler enthält.

3.   Klicken Sie auf Einfügen.

Firmwareversionen bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetypen > und selektieren Sie einen Gerätetyp > Firmware > Firmwareversionen.

2.   Klicken Sie auf das Aktionssymbol  neben der Firmwareversion und selektieren Sie Bearbeiten.

Wenn Sie eine Firmwareversion bearbeiten, können Sie eine neue Firmwaredatei hochladen, die neue Firmwaredatei ersetzt dann die bestehende.

Wenn der aktuelle Status der Firmwareversion Ghost ist, können Sie den Status der Version in Test oder Definitiv ändern, aber nicht den Namen der Version. Wenn der Status Test oder Definitiv ist, können Sie den Namen der Version, aber nicht den Status ändern. Veraltete Versionen können nicht mehr bearbeitet werden.

Firmwareversion als Definitiv einstellen

Eine Firmware-Version mit dem Status Definitiv kann verwendet werden, um Geräte vor Ort zu aktualisieren.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetypen > und selektieren Sie einen Gerätetyp > Firmware > Firmwareversionen.

2.   Klicken Sie auf das Aktionssymbol  neben einer Testfirmware und selektieren Sie Als Definitiv einstellen.

Veraltete Firmware-Versionen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetypen > und selektieren Sie einen Gerätetyp > Firmware > Firmwareversionen.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Veraltet.

Wenn die Firmwareversion veraltet ist, wird sie noch immer in der Tabelle angezeigt, kann aber nicht mehr auf Geräte geladen werden.
