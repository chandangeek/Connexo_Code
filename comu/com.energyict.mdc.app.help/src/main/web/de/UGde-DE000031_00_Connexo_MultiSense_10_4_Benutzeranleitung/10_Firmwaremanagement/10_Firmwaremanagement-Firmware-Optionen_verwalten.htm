Firmware-Optionen verwalten

Firmwaremanagement-Optionen verstehen

Firmwaremanagement-Optionen bestimmen, ob es möglich ist, Firmware hochzuladen und wie sie aktiviert wird. Welche Optionen verfügbar sind, wird vom Kommunikationsprotokoll definiert, welches der Gerätetyp verwendet. Sie können diese Firmwareeinstellungen am Gerätetypen weiter bearbeiten.

Es gibt drei Firmwaremanagement-Optionen. Wenn Sie die Firmware-Version hochgeladen haben, haben Sie die Möglichkeit die Firmware später, sofort oder an einem Aktivierungsdatum zu aktivieren. Bei der Option später aktivieren, müssen Sie die Firmware später aktivieren. Sie können dies manuell tun oder einen Befehl verwenden. Wenn Sie sich entscheiden, die Firmware mit einem Aktivierungsdatum zu aktivieren, müssen Sie das Datum angeben, an dem die Firmware aktiviert werden muss.

Beim Hochladen einer Firmwareversion wird eine Kommunikationsaufgabe verwendet und wie vorher beschrieben, wird von der Verbindungsstrategie der Verbindungsmethode der Kommunikationsaufgabe bestimmt, wann diese Kommunikationsaufgabe ausgeführt wird, siehe “How connections and communication work in Connexo” für weitere Informationen.

Wenn ein Firmware-Upgrade an einem Gerät aufgrund einer schlechten Verbindung fehlerhaft ist, erfasst ein integrierter Resume-Mechanismus den ersten fehlenden Block und versucht erneut alles zum Gerät zu schicken, beginnend mit diesem Block.

Firmwaremanagement-Optionen anzeigen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetypen > und selektieren Sie einen Gerätetyp > Firmware > Firmwaremanagementoptionen.

2.   Diese Seite zeigt die folgenden Informationen:

Parameter

Beschreibung

Zugelassenes Firmwaremanagement

Gibt an, ob ein Hochladen aus der Ferne möglich ist.

Firmware-Verwaltungsoptionen

Zeigt, welche Optionen möglich sind, um Firmware auf diesen Gerätetypen hochzuladen.

Firmwaremanagement-Optionen bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetypen > und selektieren Sie einen Gerätetyp > Firmware > Firmwaremanagementoptionen.

2.   Klicken Sie auf Bearbeiten. Selektieren Sie die folgenden Felder:

Parameter

Beschreibung

Zugelassenes Firmwaremanagement

Zeigt an, ob es möglich sein sollte, Firmware aus der Ferne hochzuladen.

Firmware-Verwaltungsoptionen

Selektierten, welche Optionen für diesen Gerätetypen möglich sind. Die verfügbaren Optionen werden vom Kommunikationsprotokoll definiert, welches der Gerätetyp verwendet.

3.   Klicken Sie auf Speichern.

3

Anmerkung

Das vom Gerätetyp verwendete Kommunikationsprotokoll bestimmt, ob es möglich ist, die Upgradeoptionen zu bearbeiten. Wenn das nicht möglich ist, ist die Schaltfläche Bearbeiten deaktiviert.
