Daten des Datenaufzeichnungsgeräts überwachen

Kanaldaten am Datenaufzeichnungsgerät anzeigen

5

Schritte

1.   Selektieren Sie ein Datenaufzeichnungsgerät mit Suche, siehe “Advanced search”.

2.   Öffnen Sie Datenquellen >Kanäle im Menü auf der linken Seite.

3.   Selektieren Sie den Karteireiter Ablesungen.

4.   Siehe “Monitor channel data” für weitere Informationen, wie die Kanaldaten angezeigt werden können.

Wenn das Datenaufzeichnungsgerät mit einem Datenaufzeichnungsnebengerät verlinkt ist, sind die Balken der Daten in der Grafik grau. Wenn Sie die Maus über die Daten halten, zeigt das Pop-up-Menü die Nebengeräte, die mit diesen Daten verlinkt sind. Wenn Sie einen Wert in der Tabelle mit allen Ablesungen selektieren, finden Sie einen Link zum Datenaufzeichnungsgerät der Nebengeräte im Karteireiter Allgemein im Detailfenster im oberen Teil der Seite.

Registerdaten am Datenaufzeichnungsgerät anzeigen

5

Schritte

1.   Selektieren Sie ein Datenaufzeichnungsgerät mit Suche, siehe “Advanced search”.

2.   Öffnen Sie Datenquellen >Register im Menü auf der linken Seite.

3.   Selektieren Sie den Karteireiter Ablesungen.

4.   Siehe “Monitor register data” für weitere Informationen, wie die Registerdaten angezeigt werden können.

Wenn Sie einen Wert in der Tabelle mit allen Ablesungen selektieren, finden Sie einen Link zum Datenaufzeichnungsgerät der Nebengeräte im Karteireiter Allgemein im Detailfenster im oberen Teil der Seite.
