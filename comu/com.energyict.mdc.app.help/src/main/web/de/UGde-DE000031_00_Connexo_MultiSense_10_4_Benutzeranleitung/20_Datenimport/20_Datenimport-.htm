Datenimport

Wenn Sie Importieren möchten, erklärt dieses Kapitel alle Eigenschaften in MultiSense, wie Importservices, die konfiguriert werden müssen, sodass Sie Ihre Daten importieren können. Im Kapitel “Manage device groups” werden die Gerätegruppen definiert, deren Daten importiert werden müssen.

Was möchten Sie tun?

Datenexport

1. Add a static device group oder Add a dynamic device group um alle Geräte zu grup­pieren, die zusammen angezeigt werden sollten.

2. Add export task for device readings um zu definieren, wann Daten exportiert werden und in welchem Format.

3. Schedule data export in the data export task configurationoder Export on request.
