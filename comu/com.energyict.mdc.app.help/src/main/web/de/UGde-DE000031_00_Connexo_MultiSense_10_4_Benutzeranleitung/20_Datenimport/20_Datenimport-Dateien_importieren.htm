Dateien importieren unter Verwendung von Importverzeichnissen

5

Schritte

1.   Speichern Sie die Datei mit den Informationen im Importverzeichnis. Sie finden den genauen Speicherplatz des Importverzeichnisses in den Importservicedetails. Weitere Informationen finden Sie hier  “Details des Importservice anzeigen”.

2.   Während die Datei importiert wird, wird sie in das Verzeichnis laufend verschoben.

3.   Wenn der Import erfolgreich war, wird die Datei ins Verzeichnis Erfolg verschoben und wenn ein Fehler aufgetreten ist, in das Verzeichnis Fehler.

Dateien über die Schnittstelle importieren

Diese Option ist nur möglich, wenn das für den Importservice in der Admin: Anwendung konfiguriert wurde.

5

Schritte

1.   Öffnen Sie Arbeitsbereich > Datenaustausch > Datei für Import hochladen.

2.   Wählen Sie einen Importservice aus der Liste. Es sind nur Importservices verfügbar, die spezifisch konfiguriert wurden.

Die Details des Importservice werden angezeigt.

3.   Selektieren Sie die Datei.

4.   Klicken Sie auf Hochladen.
