Benutzerdefinierte Attribute verstehen

Connexo bietet die Möglichkeit verschiedene zusätzliche benutzerdefinierte Attribute zu Ihrem System zu fügen. Das ermöglicht zusätzliche Informationen, die nicht von Connexo MultiSenses Basis CIM-Modell gedeckt werden, aber wichtig für den Anwender sind, beispielsweise für die Integration mit anderen Anwendungen, Prozessen, Operationen usw. So können Sie die Anwendung an Ihre besonderen regionalen Bedürfnisse anpassen.

In Connexo MultiSense können Sie benutzerdefinierte Attribute zu Geräten, Kanälen und Registern fügen. Jeder Attributsatz wird von einer Anzahl Rechte definiert.

In Connexo MultiSense ist es möglich einige benutzerdefinierten Attributsätze zu variieren, weitere Informationen unter “Benutzerdefinierte Attribute-Sätze im Laufe der Zeit verwalten”.
