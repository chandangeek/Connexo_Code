Gerätekonfigurationen konfigurieren

Gerätekonfigurationen anzeigen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetyp.

2.   Selektieren Sie einen Gerätetypen, indem Sie auf den Gerätetypnamen klicken oder einen neuen erstellen, wie in“Einen Standardgerätetyp einfügen” beschrieben.

3.   Klicken Sie auf Konfigurationen im Menü an der linken Seite.

Sie sehen alle Gerätekonfigurationen des Gerätetypen in der Übersichtstabelle. Selektieren Sie eine Konfiguration, um seine Details im Detailfenster unten anzuzeigen. Hier sehen Sie die Gerätekonfigurationsparameter wie die Datenprotokollierungsfunktion. Sie sehen auch den Status der Konfiguration gemeinsam mit Links zu seinen Datenquellen.

Gerätekonfigurationen hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetyp.

2.   Selektieren Sie einen Gerätetypen, indem Sie auf den Gerätetypnamen klicken oder einen neuen erstellen, wie in“Einen Standardgerätetyp einfügen” beschrieben.

3.   Klicken Sie auf Konfigurationen im Menü an der linken Seite.

4.   Klicken Sie auf Gerätekonfiguration hinzufügen. Ergänzen Sie die folgenden Felder:

Parameter

Beschreibung

Name

Ein eindeutiger Name.

Beschreibung der Konfiguration

Keine obligatorische Beschreibung der Gerätekonfiguration.

Direkt adressierbar

Wenn das Gerät nicht direkt mit dem Kommunikationsserver kommunizieren kann, d.h. im Fall eines Nebengeräts, ist diese Option deaktiviert.

Das Gerät ist ein Gateway

Wenn das Gerät keine Gateway-Funktion hat, ist diese Option deaktiviert.

Gateway-Typ

Selektieren Sie den Typ des Gateways:

o Home Area Network Gateway: Verbindet mit einer geringen Anzahl Nebengeräten in einer häuslichen Umgebung.

o Local Area Network Gateway: Verbindet mit einer großen Anzahl Nebengeräten in einer lokalen (größeren) Umgebung.

Je nach Art des Gateways, das Sie auswählen, ist das Aussehen der Zielseite unterschiedlich.

Datenprotokollierungsfunktion

Selektieren Sie, ob das Gerät als Datenprotokollierer handeln kann und teilen Sie seine Kanäle und Register mit den Datenprotokollierungs-Nebengeräten.

Wenn Sie diese Option selektieren, sind Sie in der Lage Nebengeräte zu verbinden oder neue Nebengeräte an diesem Datenprotokollierer zu erstellen. Weitere Informationen finden Sie in “Link data logger slave to data logger”.

Diese Option ist für Geräte mit dem Datenprotokollier-Nebengerätetypen nicht verfügbar.

Funktionalität des Geräts mit mehreren Strukturebenen

Selektieren Sie, wenn Sie mit einem einzelnen materiellen Gerät mehrere Belastungen messen möchten, wie Hauptstrom, Boiler, Solarenergie und andere.

Wenn Sie diese Option selektieren, sind Sie in der Lage Nebengeräte zu verbinden oder neue an diesem Gerät pro Belastung zu erstellen.

Daten beim Speichern validieren

Selektieren Sie, wenn Sie die Daten ab dem Moment, wenn sie im System verfügbar sind, validieren möchten.

Wenn Sie ja selektieren, werden die Daten validiert, wenn ein Gerät Übergänge von einem Status in den anderen macht, wobei die Auto-Aktion “Validierung aktiviert” selektiert wurde.

Wenn diese Option aktiv ist, ist das im Validierungsstatus des Geräts sichtbar.

5.   Klicken Sie auf Hinzufügen.

6.   Klicken Sie auf das Aktionssymbol  zum Aktivieren der Gerätekonfiguration. oder klicken Sie auf die Aktionstaste  auf der Übersichtsseite der Gerätekonfiguration.

7.   Fügen Sie Register-, Belastungsprofil und Logbuch-Konfigurationen hinzu und konfigurieren Sie.

3

Anmerkung

Wenn die Gerätekonfiguration aktiviert ist, kann sie zum Erstellen von Geräten verwendet werden. Beachten Sie bitte, dass wenn ein Gerät einmal eine bestimmte Gerätekonfiguration verwendet, diese Konfiguration nicht länger deaktiviert werden kann.

Gerätekonfiguration bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetyp > und selektieren Sie einen Gerätetyp > Konfigurationen > Konfigurationen.

2.   Klicken Sie das Aktionssymbol  und selektieren Sie Bearbeiten

3.   Die Gerätekonfigurationsparameter ändern. Wenn die Konfiguration aktiv ist, können nur der Name und die Beschreibung geändert werden.

4.   Klicken Sie auf Speichern.

Gerätekonfiguration löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetyp > und selektieren Sie einen Gerätetyp > Konfigurationen > Konfigurationen.

2.   Klicken Sie das Aktionssymbol  und selektieren Sie Löschen

3.   Klicken Sie auf Löschen. Aktive Konfigurationen können nicht gelöscht werden.

Gerätekonfigurationen klonen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetyp > klicken Sie auf einen Gerätetypnamen >Konfigurationen.

2.   Klicken Sie auf das Aktionssymbol  neben den Konfigurationen, die Sie klonen möchten und selektieren Sie Klonen.

3.   Geben Sie der neuen Gerätekonfiguration einen eindeutigen Namen und klicken Sie auf Klonen.

Die geklonten Gerätekonfigurationen haben alle dieselben Eigenschaftenwie die Originalgerätekonfiguration, aber der Status ist Inaktiv.
