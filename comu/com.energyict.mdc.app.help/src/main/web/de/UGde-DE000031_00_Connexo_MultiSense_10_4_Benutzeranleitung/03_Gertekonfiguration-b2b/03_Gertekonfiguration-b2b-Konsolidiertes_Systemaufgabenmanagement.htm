Konsolidiertes Systemaufgabenmanagement

Im Laufe dessen Connexo MultiSense können eine Anzahl Systemaufgaben  hinzugefügt und überwacht werden. Beispielsweise Datenbewertungs- und Datenvalidierungsaufgaben. Sie können die Seite des konsolidierte Systemaufgabenmanagements anzeigen und all diese Systemaufgaben gemeinsam auf einer Seite verwalten. Beachten Sie, dass die Datenbewertungs– und -validierungsaufgaben ebenfalls in  “View validation tasks” und “View estimation tasks” überwacht werden müssen. Es gibt jedoch auch Systemaufgaben, die nur in diesem konsolidierten Aufgabenmanagement gesehen werden können.

Alle Systemaufgaben in Connexo MultiSense anzeigen

Sie können eine Übersicht der Systemaufgaben in  Connexo MultiSense erstellen, die für Sie zu einem bestimmten Zeitpunkt relevant sind.

5

Schritte

1.   Öffnen Sie Verwaltung > Aufgabenmanagement > Aufgaben..

2.   Im Filterbalken im oberen Teil, geben Sie die notwendigen Kriterien ein, um die gezeigten Systemaufgaben einzuengen auf das, was Ihren Bedürfnissen entspricht. Vergessen Sie nicht auf Anwenden zu klicken, um die gefilterten Daten anzuzeigen.

Die Tabelle zeigt alle Systemaufgaben, die derzeit verfügbar sind in  Connexo MultiSense. Sie sehen den Namen der Warteschleife, in der sie sind und wann die nächste Ausführung geplant ist.

3.   Klicken Sie auf den Namen einer Aufgabe, um die Details anzuzeigen.

Die Details hängen vom Aufgabentyp ab, der selektiert wurde.

Die Vergangenheit einer Systemaufgabe aus der konsolidierten Übersicht anzeigen

Sie können die Vergangenheit einer Systemaufgabe sehen, wenn diese für diesen Aufgabentyp verfügbar ist. Beispielsweise eine Bewertungsaufgabe.

5

Schritte

1.   Öffnen Sie Verwaltung > Aufgabenmanagement > Aufgaben..

2.   Selektieren Sie Vergangenheit anzeigen4] aus dem Aktionssymbol  in der Zeile der benötigten Aufgabe.

Auf der Vergangenheitsseite sehen Sie eine Übersicht der Instanzen der selektierten Aufgabe mit Datum, Zeit, Dauer und Status.

3.   Selektieren Sie Log anzeigen4] aus dem Aktionssymbol  in der Zeile der benötigten Aufgabe, um zu sehen, was während jeder Instanz passiert ist.

Die Logseite zeigt alle Nachrichten, die während der Ausführung der Aufgabeninstanz protokolliert wurden.

Fügen Sie eine Systemaufgabe aus der konsolidierten Übersicht ein

In der konsolidierten Übersicht von  Connexo MultiSense Systemaufgaben, können Sie Systemaufgaben einfügen.

3

Anmerkung

Sie können auf dieser Seite keine Exportaufgaben einfügen. Für diesen Aufgabentyp müssen Sie noch immer zur Exportaufgabenseite gehen. Siehe “Manage data export tasks”.

5

Schritte

1.   Öffnen Sie Verwaltung > Aufgabenmanagement > Aufgaben.

2.   Klicken Sie die Taste Aufgabe hinzufügen.

3.   Selektieren Sie den Aufgabentyp, den Sie einfügen möchten.

Abhängig vom eingegebenen Aufgabentyp werden die verbleibenden Felder zum Bildschirm gefügt.

4.   Füllen Sie alle benötigten Felder aus.

5.   Selektieren Sie bei Bedarf eine Nachfolgeaufgabe, die nach Beendigung der selektierten Aufgabe ausgelöst wird. Für diesen Zweck verwenden Sie das Feld Gefolgt durch.

6.   Klicken Sie auf Hinzufügen.

Nun wird die neue Systemaufgabe in der Übersichtstabelle angezeigt. Sie können die hinzugefügte Aufgabe ebenfalls auf der entsprechende Übersichtsseite dieser speziellen Aufgabe sehen.

Eine Systemaufgabe aus der konsolidierten Übersicht bearbeiten

In der konsolidierten Übersicht von  Connexo MultiSense Systemaufgaben, können Sie Systemaufgaben bearbeiten, um eventuelle Fehler zu korrigieren.

3

Anmerkung

Sie können auf dieser Seite keine Exportaufgaben bearbeiten. Für diesen Aufgabentyp müssen Sie noch immer zur Exportaufgabenseite gehen. Siehe “Manage data export tasks”.

 

5

Schritte

1.   Öffnen Sie Verwaltung > Aufgabenmanagement > Aufgaben.

2.   Selektieren Sie Vergangenheit anzeigen4] aus dem Aktionssymbol  in der Zeile der benötigten Aufgabe.

3.   Die Bearbeiten-Seite zeigt die Felder, die für den entsprechenden Aufgabentyp relevant sind.

4.   Klicken Sie Speichern.

Eine Systemaufgabe aus der konsolidierten Übersicht löschen

In der konsolidierten Übersicht von  Connexo MultiSense Systemaufgaben, können Sie Systemaufgaben löschen, um eventuelle Fehler zu korrigieren.

3

Anmerkung

Sie können auf dieser Seite keine Exportaufgaben löschen. Für diesen Aufgabentyp müssen Sie noch immer zur Exportaufgabenseite gehen. Siehe “Manage data export tasks”.

5

Schritte

1.   Öffnen Sie Verwaltung > Aufgabenmanagement > Aufgaben.

2.   Selektieren Sie Löschen aus dem Aktionssymbol in der Zeile der benötigten Aufgabe.

3.   Klicken Sie auf Löschenum die Löschung zu bestätigen.

Eine Aufgabe aus der konsolidierten Übersicht ausführen

Sie können eine Systemaufgabe zur sofortigen Ausführung anfragen, wenn diese für den selektierten Aufgabentyp verfügbar ist. Beispielsweise eine Bewertungsaufgabe.

3

Anmerkung

Sie können auf dieser Seite keine Exportaufgaben ausführen. Für diesen Aufgabentyp müssen Sie zur Exportaufgabenseite gehen. Siehe “Manage data export tasks”.

5

Schritte

1.   Öffnen Sie Verwaltung > Aufgabenmanagement > Aufgaben.

2.   Selektieren Sie Starten aus dem Aktionssymbol in der Zeile der benötigten Aufgabe.

3.   Klicken Sie auf Start, um die selektierte Aufgabe in die Warteschleife zu schieben.
