Registergruppen verstehen

Register können zusammen gruppiert werden. Registergruppen werden zur Erfassung verschiedener Registerablesungen, die zusammengehören (z.B. Kommunikationsaufgaben) verwendet. Eine Gruppe enthält alle nötigen Informationen, die für einen bestimmten Prozess benötigt werden. z.B. werden Statusregister verwendet um alle relevanten Informationen zu erfassen, wenn es ein Problem mit einem Gerät gibt. Diese Gruppe enthält Register für Firmwareversionen, Batteriestatus usw. Register können nur zu einer Gruppe gehören.

Registergruppen hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement.

2.   Klicken Sie auf Registergruppen.

3.   Klicken Sie auf das Aktionssymbol um Registergruppen zu bearbeiten oder zu löschen.

4.   Klicken Sie auf Registergruppen hinzufügen. Geben Sie der Gruppe einen eindeutigen Namen und selektieren Sie Registertypen.

5.   Klicken Sie auf Speichern.

3

Anmerkung

Um die Ablesetypendetails anzuzeigen, klicken Sie auf das Informationssymbol  neben dem Register in der Übersichtstabelle.

Registergruppen bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement> Registergruppen.

2.   Klicken Sie das Aktionssymbol  und selektieren Sie Bearbeiten

3.   Ändern Sie den Namen und/oder (de)selektieren Sie Registertypen.

4.   Klicken Sie auf Speichern.

Registergruppen löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement> Registergruppen.

2.   Klicken Sie das Aktionssymbol  und selektieren Sie Löschen

3.   Klicken Sie auf Löschen.
