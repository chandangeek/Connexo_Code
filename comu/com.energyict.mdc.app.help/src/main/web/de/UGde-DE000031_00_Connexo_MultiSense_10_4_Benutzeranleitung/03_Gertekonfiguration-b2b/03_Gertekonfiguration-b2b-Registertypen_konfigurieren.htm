Registertypen konfigurieren

Ein Register ist ein Platzhalter für Daten über ein Gerät oder vom Gerät gemessene Daten. Es enthält Momentaufnahmen-Werte, die zu bestimmten Zeitmomenten erfasst werden. Beispiele für Register sind A+ Regime 1, Firmwareversion, Spannung L1 usw.

Ein Registertyp enthält alle wichtigen Registerparameter, so dass dieser Registertyp für verschiedene Gerätetypen und Gerätekonfigurationen wiederverwendet werden kann, aber er enthält keine Werte.

Ein Ablesetyp bietet eine detaillierte Beschreibung eines Ablesewerts. Er wird unter Verwendung von 18 wichtigen Attributen, die mit einem Punkt voneinander getrennt sind, beschrieben und von der CIM-Norm definiert. Weitere Informationen zu Ablesetypen entnehmen Sie bitte der Admin-Gebrauchsanweisung Kapitel Systemkonfiguration: Ablesetypen. Ablesetypen werden hauptsächlich verwendet, um im Datenmanagementprozess nach der Datenerfassung Daten zu beschreiben.

OBIS bedeutet Object Identification System und bietet Standardidentifizierer für alle Daten, die von den Messgeräten aufgrund den DLMS-Normen stammen. Eine OBIS-Kennung verzeichnet immer einen Ablesetypen, aber ein Ablesetyp verzeichnet nicht immer eine OBIS-Kennung. OBIS Kennungen beschreiben die Daten, die vom Gerät stammen und werden nicht für weitere Verarbeitung verwendet

 

 

 

 

 

 

 

Registertypen hinzufügen

Hier können Sie Registertypen hinzufügen, die Sie später zu einem Gerätetypen fügen können.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement> Registertypen.

2.   Klicken Sie auf die Taste Registertyp hinzufügen.

3.   Ergänzen Sie die folgenden Felder:

Parameter

Beschreibung

OBIS-Code

Geben Sie den sechsstelligen OBIS-Code ein. Die Drop-Down-Liste des Felds Ablesetyp basiert hauptsächlich auf dem eingegebenen OBIS-Code. Wenn es keine passenden Ablesetypen gibt, klicken Sie bitte auf die Taste Ablesetyp hinzufügen und geben auf der Seite Ablesetyp einfügen den benötigten Ablesetypen ein. Die Attribute, die dem eingegebenen OBIS-Code entsprechen, sind bereits vollständig. Siehe “Add reading types”.

 

Sie müssen Administratorrechte haben, um diese Taste sehen zu können. Kontaktieren Sie Ihren Connexo Administrator, wenn Sie die Taste  Ablesetyp hinzufügennicht sehen.

Ablesetyp

Selektieren Sie einen Ablesetypen aus der Drop-Down-Liste, beginnend mit dem Namen des Ablesetypen um die Drop-Down-Liste zu filtern. Nach Auswahl des Ablesetypen wird automatisch der entsprechende OBIS-Code vervollständigt.

4.   Klicken Sie auf Hinzufügen.

3

Anmerkung

Um die Ablesetypendetails eines Registers anzuzeigen, klicken Sie auf das Informationssymbol neben dem Ablesetypen in der Übersichtstabelle.

Wenn ein Registertyp von einem bestimmten Gerätetypen verwendet wird, kann nur der Name des Registertypen bearbeitet werden. Damit werden schädliche Auswirkungen auf den Datenerfassungsprozess verhindert.
