Gerätegruppen verwalten

Gerätegruppen verstehenEine statische Gerätegruppe ist eine Gerätegruppe, die aus einem Suchergebnis auf der Grundlage verschiedener Suchkriterien ausgewählt wurde. Die Zusammensetzung dieser Gruppe ändert sich im Laufe der Zeit nicht. Um neue Geräte in die Gruppe aufzunehmen, muss die Suche wiederholt und Geräte manuell eingefügt werden.

Es wird eine dynamische Gerätegruppe auf der Grundlage verschiedener Suchkriterien gebildet. Einzelne Geräte können nicht selektiert werden, alle Geräte der Suchergebnisse sind Teil der Gruppe. Diese Gruppe ist dynamisch, da immer wenn ein neues Gerät, welches den Suchkriterien entspricht, zum System hinzugefügt wird, es auch zur Gerätegruppe hinzugefügt wird.

Gerätegruppen anzeigen

5

Steps

1.   Öffnen Sie Geräte > Gerätegruppen> klicken Sie auf Gerätegruppen.

2.   Klicken Sie auf den Gerätegruppennamen, um die Detailseite anzuzeigen. Diese Seite zeigt alle Geräte, die in der selektieren Gerätegruppe enthalten sind.

3.   Ändern Sie die angezeigten Spalten, indem Sie auf die Taste Spalten über der Ergebnistabelle klicken. Kreuzen Sie die Spalten an, die Sie sehen möchten und entfernen Sie das Kreuz bei Spalten, die Sie verbergen möchten.

Fügen Sie eine statische Gerätegruppe hinzu

5

Steps

1.   Öffnen Sie Geräte > Gerätegruppen> klicken Sie auf Gerätegruppen.

2.   Klicken Sie auf Gerätegruppe einfügen.

3.   In Schritt 1 des Assistenten geben Sie der Gerätegruppe einen eindeutigen Namen und selektieren statische Gerätegruppe.

4.   Klicken Sie auf Nächste.

5.   In Schritt 2 ergänzen Sie die Filterkriterien im Menü im oberen Teil der Seite, um die Geräte herauszufiltern, die Sie zu Ihrer Gruppe hinzufügen möchten.

6.   Klicken Sie auf Vorschau. Die angewendeten Filter werden im oberen Teil der Seite angezeigt. Sie können einen Filter entfernen, indem Sie das Löschsymbol 8 anklicken oder auf Alle löschen klicken, um alle Filter zu löschen.

7.   Selektieren Sie Alle Geräte, wenn Sie alle gefilterten Geräte zur Gruppe fügen möchten oder selektieren Sie Selektierte Geräte um nur einige Geräte zu selektieren.

8.   Klicken Sie auf Nächste.

9.   Bestätigen Sie die Auswahl und klicken Sie auf Bestätigen. Diese Seite zeigt die Anzahl der Geräte in der neuen Gerätegruppe.

10.   Klicken Sie auf Beenden.

Fügen Sie eine dynamische Gerätegruppe hinzu

5

Steps

1.   Öffnen Sie Geräte > Gerätegruppen> klicken Sie auf Gerätegruppen.

2.   Klicken Sie auf Gerätegruppe einfügen.

3.   In Schritt 1 des Assistenten geben Sie der Gerätegruppe einen eindeutigen Namen und selektieren dynamische Gerätegruppe.

4.   Klicken Sie auf Nächste.

5.   In Schritt 2 ergänzen Sie die Filterkriterien im Menü im oberen Teil der Seite, um die Geräte herauszufiltern, die Sie zu Ihrer Gruppe hinzufügen möchten.

6.   Klicken Sie auf Vorschau. Die angewendeten Filter werden im oberen Teil der Seite angezeigt. Sie können einen Filter entfernen, indem Sie das Löschsymbol 8 anklicken oder auf Alle löschen klicken, um alle Filter zu löschen.

7.   Klicken Sie auf Nächste.

8.   Bestätigen Sie die Gerätegruppe, indem Sie auf Bestätigen klicken. Auf dieser Seite sehen Sie die Anzahl der verwendeten Filterkriterien und die Gesamtanzahl der Geräte in der Gerätegruppe zu diesem Zeitpunkt. Neue Geräte, die den Filterkriterien entsprechen, werden automatisch zu dieser Gerätegruppe gefügt.

9.   Klicken Sie auf Beenden.

Eine Gerätegruppe bearbeiten

5

Steps

1.   Öffnen Sie Geräte > Gerätegruppe> Gerätegruppen.

2.   Klicken Sie das Aktionssymbol  und selektieren Sie Bearbeiten

3.   Vervollständigen Sie die Filterfelder und klicken Sie auf Vorschau.

a. Bei einer statischen Gerätegruppe können Sie sich entscheiden alle Geräte zu selektieren, die vom Filter abgewiesen werden oder nur einige auszuwählen.

b. Bei einer dynamischen Gerätegruppe werden alle Ergebnisse zur Gerätegruppe gefügt.

4.   Klicken Sie auf Nächste.

5.   Bestätigen Sie durch klicken auf Bestätigen.

6.   Klicken Sie auf Beenden.

Eine Gerätegruppe löschen

5

Steps

1.   Öffnen Sie Geräte > Gerätegruppe> Gerätegruppen.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Löschen.

3.   Klicken Sie auf Löschen.

3

Anmerkung

Gerätegruppen können nicht gelöscht werden, wenn sie beispielsweise in einer Exportaufgabe verwendet werden.
