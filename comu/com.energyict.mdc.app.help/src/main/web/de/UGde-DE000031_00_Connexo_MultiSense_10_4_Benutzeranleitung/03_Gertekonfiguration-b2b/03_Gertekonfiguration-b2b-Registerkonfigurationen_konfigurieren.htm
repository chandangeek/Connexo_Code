Registerkonfigurationen konfigurieren

Registerkonfigurationen hinzufügen

1

Important

Fügen Sie Registertypen zu den Gerätetypen, sodass Sie in der Lage sind Registerkonfigurationen hinzuzufügen und zu konfigurieren.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > klicken Sie auf Gerätetyp > und selektieren Sie einen Gerätetyp > Konfigurationen > Selektieren Sie eine Konfiguration.

2.   Klicken Sie auf Registerkonfigurationen im Menü an der linken Seite.

3.   Klicken Sie auf Registerkonfiguration hinzufügen. Ergänzen Sie die folgenden Felder:

Parameter

Beschreibung

Registertyp

Selektion aus der Liste der Registertypen, die vorher zum Gerätetypen hinzugefügt wurden

OBIS-Kennzahl

Die aufgeführte OBIS-Kennzahl ist vom Registertyp abgeleitet, kann aber übersprungen werden, wenn die OBIS-Kennung der Gerätekonfiguration sich von der OBIS-Kennzahl des Gerätetypen unterscheidet.

Wert speichern als

Nummer: Registerdaten als nummerischen Wert speichern

Text: Registerdaten als Text speichern

Anzahl der Dezimalstellen

Anzahl der Dezimalstellen hinter dem Komma, nur verfügbar, wenn der Datentyp der Registrierung eine Zahl ist.

Überlaufwert

Der Überlaufwert definiert den maximalen Wert, der gespeichert werden kann.

Multiplikator verwenden

Angeben, ob ein Multiplikator verwendet werden sollte. Nur verwenden, wenn die Verbrauchsdaten mit einem bestimmten Wert multipliziert werden, um die aktuelle Verbrauchsrate zu erhalten.

Der Wert des Multiplikators kann am Gerät eingestellt werden.

Erfasster Ablesetyp

Der Ablesetyp der Daten, die vom Gerät erfasst werden.

Berechneter Ablesetyp

Der Ablesetyp der berechneten Daten. Dieser Ablesetyp wird verwendet, wenn Werte infolge eines Multiplikators berechnet werden oder zur Berechnung von Deltawerten, wenn Sammelwerte erfasst wurden.

4.   Klicken Sie auf Hinzufügen.

3

Anmerkung

Registerkonfigurationen können nur  zu inaktiven Gerätekonfigurationen hinzugefügt werden. So können negative Auswirkungen auf den Datenerfassungsprozess vermieden werden.

Auf der Übersichtsseite des Registers werden die angewendeten Validierungsregeln angezeigt. Ab hier können Sie die Validierungsregelseite öffnen. Weitere Informationen finden Sie hier“Data validation”.

Registerkonfiguration bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > klicken Sie auf Gerätetyp > und selektieren Sie einen Gerätetyp > Konfigurationen > Selektieren Sie eine Konfiguration > klicken Sie auf Registerkonfigurationen im Menü an der linken Seite.

2.   Klicken Sie das Aktionssymbol  und selektieren Sie Bearbeiten

3.   Die Registerkonfigurationsparameter ändern.

4.   Klicken Sie auf Speichern.

Registerkonfiguration löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > klicken Sie auf Gerätetyp > und selektieren Sie einen Gerätetyp > Konfigurationen > Selektieren Sie eine Konfiguration > klicken Sie auf Registerkonfigurationen im Menü an der linken Seite.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Löschen.

3.   Klicken Sie auf Löschen. Wenn die Gerätekonfiguration aktiv ist, können Registerkonfigurationen nicht gelöscht werden.
