Logbuchtypen konfigurieren

Logbuchtypen verstehenEin Logbuch kategorisiert Ereignisse, die an einem Gerät aufgetreten sind.  Ein einziges Logbuch kann Verschiedene Arten von Veranstaltungen enthalten. Ein Gerät kann mehr als ein Logbuch haben aber gängige intelligente Messgeräte haben mindestens eines. Jedes Logbuch hat seinen eigenen Identifizierer, der seine Inhalte vom Gerät zurückholt. Der Inhalt des Logbuchs wird bei der regelmäßigen Kommunikation mit einem Gerät aufgezeichnet. Typische Ereignisse, die aufgezeichnet werden, sind Stromausfälle, Verbindungsstatus, Firmwareaufzeichnung usw.

Ein Logbuchtyp enthält alle wichtigen Registerparameter, so dass dieser Registertyp fürverschiedene Gerätetypen und Gerätekonfigurationen wiederverwendet werden kann, aber er enthält keine Werte.

Logbuchtypen hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement> Logbuchtypen.

2.   Klicken Sie auf Hinzufügen Logbuchtyp. Ergänzen Sie die folgenden Felder:

Parameter

Beschreibung

Name

Ein eindeutiger Name.

OBIS-Code

Bietet die sechs Attribute des OBIS-Codes, getrennt mit “.”.

3.   Klicken Sie auf Hinzufügen.

3

Anmerkung

Wenn ein Logbuchtyp von einem Gerätetypen verwendet wird, kann nur der Name des Logbuchtypen bearbeitet werden. Damit soll verhindert werden, dass ein Gerät infolge der Änderung eventuell nicht mehr funktional ist.

Logbuchtyp bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement> Logbuchtypen.

2.   Klicken Sie das Aktionssymbol  und selektieren Sie Bearbeiten

3.   Parameter des Logbuchs ändern.

4.   Klicken Sie auf Speichern.

Logbuchtypen löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement> Logbuchtypen.

2.   Klicken Sie das Aktionssymbol  und selektieren Sie Löschen

3.   Klicken Sie auf Löschen.
