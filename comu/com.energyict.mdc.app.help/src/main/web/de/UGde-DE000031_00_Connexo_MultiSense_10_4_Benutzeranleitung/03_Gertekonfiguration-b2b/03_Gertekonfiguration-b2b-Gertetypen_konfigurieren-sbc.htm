Gerätetypen konfigurieren

Einen Standardgerätetyp einfügen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetypen.

2.   Klicken Sie auf Gerätetyp hinzufügen. Ergänzen Sie die folgenden Felder:

Parameter

Beschreibung

Typ

Klicken Sie auf Standardgerätetyp hinzufügen.

Kommunikationsprotokoll

Protokoll, welches die Kommunikation mit dem Server gestattet.

Name

Ein eindeutiger Name.

Gerätelebenszyklus

Selektieren Sie den Gerätelebenszyklus, den alle Geräte dieses Gerätetyps verwenden. Weitere Informationen entnehmen Sie bitte “Device life cycle management”.

3.   Klicken Sie auf Hinzufügen.

Einen Gerätetypen bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetypen.

2.   Klicken Sie das Aktionssymbol  und selektieren Sie Bearbeiten

3.   Die Gerätetypparameter ändern. Wenn der Gerätetyp aktive Gerätekonfigurationen hat, kann nur der Name des Gerätetyps geändert werden.

4.   Klicken Sie auf Speichern.

Das Gerätesymbol zu einem Gerätetyp fügen

Sie können ein Gerätesymbol zu einem Gerätetypen fügen, um eine visuelle Darstellung des Geräts aufzunehmen. Dieses Symbol wird auch auf der Zielseite jedes Geräts, das mit diesem Gerätetypen verbunden ist, angezeigt.

5

Schritte

1.   Öffnen Sie den Gerätetypen für den Sie ein Gerätesymbol bestimmen möchten. Öffnen Sie Verwaltung > Gerätemanagement > Gerätetypenund selektieren Sie den benötigten Gerätetypen.

2.   Klicken Sie auf das Aktionssymbol und selektieren Sie Gerätesymbol hinzufügen.

3.   Klicken Sie auf der Seite Gerätesymbol hinzufügen auf Datei selektieren um ein Bild von Ihrer Festplatte oder einem anderen Speicherort zu selektieren.

Die selektierte Datei sollte nicht größer als 500 kB sein. Das Dateiformat muss entweder JPG oder PNG sein.

4.   Klicken Sie auf Hinzufügen.

Das neue Gerätesymbol wird nun auf der Detailseite des Gerätetyps angezeigt und auf der Zielseite des Gerätes, das mit diesem Gerätetypen zusammenhängt.

3

Anmerkung

Sie können auch ein Gerätesymbol von der Gerätetyp-Detailseite hinzufügen, indem Sie Gerätesymbol hinzufügen von der Aktionstaste selektieren.

Ein Gerätesymbol an einem Gerätetypen bearbeiten oder löschen

Sie können ein Gerätesymbol von einem Gerätetypen bearbeiten oder löschen, wenn das bestehende Bild nicht mehr das aktuelle Gerät repräsentiert. Beachten Sie, dass diese Änderung sich auch auf den Zielseiten aller Geräte widerspiegeln wird, die mit diesem Gerätetypen zusammenhängen.

5

Schritte

1.   Öffnen Sie den Gerätetypen, für den Sie das Gerätesymbol bearbeiten oder löschen möchten. Öffnen Sie Verwaltung > Gerätemanagement > Gerätetyp und selektieren Sie den gewünschten Gerätetypen.

2.   Klicken Sie auf das Aktionssymbol und selektieren Sie Gerätesymbol bearbeiten.

3.   Klicken Sie auf der Seite Gerätesymbol hinzufügen auf Datei selektieren um ein Bild von Ihrer Festplatte oder einem anderen Speicherort zu selektieren.

Die selektierte Datei sollte größer als 500 kB sein. Das Dateiformat muss entweder JPG oder PNG sein.

4.   Klicken Sie Speichern.

5.   Klicken Sie auf das Aktionssymbol und selektieren Sie Gerätesymbol löschen.

6.   Klicken Sie auf Löschenum die Löschung zu bestätigen.

Gerätetypen löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetypen.

2.   Klicken Sie das Aktionssymbol  und selektieren Sie Löschen

3.   Klicken Sie auf Löschen.

Datenquellen zu einem Gerätetypen fügen

1

Important

Damit Sie Registertypen, Belastungsprofiltypen und Logbuchtypen zu einem Gerätetypen fügen können, müssen diese zuerst konfiguriert werden. Weitere Informationen finden Sie in “Datenquellen konfigurieren”.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätemanagement > Gerätetypen > selektieren Sie einen Gerätetypen.

2.   Klicken Sie im Menü auf der linken Seite auf Datenquellen > Registertypen > Registertyp hinzufügen.

3.   Selektieren Sie die Registertypen und klicken Sie auf Hinzufügen.

4.   Klicken Sie im Menü auf der linken Seite auf Datenquellen > Belastungsprofile > Belastungsprofiltypen hinzufügen.

5.   Selektieren Sie die Belastungsprofiltypen und klicken Sie auf Hinzufügen.

6.   Klicken Sie im Menü auf der linken Seite auf Datenquellen > Logbuchtypen > Logbuchtyp hinzufügen.

7.   Selektieren Sie die Logbuchtypen und klicken Sie auf Hinzufügen.
