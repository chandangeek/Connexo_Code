Kommunikations-Portpool konfigurieren

Ausgangs-Kommunikationsportpools hinzufügen

5

Schritte  

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikations-Portpools.

2.   Klicken Sie auf Kommunikations-Portpools hinzufügen > Ausgang. Füllen Sie die folgenden Felder aus:

Parameter

Beschreibung

Name

Ein einmaliger Name

Typ

Selektieren Sie einen Kommunikationstyp.

3.   Klicken Sie auf Einfügen.

4.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Aktivieren.

Eingangs-Kommunikationsportpools hinzufügen 

5

Schritte  

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikations-Portpools.

2.   Klicken Sie auf Kommunikations-Portpool hinzufügen > Eingang. Füllen Sie die folgenden Felder aus:

Parameter

Beschreibung

Name

Ein einmaliger Name

Typ

Selektieren Sie einen Kommunikationstyp.

Protokolldetektion

Ermittlungsprotokoll, das zur Erfassung von Geräten verwendet wurde, die  MultiSense kontaktieren oder Daten schicken.

Protokollerkennungsdetails

Abhängig von der Protokolldetektionsmethode, die Sie selektieren, könnte es sein, dass Sie verschiedene zusätzliche Parameter angeben müssen.

3.   Klicken Sie auf Einfügen.

4.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Aktivieren.

Kommunikations-Portpools bearbeiten

5

Schritte  

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikations-Portpools.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Bearbeiten.

3.   Ändern Sie die Parameter des Kommunikations-Portpools.

4.   Klicken Sie auf Speichern.

3

Anmerkung

Wenn ein Kommunikations-Portpool konfiguriert und gespeichert wurde, kann der Parameter-Typ nicht mehr bearbeitet werden. Damit sollen mögliche schädliche Einflüsse auf die Geräte, die diesen Portpool verwenden, verhindert werden.

Kommunikations-Portpools löschen 

5

Schritte  

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikations-Portpools.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Löschen.

3.   Klicken Sie auf Löschen.
