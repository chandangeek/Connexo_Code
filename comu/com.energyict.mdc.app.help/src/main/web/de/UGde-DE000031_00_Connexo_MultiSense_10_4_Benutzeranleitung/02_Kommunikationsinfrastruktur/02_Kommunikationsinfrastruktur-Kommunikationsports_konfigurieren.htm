Kommunikationsports konfigurieren

Eingangs-Kommunikationsport(s) hinzufügen

5

Schritte  

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikationsserver >klicken Sie auf einen Kommunikationsservernamen.

2.   Klicken Sie auf Kommunikationsports im Menü auf der linken Seite oder klicken Sie auf den Link der Kommunikationsport, der auf der Übersichtsseite bereitgestellt wird.

3.   Klicken Sie auf Kommunikationsport hinzufügen > Eingang. Füllen Sie die folgenden Felder aus:

Parameter

Beschreibung

Name

Einmaliger Name in MultiSense für die Port.

Typ

Seriell, Servlet, TCP, UDP.

Gleichzeitige Verbindungen

Anzahl der Threads, die gleichzeitig an dieser Port laufen können. Diese Option ist nicht für physikalische Ports wie Modems verfügbar.

Eingangs-Kommunikations-Portpool

Selektieren Sie einen Kommunikations-Portpool aus der Liste der konfigurierten Eingangs-Kommunikations-Portpools.

4.   Klicken Sie auf Einfügen.

5.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Aktivieren.

3

Anmerkung

Abhängig vom Typ müssen zusätzliche Felder ausgefüllt werden.

Kommunikationsports ohne zugewiesenem Kommunikations-Portpool können hinzugefügt, jedoch nicht aktiviert werden.

Ausgangs-Kommunikationsport(s) hinzufügen

5

Schritte  

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikationsserver > klicken Sie auf einen Kommunikations-Servernamen.

2.   Klicken Sie auf Kommunikationsports im Menü an der linken Seite oder klicken Sie auf den Link des Kommunikationsports, der auf der Übersichtsseite angegeben wird.

3.   Klicken Sie auf Kommunikationsport hinzufügen > Ausgang. Füllen Sie die folgenden Felder aus:

Parameter

Beschreibung 

Name

Ein einmaliger Name.

Typ

Seriell, TCP, UDP.

Gleichzeitige Verbindungen

Anzahl der Threads, die gleichzeitig an dieser Port laufen können. Diese Option ist nicht für physikalische Ports wie Modems verfügbar.

Ausgangs-Kommunikations-Portpool

Klicken Sie auf Kommunikations-Portpools hinzufügen um einen oder mehrere Portpools zu selektieren.

Klicken Sie auf Kommunikations-Portpools verwalten um einen neuen Ausgangs-Portpool zu erstellen.

4.   Klicken Sie auf Einfügen.

5.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Aktivieren.

3

Anmerkung

Abhängig vom Kommunikationstyp müssen verschiedene zusätzliche Felder ausgefüllt werden.

Kommunikationsports ohne zugewiesenen Kommunikations-Portpool können hinzugefügt, jedoch nicht aktiviert werden.

Kommunikationsports bearbeiten

5

Schritte  

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikationsserver > klicken Sie auf einen Kommunikationsserver-Namen.

2.   Klicken Sie auf Kommunikationsports im Menü an der linken Seite oder klicken Sie auf den Link des Kommunikationsports, der auf der Übersichtsseite angegeben wird.

3.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Bearbeiten.

4.   Ändern Sie die Parameter des Kommunikationsports.

5.   Klicken Sie auf Speichern.

Kommunikationsports löschen

5

Schritte  

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikationsserver > klicken Sie auf einen Kommunikationsserver-Namen.

2.   Klicken Sie auf Kommunikationsports im Menü an der linken Seite oder klicken Sie auf den Link des Kommunikationsports, der auf der Übersichtsseite angegeben wird.

3.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Löschen.

4.   Klicken Sie auf Löschen.

Kommunikationsports an Kommunikations-Portpools verwalten

5

Schritte  

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikations-Portpools > klicken Sie auf einen }Kommunikations-Portpool-Namen.

2.   Klicken Sie auf Kommunikationsport im Menü auf der linken Seite oder klicken Sie auf den Link des Kommunikationsports auf der Übersichtsseite.

3.   Kommunikationsports mithilfe des Aktionssymbols löschen.

4.   Klicken Sie auf Kommunikationsport hinzufügen.

5.   Selektieren Sie ein oder mehrere Kommunikationsports.

6.   Klicken Sie auf Einfügen.
