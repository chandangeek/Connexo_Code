5.   Klicken Sie auf Hinzufügen.

 

1

Wichtig

In den Dateinamen, Dateierweiterungen oder Dateispeicherorten dürfen die folgenden Zeichen nicht verwendet werden, es sei denn, sie sind Bestandteil eines verfügbaren Etiketts.

Ausgangsergebnis

Jede kreierte Datei enthält die folgenden Felder:

  • Intervall-Zeitstempel: ist im ISO-Format. Das ist das vollständige Datum plus Stunden, Minuten, Sekunden und einer Dezimalfraktion einer Sekunde (wenn anders als 0). Das Format ist JJJJ-MM-TTThh:mm:ss.sTZD.

  • Geräte MRID: ist ein Text-Feld.

  • Gerätename: ist ein Text-Feld. Der Name darf nicht das selektierte Trennzeichen (“,” oder “;”) enthalten.

  • Ablesetyp: Die Ablesetypen werden pro Gerät gruppiert und die Gerätegruppe bestimmt die Reihenfolge. Innerhalb eines Ablesetyps werden die Werte chronologisch auf dem Intervallzeitstempel geordnet: der älteste Intervallzeitstempel zuerst.

  • Messgerätwert: Der Wert ist eine Zahl. Neue Daten werden mit dem definierten  Identifizierer-Etikett markiert und die aktualisierten Daten werden mit dem Update-Identifizierer-Etikett markiert.

  • Ergebnis der Validierung: Kann die folgenden Ergebnisse haben:

  • Wenn die Daten nicht validiert wurden, erscheint ein leerer String.

  • Wenn die Daten validiert wurden und gültig sind, wird der String “gültig” angezeigt.

  • Wenn Daten fehlen, wird der String  “fehlend” angezeigt.

  • Wenn Daten vorhanden und suspekt sind, wird der String  “suspekt” angezeigt.

Beispiel:

  • 2015-06-01T00:00:00+02:00;3d2d2c2b-6c4d-45d0-9285-46ef25d4d87c;EDR513PX468003;0.0.2.1.1.1.12.0.0.0.0.0.0.0.0.0.72.0;3771825;gültig

3

Anmerkung

Es wird empfohlen immer das <Identifizierer>-Etikett zum Dateinamen oder dem Dateispeicherplatz zu fügen und es ist erforderlich dies zu tun, wenn Updates in verschiedene Dateien oder Verzeichnisse exportiert werden.

Beispiel:

  • 2015-07-15T08:22:20.500+01:00,0.36.114.24,3d2d2c2b-6c4d-45d0-9285-46ef25d4d87c,SPE001
