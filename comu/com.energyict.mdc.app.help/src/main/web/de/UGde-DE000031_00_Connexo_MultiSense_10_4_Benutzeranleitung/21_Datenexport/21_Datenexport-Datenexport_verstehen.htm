Datenexport in Connexo

MultiSense ist die Messdatenerfassungsanwendung von Connexo. Wenn MultiSense Daten von Messgeräten erfasst hat, prüfen Sie bitte, ob die Daten gültig sind und schätzen Sie die Daten, wenn notwendig, können Sie diese Daten exportieren. Sie haben auch die Möglichkeit unbearbeitete Daten zu exportieren. Es könnte notwendig sein, dass Sie Daten an andere Parteien und Systeme  für weitere Verarbeitung exportieren, beispielsweise zur Verwendung der Daten zur Rechnungslegung.

In MultiSense gibt es verschiedene Standardexporteure, mit denen Sie verschiedene Formate exportieren können. Es gibt jedoch eine Möglichkeit zusätzliche nutzerdefinierte Exporteure hinzuzufügen.  Damit können Sie alle benötigten Daten exportieren.

Wie Datenexport konfiguriert wird

Es gibt fünf wichtige Funktionen beim Konfigurieren von Datenexportaufgaben: den Datenselektor, den Formatierer, das Ziel, die Planung und die Überwachung.

Mit dem  Datenselektor können Sie definieren, welche Daten MultiSense unter Verwendung von Datentypen, wie Ablesetypen, Ereignistypen usw. exportiert werden. Standardmäßig sind bei Installation von Connexo verschiedene Datenselektoren verfügbar. Diese werden mit dem Tag [STD] gekennzeichnet. Datenselektoren ohne diesen Tag sind benutzerdefiniert. Für weitere Informationen zu diesen Selektoren kontaktieren Sie bitte Ihren Systemadministrator.

Je nach selektiertem Datenselektor, können optionale Möglichkeiten verfügbar werden, beispielsweise:

  • Selektieren Sie, ob Sie gültige Daten oder ununterbrochene Daten möchten,

  • Selektieren Sie eine Gerätegruppe, wenn Sie nur Daten von einer beschränkten Gruppe von Geräten möchten,

  • Selektieren Sie den Exportzeitraum (auf der Grundlage relativer Perioden, können Sie relative Zeitabschnitte selbst erstellen oder diese verwenden, die standardmäßig verfügbar sind).

Der Formatierer speichert alle Informationen, die exportiert werden müssen, in einem speziellen Format. Mit dem Ziel können Sie angeben, wohin die Datei exportiert wird. Eine Datei kann mehrere Male in denselben Zielordner oder an verschiedene Bestimmungen exportiert werden. Beispiele für Ziele sind E-Mail, ftp, usw.

Letztendlich können Sie in MultiSense wählen, Exportaufgaben manuell zu starten oder einen Plan hinzuzufügen, damit die Aufgaben zu festen Zeitpunkten ausgeführt werden. Sie können alle Aufgaben überwachen und ihre Vergangenheit und Protokollierung verfolgen. Das hilft Ihnen Fehler zu identifizieren und zu beseitigen, wenn Datenexportaufgaben fehlschlagen.
