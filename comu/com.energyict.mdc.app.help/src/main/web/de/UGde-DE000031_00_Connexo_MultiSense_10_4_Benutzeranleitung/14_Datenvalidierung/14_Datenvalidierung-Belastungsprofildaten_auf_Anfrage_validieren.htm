Belastungsprofildaten auf Anfrage validieren

Belastungsprofildaten auf Anfrage von der Übersichtsseite validieren 

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Belastungsprofile im Menü an der linken Seite.

3.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Jetzt validieren.

4.   Selektieren Sie das Startdatum für die Validierung der Belastungsprofildaten und klicken Sie auf  Validieren.

3

Anmerkung

Die Option jetzt validieren ist nur verfügbar, wenn die Validierung am Gerät aktiv ist und die Validierungsregeln hinzugefügt wurden.  Weitere Informationen finden Sie in “Validierungsstatus an einem Gerät verwalten”.

Belastungsprofildaten auf Anfrage von der Detailseite validieren

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Belastungsprofile im Menü an der linken Seite.

3.   Klicken Sie auf den Belastungsprofilnamen.

4.   Selektieren Sie den Karteireiter Spezifikationen.

5.   Klicken Sie auf die Aktionstaste }  und selektieren Sie Jetzt validieren.

6.   Selektieren Sie das Startdatum für die Validierung der Belastungsprofildaten und klicken Sie auf  Validieren.

3

Anmerkung

Die Option jetzt validieren ist nur verfügbar, wenn die Validierung am Gerät aktiv ist und die Validierungsregeln hinzugefügt wurden. Weitere Informationen finden Sie in “Validierungsstatus an einem Gerät verwalten”.
