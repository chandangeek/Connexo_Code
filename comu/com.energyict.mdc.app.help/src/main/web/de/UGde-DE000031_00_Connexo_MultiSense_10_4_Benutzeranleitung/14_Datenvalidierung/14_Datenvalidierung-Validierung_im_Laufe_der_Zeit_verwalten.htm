Validierung im Laufe der Zeit verwalten

Verschiedene Versionen der Validierungsregelsätze können erstellt werden, um die Validierungsdauer der Validierungsregel zu definieren, sodass sie zwischen anwendbaren Regeln laut Kalender wechseln können.

Versionen des Validierungsregelsatzes anzeigen

5

Schritte

1.   Öffnen Sie Verwaltung > Validierung > Validierungsregelsätze > einen Validierungsregelsatz selektieren.

Auf der Übersichtsseite sehen Sie die verschiedenen Versionen, die Sie vorher konfiguriert haben. Der Zeitraum der Version gibt an, wann diese Version für Validierung verwendet wurde und wird.

2.   Selektieren Sie eine Version, sodass Sie sehen, welche Validierungsregeln gemeinsam mit den Validierungregeldetails enthalten sind. Sie können diese Details auch sehen, wenn Sie auf den Versionsnamen des Validierungregelsatzes klicken und dann die Validierungsregeln im Menü an der linken Seite selektieren.

3

Anmerkung

Es sollte immer mindestens eine Version pro Validierungsregelsatz verfügbar sein. Der Zeitraum dieser Version ist immer. Wenn neue Versionen hinzugefügt werden, wird der Zeitraum entsprechend der neuen Version aktualisiert.

Versionen des Validierungsregelsatzes hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Validierung > Validierungsregelsätze >  > klicken Sie auf den Namen eines Validierungsregelsatzes > klicken Sie auf Validierungsregeln im Menü an der linken Seite.

2.   Klicken Sie auf Version einfügen. Füllen Sie die folgenden Felder aus:

Parameter

Beschreibung

Beschreibung

Geben Sie eine kurze Beschreibung, um die Verwendung der Version des Validierungsregelsatzes zu erläutern.

Anfang

Definieren Sie, wann die neue Version starten sollte.

3.   Klicken Sie auf Einfügen.

4.   Fügen Sie Validierungsregeln ein, wie in “Validierungsregeln hinzufügen” beschrieben.

Versionen des Validierungsregelsatzes klonen

5

Schritte

1.   Öffnen Sie Verwaltung > Validierung > Validierungsregelsätze >  > klicken Sie auf den Namen eines Validierungsregelsatzes > klicken Sie auf Validierungsregeln im Menü an der linken Seite.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Klonen. Bearbeiten Sie die folgenden Felder:

Parameter

Beschreibung

Beschreibung

Geben Sie eine kurze Beschreibung, um die Verwendung der Version des Validierungsregelsatzes zu erläutern.

Anfang

Definieren Sie, wann die neue Version starten sollte.

3.   Klicken Sie auf Einfügen.

4.   Die Validierungsregeln nach Bedarf bearbeiten, löschen oder hinzufügen, weitere Informationen unter “Validierungsregeln hinzufügen”.

Versionen des Validierungsregelsatzes bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Validierung > Validierungsregelsätze >  > klicken Sie auf den Namen eines Validierungsregelsatzes > klicken Sie auf Validierungsregeln im Menü an der linken Seite.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Bearbeiten.

3.   Bearbeiten Sie die Version und klicken Sie Speichern.

Versionen des Validierungsregelsatzes löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Validierung > Validierungsregelsätze >  > klicken Sie auf den Namen eines Validierungsregelsatzes > klicken Sie auf Validierungsregeln im Menü an der linken Seite.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Löschen.

3.   Es wird eine Warnmeldung angezeigt, klicken Sie auf  Löschen.
