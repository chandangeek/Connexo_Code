Validierungsregeln hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Validierung > Validierungsregelsätze > klicken Sie auf einen Validierungsregelsatz > klicken Sie auf Validierungsregeln im Menü an der linken Seite.

2.   Klicken Sie auf Validierungsregel einfügen. Füllen Sie die folgenden Felder aus:

Parameter

Beschreibung

Name

Ein eindeutiger Name

Prüfmechanismus

Selektieren Sie, wie die Validierung durchgeführt werden muss. Je nach selektiertem Prüfer müssen eventuell weitere Parameter definiert werden.

Ablese-Typ(en):

Selektieren Sie die zu validierenden Daten. Mehrere Ablesetypen können wie in “Ablesetypen hinzufügen” beschrieben hinzugefügt werden.

Datenqualitätsstufe

fehlerverdächtig: Die Daten sind ungültig und müssen beachtet werden. Dieser Wert unterliegt der Bewertung. Diese Werte werden in Rot auf der grafischen Darstellung der Daten angezeigt.

Informativ: Die Daten sind gültig, könnten jedoch ein Hinweis darauf sein, dass irgendetwas schief gelaufen ist. Dieser Wert wird nicht bewertet. Diese Werte werden in Gelb auf der grafischen Darstellung der Daten angezeigt.

3.   Klicken Sie auf Einfügen.

4.   Klicken Sie auf das Aktionssymbol, um die Regel zu aktivieren oder zu inaktivieren.

3

Anmerkung

Die folgenden Validierungsfunktionen sind standardmäßig verfügbar, sie werden mit dem [STD] Label gekennzeichnet. Validierfunktionen ohne dieses Label sind kundendefinierte Validierfunktionen. Für weitere Informationen zu diesem Prüfern kontaktieren Sie bitte Ihren Systemadministrator.

Prüfer

Beschreibung

Prüfen Sie die fehlenden Werte[STD]

Prüfen Sie nach fehlenden Daten. Diese Daten können in einem späteren Stadium durch Bewertung und Bearbeitung angeboten werden.

Ablesequalität [STD]

Ablesequalitäten sind Anmerkungen, die zu den Werten hinzugefügt werden, um anzugeben, dass zur Zeit des Ablesens etwas aufgetreten ist, beispielsweise ein Stromausfall.

Der Prüfer der Ablesequalität prüft die Intervalldaten und Registerablesungen, wenn sie eine oder mehrere der angegebenen Ablesequaltäten enthalten. Beachten Sie, dass Sie für System und Indexcode der Ablesequalitäten den Joker “*” verwenden können. In diesem Fall wird eine Ablesung mit einer Ablesequalität mit der korrekten Kategorie als fehlerverdächtig, unabhängig von ihrem System oder Ihrem Indexcode markiert wird.

Klicken Sie Ablesequalität hinzufügen, um die Ablesequalitäten anzugeben, die mit diesem Prüfer geprüft werden.

Zunahme registrieren [STD]

Verbrauch im Laufe der Zeit prüfen. Wenn der Wert des Registers niedriger oder gleich dem vorigen Registerwert ist, dann werden diese Daten als fehlerverdächtig gekennzeichnet.

Prüfen Sie Fehler gleiche Daten, wenn Sie gleiche Werte als fehlerverdächtig kennzeichnen möchten.

Grenzwert-Verletzung [STD]

Prüft jeden Intervallwert in Abhängigkeit eines Mindest- und Höchstwerts. Werte außerhalb dieser Grenzen werden als fehlerverdächtig gekennzeichnet.

Sie können die Grenzwertparameter auf der Validierungskonfigurationsseite des Kanals bearbeiten.

Fortlaufenden Nullen [STD]

Dieser Prüfer gestattet Ihnen Daten zu kennzeichnen, die aus fortlaufenden Nullen in einer vordefinierten Periode bestehen. Diese Regel funktioniert nur für Delta-Ablesetypen.

Füllen Sie die folgenden Parameter aus, um diesen Bewerter zu konfigurieren:

o Mindest- und Höchstperiode: Definiert den Zeitraum, der nach fortlaufenden Nullen gesucht wird. Wenn der Zeitraum größer als der Mindestzeitraum und kleiner als die Höchstperiode ist werden Daten aus diesem Zeitraum als fehlerverdächtig markiert. Wenn der Zeitraum kleiner als der Mindestzeitraum oder größer als die Höchstperiode ist, wird dieser Block als gültig markiert.

  • Mindestgrenzwert : Für Belastungen, die nie Null sind, können Sie den Mindestgrenzwert konfigurieren. Werte kleiner oder gleich diesem Grenzwert werden als Null gezählt.

  • Rückwirkend prüfen: Wenn Sie diese Option ankreuzen, werden Werte vor dem letzten Prüfdatum berücksichtigt, wenn sie in das definierte Zeitfenster fallen.    Connexo MultiSense prüft nach fortlaufenden Nullen in dem Zeitraum, der vom letzten geprüften Datum minus dem Höchstzeitraum berechnet wurde. Werte, die vom vorigen Validierungsdurchgang als gültig gekennzeichnet wurden, können jetzt fehlerverdächtig werden.

Zählervorlauf [STD]

Verwenden Sie diesen Prüfer, um die Summe der Intervallwerte gegenüber zwei Messgerätregisterwerten geprüft haben. Wenn der Unterschied höher als das definierte Maximum ist, kennzeichnet die Regel alle Werte zwischen den zwei Registerablesungen als fehlerverdächtig.

Füllen Sie die folgenden Parameter aus, um diesen Bewerter zu konfigurieren:

o Referenz Ablesetyp: Selektieren Sie das Register mit dem die Kanalablesungen verglichen werden.

  • Maximale absolute Differenz: Definieren Sie die absolute oder relative Differenz zwischen der Summe der Intervallkanaldaten und der Differenz zwischen den zwei Registerablesungen.

  • Referenzzeitraum: Überspringen von Ablesungen, wenn für einen Zeitraum, der länger ist als der Referenzzeitraum keine kumulativen Registerablesungen gefunden wurden.

  • Mindestgrenzwert : Stellen Sie einen Mindestgrenzwert ein, um die Intervalle, für die der Differenz zwischen den zwei kumulativen Registerablesungen zu klein ist, zu überspringen, um so zu vermeiden, dass kleinerer Verbrauchsmengen verglichen werden.

Validierungsregeln bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Validierung > Validierungsregelsätze > klicken Sie auf einen Validierungsregelsatz >Validierungsregeln.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Bearbeiten.

3.   Parameter der Validierungsregel ändern.

4.   Klicken Sie auf Speichern.

Den Grenzwert für die Verstoßregeln auf der Kanalvalidierungsseite bearbeiten

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Öffnen Sie Datenquellen im Menü auf der linken Seite und klicken Sie auf Kanäle. Selektieren Sie einen Kanal und selektieren Sie den Karteireiter Validierungskonfiguration im oberen Teil der Seite.

3.   Die zu bearbeitenden Parameter werden mit einem leeren Bleistiftsymbol gekennzeichnet .

4.   Klicken Sie auf die Aktionstaste und selektieren Sie “Validierungsregel” bearbeiten.

5.   Ändern Sie die Grenzwerte und klicken Sie Speichern.

6.   Geänderte Grenzwerte sind mit einem ausgefüllten Bleistiftsymbol gekennzeichnet .

Ablesetypen hinzufügen

5

Schritte

1.   Klicken Sie auf Ablesetypen einfügen.

2.   Füllen Sie die Filterfelder aus, um die korrekten Ablesetypen zu finden.

3.   Klicken Sie auf Anwenden.

Heben Sie die Auswahl von Items im Drop Down Menü auf, um einzelne Filter zu löschen oder klicken Sie auf Alle löschen , um alle angewendeten Filter zu löschen.

Sie sehen die Details der Ablesetypen, indem Sie auf das Informationssymbol klicken .

5.   Klicken Sie auf Einfügen.
