Was ist Kommunikationstopologie

Wenn Ihr Netzwerk aus einer großen Anzahl an Geräten besteht, können die Kosten ansteigen, wenn Sie jedes einzelne Gerät verbinden möchten. Für die Senkung dieser Verbindungskosten können Sie die Anzahl der Verbindungen reduzieren. Das erreichen Sie, indem Sie verschiedene Geräte miteinander verbinden und die Informationen von mehreren Geräten an ein einzelnes Gerät schicken, welches wiederum mit dem Datenerfassungssystem verbunden wird. So sind beispielsweise ein Gas- und Stromzähler nicht mit dem Internet, sondern mit einem Gateway verbunden. Dieses Gateway überträgt dann alle Messinformationen an das Datenerfassungssystem.

Kommunikationstopologie in  MultiSense

Definieren der Kommunikationstopologie in  MultiSense beginnt, wenn Sie eine Gerätekonfiguration konfigurieren.  An dieser Stellen müssen Sie angeben, ob das Gerät direkt ansprechbar ist und darum direkt unter Verwendung einer Verbindungsmethode mit MultiSense verbunden werden kann. Wählen Sie zweitens, ob das Gerät ein Gateway ist oder nicht und welcher Typ. Wenn im Kommunikationsprotokoll definiert ist, dass das Gerät kein Gateway sein kann, ist diese Möglichkeit in der Gerätekonfiguration deaktiviert.

Auf der Zielseite des Zählers oder Gateways können Sie dann das Gerät zum Verbinden und Daten übertragen selektieren, das ist das Master-Gerät. Auf dieser Zielseite können Sie auch ansehen, welche Geräte ihre Daten an dieses Gerät weiterleiten und es darum als Master-Gerät verwenden. Die Slave-Geräte werden in einer Tabelle oder Liste angezeigt.
