Erneuerungsprozess für den Schlüssel

Schlüsselerneuerung verstehen

Wenn ein Schlüssel sein Ablaufdatum erreicht hat, muss er erneuert werden. In Connexo MultiSense können Sie unter Verwendung der Schlüssel Aktiv und Passiv verwalten. Der aktuelle Schlüssel ist unter Aktiver Schlüssel gespeichert, während der neue Schlüssel unter Passiver Schlüssel gespeichert ist.

Der Schlüsselerneuerungsprozess besteht aus den folgenden Schritten:

1.   Einen neuen Schlüssel erzeugen -> Passiver Schlüssel.

2.   Schicken Sie den neuen Schlüssel an das Gerät, verwenden Sie dabei ein Kommando. Für diese Kommunikation wird der alte Schlüssel verwendet.

3.   Ersetzen Sie den aktuellen Schlüssel durch den neuen Schlüssel -> aktiver Passiver Schlüssel. Der Passive Schlüssel wird nun der Aktive Schlüssel und der aktive Schlüssel wird als passiver Schlüssel gespeichert. Wenn die Kommunikation mit dem neuen Schlüssel nicht funktioniert, kann der alte Schlüssel wieder hergestellt werden.

4.   Führen Sie Kommunikationstests durch, um zu prüfen, ob kein Schlüssel funktioniert.

5.   Wenn die Kommunikation erfolgreich ist, löschen Sie den passiven Schlüssel, wenn nicht aktivieren Sie den passiven Schlüssel erneut.

Sie können diesen Prozess manuell ausführen oder den Standardprozess verwenden, der standardmäßig verfügbar ist: Wie startet man einen Prozess am Gerät, siehe “Start processes on devices”.

Nachsorge an Geräte, die beim erneuern des Schlüssels fehlerhaft waren

Sie können Geräte überwachen, die bei der Schlüsselerneuerung mittels Anwenderaufgaben fehlerhaft waren. Wenn eine Schlüsselerneuerung fehlerhaft ist, wird eine Anwenderaufgabe erzeugt, um ihn aufzufordern, die entsprechenden Handlungen auszuführen. Wenn dies erledigt ist, können Sie die Anwenderaufgabe fertig stellen, sodass sie von Ihrer Aufgabenliste verschwindet. Weitere Informationen zu Benutzeraufgaben finden Sie in “Task management”.

Einen passiven Schlüssel erzeugen

Diese Option ist nur möglich, wenn der aktive Schlüsselwert definiert ist.

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Öffnen Sie im Menü an der linken Seite Kommunikation > Sicherheitszugriffsberechtigte.

3.   Klicken Sie auf das Aktionssymbol  neben dem Sicherheitszugriffsberechtigten und selektieren Sie Bearbeiten.

Es wird automatisch ein neuer passiver Schlüssel erzeugt.

Passiven Schlüssel aktivieren

Die Aktivierung des passiven Schlüssels in Connexo MultiSense hat keinerlei Einfluss auf den Schlüssel am Gerät vor Ort  Damit Sie in der Lage sind, den effektiven Schlüssel im Gerät selbst zu ändern, müssen Sie ein Gerätekommando an dieses Gerät schicken.

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Öffnen Sie im Menü an der linken Seite Kommunikation > Sicherheitszugriffsberechtigte.

3.   Klicken Sie auf das Aktionssymbol  neben dem Sicherheitszugriffsberechtigten und selektieren Sie Passiven Schlüssel aktivieren.

Wenn Sie den passiven Schlüssel aktivieren, wird der derzeitige aktive Schlüssel der passive Schlüssel. Dieser Schlüssel wird nicht gelöscht, bis Sie die Löschaktion ausdrücklich ausführen. Damit können Sie den Schlüssel wiederherstellen, sollte die Kommunikation mit dem neuen Schlüsselwert misslingen.

Passiven Schlüssel löschen

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Öffnen Sie im Menü an der linken Seite Kommunikation > Sicherheitszugriffsberechtigte.

3.   Klicken Sie auf das Aktionssymbol  neben dem Sicherheitszugriffsberechtigten und selektieren Sie Passiven Schlüssel löschen.

4.   Bestätigen Sie die Handlung, indem Sie Löschen anklicken.
