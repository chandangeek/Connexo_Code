Aufgabenmanagement

Eine einzelne Aufgabe ausführen

5

Schritte

1.   Öffnen Sie Arbeitsplatz > Aufgabenmanagement > Aufgaben oder selektieren Sie eine der anderen vorgefilterten Ansichten.

2.   Selektieren Sie eine Aufgabe.

3.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Aufgabe ausführen.

4.   Füllen Sie die erforderlichen Parameter aus und klicken Sie auf Start, um die Aufgabe zu beginnen. Nach dem Start haben Sie die Möglichkeit die Aufgabe auszuführen oder zu speichern.

3

Anmerkung

Wenn Sie im Detailfenster eines Prozesses an einem Objekt auf eine Aufgabe klicken, werden Sie zum Aufgabenausführungsfenster weitergeleitet. Hier können Sie die Aufgabe ausführen.

Sammelaktionen für Aufgaben ausführen

5

Schritte

1.   Öffnen Sie Arbeitsplatz > Aufgabenmanagement > Aufgaben oder selektieren Sie eine der anderen vorgefilterten Ansichten.

2.   Filtern Sie die Aufgaben bei Bedarf.

3.   Klicken Sie auf Sammelaktionen.

4.   In Schritt eins selektieren Sie die Aufgaben für die Sammelaktion.

Alle Aufgaben bedeutet alle Aufgaben aus der Aufgabenliste im vorigen Bildschirm. Wenn Sie diese Liste gefiltert haben, gilt dieser Filter noch immer. Selektieren Sie ausgewählte Aufgaben, um eine kundenspezifische Auswahl der Aufgaben in einer Tabelle zu erstellen.

5.   Klicken Sie auf Nächste.

6.   In Schritt zwei müssen Sie die Art der Handlung selektieren. Selektieren Sie Aufgabenmanagement, um die ausgewählten Aufgaben zu verwalten und Berechtigte, Fälligkeitsdatum und Priorität zu ändern oder Aufgabenausführung, um die ausgewählten Aufgaben auszuführen.

7.   Klicken Sie auf Nächste.

8.   In Schritt drei geben Sie die Parameter für die in Schritt  zwei ausgewählte Handlung an. Das können beispielsweise der Berechtigte des Aufgabenmanagement oder bestimmte Aufgabendetails sein, die zum Ausführen der Aufgabe benötigt werden.

9.   Klicken Sie auf Nächste.

10.   Bestätigen Sie die Sammelaktion in Schritt vier, indem Sie auf Bestätigen klicken.

11.   Schritt fünf zeigt das Ergebnis der Handlung.

12.   Klicken Sie auf Beenden.
