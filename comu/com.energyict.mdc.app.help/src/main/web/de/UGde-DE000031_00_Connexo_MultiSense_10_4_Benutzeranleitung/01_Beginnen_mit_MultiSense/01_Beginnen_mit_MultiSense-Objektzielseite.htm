Objektzielseite

Die Objektzielseite verstehen

Die Objektzielseite ist die erste Seite, die Sie sehen, wenn Sie auf den Link eines Objekts klicken, beispielsweise in den Suchergebnissen. Auf dieser Seite können Sie schnell die wichtigsten Informationen für das Objekt überblicken. Sie können seine Möglichkeiten und seine Leistungen sehen. Detailliertere Informationen finden Sie auf dieser Seite, indem Sie eine der links im vertikalen Navigationsbalken angegebenen Links anklicken. Die Widgets enthalten auch viele Links, die Sie anklicken können, um detailliertere Informationen anzuzeigen.

Die Zielseite des Objekts zeigt immer eine Anzahl der Standardwidgets. Die anderen Widgets hängen vom Typ des angezeigten Objekts ab. Wenn beispielsweise ein Gerät nicht direkt ansprechbar ist, sehen Sie kein Verbindungswidget.
