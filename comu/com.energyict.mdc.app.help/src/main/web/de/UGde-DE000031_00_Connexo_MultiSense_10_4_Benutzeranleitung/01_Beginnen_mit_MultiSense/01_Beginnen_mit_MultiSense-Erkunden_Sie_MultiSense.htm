Allgemeines Layout

Alle Seiten in MultiSense haben ein gleiches Layout. Um die Verwendung der Anwendung zu erleichtern, wird hier das allgemeine Layout erklärt.

 

 

Abbildung: Connexo Layout

A. Anwendungszentrum.

B. Verschiedene Menüs: Shortcut zu Dashboard-, Arbeitsplatz-, Geräte-und Verwaltungs-Seiten.

C. Navigation komprimieren oder vergrößern.

D. Zur Startseite zurückkehren.

E. Suche: Schnelle Verbindung zur Suchseite.

F. Online-Hilfe und "Über"-Informationen.

G. Logout: Platz in der Anwendung zum Abmelden.

H. Arbeitsbereich: Änderungen hängen von der Funktion ab, die Sie anzeigen lassen.

Übersicht Seitenaufbau

Wenn Sie auf eine spezielle Funktion in MultiSense klicken, öffnet sich in den meisten Fällen eine Übersichtsseite dieser Funktion. In der Anwendung haben diese Übersichtsseiten ein gleichartiges Aussehen.

Die nachfolgend gezeigte Abbildung und der Text zeigen und erklären das Layout der Übersichtsseiten gemeinsam mit den Handlungen, die Sie unter Verwendung der Symbole auf der Seite ausführen können

.

 

 

Abbildung: Übersichtsseite Layout.

A. Brotkrumen: zeigt, wo Sie sich in der Anwendung befinden. Ermöglicht einfache Navigation zu Seiten, die höher in der Struktur sind.

B. Das Menü zeigt die verschiedenen verfügbaren Komponenten, die mit der Hauptfunktion verbunden sind.

C. Übersichtstabelle: Diese Tabelle zeigt alle Objekte, die bereits zum System gefügt wurden. Es können verschiedene Handlungen mit den Items in der Tabelle ausgeführt werden, z.B. können sie bearbeitet, gelöscht, aktiviert, angezeigt usw. werden. In den meisten Fällen ist der Wert in Name ein Link, der auf die Einzelseite des Items verweist.

D. Detailfenster: Zeigt mehr Details eines Items, das in der Tabelle selektiert wurde. Standardmäßig ist das erste Item selektiert.

E. Hinzufügen-Taste: Taste, mit der neue Items zur Tabelle gefügt werden können. Kann auch eine andere Handlung sein, je nachdem welche Funktion Sie zu dem Zeitpunkt ansehen.

F. Aktionssymbol: Wenn Sie auf dieses Symbol klicken, erscheinen die verschiedenen Handlungen, die beim selektierten Item ausgeführt werden können.

G. Aktionstaste: Wenn Sie auf diese Taste klicken, erscheinen die verschiedenen Handlungen, die beim selektierten Item ausgeführt werden können. Hat dieselbe Funktion wie das Aktionssymbol.

H. Raster-Exporttaste: Wenn Sie auf diese Taste klicken, wird der Inhalt der Tabelle in eine CSV-Datei exportiert.
