Bewertungsregeln konfigurieren

Bewertungsregel-Satz hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Datenschätzung > Bewertungsregelsätze.

2.   Klicken Sie auf Bewertungsregelsatz einfügen. Füllen Sie die folgenden Felder aus:

Parameter

Beschreibung

Name

Geben Sie dem Bewertungsregelsatz einen Namen.

Beschreibung

Geben Sie eine kurze Beschreibung für diesen Bewertungsregelsatz.

3.   Klicken Sie auf Einfügen.

Bewertungsregelsatz bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Datenschätzung > Bewertungsregelsätze.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Bearbeiten.

3.   Parameter des Bewertungsregelsatzes ändern.

4.   Klicken Sie auf Speichern.

Bewertungsregelsatz löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Datenschätzung > Bewertungsregelsätze.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Löschen.

3.   Klicken Sie auf Löschen.

Bewertungsregel einfügen

5

Schritte

1.   Öffnen Sie Verwaltung > Datenschätzung > Bewertungsregelsätze. > klicken Sie auf einen Bewertungsregelsatz > klicken Sie auf Bewertungsregel im Menü auf der linken Seite.

2.   Klicken Sie auf Bewertungsregel einfügen. Füllen Sie die folgenden Felder aus:

Parameter

Beschreibung

Bewertungsregel

Geben Sie der Bewertungsregel einen Namen.

Bewertungsfunktion

Selektieren Sie die Bewertungsfunktion. Dieser Parameter definiert, wie die Bewertung ausgeführt wird. Je nach Bewertungsfunktion, die Sie selektieren, könnte es notwendig sein, weitere Parameter zu konfigurieren. Weitere Informationen finden Sie in “Standardbewertungsfunktionen”.

Ablesetypen

Selektieren Sie die Ablesetypen, um den Datentyp zu definieren, der bewertet wird. Ablesetypen bieten die Verbindung zu den Gerätekonfigurationen.

3.   Klicken Sie auf Hinzufügen.

4.   Klicken Sie auf das Aktionssymbol  neben einer Bewertungsregel und selektieren Sie Aktivieren.

Bewertungsregeln bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Datenschätzung > Bewertungsregelsätze. > klicken Sie auf einen Bewertungsregelsatz > klicken Sie auf Bewertungsregel im Menü auf der linken Seite.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Bearbeiten.

3.   Parameter der Bewertungsregel ändern.

4.   Klicken Sie auf Speichern.

Bewertungsregeln löschen

1.   Öffnen Sie Verwaltung > Datenschätzung > Bewertungsregelsätze. > klicken Sie auf einen Bewertungsregelsatz > klicken Sie auf Bewertungsregel im Menü auf der linken Seite.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Löschen.

3.   Klicken Sie auf Löschen.

Ändern der Reihenfolge der Bewertungsregeln in einem Bewertungsregelsatz

Es ist wichtig, die Reihenfolge der Bewertungsregeln im Satz einzustellen, da fehlerverdächtige Werte nur mit einer Bewertungsregel bewertet werden können. Darum sollte die Bewertungsregel, die zu den realistischsten Werten führt, im Satz ganz oben stehen.

5

Schritte

1.   Öffnen Sie Verwaltung > Datenschätzung > Bewertungsregelsätze. > klicken Sie auf einen Bewertungsregelsatz > klicken Sie auf Bewertungsregel im Menü auf der linken Seite.

2.   Klicken Sie auf Reihenfolge bearbeiten.

3.   Verschieben Sie die Bewertungsregel mit ziehen und ablegen nach oben oder unten und lassen Sie sie los, wenn ein grünes Kreuz erscheint.

4.   Klicken Sie auf Rückgängig machen um die Reihenfolge zu ändern oder klicken Sie auf Bestellung speichern um die Änderungen zu speichern.
