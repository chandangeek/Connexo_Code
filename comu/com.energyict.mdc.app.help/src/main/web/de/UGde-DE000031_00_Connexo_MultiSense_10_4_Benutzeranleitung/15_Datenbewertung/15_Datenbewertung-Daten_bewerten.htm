Bewertung planen

Bewertungsaufgaben anzeigen

5

Schritte

1.   Öffnen Sie Verwaltung > Datenbewertung > Bewertungsregeln.

Die Übersichtsseite zeigt eine Tabelle mit allen konfigurierten Bewertungsaufgaben. Sie sehen den Status jeder Aufgabe und wenn der nächste Durchgang geplant ist. Das Detailfenster unter der Tabelle zeigt die Details der Bewertungsaufgabe, die in der Tabelle ausgewählt wurde.

Bewertungsaufgaben einfügen

5

Schritte

1.   Öffnen Sie Verwaltung > Datenbewertung > Bewertungsregeln.

2.   Klicken Sie auf Bewertungsaufgabe einfügen. Füllen Sie die folgenden Felder aus:

Kategorie

Parameter

Beschreibung

 

o Name

o Eindeutiger Name

 

Protokollierebene

Die Logebene definieren, die für die Ausführung der Bewertungsaufgaben benötigt wird. Optionen sind Fehler, Warnung, Information, Debug und Trace. Wobei die Ebene Fehler nur Protokolle erzeugt, wenn Fehler auftreten und Trace eine detaillierte Beschreibung aller Ausführungen bietet.

Warnung ist die Standardlogebene.

Datenquellen

o Gerätegruppe

o Selektieren Sie die Gerätegruppe, deren Daten Sie bewerten möchten.

Planen

o Wiederauftreten

o Start am

  • Das Wiederauftreten der Aufgabe angeben.

  • Anfangsdatum der Aufgabe definieren.

Datenoptionen

o Bewertungszeitraum

o Den Datenzeitraum definieren, der bewertet werden muss.

3.   Klicken Sie auf Einfügen.

Bewertungsaufgabe bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Datenbewertung > Bewertungsregeln.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Bearbeiten.

3.   Parameter der Bewertungsaufgabe ändern.

4.   Klicken Sie auf Speichern.

Bewertungsaufgaben löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Datenbewertung > Bewertungsregeln.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Löschen.

3.   Klicken Sie auf Löschen.
