Verwendungsstellen erstellen und bearbeiten

Erstellen und Aktualisieren von Verwendungspunkten mithilfe von Dateiimport

Verwenden Sie den Importrahmen im mehrere Verwendungspunkte von einer csv-Datei hinzuzufügen oder zu aktualisieren. Weitere Informationen zu Importdiensten finden Sie in “Data import”. Für die Konfiguration siehe Importdienste im Kapitel Systemkonfiguration in der Admin: Gebrauchsanweisung.

1

Wichtig

Voraussetzungen:

  • Der Standardimportservice Verwendungspunkt-Importer muss korrekt konfiguriert und aktiviert werden.

  • Die entsprechenden Verzeichnisse für die Importdatei muss auf dem Connexo-Server verfügbar sein.

  • Der Importservice des Verwendungspunkts muss auf einem Anwendungsserver gespeichert werden.

Start der messtechnischen Konfiguration

Nein

Datum

Anfangsdatum der messtechnischen Konfigurationsversion am Verwendungspunkt. Nur anwendbar für die Erstellung neuer Verwendungspunkte. Der Wert wird übersprungen, wenn die Zeile einen bestehenden Verwendungspunkt aktualisieren soll.

Die Daten des Verwendungspunkts müssen in einer CSV-Datei verfügbar sein. Die erste Zeile in dieser Datei muss die Spaltentitel enthalten und die Felder müssen mit einem Semikolon oder einem Komma getrennt sein. Die folgenden Felder werden in der Importdatei des Verwendungspunkts erwartet.

3

Anmerkung

Die messtechnischen Konfigurationszeilen gelten nur für die Erstellung neuer Verwendungspunkte. Der Wert wird übersprungen, wenn die Zeile einen bestehenden Verwendungspunkt aktualisieren soll. Es ist nicht möglich die messtechnische Konfiguration bestehender Verwendungspunkte zu aktualisieren, da ein Verwendungspunkt mehrere messtechnische Konfigurationspunkte haben kann. Es wäre nicht deutlich, welche Version aktualisiert werden muss.

5

Schritte

1.   Speichern Sie die CSV-Datei mit den Verwendungspunkten im Importverzeichnis.

2.   Der Anwendungsserver holt sich die Importdatei für weitere Verarbeitung. Zu diesem Zeitpunkt wird die Importdatei in das Verzeichnis laufendverschoben.

3.   Nach Beendigung des Imports, wird die Importdatei entweder in das VerzeichnisErfolgreich oder das Verzeichnis Fehlerverschoben.

4.    Prüfen Sie die erstellten Verwendungspunkte mit der Suchfunktion.  Siehe “Advanced search”.

5.   Wenn es ein Problem gab, können Sie das Problem beseitigen, indem Sie im Log in der Vergangenheit des Importservice suchen. Siehe “Monitor data import”.

Einen Verwendungspunkt manuell hinzufügen

Neben dem gleichzeitigen Import einiger Verwendungspunkte mithilfe eines Dateiimporters, können Sie einen Verwendungspunkt in Connexo MultiSense auch erstellen, indem Sie die Daten manuell in ein Formular eingeben. Wenn der Verwendungspunkt erstellt wurde, können Sie Geräte damit verbinden.

5

Schritte

1.   Öffnen Sie Verwendungspunkte > Einen Verwendungspunkt erstellen.

2.   Füllen Sie die Felder im Bildschirm aus. Alle Felder – außer Name – sind Pflichtfelder.

Parameter

Beschreibung

Name

Einzigartiger Name für den Verwendungspunkt. Dieses Feld ist einzigartig für jeden Verwendungspunkt. Wenn der Verwendungspunkt erstellt wird, erzeugt Connexo automatisch einen konformen CIM.

Dienstkategorie

Dropdown mit allen aktiven Dienstkategorien im System.

Erstellt

Datum der tatsächlichen Erstellung des Verwendungspunkts. Der Standardwert ist Nun.

Koordinaten

Definieren Sie die genauen Koordinaten des Verwendungspunkts in Grad, Minuten und Sekunden.

Standort

Beginnen Sie die Adresse des Verwendungspunkts einzugeben oder selektieren Sie Input location um alle Adressdaten zu definieren.

3.   Klicken Sie auf Hinzufügen.

Sie werden zur Zielseite des Verwendungspunkts, den Sie soeben erstellt haben, weitergeleitet.

Einen Verwendungspunkt bearbeiten

Bestehende Verwendungspunkte können in Connexo MultiSense bearbeitet werden, um die bei der Erstellung eventuell gemachten Fehler zu korrigieren.

5

Schritte

1.   Einen Verwendungspunkt suchen, wie in “Search for usage points”beschrieben.

2.   Klicken Sie auf den Verwendungspunkt, um seine Zielseite zu öffnen.

3.   Selektieren SieBearbeiten aus der Schaltfläche Handlungen.

4.   Bearbeiten Sie die relevanten Felder.

 und Dienstkategorie können nicht bearbeitet werden.

5.   Klicken Sie Speichern.

Sie werden wieder zur Zielseite des Verwendungspunkts geleitet, die die angepassten Daten anzeigt.
