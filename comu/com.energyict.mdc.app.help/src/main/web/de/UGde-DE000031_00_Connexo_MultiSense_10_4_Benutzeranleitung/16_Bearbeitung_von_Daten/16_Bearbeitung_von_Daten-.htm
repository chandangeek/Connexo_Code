Bearbeitung von Daten

In einigen Fällen ist es notwendig, manuell Daten zu ändern oder hinzuzufügen. Beispielsweise Korrektur falscher Werte oder Vervollständigung fehlender Werte, die nach Validierung aufgenommen wurden. Manchmal müssen Sie auch Daten von manuellen Messgeräteablesungen oder für Testzwecke hinzufügen. In anderen Fällen könnte es notwendig sein, nicht nutzbare oder falsche Werte zu widerrufen.

Dieses Kapitel beschreibt die verschiedenen Schritte, die Sie durchführen müssen, um die oben beschriebenen Handlungen auszuführen und Ihre Daten gültig zu machen.

Was möchten Sie tun?

Korrigieren von fehlerverdächtigen Daten

In MultiSense können Sie einfach Werte einfügen, wenn Daten fehlen oder ungültige Daten bearbeiten oder löschen. Um die Daten zu identifizieren, die angepasst werden müssen, können Sie Validierungsregeln verwenden. Führen Sie die folgenden Handlungen aus, um Ihre Daten gültig zu machen:

1. Configure validation rules um fehlende und falsche Werte in den erfassten Daten zu er­mitteln.

2. View suspect data.

3. Registerdaten bearbeiten durch Bearbeiten, Hinzufügen oder Löschen von Werten und Kanaldaten bearbeiten durch Bearbeiten oder Löschen von Werten.

4. Validate register data on request oder Validate channel data on request zum Anzeigen, ob die Daten nun gültig sind.
