Ablesungen importieren

1

Important

Mit diesem Importer können Sie nur die täglichen Ablesungen importieren.

Um Geräteablesungen importieren zu können, benötigen Sie eine CSV-Datei. Die erste Zeile dieser Datei muss die Überschrift sein und die Felder sind mit "," oder ";" getrennt. Die folgenden Informationen müssen in dieser Datei enthalten sein:

Feld

Erforderlich

Datenformat

Geräteidentifizierer

Ja

String

Ablesedatum

Ja

Datum Wie im Importservice definiert. Standardformat ist “TT/MM/JJJJ hh:mm”

Ablesetyp

Ja

String

Wert

Ja

Nummer. Sollte den Konfigurationseinstellungen (Überschuss, Anzahl der Ziffern, Anzahl der Kommastellen) entsprechen.

Der Geräteidentifizierer kann die MRID oder der Name des Geräts sein. Der Importer prüft zuerst die MRID und wenn diese nicht abgerufen werden kann, sucht er nach Geräten mit passenden Namen.

Es ist möglich , verschiedene Ablesungen vom selben Gerät hinzuzufügen verschiedene Reihen mit demselben Namen zu erstellen, aber mit verschiedenen Werten. Alle Werte sollten den Konfigurationseinstellungen entsprechen, und sowohl der Überschuss als auch die Anzahl Ziffern und Kommastellen sollten definiert sein.

Beispiel:

  • Geräteidentifikation; Ablesedatum; Ablesetyp MRID;Ablesewert;;;;
    SPE010000010080;25/06/2015  12:00:00;0.0.0.1.1.1.12.0.0.0.0.0.0.0.0.3.72.0;1,335,010,534.12;;;;
    SPE010000010080;25/06/2015  12:00:00;0.0.0.1.19.1.12.0.0.0.0.0.0.0.0.3.72.0;497,899,785;;;;
    SPE010000010081;25/06/2015  12:00:00;0.0.0.1.1.1.12.0.0.0.0.0.0.0.0.3.72.0;1,335,010,542;0.0.0.1.19.1.12.0.0.0.0.0.0.0.0.3.72.0;1,335,010,542;0.0.0.1.1.1.12.0.0.0.0.0.0.0.0.0.72.0;497,899,785

5

Schritte

1.   Speichern Sie die CSV-Datei mit den Ablesungen in das Importverzeichnis.

2.   Wenn der Import erfolgreich abgeschlossen ist, wird die Original CSV in das Erfolgsverzeichnis verschoben. Die definierten Werte werden nun zu den Geräten in MultiSense eingefügt.

Kanalablesungen zu Geräten ohne Ablesungen fügen

In MultiSense können Sie manuell Ablesungen zu einem Gerät fügen,  welches nie abgelesen wurde. Dafür müssen Sie die nachfolgenden Handlungen ausführen. Diese Handlung ist auch für Register verfügbar, aber hier müssen Sie jeweils einen Wert einfügen.

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kanäle im Menü an der linken Seite.

3.   Selektieren Sie einen Kanal.

4.   Selektieren Sie im Filter im oberen Teil der Seite Heute als Messzeit. Klicken Sie auf Anwenden.

Es erscheint eine Informationsnachricht, die angibt, dass keine Ablesungen verfügbar sind.

5.   Klicken Sie auf Ablesung hinzufügen.

6.   Klicken Sie auf den kalkulierten Wert in der Tabelle und geben Sie den Wert ein oder Kopieren/Einfügen Sie mehrere Werte von einer anderen Quelle.

7.   Klicken Sie Änderungen speichern.

Ablesungen von der Referenz kopieren

In MultiSense haben Sie die Möglichkeit die Ablesungen eines anderen Geräts mit dem selben oder einem anderen Behandlungstypen zu kopieren.

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kanäle im Menü an der linken Seite.

3.   Klicken Sie auf die Sammelaktions-Taste und selektieren Sie Kopieren der Referenz.

4.   Ergänzen Sie die folgenden Felder:

Parameter

Beschreibung

Gerät überprüfen

Selektieren Sie das Gerät, um die Daten zu kopieren.

Messtyp überprüfen

Selektieren Sie den Ablesetypen der Daten zum kopieren.

Anfangsdatum/-zeit

Selektieren Sie die Datum/Zeit für das Intervall der zu kopierenden Daten.

Fehlerverdächtige Werte zulassen

Wenn geprüft, werden auch fehlerverdächtige Daten kopiert.

Vollständiger Zeitraum

Wenn geprüft, werden keine Daten kopiert, wenn es fehlende Werte in den erfassten Daten des Kontrollgeräts in der ausgewählten Zeitspanne gibt.

Kommentar zur Bewertung

Selektieren Sie den Kommentar, der zur Ablesung gefügt wird. Standardmäßig ist “Originalkommentar behalten” ausgewählt. Sie können das löschen, indem Sie die Taste neben dem Drop-Down-menü anklicken.

5.   Klicken Sie auf Kopieren.

6.   Klicken Sie Änderungen speichern.
