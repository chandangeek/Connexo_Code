Kommunikation

Kommunikationen kombiniert erneut auslösen

5

Schritte

1.   Öffnen Sie Arbeitsplatz > Kommunikation > Kommunikationen oder öffnen Sie Arbeitsplatz > Kommunikation > Kommunikationsübersicht > selektieren Sie eine Kommunikationsgruppe beispielsweise die unterbrochenen Verbindungen.

2.   Klicken Sie auf Sammelaktion.

3.   Selektieren Sie für die Handlung alle Ergebnisse in der Tabelle oder eine benutzerdefinierte Auswahl.

4.   Klicken Sie auf Nächste.

5.   Selektieren Sie die Aktion Start oder Jetzt starten.

  • Selektieren Sie Starten, wenn die Aufgabe beim nächsten Mal, wenn eine Verbindung hergestellt wird, entsprechend der Verbindungsstrategie und -planung ausgeführt werden kann.

  • Selektieren Sie Jetzt starten, wenn Sie die Aufgabe sofort ausführen möchten. Das ist nur möglich, wenn die Verbindungsstrategie der Verbindungsmethode schnellstmöglich ist. In diesem Fall wird eine neue Verbindung eingerichtet.

6.   Klicken Sie auf Nächste.

7.   Im nächsten Schritt werden Sie gebeten die Handlung zu bestätigen, klicken Sie auf Bestätigen.

Klicken Sie auf Beenden.
