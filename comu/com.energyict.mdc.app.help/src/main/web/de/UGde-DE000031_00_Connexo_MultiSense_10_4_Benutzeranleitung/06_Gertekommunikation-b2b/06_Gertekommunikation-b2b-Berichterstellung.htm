Berichterstellung

Verschiedene Standardberichte sind verfügbar in Connexo MultiSense.

Einen Kommunikationsbericht erstellen

5

Schritte

1.   Öffnen Sie Arbeitsplatz > Datenkommunikation> Kommunikationen.

2.   Klicken Sie auf Bericht erzeugen .

3.   Die Berichtkategorie Gerätekommunikation wurde ausgewählt. Selektieren Sie eine Bericht-Unterkategorie aus der Liste der Standardberichte.

Bericht

Beschreibung

Gerätekommunikationsaufgabe

Listet alle Kommunikationsaufgaben für die Geräte in den/der selektierten Gerätegruppe/n auf.

Gateway-Leistung

Listet alle Gateways gemeinsam mit ihren Leistungsstatistiken, berechnet über den festgelegten Zeitraum der Erstellung des Berichts auf.

Messgeräteleistung

Zeigt alle Geräte einer Gerätegruppe und den Gateway, mit dem sie verbunden sind, gemeinsam mit ihren Leistungsstatistiken. Alle Statistiken werden über den Zeitraum berechnet, der bei Erstellung des Berichts angegeben wurde.

Punkt-zu-Punkt-Leistung

Dieser Bericht kombiniert Kommunikations- und Verbindungsstatistiken über den Zeitraum, der bei Erstellung des Berichts angegeben wurde.

4.   Klicken Sie auf Nächste.

5.   Füllen Sie die Obligatorischen Filter aus. Mit diesen Filtern können Sie die Geräte selektieren, deren Daten Sie anzeigen möchten. Stellen Sie die In Bericht Filter so ein, dass die Daten der Geräte, die Sie mit den obligatorischen Filtern selektiert haben, gefiltert werden. Die “In Bericht” Filter können auch eingestellt werden, nachdem der Bericht erstellt wurde.

6.   Klicken Sie auf Nächste.

7.   Wenn alle Filter korrekt sind, klicken Sie auf Bericht erstellen.

Der Bericht wird auf einer neuen Seite geöffnet. Die Aufzeichnungen im Bericht können unter Einstellung der In Bericht Filter und klicken auf Filter anwenden gefiltert werden.

Navigieren Sie zur nächsten Seite des Berichts, indem Sie auf   klicken. Sie zeigen die Ergebnisse in einer Tabelle an, wenn Sie auf  oder  klicken, um die Ergebnisse in einer Grafik anzuzeigen. Klicken Sie auf , um den Bericht als CSV, Word, Excel, PDF, RTF oder Text-Dokument zu exportieren.

Einen Verbindungsbericht erstellen

5

Schritte

1.   Öffnen Sie Arbeitsplatz > Datenkommunikation> Verbindungen.

2.   Klicken Sie auf Bericht erzeugen .

3.   Die Berichtkategorie Geräteverbindungen ist ausgewählt. Selektieren Sie eine Bericht-Unterkategorie aus der Liste der Standardberichte.

Bericht

Beschreibung

Geräteverbindungen

Liste aller Verbindungen und Verbindungsdetails für alle Geräte in der/den selektierten Gerätegruppe/n.

4.   Klicken Sie auf Nächste.

5.   Füllen Sie die Obligatorischen Filter aus. Mit diesen Filtern können Sie die Geräte selektieren, deren Daten Sie anzeigen möchten. Stellen Sie die In Bericht Filter so ein, dass die Daten der Geräte, die Sie mit den obligatorischen Filtern selektiert haben, auch gefiltert werden. Die “In Bericht” Filter können auch eingestellt werden, nachdem der Bericht erstellt wurde.

6.   Klicken Sie auf Nächste.

7.   Wenn alle Filter korrekt sind, klicken Sie auf Bericht erstellen.

Der Bericht wird auf einer neuen Seite geöffnet. Die Aufzeichnungen im Bericht können unter Verwendung der In Bericht Filter und klicken auf Filter anwenden gefiltert werden.

Navigieren Sie zur nächsten Seite des Berichts, indem Sie auf   klicken. Sie zeigen die Ergebnisse in einer Tabelle an, wenn Sie auf  oder  klicken, um die Ergebnisse in einer Grafik anzuzeigen. Klicken Sie auf , um den Bericht als CSV, Word, Excel, PDF, RTF oder Text-Dokument zu exportieren.
