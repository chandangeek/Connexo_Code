Verbindungen filtern

Verbindungen von der Übersichtsseite filtern

5

Schritte

1.   Öffnen Sie Arbeitsplatz > Datenkommunikation> Verbindungsübersicht.

2.   Klicken Sie auf einen der Links, die auf der Übersichtsseite angezeigt werden.

Sie gelangen zur Verbindungsseite, wo ein Filter angewendet wurde, und nur die Verbindungen gezeigt werden, die dem von Ihnen selektierten Link entsprechen.

3

Anmerkung

Wenn Sie auf einen der Parameter in der Verbindungszusammenfassung klicken, werden mehrere Filter kombiniert.

Parameter

Filter

Erfolg

Warten

Laufend

Ausstehend + Laufend + Erneut versuchen

Fehlgeschlagen

Nie abgeschlossen + fehlgeschlagen

Verbindungen mit der Filteroption filtern

5

Schritte

1.   Öffnen Sie Arbeitsplatz > Datenkommunikation> Verbindungen.

2.   Selektieren Sie einen oder mehrere Werte für einen oder mehrere Filterobjekte aus dem Filtermenü im oberen Teil der Seite.

3.   Klicken Sie auf Anwenden.

Die angewendeten Filter werden im oberen Teil der Seite angezeigt.

4.   Klicken Sie auf Alle löschen im Filterfenster oder in der oberen rechten Ecke, um alle angewendeten Filter zu löschen.

Einzelne Filter können gelöscht werden, indem die Selektion des Objekts im Drop-Down-Menü entfernt wird.
