Verbindungsmethoden an einem Gerät konfigurieren

Verbindungsmethoden an einem Gerät verstehen

Geräteverbindungsmethoden werden von den Gerätekonfigurationen übernommen. Alle Parameter und Verbindungsdetails sind gleich, obwohl es in Connexo MultiSense möglich ist, diese Einstellungen auf Geräteebene zu ändern. Das wird benötigt, um die Verbindungsdetails anzugeben, die gerätespezifisch sind, wie IP-Adressen.

Ausgangsverbindungsmethoden an einem Gerät konfigurieren

1

Important

Alle Parameter der Verbindungsmethode des Gerätes werden aus der Gerätekonfiguration übernommen. Stellen Sie sicher, dass Verbindungsmethoden in der Gerätekonfiguration konfiguriert werden.

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Verbindungsmethoden im Menü an der linken Seite.

3.   Klicken Sie auf Ausgangsverbindungsmethode hinzufügenoder Verbindungsmethode hinzufügen  > Ausgangsverbindungsmethode hinzufügen. Ergänzen Sie die folgenden Felder:

Parameter

Beschreibung

Konfiguration der Verbindungsmethode

Selektieren Sie eine Verbindungsmethode, die in der Gerätekonfiguration konfiguriert wird. Diese Methode kann nur einmal pro Gerät verwendet werden. Wenn dies gespeichert wurde, kann dieser Parameter nicht mehr geändert werden.

Protokolldialekt

Fügen Sie einen der Protokolldialekte hinzu, die am Gerät verfügbar sind.

Kommunikationsanschlusspool

Selektieren Sie die Kommunikationsanschlusspools, die in der selektierten Konfiguration verfügbar sind.

Verbindungsstrategie

Sobald wie möglich: Immer dann verbinden, wenn eine Kommunikationsaufgabe eine Verbindung benötigt.

Verbindungen minimieren : nur zur von Ihnen angegebenen Zeit mit dem Gerät verbinden.

Verbindungszeitplan

Nur verfügbar, wenn “Verbindungen minimieren” selektiert wurde. Der Plan, wann eine Verbindung hergestellt werden soll.

Verbindungsfenster

Ist das Zeitfenster, in dem es gestattet ist, eine Verbindung herzustellen und Kommunikationsaufgaben auszuführen.

Anzahl der gleichzeitigen Verbindungen

Gestattet mehrere Verbindungen gleichzeitig, um parallele Planung von Geräten zu ermöglichen. Nicht verfügbar, wenn “Verbindungen minimieren” selektiert wurde.

Standardmäßig ist 1, maximal sind 16 gleichzeitige Verbindungen gestattet.

Aktiv

Die Verbindungsmethode aktivieren.

4.   Vervollständigen oder ändern Sie die Verbindungsdetails.

5.   Klicken Sie auf Hinzufügen.

6.   Klicken Sie auf das Aktionssymbol  um die Verbindungsmethode zu aktivieren und als Standard einzustellen.

3

Anmerkung

Es ist möglich eine unvollständige Verbindungsmethode zu speichern, wenn nicht alle erforderlichen Felder ausgefüllt sind. In diesem Fall wird eine Nachricht angezeigt, um zu bestätigen, ob Sie wirklich die unvollständige Verbindungsmethode speichern möchten. Die Verbindungsmethode erscheint in der Übersichtstabelle, hat aber noch immer den Status unvollständig.

Verbindungsmethode an einem Gerät bearbeiten

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Verbindungsmethoden im Menü an der linken Seite.

3.   Klicken Sie das Aktionssymbol  und selektieren Sie Bearbeiten

4.   Die Parameter der Verbindungsmethode ändern.

5.   Klicken Sie auf Speichern.

4.   Klicken Sie auf Löschen.
