Kommunikationsaufgaben konfigurieren (Verwaltung)

Kommunikationsaufgaben hinzufügen

1

Important

Damit Kommunikationsaufgaben hinzugefügt werden können, müssen Handlungen betreffend Registern, Belastungsprofilen oder Logbüchern, Datenquellen zuerst konfiguriert werden. Das ist mit  “Configure register types”, “Configure load profile types” und “Configure logbook types” möglich.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikationsaufgaben.

2.   Klicken Sie auf Kommunikationsaufgabe hinzufügen.

3.   Geben Sie der Kommunikationsaufgabe einen eindeutigen Namen und klicken Sie auf Hinzufügen.

4.   Klicken Sie auf der Übersichtsseite der Kommunikationsaufgabe auf Aktionen im Menü auf der linken Seite.

5.   Klicken Sie auf Aktion einfügen. Selektieren Sie eine Kategorie aus dem Menü und spezifizieren Sie dann die Handlung

Kategorie

Beschreibung

Standardüberprüfung

Siehe “Eine Basiskontrollaktion zu einer Kommunikationsaufgabe fügen”.

Anmelden

Siehe “Ein Register zu einer Kommunikationsaufgabe fügen”.

Lastprofil

Siehe “Ein Belastungsprofil zu einer Kommunikationsaufgabe fügen”.

Logbuch

Siehe “Ein Logbuch zu einer Kommunikationsaufgabe fügen”.

Uhr

Siehe “Eine Zeithandlung zu einer Kommunikationsaufgabe fügen”.

Topologie

Siehe “Eine Topologieaktion zu einer Kommunikationsaufgabe fügen”.

Statusdaten

Siehe “Eine Statusinformationsaktion zu einer Kommunikationsaufgabe fügen”.

6.   Die Attribute der Handlung vervollständigen.

7.   Klicken Sie auf Einfügen.

3

Anmerkung

Gerätekommandos werden für unregelmäßige Einstellungen der Geräteparameter verwendet und diese können über die Verbindung der Kommunikationsaufgabe einer Kommunikationsaufgabe hinzugefügt werden. Weitere Informationen finden Sie in “Add device command categories to a communication task”.

Kommunikationsaufgaben bearbeiten

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikationsaufgaben.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Bearbeiten um den Namen der Kommunikationsaufgabe zu ändern.

3.   Klicken Sie auf den Namen der Kommunikationsaufgabe und klicken Sie auf Aktionen im Menü auf der linken Seite, um die Aktionen dieser Aufgabe zu ändern.

4.   Klicken Sie auf Speichern.

Kommunikationsaufgaben löschen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikationsaufgaben.

2.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Löschen.

3.   Klicken Sie auf Löschen.

Eine Basiskontrollaktion zu einer Kommunikationsaufgabe fügen

Wenn Sie diese Handlung zu einer Kommunikationsaufgabe fügen, können Sie die Seriennummer und die Uhr des Geräts prüfen. Diese Kommunikationsaufgabe wird zuerst von der Kommunikationsmaschine ausgeführt, bevor die Messgerätedaten ausgelesen werden. Wenn diese Aufgabe fehlerhaft ist, werden alle anderen Aufgaben nicht ausgeführt. Das verhindert, dass fehlerhafte Daten ausgelesen und gespeichert werden.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikationsaufgaben > selektieren Sie eine Kommunikationsaufgabe in der Tabelle  > klicken Sie auf   Aktionen im Menü an der linken Seite.

2.   Klicken Sie auf Aktion einfügen.

3.   Selektieren Sie Basisprüfung im Feld Kategorie.

4.   Selektieren Sie die Aktion Prüfen.

5.   Definieren Sie die Attribute der Handlung:

Parameter

Beschreibung

Seriennummer überprüfen

Prüfen Sie, ob die Seriennummer des Geräts korrekt ist.

Taktabweichung lesen

Prüfen Sie, den Zeitunterschied zwischen dem Gerät und dem Kommunikationsserver.

Maximale Zeitdifferenz

Definieren Sie den maximalen Zeitunterschied, der zwischen dem Gerät und dem Kommunikationsserver gestattet ist. Wenn der Zeitunterschied größer als gestattet ist, erzeugt die Kommunikationsaufgabe einen Fehler.

Sie können diese Einstellung nur konfigurieren, wenn Sie den Zeitunterschied selektieren.

6.   Klicken Sie auf Hinzufügen.

Ein Register zu einer Kommunikationsaufgabe fügen

Diese Aktion zu einer Kommunikationsaufgabe fügen, ermöglicht Ihnen eine Auswahl von Registern an Ihren Geräten abzulesen.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikationsaufgaben > selektieren Sie eine Kommunikationsaufgabe in der Tabelle  > klicken Sie auf   Aktionen im Menü an der linken Seite.

2.   Klicken Sie auf Aktion einfügen.

3.   Selektieren Sie Register im Feld Kategorie.

4.   Selektieren Sie die Aktion Ablesen.

5.   Selektieren Sie eine oder mehrere Registergruppe(n).

6.   Klicken Sie auf Hinzufügen.

3

Anmerkung

Wenn ein Register nicht zu einer Gerätekonfiguration oder einem Gerätetypen gefügt wird, versucht der Kommunikationsserver nicht, dieses Register am Gerät abzulesen. Das bedeutet, dass Sie keine verschiedenen Kommunikationsaufgaben für verschiedene Gerätetypen erstellen müssen.

Ein Belastungsprofil zu einer Kommunikationsaufgabe fügen

Diese Aktion zu einer Kommunikationsaufgabe fügen, ermöglicht Ihnen eine Auswahl von Belastungsprofilen und Kanälen an Ihren Geräten abzulesen.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikationsaufgaben > selektieren Sie eine Kommunikationsaufgabe in der Tabelle  > klicken Sie auf   Aktionen im Menü an der linken Seite.

2.   Klicken Sie auf Aktion einfügen.

3.   Selektieren Sie Register im Feld Kategorie.

4.   Selektieren Sie die Aktion Prüfen.

5.   Selektieren Sie eine oder mehrere Belastungsprofiltyp(en).

6.   Selektieren Sie eine der Optionen für die Parameter Ein Intervall als schlechte Zeit kennzeichnen, Messgerätereignisse von Ablesequalitäten und Fehler, wenn Profilkonfiguration nicht passt.

Weitere Informationen zu diesen Parametern können Sie ablesen, wenn Sie mit der Maus über das Informationssymbol gehen .

7.   Klicken Sie auf Hinzufügen.

4.   Selektieren Sie die Aktion Prüfen.

Eine Zeithandlung zu einer Kommunikationsaufgabe fügen

Wenn Sie diese Handlung zu einer Kommunikationsaufgabe fügen, können Sie die Uhr Ihrer Geräte einstellen, forcieren oder synchronisieren.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikationsaufgaben > selektieren Sie eine Kommunikationsaufgabe in der Tabelle  > klicken Sie auf   Aktionen im Menü an der linken Seite.

2.   Klicken Sie auf Aktion einfügen.

3.   Selektieren Sie Uhr im Feld Kategorie.

4.   Selektieren Sie die Aktion.

Wenn Sie die Aktion Einstellen selektieren, definieren Sie die minimale und maximale Zeitdifferenz.

Wenn Sie die Aktion Synchronisieren selektieren, definieren Sie die minimale und maximale Zeitdifferenz, sowie die maximale Zeitverschiebung.

5.   Klicken Sie auf Hinzufügen.

Eine Topologieaktion zu einer Kommunikationsaufgabe fügen

Wenn Sie diese Handlung zu einer Kommunikationsaufgabe fügen, können Sie die Topologie Ihrer Geräte aktualisieren oder verifizieren.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikationsaufgaben > selektieren Sie eine Kommunikationsaufgabe in der Tabelle  > klicken Sie auf   Aktionen im Menü an der linken Seite.

2.   Klicken Sie auf Aktion einfügen.

3.   Selektieren Sie Topologie im Feld Kategorie.

4.   Selektieren Sie die Aktion Aktualisieren oder Verifizieren.

Die Aktion Aktualisieren erfasst die Topologieinformation des Geräts und aktualisiert bei Bedarf. Wenn ein unbekanntes Gerät erfasst wird, führt das zu einem unbekannten Geräteereignis und einer Konfigurations-Warnmeldung.

Die Aktion Verifizieren erfasst die Topologieinformation des Geräts und protokolliert, ob die Informationen noch immer aktuell sind oder nicht. Wenn das nicht aktuell ist, entsteht eine Konfigurationswarnung

5.   Klicken Sie auf Hinzufügen.
