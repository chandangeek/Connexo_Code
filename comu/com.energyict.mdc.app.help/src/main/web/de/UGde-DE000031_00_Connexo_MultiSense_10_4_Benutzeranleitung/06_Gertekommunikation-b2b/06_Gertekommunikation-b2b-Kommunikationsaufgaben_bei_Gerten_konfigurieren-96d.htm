Kommunikationsaufgaben bei Geräten konfigurieren

Kommunikationsaufgaben an einem Gerät anzeigen

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kommunikationsaufgaben im Menü auf der linken Seite.

Alle Kommunikationsaufgaben, die das Gerät benötigt und ausführen konnte, werden in der Tabelle angezeigt. Detail jeder Kommunikationsaufgabe werden im nachfolgenden Detailfenster angezeigt.

Unter Verwendung einer Kommunikationsaufgabe Daten auf Anfrage erfassen

  • Weitere Informationen, wie man die Dringlichkeit einer Kommunikationsaufgabe ändern kann, siehe “Change urgency of a communication task”.

  • Um eine Kommunikationsaufgabe auf Nachfrage zu starten, siehe  “Run a communication task on request from the communication task overview” und “Run a communication task on request using the communication planning”.

  • Um die Verbindungsmethode einer Kommunikationsaufgabe zu ändern, siehe “Verbindungsmethode einer Kommunikationsaufgabe an einem Gerät ändern”

Verbindungsmethode einer Kommunikationsaufgabe an einem Gerät ändern

5

Schritte

1.   Selektieren Sie ein Gerät mit Suche, siehe “Advanced search”.

2.   Klicken Sie auf Kommunikationsaufgaben im Menü auf der linken Seite.

3.   Klicken Sie auf das Aktionssymbol  und selektieren Sie Verbindungsmethode ändern.

4.   Selektieren Sie die Verbindungsmethode und klicken Sie auf  Speichern.

3

Anmerkung

Wenn Sie die Verbindungsmethode einer Kommunikationsaufgabe ändern, die Teil eines gemeinsamen Plans ist, wird die Verbindungsmethode aller Aufgaben in diesem Plan geändert.
