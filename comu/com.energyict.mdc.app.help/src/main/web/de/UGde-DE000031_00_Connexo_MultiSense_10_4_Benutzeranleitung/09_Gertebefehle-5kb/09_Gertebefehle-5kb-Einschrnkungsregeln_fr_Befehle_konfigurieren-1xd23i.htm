Einschränkungsregeln für Befehle konfigurieren

Eine Einschränkungsregel für Befehle hinzufügen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Einschränkungsregeln für Befehle.

2.   Klicken Sie auf Eine Einschränkungsregel für Befehle hinzufügen und vervollständigen Sie die nachfolgenden Parameter:

Parameter

Beschreibung

Name

Geben Sie einen einmaligen Namen.

Tagesgrenzwert

Definieren Sie die maximale Anzahl Instanzen aller ausgewählten Befehle, die pro Tag verschickt werden können. Die Menge muss kleiner als die Einschränkung der Woche und des Monats sein.

Wochengrenzwert

Definieren Sie die maximale Anzahl Instanzen aller ausgewählten Befehle, die pro Woche verschickt werden können. Die Menge muss kleiner als die Einschränkung des Monats sein.

Monatsgrenzwert

Definieren Sie die maximale Anzahl Instanzen aller ausgewählten Befehle, die pro Monat verschickt werden können.

Befehle

Selektieren Sie die Befehle, für welche die Regel gilt. Sie können nach Kategorie filtern. Klicken Sie auf Einfügen.

3.   Klicken Sie auf Einfügen.

Eine Einschränkungsregel für Befehle bearbeiten

1

Wichtig

Eine Einschränkungsregel für Befehle zu bearbeiten, muss vom/von den Genehmigungsberechtigten genehmigt werden, bevor die Änderung effektiv wird.

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Einschränkungsregeln für Befehle.

2.   Klicken Sie auf das Aktionssymbol  neben der Einschränkungsregel für Befehle, die Sie bearbeiten möchten und selektieren Sie Bearbeiten.

3.   Ändern Sie die Parameter und klicken Sie Speichern.

1

Wichtig

5

Schritte

3.   Klicken Sie als Bestätigung auf {1}Löschen{2}.
