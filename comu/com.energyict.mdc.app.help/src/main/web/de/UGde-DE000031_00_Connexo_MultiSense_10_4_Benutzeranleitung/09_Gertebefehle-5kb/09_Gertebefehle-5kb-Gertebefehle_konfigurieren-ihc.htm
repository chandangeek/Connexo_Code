Kategorien für Befehle selektieren

Neue Kategorien für Gerätebefehle zu einer Kommunikationsaufgabe fügen

5

Schritte

1.   Öffnen Sie Verwaltung > Gerätekommunikation > Kommunikationsaufgaben > selektieren Sie eine Kommunikationsaufgabe in der Tabelle oder fügen Sie eine neue hinzu, wie in “Add communication tasks” beschrieben> klicken Sie auf Kommandokategorien im Menü an der linken Seite.

2.   Klicken Sie Kategorien für Befehle einfügen.

3.   Selektieren Sie eine oder mehrere Kategorien für den Befehl.

4.   Klicken Sie auf Einfügen.
