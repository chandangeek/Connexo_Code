Dateimanagement unter Verwendung von Gerätebefehlen

Dateimanagement verstehen

Dateimanagement in MultiSense

In Connexo MultiSense können Dateien zu Gerätetypen gefügt werden, sodass diese bei Bedarf zu Geräten gefügt werden können. Das vom Gerätetyp verwendete Kommunikationsprotokoll bestimmt, ob es möglich ist Dateien zu verwalten. Wenn das Kommunikationsprotokoll mindestens einen Befehl unterstützt, der eine bestimmte Datei erfordert, ist das Dateimanagementmenü für den Gerätetypen verfügbar.

Wenn Dateimanagement möglich ist, ist der Menüpunkt Dateimanagement im Navigationsmenü des Gerätetypen verfügbar. Dateimanagement hat zwei Karteireiter: Spezifikationen und Dateien. Im Karteireiter  Spezifikationen können Sie die Dateimanagementoptionen bearbeiten. Im Karteireiter  Dateien können Sie alle Dateien verwalten und das  Dateimanagement aktivieren, wenn es inaktiv ist.
