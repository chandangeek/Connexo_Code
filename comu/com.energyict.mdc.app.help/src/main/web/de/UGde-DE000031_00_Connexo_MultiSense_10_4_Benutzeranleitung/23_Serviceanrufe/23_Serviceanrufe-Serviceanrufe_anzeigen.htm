Serviceanrufe anzeigen

Alle Serviceanrufe anzeigen

In diesem Bildschirm haben Sie eine Übersicht aller Serviceanrufe in Connexo. Sie finden schnell den Serviceanruf, den Sie suchen und sehen dessen Spezifikationen. Von dieser Seite aus können Sie sich durch alle Details klicken, oder – wenn zutreffend – zu einer Übersicht seiner Kind-Serviceanrufe. Damit können Sie die Serviceanrufe in Connexo überwachen und verfolgen.

3

Anmerkung

In der Liste der Serviceanrufe werden die Serviceanrufe der oberen Ebene (ohne Eltern) zuerst angezeigt und nach Modifikationsdatum sortiert. Die neusten stehen dabei zuerst.  Dann werden die Kind-Serviceanrufe mit den neusten zuerst angezeigt. Ihre Ebene in der Hierarchie wird nicht berücksichtigt.

5

Schritte

1.   Öffnen Sie Arbeitsplatz > Serviceanrufe.

2.   Im Filterbalken finden Sie schnell Ihren Serviceanruf, indem Sie filtern nach:

a. ID: Sowohl Serviceanruf-ID als auch externe Referenz ID, d.h. ID vom System des Drittanbieters.

b. Typ.

c. Status.

d. Eingangsdatum.

e. Modifikationsdatum.

3.   Die folgenden Spalten sind in der Übersichtstabelle des Serviceanrufs verfügbar.

Parameter

Beschreibung

Serviceanruf

Von Connexo bereitgestellte ID für jeden Serviceanruf.

Externe Referenzen

Vom Anforderer mitgeteilte ID für den Serviceanruf, d.h. das System des Drittanbieters.

Typ

Der Typ des Serviceanrufs wie von Ihrem System und Lieferteam definiert. Siehe “Serviceanruftypen verstehen”.

Status

Der Status des Serviceanrufs der oberen Ebene. Siehe “Lebenszyklen von Serviceanruftypen verstehen”. Beachten Sie, dass der Status von der Aktivität des Kind-Serviceanrufs abhängt. Wenn einer von ihnen noch in laufendem Zustand ist, wird das im Eltern-Serviceanruf wiedergespiegelt.

Eingangsdatum

Datum, wann der Serviceanruf im System eingegangen ist.

Modifikationsdatum

Datum, wann eine Handlung zuletzt versucht wurde oder der Status geändert wurde.

4.   Die folgenden Felder sind im Detailfenster verfügbar.

Parameter

Beschreibung

Serviceanruf der oberen Ebene

ID des ersten Serviceanrufs in der Struktur der geschachtelten Serviceanrufe, zu dem der selektierte Serviceanruf gehört. Dieser Serviceanruf hat keinen Eltern-Serviceanruf. Wenn es nur eine Ebene über dem selektierten Serviceanruf gibt, d.h. Eltern-Serviceanruf und Serviceanruf der oberen Ebene sind identisch, dann wird der Serviceanruf der oberen Ebene nicht angezeigt. Wenn es keinen Serviceanruf der oberen Ebene gibt, wird dieses Feld auch nicht angezeigt.

Eltern-Serviceanruf

ID des Serviceanrufs, der der Eltern-Serviceanruf des selektierten Serviceanrufs ist. Wenn es keinen Eltern-Serviceanruf gibt, wird dieses Feld nicht angezeigt.

Herkunft

Anzeige des Systems des Drittanbieters, von dem der Serviceanruf stammt.

Kinder

Anzahl der Kinder des selektierten Serviceanrufs, d.h. die Serviceanrufe, die sich unter dem selektierten Serviceanruf in der Hierarchie der geschachtelten Serviceanrufe befinden. Wenn Sie diesen Wert anklicken, wird die Zielseite des Eltern-Serviceanrufs geöffnet, die alle seine Kind-Serviceanrufe aufführt.

Status

Prozentsatz der Kind-Serviceanrufe für jeden Status.

Anzeige einer Zielseite des Serviceanrufs mit Kindern

Auf der Zielseite des Serviceanrufs sehen Sie die Details des Serviceanrufs. Wenn der selektierte Serviceanruf Kind-Serviceanrufe hat, hat der Bildschirm zwei Karteireiter. Standardmäßig wird die Übersicht der Kinder des Serviceanrufs gezeigt. In dem anderen Karteireiter können Sie die Details und Protokollierangaben der Eltern-Serviceanrufe sehen. Dieser Bildschirm bietet eine praktische Zusammenfassung eines Eltern-Serviceanrufs.

Serviceanrufe an einem Objekt anzeigen

Wenn Sie den Status eines laufenden Serviceanrufs an einem Objekt, wie einem Gerät oder einer Verwendungsstelle ansehen, können Sie bestimmen, ob spätere Handlungen an diesem Objekt durchgeführt werden können. Es ist auch interessant, die Vergangenheit der Serviceanrufe an einem Objekt einzusehen, um zu überprüfen, was in der Vergangenheit geschehen ist.

5

Schritte

1.   Öffnen Sie ein Objekt über Suchen.

2.   Klicken Sie in der Zielseite des Objekts auf Serviceanrufe im linken vertikalen Menü.

3.   Standardmäßig werden die Laufenden Serviceanrufe angezeigt.

Die folgenden Felder sind in einer aufgerufenen Tabelle verfügbar:

Parameter

Beschreibung

Servicanruf-ID

ID des selektierten Serviceanrufs.

Externe Referenzen

4.   Klicken Sie auf den Karteireiter Vergangenheit um die Serviceanrufe anzuzeigen, die beim selektierten Objekt nicht mehr laufen.

Die folgenden Felder sind in einer aufgerufenen Tabelle verfügbar:

Parameter

Beschreibung

Serviceanrufe

System-ID des Serviceanrufs mit einem Link zur entsprechenden Serviceanruf-Zielseite.

Externe Referenzen

Vom Anforderer mitgeteilte ID für den Serviceanruf, d.h. das System des Drittanbieters.

Die Tabelle kann gefiltert werden nach:

a. IID: Sowohl Serviceanruf-ID als auch externe ReferenzID, d.h. ID vom System des Drittanbieters.

b. Status.

c. Eingangsdatum. Standardmäßig werden die Serviceanrufe der letzten 60 Tage angezeigt.

Anzeige von Kommandos an einem Gerät, das von einem Serviceanruf verfolgt wird.

Siehe “Monitor commands on a device” für weitere Informationen zu Kommandos an einem Gerät.

5

Schritte

1.   Öffnen Sie ein Gerät über Suchen.

2.   Selektieren Sie im linken vertikalen Menübalken Kommandos und selektieren Sie das gewünschte Kommando.

3.   Suchen Sie im Detailfenster das Feld Serviceanruf. Der Wert dieses Felds ist die entsprechende Serviceanruf-ID. Klicken Sie darauf, um seine Zielseite zu öffnen.
