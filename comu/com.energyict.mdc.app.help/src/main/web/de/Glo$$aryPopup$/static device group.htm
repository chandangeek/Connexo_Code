static device group

ist eine Gerätegruppe, die aus einem Suchergebnis auf der Grundlage verschiedener Suchkriterien ausgewählt wurde. Die Zusammensetzung dieser Gruppe ändert sich im Laufe der Zeit nicht. Um neue Geräte in die Gruppe aufzunehmen, muss die Suche wiederholt und Geräte manuell eingefügt werden.
