rule template

Eine Regelvorlage ist eine Sammlung von Parametern, die für eine bestimmte Regel typisch ist. Die genauen Werte der Parameter werden in der Regel selbst definiert. Ein Beispiel für eine solche Vorlage ist die Vorlage zur Schaffung eines Problems, wenn ein Rückgang festgestellt wird. Alle Parameter werden angezeigt, aber der Anwender muss den Ablesetypen, den Trendzeitraum und den Grenzwert angeben.
