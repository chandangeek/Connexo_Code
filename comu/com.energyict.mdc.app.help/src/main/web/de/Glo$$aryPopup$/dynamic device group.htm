dynamic device group

Wird auf der Grundlage verschiedener Suchkriterien gebildet. Einzelne Geräte können nicht selektiert werden, alle Geräte der Suchergebnisse sind Teil der Gruppe. Diese Gruppe ist dynamisch, da immer wenn ein neues Gerät, welches den Suchkriterien entspricht zum System gefügt wird, es auch zur Gerätegruppe gefügt wird.
