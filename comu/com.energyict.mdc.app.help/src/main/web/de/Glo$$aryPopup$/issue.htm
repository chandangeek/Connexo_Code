issue

Ein Problem präsentiert ein Problem, das während der ausführung einer manuellen oder automatisierten Aufgabe auftritt. Normalerweise ist menschliches Einschreiten notwendig, um das Problem zu analysieren und zu beseitigen, obwohl einige Probleme einfach beseitigt werden können, indem man es automatisch erneut versucht. Die Details eines Problems und wie man es beseitigt hängen vom Aufgabentyp ab, von dem sie stammt. Probleme können einem einzelnen Anwender oder Anwendergruppe zugewiesen werden, um sie zu beseitigen.
