data logger

Ein Datenaufzeichnungsgerät ist ein Gerät, welches Daten von einem anderen Gerät in seinen eigenen Kanälen speichert. Sie wissen nicht sofort, welche Daten diese Nebengeräte haben werden. Das Datenaufzeichnungsgerät speichert keinerlei Informationen seiner Datenaufzeichnungs-Nebengeräte. Sie müssen die Nebengeräte nicht genau angeben, bevor Sie Daten erfassen können. Sie können die Nebengeräte später verlinken.
