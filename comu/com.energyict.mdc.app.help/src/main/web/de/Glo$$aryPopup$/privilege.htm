privilege

Bestimmt, welche Rechte ein Anwender hat, um bestimmte Handlungen bei einer bestimmten Ressource auszuführen. Das kann beispielsweise das Ansehen, Hinzufügen, Bearbeiten oder Löschen eines Kommunikationsservers sein. Eine Ressource ist eine Komponente einer Anwendung, z.B. Rolle ist eine Ressource sowie eine Problemregel, Lizenzen und Kommunikationsserver.
