Change the privileges of a key security accessor

You can change the security privileges to determine who can view and edit the key security accessors.

5

Steps

1.   Go to Administration > Security > Security accessors.

2.   Click the actions icon  next to the needed key security accessor and select Change privileges.

3.   In the pop-up, select the needed view and edit privileges.

4.   Click Save.

5.   Click the information icon  next to the fields View privileges and Edit privileges in the details pane of the current security accessor to see what user roles have these privileges.
