Execute a transition on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   On the overview page click the actions button and select a transition.

3.   Define the transition date.

4.   Click Next.

The status page shows the ongoing transition and reports any errors when the transition was unsuccessful.

5.   Click Finish.

Add device

5

Steps

1.   Go to Devices > Device lifecycle management.

2.   Click Add device. Complete the following fields:

Parameter

Description

Name

Give a unique name to the device. Once the device is created Connexo will automatically create a CIM compliant  (Master Resource IDentifier)

Serial number

The serial number of the device.

Device configuration

oFirst select a device type.

oSelect a configuration that you defined previously for the selected device type. Only active device configurations are shown.

Manufacturer

Manufacturer of the default, non-mandatory field.

Model number

Model number of the device, non-mandatory field.

Model version

Model version of the device, non-mandatory field.

Shipment date

Date on which the device arrived in the warehouse.

Year of certification

Date the device was last certified.

Batch

Batch number of the device

3.   Click Add.

3

Note

When you add a device and select a device configuration, then this device will inherit all parameters of the device configurations, as well as what the default connection method is.
