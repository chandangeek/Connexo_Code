Default device life cycle stages

Default stages

The life cycle of a device is divided into different stages. These stages can contain different device states. Three stages are available by default in MultiSense and these cannot be altered. The following stages are available: Pre-operational, Operational and Post-operational. These stages impose different restrictions on what can be done on a device.

 

Device options and the device life cycle stage

Depending on the stage the device state is currently residing in, different actions may not be possible or some information will not be displayed.
