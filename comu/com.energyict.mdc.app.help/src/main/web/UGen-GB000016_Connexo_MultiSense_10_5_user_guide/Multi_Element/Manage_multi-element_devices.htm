Manage multi-element devices

Add multi-element device

5

Steps

1.   See “Configure device configurations” how to a device configurations.

2.   Select the option Multi-element functionality to create a multi-element device configuration.

3.   Add load profile, register and logbook configurations.

4.   Add or import devices with this configuration create new multi-element devices in MultiSense.

Change device configuration of a multi-element device

5

Steps

1.   Select a multi-element device using Search, see “Advanced search”.

2.   Click the action button  on the landing page and select Change device configuration.

3.   Select the new device configuration.

You can go from a three element configuration to a two element configuration if there are no slave devices linked to the third element. If this is not the case then a conflicting device mapping will be generated.

4.   Click Save.

Add multi-element specific device life cycle

MultiSense foresees a multi-element specific pretransition check where it inspects whether there are still operational slaves linked to the multi-element meter that you wish to decommission.

5

Steps

1.   Add or clone a device life cycle as described here “Manage device life cycles”.

2.   Edit the Decommission transition as described in “Edit transition” and select the pretransition check Decommission: Still multi-element slaves linked to the device.

3.   Click Save.
