Import files using import folders

5

Steps

1.   Place the file containing the information in the import folder. You can find the exact location of the import folder in the import service details see “View import service details” for more information.

2.   While the file is being imported it is moved to the In progress folder.

3.   If import was successful it is moved to the Success folder, if it failed it is moved to the Failure folder.

Import files via the interface

This option is only possible of this was configured for the import service in the Admin application.

5

Steps

1.   Go to Workspace > Data exchange > Upload file for import.

2.   Select an import service from the list, only those import services that have been specifically configured are available.

The import service details are shown.

3.   Select the file.

4.   Click Upload.
