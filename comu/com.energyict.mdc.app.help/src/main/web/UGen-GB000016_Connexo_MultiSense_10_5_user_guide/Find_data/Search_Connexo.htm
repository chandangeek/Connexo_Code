Search Connexo

Understand the search feature

In Connexo MultiSense it is possible to search for different types of objects using an extensive set of parameters. The search feature can be accessed by pressing Search in the upper right corner of the page. The search function is available on every page throughout the application.

On the search page the search criteria are located on top of the page. The applied search criteria are indicated by a number on the criteria button itself. The number designates the amount of elements of that criteria that are selected. Criteria can be removed by clicking the criteria and deselecting or removing the criteria.

Different search criteria are always shown by default. Which criteria depends on the item you are searching for. You also have to option to add more criteria.

The results of the search are shown in a table below the criteria. By clicking the button Columns, you have the option to change which columns are displayed. Click the name of the column to sort the search results based on the parameter. Click again to change the order.

Important

Only the rows that are visible on the current page are sorted. The complete search result may span over multiple pages as indicated under the table containing the search result. The data on the other pages isn’t included in the sort. So, the first row on the current page isn’t necessarily the first row in the complete search result.

From the search results you can navigate to a specific item by clicking its name. There is also an option to perform actions on multiple items at once using the feature Bulk action. Finally, you can calculate the total number of found items by clicking the button Count.
