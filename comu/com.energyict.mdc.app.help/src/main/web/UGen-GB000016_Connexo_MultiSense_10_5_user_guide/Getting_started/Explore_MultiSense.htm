Understand the interface

General layout

All pages in MultiSense have a similar layout. To facilitate using the application the general layout of the application is given here.

Figure: Connexo layout.

 

A. Application center

B. Different menus: shortcut to the Workspace, Devices and Administration pages.

C. Collapse or expand navigation.

D. Return to homepage

E. Search: quick link to the search page

F. Online help and “About” information

G. Logout: place in the application to logout

H. Work area: changes depending on the feature you are viewing.

Overview page layout

When you click a specific feature within MultiSense, then in most cases an overview page of that feature will open. Throughout the application these overview pages have a similar layout.

The figure and text below show and explain the layout of the overview pages together with the actions that you can perform using the icons on the page.

Figure: Overview page layout.

 

A. Breadcrumb: represents where you are located in the application. Allows easy navigation to pages that are higher in the structure.

B. Menu displaying the different available components that are linked to the main feature.

C. Overview table: this table shows all the items that have already been added to the system. Different actions can be performed on the items in the table, e.g. they can be edited, removed, activated, viewed, etc. In most cases, the value in Name is a link referring to the item’s detail page.

D. Details pane: shows some more details of an item that is selected in the table. By default the first item is selected.

E. Add button: button designated to add new items to the table, can also be another action depending on the feature you are viewing at the moment.

F. Actions icon: clicking this icons shows the different actions that can be performed on the selected item.

G. Actions button: clicking this button shows the different actions that can be performed on the selected item. Has the same functionality as the actions icon.

H. Grid export button: clicking this button will export the content of the table to a CSV file.
