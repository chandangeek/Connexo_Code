Understand the dashboard

The dashboard in MultiSense

The dashboard allows you to perform your daily task quick and easy. Here you can find the latest status of the items that you are responsible for. The overview shows the issues assigned to you together with the server status, active connections and communications and connection and communication KPIs. You can add devices and device groups to your favourites so you can navigate to them directly.

View the dashboard

5

Steps

1.   Go to Dashboard using the vertical navigation.

The operator dashboard consists out of different widgets that allow you to track all items that acquire you attention and allow you to monitor the data collection process. The following widgets are shown:

Widget

Description

Open data collection issues

Shows all open issues that are assigned to the operator that is logged in.

My flagged devices

Quick links to the devices you flagged as favourite.

My favourite device groups

Select device groups to add to the dashboard so that they can be accessed more quickly.

Communication servers

Shows the status of all communication server.

Active connections

Shows the status of all active connections.

Active communications

Shows the status of all active communications.

2.   Refresh the dashboard by clicking Refresh in the upper right corner.
