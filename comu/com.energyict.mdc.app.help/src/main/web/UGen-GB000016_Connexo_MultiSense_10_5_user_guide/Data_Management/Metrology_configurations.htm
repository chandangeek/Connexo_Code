Metrology configurations

Manage metrology configurations

Understand metrology configuration

You can compare the metrology configuration of a usage point to the device configuration of a device. It acts like a template that helps to define the usage point setup and defines the input and output of this usage point. It controls the link between a device and a usage point because it defines the type of data a device should have before you can link it to the usage point. The output of the device needs to be the same as the input of the usage point.

Typically, many usage points will use the same metrology configuration.

View metrology configurations

Get an overview of all metrology configurations to efficiently manage them.

5

Steps

1.   Go to Administration > Usage point management > Metrology configurations.

2.   The table on the page Metrology configurations shows the available metrology configurations sorted by Name.

3.   The details pane under the table shows further details about the selected metrology configuration such as Service category and Reading types. Click the information icon next to a reading type to see its details.

Add a metrology configuration

Create a new metrology configuration to configure usage points that comply with the current market requirements.

5

Steps

1.   Go to Administration > Usage point management > Metrology configurations.

2.   Click Add metrology configuration.

3.   On the Add metrology configuration page, complete the required fields.

Name must contain a unique value.

4.   Click the Add reading types button to specify the required reading types.

5.   On the Add reading types page, use the filter panel to narrow down on the set of reading types that you need to select. Don’t forget to click Apply to show the filtered reading types or Clear all to remove the filter. By default, only the reading types that can be added are shown.

6.   Select reading types and click Add.

7.   Remove reading types by clicking the remove icon . Note that the reading type will be removed immediately without any confirmation.

8.   Click Add.

The new metrology configuration is shown in the overview table with the status Inactive.

Activate/Deactivate metrology configurations

A metrology configuration must be active to make it available to usage points. Newly added metrology configurations are inactive by default.

Steps

1.   Go to Administration > Usage point management > Metrology configurations.

2.   Select the desired metrology configuration in the table on the page Metrology configurations.

3.   Select Activate/Deactivate from the action icon  at the end of the selected line or from the Actions button in the details pane.

3

Note

Metrology configurations that are still used by at least one usage point can’t be deactivated. In this case, use Search to find the usage points that are using the concerned metrology configuration.

Edit metrology configuration

Update existing metrology configurations to modify attributes and associated reading types. For inactive metrology configurations all fields can be edited. In case of active metrology configurations only Name and Description can be edited.

5

Steps

1.   Go to Administration > Usage point management > Metrology configurations.

2.   Select the desired metrology configuration.

3.   Select Edit after clicking the actions icon  at the end of the selected line or from the Actions button in the details pane.

4.   On the Edit metrology configuration page make the needed edits.

5.   Click Save.

The adjusted metrology configuration is shown in the overview table.

Remove metrology configuration

Remove inactive metrology configurations to get rid of unused metrology configurations. Active metrology configurations can’t be removed.

5

Steps

1.   Go to Administration > Usage point management > Metrology configurations.

2.   Select the desired metrology configuration.

3.   Select Remove after clicking the actions icon  at the end of the selected line or from the Actions button in the details pane.

4.   Click Confirm.

The metrology configuration is no longer present in the Metrology configurations. It is no longer available.
