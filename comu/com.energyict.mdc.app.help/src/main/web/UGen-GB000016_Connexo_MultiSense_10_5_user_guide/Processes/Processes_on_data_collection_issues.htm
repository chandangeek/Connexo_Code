Monitor processes on data collection issues

5

Steps

1.   Go to Workspace > Issues.

2.   Select the desired issue.

3.   On the issue scroll down to the section Contextual information and click the tab Processes.

4.   All running and completed processes are listed with the most recent ones on top.

We see the following data.

Parameter

Description

Process status

Filled circle is a completed process. Open circle with black border is a running process.

Started on

Process creation date and time.

Process ID

Click this link to see all active and completed processes for the selected issue including a status overview and the corresponding variables.

Process name

Name of the process

Version

Process version

Started by

Person who started the process.

Running processes also show open tasks with the following data.

Parameter

Description

Task name

Name of the task in the process.

Task status

Current status of this task.

Task assignee

Person who is asked to execute the task.

5.   You can click the task to perform it. See “Perform tasks”.

Start processes on data collection issues

5

Steps

1.   Go to Workspace > Issues.

2.   Select the desired issue.

3.   Click the action icon  and select Start process.

4.   Alternatively, open the issue and select Start process from the Actions button.

5.   Select one of the processes available in the drop down menu.

6.   Define the process parameters - if applicable.

7.   Click Start.

3

Note

The available processes are configured in Connexo Admin by editing a process in Process management. Refer to the Connexo Admin guide for more information.
