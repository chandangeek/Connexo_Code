Configure register types

 

 

Add register types

Here you can add register types that you can later add to a device type.

5

Steps

1.   Go to Administration > Device management> Register types.

2.   Click the button Add register type.

3.   Complete the following fields:

Parameter

Description

OBIS code

Enter the 6 figure OBIS code. The drop-down list of the field Reading type is populated based on the entered OBIS code. If there aren’t any matching reading types, then click the button Add reading type and add the needed reading type on the Add reading type page. The attributes that match the entered OBIS code are already completed. See “Add reading types”.

 

You must have administrator privileges to see this button. Contact your Connexo administrator if you don’t see the button Add reading type.

Reading type

Select a reading type from the drop-down list or start typing the reading type name to filter the drop-down list. After selecting the reading type, the corresponding OBIS code is automatically completed.

4.   Click Add.

3

Note

To see the reading type details of a register click the information icon  next to the reading type in the overview table.

When a register type is being used by a certain device type then only the name of the register type can be edited. This is to prevent any harmful effects on the data collection process.

Remove register types

5

Steps

1.   Go to Administration < Device management> Register types.

2.   Click the actions icon  and select Remove.
