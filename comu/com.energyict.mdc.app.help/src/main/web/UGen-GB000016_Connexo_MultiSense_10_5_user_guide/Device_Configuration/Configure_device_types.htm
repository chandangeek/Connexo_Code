Configure device types

Add a standard device type

5

Steps

1.   Go to Administration > Device management > Device types.

2.   Click Add device type. Complete the following fields:

Parameter

Description

Type

Select Standard device type.

Communication protocol

Protocol to allow communication with the server.

Name

A unique name.

Device life cycle

Select the device life cycle all devices of this device type will use. For more information see “Device life cycle management”.

3.   Click Add.

Edit a device type

5

Steps

1.   Go to Administration > Device management > Device types.

2.   Click the actions icon and select Edit.

3.   Change the device type parameters. If the device type has active device configurations, then only the name of the device type can be altered.

4.   Click Save.

Add device icon to a device type

You can add a device icon to a device type to include a visual representation of the device. This icon will also be shown on the landing page of each device that is related to this device type.

5

Steps

1.   Open the device type for which you want to specify a device icon. Go to Administration > Device management > Device type and select the needed device type.

2.   Click the action icon and select Add device icon.

3.   On the Add device icon page, click Select file to select an image from your hard disk or any other location.

The selected file shouldn’t be larger than 500kB. The file format needs to be either JPG or PNG.

4.   Click Add.

The new device icon is now shown on the details page of the device type and on the landing page of the devices that are related to this device type..

3

Note

You can also add a device icon from the device type details page by selecting Add device icon from the actions button.

Edit or remove a device icon on a device type

You can edit or remove a device icon from a device type if the existing image no longer represents the actual device. Note that this change will be reflected also on the landing pages of all devices that are related to this device type.

5

Steps

1.   Open the device type for which you want to edit or remove the device icon. Go to Administration > Device management > Device type and select the needed device type.

2.   Click the action icon and select Edit device icon.

3.   On the Add device icon page, click Select file to select another image from your hard disk or any other location.

The selected file shouldn’t be larger than 500kB. The file format needs to be either JPG or PNG.

4.   Click Save.

5.   Click the action icon and select Remove device icon to remove the existing device icon.

6.   Click Remove to confirm the removal.

Remove device types

5

Steps

1.   Go to Administration > Device management > Device types.

2.   Click the actions icon  and select Remove.

3.   Click Remove.

Add data sources to a device type

1

Important

To be able to add register types, load profile types and logbook types to a device type, these first need to be configured. See “Configure data sources” for more information.

5

Steps

1.   Go to Administration > Device management > Device types > select a device type.

2.   In the menu on the left click Data sources > Register types > Add register type.

3.   Select the register types and click Add.

4.   In the menu on the left click Data sources > Load profiles > Add load profile type.

5.   Select the load profile types and click Add.

6.   In the menu on the left click Data sources > Logbook types > Add logbook type.

7.   Select the logbook types and click Add.
