Understand validation tasks

Data can be validated on store, but there is also an option to schedule the validation. By using validation tasks the validation of your data can be scheduled to be performed at a specific frequency.

Validation tasks can only be defined for device groups, meaning you will have to configure device groups before you can use the validation task functionality. For more information on device groups see “Manage device groups”.

1.   Go to Administration > Validation > Validation tasks.
