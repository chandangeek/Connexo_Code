Validate load profile data on request

Validate load profile data on request from the overview page

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Load profiles in the menu on the left

3.   Click the actions icon  and select Validate now.

4.   Select the start date for validation of the load profile data and click Validate.

3

Note

The option Validate now is only available when validation on the device is active and validation rules have been added. See “Manage validation status on a device” for more information.

Validate load profile data on request from the detail page

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Load profiles in the menu on the left

3.   Click a load profile name.

4.   Select the Specifications tab.

5.   Click the actions button  and select Validate now.

6.   Select the start date for validation of the load profile data and click Validate.

3

Note

The option Validate now is only available when validation on the device is active and validation rules have been added. See “Manage validation status on a device” for more information.
