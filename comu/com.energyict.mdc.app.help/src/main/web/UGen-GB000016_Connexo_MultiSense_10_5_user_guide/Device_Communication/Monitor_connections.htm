Monitor connections

Connection overview

5

Steps

1.   Go to Workspace > Data communication > Connections overview.

The connections overview page is divided in four parts showing different types of connection related data. In the upper right conner the time stamp is displayed as well as the option the refresh the view (E). You have the option to filter the results on a certain device group (A), see “View connection overview per device group” for more information.

More information on the displayed data is given in the table below.

Section

Options

Description

Summary

oActive connections (B)

Summary of all connections, showing amount of successful connections, successful connections but with at least one failed communication task, ongoing connections and failed connections. Also shows the KPI when you select a device group.

oActive communication servers (C)

Shows which communication servers are down and causing communication problems. Server status is shown when hovering over the server.

oQuick links (D)

 

Links to view all connections and the communication overview and different reports.

Connections over time (F)

oDevice groups

A graphical representation of the successful, ongoing and failed connections relative to the defined target (KPI).

Connections can be omitted or added to the graph by clicking the labels above the graph.

Overview (G)

oPer status

Detailed overview of the current status of all connections. Fore more information on the different statuses see the figure below.

oPer last result

Detailed overview of the last status of all connections.

Breakdown (H)

oPer communication pool

 

Breakdown of the number of failed, successful and pending connections per communication port pool, device type and connection method. This allows identification of problems related to a specific component.

Hovering over the bar shows the amount of pending, successful and failed connections.

View connection overview per device group

1

Important

Device groups can be defined as described in “Manage device groups”.

5

Steps

1.   Go to Workspace > Data communication > Connections overview.

2.   Select a device group from the filter on top of the page.

All connections for this device group are shown together with a graphical representation of all connections over time and the configured connection KPI.
