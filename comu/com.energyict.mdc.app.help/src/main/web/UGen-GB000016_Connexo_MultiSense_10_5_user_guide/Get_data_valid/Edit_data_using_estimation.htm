Edit data using estimation

Edit one or more values with estimator

In some cases data cannot be estimated using the estimation rules defined in the estimation rule set. Using issue creation rules MultiSense can create an issue to indicate which values were not estimated. See “Configure issue creation rules” for more information. One option to make these values no longer suspect is to manually estimate them. With this action you can still configure different parameters of the estimation process.

3

Note

You can also estimate non suspect readings.

 

5

Steps

1.   Select a device using Search, see “Advanced search” and go to Validation results or navigate directly to data that could not be estimated via Workspace > Data validation > Issues.

2.   Click the actions icon  of one suspect value and select Edit with estimator or select multiple values using the CTRL or SHIFT key and click Bulk action and select Edit with estimator.

3.   Select an Estimator.

4.   Define the estimation rule parameters.

5.   Click Estimate.

6.   If the estimated values are correct, click Save changes.

The estimated values will have the edit icon and will no longer be suspect.

Estimate one or more values with rule

When the estimation of certain values failed using the defined estimation rules, you can still manually validate these values using another estimation rule. The only prerequisite is that the reading type of estimation rule matches the reading type of the value.

In contrast to the action Edit with estimator, no parameters can be defined and the estimation rule is applied as is.

5

Steps

1.   Select a device using Search, see “Advanced search” and go to Validation results in the menu on the left or navigate directly to data that could not be estimated via Workspace > Data validation > Issues.

2.   Click the actions icon  of one suspect value and select Estimate with rule. For multiple values select the values using the CTRL or SHIFT key and click Bulk action and select Estimate with rule.

3.   Select an estimation rule. Only those rules that have the same reading type is the data are available.

4.   Click Estimate.

5.   If the estimated values are okay, click Save changes.

Edit estimation comment

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Go to Data sources > Channels using the menu on the left.

3.   If a value has the green flag estimation mark, you can click the actions icon  and select Edit estimation comment.

4.   Select the estimation comment.

5.   Click Edit.

The estimation comment can also be edited in different other actions. See “Correct data”, “Edit one or more values with estimator” and “Estimate one or more values with rule”
