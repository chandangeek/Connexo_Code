Add register data

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Registers in the menu on the left.

3.   Click the name of register you want to add data to.

4.   Click Add reading above the table. Complete the following fields (depending on the type of register):

Parameter

Description

Measurement time

Time when data was recorded.

Start of period

Start of the billing period (for billing registers only)

End of period

End of the billing period (for billing registers only)

Value

Value of the reading, numerical or text depending on the register.

5.   Click Add. The value will be added to the register and marked by the edit icon . When you hover over this icon the date of editing is shown.

Add channel data

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channels in the menu on the left.

3.   Click the name of channel you want to add data to.

4.   Select the calculated value in the table you wish to add.

5.   Type or copy/paste one or more values.

6.   Click Save changes.
