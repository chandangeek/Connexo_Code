Important

Add logbooks to devices via device configurations, see “Configure logbook configurations”.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Logbooks in the menu on the left.

3.   Select a logbook in the table, the details are shown in the details pane under the table.

You can also see a logbook’s specifications when you click the logbook name and select the Specifications tab.

View logbook data

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Logbooks in the menu on the left and click a logbook name.

You can also go directly to events independent of the logbook they belong to when you click Events in the menu on the left.

3.   Select an event in the table, details of this event are shown in the details pane below. In these details you can find the following information:

Parameter

Description

Event date

The date and time of the event.

Logbook

Link to the logbook the event belongs to.

Event type

A four digit CIM code, hover over the information icon to see more information. This will show you the device type, domain, subdomain and the event or action.

Device code

A specific code of the event provided by the protocol.

Event log id

The identifier of the event in the logbook.

Reading date

Time-stamp of the data collection.

Description

If a message was part of the event, this text is displayed here.

Event attributes

Specific attributes defined by the protocol.

Change next reading block start on a logbook

Change the start timestamp of the reading block that will be collected during the next collection phase of a logbook. This allows you to re-read the events in a logbook, e.g. if data is exceptionally different than usual. Or your may want to avoid collecting events from the past that you are not interested in, e.g. because you know that the device was broken or behaving suspiciously.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Logbook in the menu on the left.

3.   On the line of the desired logbook, click the actions icon  and select Change next reading block start.

4.   In the pop-up dialogue that appears, pick the next reading block start date.

5.   Click Save.

3

Note

The action Change next reading block start is also available from the Actions button on the logbook details pane and on its details page.
