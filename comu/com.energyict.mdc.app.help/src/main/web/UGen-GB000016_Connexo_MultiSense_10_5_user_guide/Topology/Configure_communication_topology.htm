Configure communication topology

Set master device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   On the landing page in the Communication topology pane there are 4 different scenario’s:

Scenario

Options

Device is directly addressable

Because the device can be addressed directly, it is not possible to assign a master device to this device. The device itself however can be a master for another device.

If you select this option, you will be able to manage connection methods on the device configuration.

Device is not directly addressable

A master device needs be assigned to allow remote communication. It is not possible to set up a connection if the device is not directly addressable and does not have a master device.

Device can be a gateway

A slave device can be assigned to this device. The most recently added slave devices are shown.

Device cannot be a gateway

The device cannot have slave devices.

3.   Click the edit icon  to add or edit the master device. Click the remove icon  to remove the master device.

4.   Select a master device from the list.

5.   Click the green checkmark.
