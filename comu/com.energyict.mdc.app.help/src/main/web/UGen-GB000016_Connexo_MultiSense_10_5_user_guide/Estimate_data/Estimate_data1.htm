View estimation tasks

5

Steps

1.   Go to Administration > Data estimation > Estimation tasks.

The overview page shows a table containing all configured estimation tasks. You can see the status of every task and when the next run is scheduled. The details pane below the table shows the details of the estimation task selected in the table.

Add estimation tasks

5

Steps

1.   Go to Administration > Data estimation > Estimation tasks.

2.   Click Add estimation task. Complete the following fields:

Category

Parameter

Description

 

oName

oUnique name

 

Log level

Define the log level needed for the execution of the estimation tasks. Options are Error, Warning, Information, Debug and Trace. Where the level error only creates loggings if errors occur and trace gives a detailed description of all executions.

Warning is the default log level.

Data sources

oDevice group

oSelect the device group to estimate the data of.

Schedule

oRecurrence

oStart on

  • Indicate the recurrence of the task.

  • Define the start date of the task.

Data options

oEstimation period

oDefine the period of data that needs to be estimated.

3.   Click Add.

Edit estimation tasks

5

Steps

1.   Go to Administration > Data estimation > Estimation tasks.

2.   Click the actions icon  and select Edit.

3.   Change the estimation task parameters.

4.   Click Save.

Remove estimation tasks

5

Steps

1.   Go to Administration > Data estimation > Estimation tasks.

2.   Click the actions icon  and select Remove.

3.   Click Remove.
