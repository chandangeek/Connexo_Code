Configure estimation rules

Add estimation rule set

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets.

2.   Click Add estimation rule set. Complete the following fields:

Parameter

Description

Name

Give an a name to the estimation rule set.

Description

Give a short description for this estimation rule set.

3.   Click Add.

Edit estimation rule sets

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets.

2.   Click the actions icon  and select Edit.

3.   Change the estimation rule set parameters.

4.   Click Save.

Remove estimation rule sets

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets.

2.   Click the actions icon  and select Remove.

3.   Click Remove.

Add estimation rules

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets. > click an estimation rule set > click Estimation rules in the menu on the left.

2.   Click Add estimation rule. Complete the following fields:

Parameter

Description

Estimation rule

Give a name to the estimation rule.

Estimator

Select the estimator this parameter defines how estimation will be performed. Depending on the estimator you select you may need to configure more parameters. See “Default estimators” for more information.

Reading types

Select the reading types to define the type of data that will be estimated. Reading types provide the link with device configurations.

3.   Click Add.

4.   Click the actions icon  next to the estimation rule and select Activate.

Edit estimation rules

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets. > click an estimation rule set > click Estimation rules in the menu on the left.

2.   Click the actions icon  and select Edit.

3.   Change the estimation rule parameters.

4.   Click Save.

Remove estimation rules

1.   Go to Administration > Data estimation > Estimation rule sets. > click an estimation rule set > click Estimation rules in the menu on the left.

2.   Click the actions icon  and select Remove.

3.   Click Remove.

Change the order of estimation rules in an estimation rule set

It is important to set the order of the estimation rules in the set because suspect values can only be estimated using one estimation rule. This is why the estimation rule that results in the most realistic values should be put on top in the set.

5

Steps

1.   Go to Administration > Data estimation > Estimation rule sets. > click an estimation rule set > click Estimation rules in the menu on the left.

2.   Click Edit order.

3.   Drag and drop the estimation rule up or down and release when the green check appears.

4.   Click Undo to cancel changing the order or click Save order to save the changes.
