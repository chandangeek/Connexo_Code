Manage time of use calendars

Configure time of use calendars on device types

Add a time of use calendar to the selected device type

Before you can start using time of use calendars on devices, they must be assigned to a device type.

5

Steps

1.   Go to Administration > Device types > Open a device type > Specifications > Time of use calendars.

2.   In the tab ToU calendars, click the button Add ToU calendars.

3.   On the Add time of use calendars page, select the desired time of use calendars.

4.   Click Add to assign the selected time of use calendars to the current device type.

Remove a time of use calendar from a device type

You can remove a time of use calendar if it is no longer required.

5

Steps

1.   Go to Administration > Device types > Open a device type > Specifications > Time of use calendars.

2.   In the tab ToU calendars, select a time of use calendar.

3.   Click the action icon  in the selected row or the click the Actions button in the details pane. Then click Remove.

3

Note

The time of use calendar can be re-added if needed. If it is no longer assigned to any device type, the time of use calendar can be removed from Connexo in Connexo Admin.

Edit time of use specifications

You can view and edit the specifications for the use of time of use calendars with the current device type.

5

Steps

1.   Go to Administration > Device types > Open a device type > Specifications > Time of use calendars.

2.   Click the tab Specifications.

3.   Click the Actions button and select Edit if you want to edit the time of use specifications.

Parameter

Description

Time of use allowed

Default value is No.

 

If time of use is NOT allowed:

  • Specification tab is the default tab.

  • The Add ToU calendars button is disabled.

  • Time of use calendar actions such as View preview and Remove are disabled.

  • On devices of the current device type, there is no time of use menu item.

 

If time of use is allowed:

  • ToU calendars tab is the default tab.

  • Time of use calendars can be added, previewed and removed.

  • There is a time of use menu item on devices of the current device type.

Time of use options

The available time of use options are defined by the protocol that is used by the device type. You can select multiple time of use options by clicking the corresponding checkboxes.
