Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Load profiles in the menu on the left.

3.   Click the actions icon   or the actions button and select View channels.

4.   Select a channel, details are shown in the details pane below.

5.   View more details by clicking the channel name or by clicking the actions icon and selecting View details.

View channel data in a table

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Load profiles in de menu on the left.

3.   Click the actions icon  and select View channels.

4.   Click the actions icon  of a channel and select View data.

5.   Click Table view in the upper right corner.

Values (for non cumulative data), Cumulative values and Delta (for cumulative data) are shown in the columns. There is also a column for the internal flags.

Details of the value of the channel together with some general information are shown in the details pane on the bottom of the page when you select the interval in the table.

View channel data in a graph

5

Steps

1.   Search for a device > click device name > click Load profiles.

2.   Click the actions icon  and select View channels.

3.   Click the actions icon next to the channel and select View data.

4.   Click Graph view in the upper right corner.

Channel data is displayed in A in the screenshot below. The unit of the channel is displayed on the left side of the graph. Details of the channel for one time point are shown when hovering over the value.

You can zoom in or out by selecting a standard value, this can be 4 hours, 1 day, 2days or 1 week (B. in the screenshot), but these are dependent of the configured interval. A second option to zoom in or out is to change the time window in the field indicated by C in the screenshot.

On the bottom of the page (D in the screenshot) there is a bar that shows where the interval that is displayed positions in a larger time frame. The displayed interval can be shifted by sliding the bar to the left or the right. You can also make the interval larger or smaller by making the scroll bar wider or smaller respectively.
