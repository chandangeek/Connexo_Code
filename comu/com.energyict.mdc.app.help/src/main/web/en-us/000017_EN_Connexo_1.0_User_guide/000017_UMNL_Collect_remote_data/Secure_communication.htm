Secure communication

Understand secure communication

To guarantee that the data to and from the devices remains secure, certain measures have to be taken. In Connexo MultiSense this is accomplished via the use of security settings.

Security settings consist out of authentication and encryption. Authentication ensures that both ends of the connection are secure while encryption insures that the information within a session is not compromised.

In the System administration app the system administrator adds privileges to different user roles. These define what a user is allowed to do within the different parts the applications. In Connexo MultiSense there is also a possibility to manage privileges on a device configuration level. This allows you to define what certain users can do on single device types. For example a group of users that work with water meters will have full access to these meter, but will not have any privileges on gas meters.
