Manage metrology configuration versions on a usage point

Add metrology configuration version to a usage point

Define versions of metrology configurations on a usage point to overview and export data provided by reading types that will be specified on these metrology configurations.

5

Steps

1.   Search for a usage point as described in “Search for usage points”.

2.   Click History in the menu on the left.

3.   Select the Metrology configuration tab.

4.   Click Add metrology configuration version above the table listing the history of metrology configuration versions.

5.   On the Add metrology configuration version screen, complete the required fields.

Parameter

Description

Metrology configuration

Select a metrology configuration. The menu only shows active metrology configurations with the same service category as the usage point.

Start date

Define the start date of when this version of the metrology configuration will be active. The tart date must be later than the start date of the previous version. If the end date of the previous version is specified, then the start date of the new metrology configuration version must be later than or the same as the end date of the previous version.

End date

Define the end date of the version or select none. The metrology configuration version will be active from the start date until the start date of the next metrology configuration version. End date of the new metrology configuration version must be later than its start date.

If the usage point already has meter activations, then reading types are validated. The reading types required by the new metrology configuration version must match the reading types provided by meters for the corresponding period of time.

6.   Click Add.

The new metrology configuration version is now shown in the table under the tab Metrology configuration on the History page.

Edit a metrology configuration version on a usage point

You can update existing metrology configuration versions on a usage point to correct eventual mistakes made during its creation.

The edit action is only available for current metrology configuration versions or for versions in the future, i.e. the end date must be in the future. Versions in the past can’t be modified.

For current metrology configuration versions the metrology configuration and start date can’t be edited. In this case only the end date can be edited.

5

Steps

1.   Select the tab Metrology configuration on the History page of the desired usage point.

2.   Select the desired metrology configuration version.

3.   Select Edit from the action icon  or click the Actions button in the details pane.

4.   On the Edit metrology configuration version page, make the desired edits.

  • Start date can’t be overlapped by that of other versions.

  • End date can’t be overlapped by that of other versions.

  • End date is later than or the same as the current date.

  • End date must be later than the start date.

  • f the usage point already has meter activations, then reading types are validated. The reading types required by the new metrology configuration version must match the reading types provided by meters for the corresponding period of time.

5.   Click Save.

The adjusted metrology configuration version is now shown in the table under the tab Metrology configuration on the History page.

Remove a metrology configuration version from a usage point

Remove versions of metrology configurations from a usage point to keep a clean list of metrology configuration versions. Eventual versions that were created by mistake can be removed.

Only metrology configuration versions in the past, i.e. with both start and end date in the future, can be removed.

5

Steps

1.   Select the tab Metrology configuration on the History page of the desired usage point.

2.   Select the desired metrology configuration version.

3.   Select Remove from the action icon  or click the Actions button in the details pane.

4.   Click Confirm.

The metrology configuration version is no longer present in the table under the tab Metrology configuration on the History page. You are no longer able to view and use data from reading types specified on this metrology configuration.

3

Note

Dates of other metrology configuration versions on the usage point remain unchanged. Gaps between versions are allowed.
