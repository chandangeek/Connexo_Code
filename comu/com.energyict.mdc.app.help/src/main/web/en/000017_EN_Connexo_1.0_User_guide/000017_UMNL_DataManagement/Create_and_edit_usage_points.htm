Create and edit usage points

Create and update usage points using file import

Use the import framework to add or update multiple usage points from a csv file. For more information on import services see “Data import” or for configuration see Import services in the chapter System configuration of the Admin user guide.

1

Important

Prerequisites:

  • The default import service Usage Point Importer must be correctly configured and activated.

  • The corresponding folders for holding the import file must be available on the Connexo server.

  • The usage point import service must be added to an application server.

ImportantThe usage point data must be available in a CSV file. The first line in this file must contain the column headers and the fields must be separated by a semicolon or a comma. The following fields are expected in the usage point import file.

3

Note

The metrology configuration lines are only applicable for the creation of new usage points. The value is skipped if the line is meant to update an existing usage point. It is not possible to update the metrology configuration of existing usage points because a usage point can have multiple metrology configuration versions. It would not be clear which version to update.

5

Steps

1.   Put the CSV file with the usage points in the import folder.

2.   The application server will pick up the import file for further processing. At this moment the import file will be moved to the folder In progress.

3.   When the import is finished, the import file is moved either to the folder Success or to the folder Failure.

4.    Check the created usage points using the search function. See “Advanced search”.

5.   If there was a problem, you can troubleshoot by looking at the log in import service history. See “Monitor data import”.

Add a usage point manually

Next to importing a number of usage points simultaneously using a file importer you can also create a usage point in Connexo MultiSense by entering its data manually on a form. Once the usage point is created, you will be able to link devices to it.

5

Steps

1.   Go to Usage points > Add usage point.

2.   Complete the fields on screen. All fields - except for Name - are required.

Parameter

Description

MRID

MRID of the usage point. This field is unique for each usage point.

Name

This is a optional field allowing you to enter a relevant name for the usage point.

Service category

Drop down containing all active service categories in the system.

Created

Date of the physical creation of the usage point. Default value is Now.

3.   Click Add.

You are directed to the landing page of the usage point you just created.

Edit a usage point

Existing usage points can be edited in Connexo MultiSense to correct eventual mistakes during creation.

5

Steps

1.   Search for a usage point as described in “Search for usage points”.

2.   Click the usage point to see its landing page.

3.   Select Edit from the Actions button.

4.   Edit the relevant fields.

MRID and Service category can’t be edited.

5.   Click Save.

You are directed back to the usage point landing page that shows the adjusted data.
