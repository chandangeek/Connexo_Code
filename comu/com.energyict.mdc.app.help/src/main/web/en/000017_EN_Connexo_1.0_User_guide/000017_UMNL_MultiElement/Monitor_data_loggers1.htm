View channel data on multi-element device

5

Steps

1.   Select a data logger device using Search, see “Advanced search”.

2.   Go to Data sources > Channels in the menu on the left. The table shows all available channels together the slave they are linked to. Click a channel name.

3.   Select the Readings tab.

4.   See “Monitor channel data” for more information on how to view channel data.

If the device is linked to a multi-element slave then the bars of data in the graphic will be grey. When you hover over the data the pop-up will show the slave devices linked to this data. If you select a value in the table with all the readings, you can find a link to the  slave devices on the General tab of the details pane on the bottom of the page.

View register data on multi-element device

5

Steps

1.   Select a multi-element device using Search, see “Advanced search”.

2.   Go to Data sources > Registers in the menu on the left. Click a register name.

3.   Select the Readings tab.

4.   See “Monitor register data” for more information on how to view register data.

If you select a value in the table with all the reading, you can find a link to the data logger slave devices on the General tab of details pane on the bottom of the page.
