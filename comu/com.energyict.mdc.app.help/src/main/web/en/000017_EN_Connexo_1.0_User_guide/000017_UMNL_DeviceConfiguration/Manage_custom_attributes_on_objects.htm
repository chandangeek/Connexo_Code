Manage custom attributes on objects

View custom attributes values on device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Go to Device attributes in the menu on the left.

This page shows all device custom attributes added and configured on this device together with a link to the versions if these are available. The custom attribute sets linked to registers and channels can be viewed on the register and channel pages.

Edit custom attributes values on device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Go to Device attributes in the menu on the left.

3.   Click the actions button  and select Edit ‘Name custom attribute set’. Every set of attributes can be edited separately.

4.   Edit the attribute values or restore the default values by clicking the restore button  or

5.   Click Save.

View custom attributes values on register

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Go to Data sources > Registers in the menu on the left.

3.   Select a register in the table, the attribute values are shown in the details pane on the bottom of the page. Or click the register name and select the Specifications tab.

Edit custom attributes values on register

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Go to Data sources > Registers in the menu on the left.

3.   Select a register in the table or click the register name and select the Specifications tab and click the actions icon  or actions button  and select Edit ‘Name custom attribute set.

4.   Edit the attribute values or restore the default values by clicking the restore button  or

5.   Click Save.
