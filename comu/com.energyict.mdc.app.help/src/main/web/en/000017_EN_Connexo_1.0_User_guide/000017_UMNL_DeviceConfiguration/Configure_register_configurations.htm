Configure register configurations

Add register configurations

1

Important

Add register types to device types to be able to add and configure register configurations.

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration.

2.   Click Register configurations in the menu on the left.

3.   Click Add register configuration. Complete the following fields:

Parameter

Description

Register type

Selection out of the list of register types that were previously added to the device type

OBIS code

The listed OBIS code is derived from the register type, can be overruled if the OBIS code of the device configuration is different than the OBIS code of the device type.

Store value as

Number: store register data as numerical value

Text: store register data as text

Number of fraction digits

Amount of digits after the decimal, only available when the data type of the register is number

Overflow value

The overflow value defines the maximum value that can be stored.

Use Multiplier

Indicate whether a multiplier should be used. Only use when the consumption data needs to be multiplied a specific value to get the actual consumption rate.

The value of the multiplier can be set on the device.

Collected reading type

The reading type of the data that is collected from the device.

Calculated reading type

The reading type of the calculated data, this reading type is used when values need to be calculated as consequence of a multiplier or to calculate delta values when bulk values were collected.

4.   Click Add.

3

Note

Register configurations can only be added to inactive device configurations. To is to prevent negative effects on the data collection process

On the overview page of the register the applied validation rules are shown. From here you can go to the validation rules page, for more information see “Data validation”.

Edit register configuration

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration > click Register configurations in the menu on the left.

2.   Click the actions icon  and select Edit.

3.   Change the register configuration parameters.

4.   Click Save.

Remove register configuration

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration > click Register configurations in the menu on the left.

2.   Click the actions icon  and select Remove.

3.   Click Remove. If the device configuration is active, register configuration can not be removed.
