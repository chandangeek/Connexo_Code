Configure load profile types

Add load profile types

5

Steps

1.   Go to Administration > Device management > Load profile types.

2.   Click Add load profile type. Complete the following fields:

Parameter

Description

Name

A unique name.

Interval

Defines the interval of all channels belonging to this load profile.

OBIS code

Provide the 6 attributes of the OBIS code separated by “.”.

Reading types

Click Add reading type to select one or more reading types and click Add. These reading types will be used to define channels on the load profile configuration.

3.   Click Add

3

Note

Only the name of the load profile type can be edited when a load profile type is used by a device type. This is to prevent any harmful effects on the data collection process.

Edit load profile type

5

Steps

1.   Go to Administration < Device management> Load profile types.

2.   Click the actions icon  and select Edit.

3.   Change the load profile parameters.

4.   Click Save.

Remove load profile type

5

Steps

1.   Go to Administration < Device management> Load profile types.

2.   Click the actions icon  and select Remove.

3.   Click Remove.
