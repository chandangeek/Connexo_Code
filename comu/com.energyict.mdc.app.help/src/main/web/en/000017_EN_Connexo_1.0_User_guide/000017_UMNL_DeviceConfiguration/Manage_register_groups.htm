Manage register groups

Understand register groups

Registers can be grouped together. Register groups are used to collect different register readings that belong together (for example in communication tasks). A group will contain all necessary information needed for a certain process. E.g. status registers are used to collect all relevant information for when there is a problem with a device. This group contains registers for firmware versions, battery status, etc. Registers can only belong to one group.

Add register groups

5

Steps

1.   Go to Administration > Device management.

2.   Click Register groups.

3.   Click the actions icon to edit or remove register groups.

4.   Click Add register group. Give a unique name to the group and select register types.

5.   Click Save.

3

Note

To see the reading type details click the information icon  next to the register in the overview table.

Edit register groups

5

Steps

1.   Go to Administration > Device management> Register groups.

2.   Click the actions icon  and select Edit.

3.   Change the name and/or (de)select register types.

4.   Click Save.

Remove register group

5

Steps

1.   Go to Administration > Device management> Register groups.

2.   Click the actions icon  and select Remove.

3.   Click Remove.
