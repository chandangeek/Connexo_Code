Configure logbook configurations

Add logbooks

1

Important

Add logbook types to device types to be able to add and configure logbook configurations.

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration.

2.   Click Logbook configuration in the menu on the left.

3.   Click Add logbook configuration.

4.   Select one or more logbook configurations.

5.   Click Manage logbooks in the upper right conner to add extra logbook types.

6.   Click Add.

3

Note

Logbook configurations can only be added to inactive device configurations. This is to prevent any negative effects on the data collection process.

Edit logbook configuration

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration > click Logbook configurations in the menu on the left.

2.   Click the actions icon  and select Edit.

3.   Overrule the logbook’s OBIS code.

4.   Click Save.

Remove logbook configuration

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration > click Logbook configurations in the menu on the left.

2.   Click the actions icon  and select Remove.

3.   Click Remove. If the device configuration is active, logbook configuration can not be removed.
