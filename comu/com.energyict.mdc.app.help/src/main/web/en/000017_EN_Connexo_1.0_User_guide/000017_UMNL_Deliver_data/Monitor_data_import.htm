Monitor data import

View import service details

5

Steps

1.   Go to Administration > Data import > Import services.

The overview table shows all configured import services. The system administrator can add and configure these import services in Admin. The table shows the file importer used together with the status of the service, the imported folder and the file pattern.

2.   Select an import service in the table, the details pane on the bottom of the page shows some general details.

3.   Click the import service name or click the actions icon  and select View details to see more extensive details.

View import service history

5

Steps

1.   Go to Administration > Data import > Import history.

2.   Click the import service name and then select History in the menu on the left or click the actions icon  and select View history.

The history shows every started import together with the duration and status. When you select an item in the table the details pane on the bottom of the page shows specific details of this import.

3.   Click the actions icon  and select View log to see a detailed logging of the history item.
