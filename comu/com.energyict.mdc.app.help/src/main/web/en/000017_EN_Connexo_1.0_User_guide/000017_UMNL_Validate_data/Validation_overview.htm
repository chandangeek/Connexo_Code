Validation overview

Understand data validation KPIs

To be able to see all suspects on all devices of a specific device group MultiSense  uses data validation KPIs. These KPIs calculate the amount of suspects of a device group periodically as defined during configuration.

View data validation KPIs

5

Steps

1.   Go to Administration > Data validation > Data validation KPIs.

The table shows all configured data validation KPI together with the device group they apply to and how often they are calculated. You can also see when the KPI was last calculated.

Add data validation KPI

5

Steps

1.   Go to Administration > Data validation > Data validation KPIs.

2.   Click Add data validation KPI. Complete the following fields:

Parameter

Description

Device group

Select the device group to calculate the data validation KPI of. You can only create one data validation KPI per device group.

Calculation frequency

Select how often the KPI should be recalculated.

3.   Click Add.

Remove data validation KPI

5

Steps

1.   Go to Administration > Data validation > Data validation KPIs.

2.   Click the remove icon next to the KPI you wish to remove.

3.   Click Remove to confirm.

View validation results in validation overview

5

Steps

1.   Go to Workspace > Data validation > Validations.

2.   Use the filter to see specific results, see “Filter results in validation overview” for more information.

The table shows all devices that comply with the filter criteria. For every device you can see its name, serial number, device type and configuration, the total amount of suspects on this device and whether all data has been validated.

Beware that the results on the overview are only renewed after the time you specified during the configuration of the data validation KPI.

3.   Click the device name link to go to the device and edit the suspect values as described here “Edit data”.

Filter results in validation overview

5

Steps

1.   Go to Workspace > Data validation > Validations.

2.   Use the filter pane on top of the page to filter the validation results. The following filters are available:

Filter criteria

Description

Device group

Select a device group of which you want to see the suspects. The device groups you see here match the data validation KPIs configured in Administration (“View data validation KPIs”). You can select multiple device groups.

Period

Specify a period for which you want to see the validation results.

Amount of suspect

Use this filter to only see device with the specified amount of suspects.

Validator

Use this filter to only see devices that contain suspects that violated validation rules using the validators specified in the filter.

3.   Click Apply.

4.   Click Clear all to remove all specified filters.
