Manage validation on a device

In Connexo MultiSense it is possible to add device configurations to validation rule sets and vice versa. This ensures that the data collected by the device will be validated.

Add validation rule sets to a device configuration

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations in the menu on the left > select a configuration > Reading quality > click Validation rule sets.

2.   Click Add validation rule sets.

3.   Add one or more validation rule sets. Clicking the name of the validation rule (set) will redirect to the corresponding page.

4.   Click Add.

3

Note

Only validation rule sets that share the same reading type(s) as the device configuration will be available.

Validation rule sets can also be added to device configurations via Administration > Validation > Validation rule sets. Automatically these rule sets will be shown on the device configuration homepage. See “Add device configurations to validation rule sets”.

Remove validation rule sets from a device configuration

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations in the menu on the left > select a configuration > Reading quality > click Validation rule sets.

2.   Click the actions icon  and select Remove.

3.   Click Remove.

Add device configurations to validation rule sets

5

Steps

1.   Go to Administration > Validation > Validation rule sets > click a validation rule set > click Device configurations in the menu on the left.

2.   Click Add device configurations. Select one or more device configurations.

3.   Click Add.

The validation rule set will also be visible on the selected device configuration(s).

3

Note

Only device configurations that have the same reading type(s) as the validation rules will be available and can be added.

Manage validation status on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Validation configuration under Reading quality in the menu on the left.

3.   Click Activate data validation.

When data validation is activated after a period of deactivation and/or the device has already readings, then select the date from which you want to start validating the data and select if you want to validate now or wait for new data. When you click Activate & Run validation will start immediately. When you select Validate data on storage, validation will only start once a new data interval is stored.

4.   Click Deactivate data validation to stop validating data after it has been collected.

3

Note

When data validation is deactivated on a device level none of the validation rules will be applied even if the validation rule(s) (sets) are active.

Depending on the state of the device it may not be possible to (de)activate validation.

Manage validation status of a single validation rule sets on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Validation configuration under Reading quality in the menu on the left.

3.   Click the actions icon  next to the validation rule set and select Activate/Deactivate.
