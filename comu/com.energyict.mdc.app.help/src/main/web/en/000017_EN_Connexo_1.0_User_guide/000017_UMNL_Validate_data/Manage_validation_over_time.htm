Manage validation over time

Different version of validation rule sets can be created to define the validity period of the validation rule so you can switch between applicable rules according to a calendar.

View validation rule set versions

5

Steps

1.   Go to Administration > Validation > Validation rule sets > select a validation rule set.

On the overview page you can see the different version that you configured previously. The period of the version shows when this version was, is and will be used for validation.

2.   Select a version to see which validation rules are included together with the validation rule details. You can also see these details when you click the validation rule set version name and then select the Validation rules in the menu on the left.

3

Note

There is always at least one version available per validation rule sets. The period of this version will be always. Once new versions are added the period will be updated according to the newly added version.

Add validation rule set version

5

Steps

1.   Go to Administration > Validation > Validation rule sets > click a validation rule set name > click Validation rules in the menu on the left.

2.   Click Add version. Complete the following fields:

Parameter

Description

Description

Give a short description to explain the use of the validation rule set version.

Start

Define when the new version should start.

3.   Click Add.

4.   Next add validation rules as described in “Add validation rules”.

Clone validation rule set version

5

Steps

1.   Go to Administration > Validation > Validation rule sets > click a validation rule set name > click Validation rules in the menu on the left.

2.   Click the actions icon  and select Clone. Edit the following fields:

Parameter

Description

Description

Give a short description to explain the use of the validation rule set version.

Start

Define when the new version should start.

3.   Click Add.

4.   Edit, remove or add validation rules as needed, see “Add validation rules” for more information.

Edit validation rule set versions

5

Steps

1.   Go to Administration > Validation > Validation rule sets > click a validation rule set name > click Validation rules in the menu on the left.

2.   Click the actions icon  and select Edit.

3.   Edit the version and click Save.

Remove validation rule set versions

5

Steps

1.   Go to Administration > Validation > Validation rule sets > click a validation rule set name > click Validation rules in the menu on the left.

2.   Click the actions icon  and select Remove.

3.   A warning message is shown, click Remove.
