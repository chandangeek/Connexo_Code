Configure validation rule sets

5

Steps

1.   Go to Administration > Validation

2.   Click Validation rule sets.

3.   Remove or edit validation rule sets via the actions icon .

4.   Click Add validation rule set to create a new set. Complete the following fields:

Parameter

Description

Name

A unique name.

Description

A description of the rule set.

5.    Click Add.

Configure validation rules

5

Steps

1.   Go to Administration > Validation > Validation rule sets

2.   Click a validation rule set.

3.   Click Validation rules in the menu on the left.

4.   Edit or remove validation rules via the actions icon .

5.   Click Add validation rule. Fill in the following fields:

Parameter

Description

Name

A unique name

Validator

Select how to perform validation. Depending on the validation rule selected, extra parameters may need to be defined.

Reading type(s)

Multiple reading types can be added as described in “Add reading type”.

6.   Click Add.

7.   Click the actions icon to activate the rule.

3

Note

The following validators are available by default:

Validator

Description

Check missing values

Check for missing data. This data can be provided in a later stage by estimation and editing.

Interval state

Check interval data for an interval flag. An interval flag represents an event or alarm that can occur during the interval, e.g. when power is down.

Register increase

Check consumption over time. If the value of the register is lower or equal to the previous register value then this data is marked as suspicious.

Check Fail equal data if you want to mark equal values as suspicious.

Threshold violation

Checks each interval value against a minimum and maximum value. Values outside these boundaries are marked as suspicious.

Add reading type

5

Steps

1.   Click Add reading types.

2.   Complete the filter fields to find the correct reading types.

3.   Click Apply.

Click the remove icon  to remove single filters or click Clear all to remove all applied filters.

4.   Select to add all reading types of the filter result or make a custom selection and select one or multiple reading types from the table.

You can see the reading type details by clicking the information icon .

5.   Click Add.
