Understand data estimation

What is data estimation

Data estimation corrects suspect values, detected by the validation process, using  ‘predefined’ algorithms. With data estimation you can estimate the data based on valid collected data or using custom values.

In some cases data cannot be estimated and needs to be edited, see “Data editing” for more information.

Data estimation in MultiSense

The configuration of data estimation is similar to that of data validation. Estimation rules are grouped in estimation rule sets. Within the rule set you define the order of the rules to select which estimation rule will be used first. Typically the rules that result in the most realistic values are put on top of the list. Once the value is estimated with one of the rules this value is no longer available for estimation by the other estimation rules in the set.

In MultiSense different default estimators are already available upon installation, these estimators still need to be configured further in the estimation rule. The estimators cover all basic needs. You can find a more detailed description of these estimators in the topic “Default estimators”.
