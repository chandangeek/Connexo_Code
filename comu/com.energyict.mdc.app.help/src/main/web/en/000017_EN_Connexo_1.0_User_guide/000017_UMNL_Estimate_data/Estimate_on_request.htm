Estimate on request

Run estimation tasks on request

5

Steps

1.   Go to Administration > Data estimation > Estimation tasks.

2.   Click the actions icon  next to the estimation task and select Run or click the estimation task click the actions button  on the overview page and select Run.

This task will be queued to be executed as soon as possible.

Manually trigger estimation of one or more values

5

Steps

1.   Select a device using Search, see “Advanced search” or navigate directly to data that could not be estimated via Workspace > Data validation > Issues.

2.   Click the actions icon  of one suspect value and select Estimate value or select multiple values using the Ctrl or Shift key and click Bulk action and select Estimate value.

3.   Select an Estimator.

4.   Define the estimation rule parameters.

5.   Click Estimate.

6.   If the estimated values are correct, click Save changes.

The estimated values will have the edit icon and will no longer be suspect.
