Monitor data estimation

View estimation task history

5

Steps

1.   Go to Administration > Data estimation > Estimation tasks.

2.   Click the estimation task and select History in the menu on the left or click the actions icon  and select View history.

The history view shows every executed estimation task its status and its duration. The details pane below the overview table shows more details of the estimation task such as when it started and finished.

3.   Click the actions icon  and select View log to see a detailed logging of the task.
