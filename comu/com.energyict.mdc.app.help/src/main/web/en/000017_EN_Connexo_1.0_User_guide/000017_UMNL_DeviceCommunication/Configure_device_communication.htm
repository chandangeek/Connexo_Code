Configure device communication

Understand device communication

In “Configure communication infrastructure” different parts of the communication infrastructure are set up. These settings are developed in such way that they can be used by different device types. This typically includes the communication servers and communication ports as well as communication tasks.

However for a device in Connexo MultiSense to be able to connect and collect data, extra parameters need to be configured. For example you need to define the connection method using a specific communication port pool and then also specify the connection details such as host and port number in the case of a TCP connection. First these settings are configured on a device configuration level, later these can be further specified or adjusted on the device itself.
