Configure connection methods on a device

Understand connection methods on a device

Device connection methods are inherited from the device configuration. All parameters and connection details are the same, however in Connexo MultiSense  it is possible to change these settings on a device level. This is needed to specify the connection details that are device specific such as IP addresses.

Add outbound connection method on a device

1

Important

All parameters of the device connection method are inherited from the device configuration. Make sure that connection methods are configured in the device configuration.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Connection methods in the menu on the left.

3.   Click Add outbound connection method or Add connection method > Add outbound connection method. Complete following fields:

Parameter

Description

Configuration connection method

Select a connection method configured on the device configuration. This method can only be used once per device. After saving, this parameters can no longer be changed.

Protocol dialect

Add one of the protocol dialects available on the device.

Communication port pool

Select communication port pools available on the selected configuration.

Connection strategy

As soon as possible: connect whenever a communication task needs a connection.

Minimize connections: only connect with the device at the scheduled time you specified.

Connection schedule

Only available when “Minimize connections” is selected. The schedule when to set up a connection.

Connection window

Is the time window during which it is allowed to set up a connection and communication tasks can be executed.

Number of simultaneous connections

Allows multiple connections at the same time to make parallel scheduling of devices possible. Not possible when “Minimize connections” is selected.

By default 1, maximum 16 simultaneous connections allowed.

Active

Activate the connection method.

4.   Complete or change Connection details.

5.   Click Add.

6.   Click the actions icon to activate the connection method and set as default.

3

Note

It is possible to save an incomplete connection method when not all acquired fields are filled in. In this case a message is shown to ask whether you are certain to save the incomplete connection method. The connection method will appear in the overview table, but will have the status incomplete.

Edit connection method on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Connection methods in the menu on the left.

3.   Click the actions icon  and select Edit.

4.   Change the connection method parameters.

5.   Click Save.

Remove connection method on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Connection methods in the menu on the left.

3.   Click the actions icon  and select Remove.

4.   Click Remove.
