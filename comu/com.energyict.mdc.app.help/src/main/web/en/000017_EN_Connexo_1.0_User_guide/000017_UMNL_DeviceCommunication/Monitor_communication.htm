Data collection KPIs

Understand data collection KPIs

Key performance indicators or KPIs are used to measure success. In the case of data collection KPIs it measures the success of data collection. More specifically it states how many connections and/or communication tasks should be performed successfully.

View data collection KPIs

5

Steps

1.   Go to Administration > Device communication > Data collection KPIs.

The overview table shows all KPIs that have already been configured. When you click on a configured KPI in the table, the details of the KPI are shown in the details pane below. The details show the device group of the KPI together with the targets for connections and communications.

3

Note

The KPIs can be viewed on the Dashboard when you filter on a specific device group. See “Connection and communication” for more information. KPIs are also visualised on the connection and communication overview see “View connection overview per device group” and “View communication overview per device group”.

Add data collection KPI

5

Steps

1.   Go to Administration > Device communication > Data collection KPIs.

2.   Click Add data collection KPI and complete the following fields:

Parameter

Description

Device group

Select the device group you want to monitor the data collection of.

Calculation frequency

Define the frequency of the KPI calculation.

Display range

Define the time range that should be displayed in “Connections over time” see “Connection overview” and “Communication overview” for more information.

The possible values in the drop down menu are determined by the selected calculation frequency.

Connection KPI

Indicate whether you want a KPI for connections and if yes define the target in percentages.

Communication KPI

Indicate whether you want a KPI for communication and if yes define the target in percentages.

3.   Click Save.

3

Note

You can only have one data collection KPI per device group.

Edit data collection KPIs

5

Steps

1.   Go to Administration > Device communication > Data collection KPIs.

2.   Click the actions icon  next to the configured KPI and select Edit.

Device group and frequency cannot be edited.

Remove data collection KPIs

5

Steps

1.   Go to Administration > Device communication > Data collection KPIs.

2.   Click the actions icon  next to the configured KPI and select Remove.

3.   Click Remove.
