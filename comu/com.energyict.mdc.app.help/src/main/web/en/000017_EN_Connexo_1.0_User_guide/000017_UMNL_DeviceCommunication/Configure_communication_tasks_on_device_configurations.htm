Configure communication tasks on device configurations

Add communication task configuration on device configuration

1

Important

To be able to add communication tasks to device configurations, first a shared communication task has to be created. See “Configure communication tasks (Administration)”.

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration.

2.   Click Communication task in the menu on the left.

3.   Click Add communication task. Complete the following fields:

Parameter

Description

Communication task

List of tasks that were defined in “Add communication tasks”. Only communication tasks that have not already been used for the device configuration are visible.

Security set

List of security settings that have been added to the device configuration. This will define the specific security requirements of the communication task.

Connection method

Select the connection method the task will use out of the list of connection methods that have been added to the device configuration. By default the default connection method is selected.

Protocol dialect

Dialects configured for that device configuration.

Urgency

 Presents a hierarchy to prioritise communication tasks. Value can go from 0 to 999, the lowest value corresponds to the highest priority. These communication tasks will be executed first.

Always execute for inbound

Indicates that all inbound connections will be accepted. Only possible when you select an inbound connection method.

4.   Click Add. By default the new communication task is set on active.

5.   Click the actions icon to (de)activate the task.

Edit communication task configuration on device configuration

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration > click Communication task in the menu on the left.

2.   Click the actions icon  and select Edit.

3.   Change the communication task parameters.

4.   Click Save.

Remove communication task configuration on device configuration

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration > click Communication task in the menu on the left.

2.   Click the actions icon  and select Remove.

3.   Click Remove.
