Monitor data import

View import service details

5

Steps

1.   Go to Administration > Data import > Import services.

The overview table shows all configured import services. The system administrator can add and configure these import services in Admin. The table shows the file importer used together with the status of the service, the imported folder and the file pattern.

2.   Select an import service in the table, the details pane on the bottom of the page shows extra information such as the folder where in progress, successful and failed import files will be moved.

3.   Click the import service name or click the actions icon  and select View details to see more extensive details.

4.   Click the tab History to see the log for this import service.

View import service history

5

Steps

1.   Go to Administration > Data import > Import history.

The history page shows every started import together with the duration and status. When you select an item in the table the details pane on the bottom of the page shows specific details of this import.

2.   Filter the history to only see the history of specific import service or for a specific time window or status using the filter on top of the page. Click Apply.

3.   Remove filters by deselecting items in the drop down menu, click Clear or remove all filters by clicking Clear all.

4.   Click the actions icon  and select View log to see a detailed logging of the history item.
