Data import

Want to import, this chapter covers all features inside MultiSense such as import services that need to be configured in order to be able to import your data. Defining device groups of which data needs to be imported is described in the chapter “Manage device groups”.

Export data

1. Add a static device group or Add a dynamic device group to group all devices that should be viewed together.

2. Add export task for device readings to define when to export data and in what format.

3. Schedule data export in the data export task configurationor Export on request.
