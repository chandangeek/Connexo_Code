Default device life cycle management

Default device life cycle

Default device life cycle diagram

MultiSense offers one default device life cycle. When you install MultiSense this device life cycle will be fully configured and available. You can choose to use this life cycle or create a new one, but be aware that you need a least one device life cycle to add device types.
