Device life cycle management

Every device in MultiSense  has a life cycle. During its life cycle a device transitions from one status to another. The transitions are linked with different actions and checks that are executed automatically to guarantee the device can function correctly in the next status.

This chapter describes how to set up and edit device life cycles and how to transition devices once everything is configured. Before explaining all the technical details, this chapter first explains all the concepts of device life cycles to help you understand how these work in MultiSense .
