Manage device life cycle states

Understand states

A device state is the current status of a device. The state defines what you can see on the device, which actions the device can execute and what you can do on the device. You can use device states to search for devices.

A device life cycle needs at least one state and an initial state. The initial state is the state a device acquires when you add it to MultiSense. The initial state can be changed at any time.

Set initial state

5

Steps

1.   Go to Administration > Device life cycle management > Device life cycles > click a device life cycle name > click States in the menu on the left.

2.   Click the actions icon  and select Set initial state.

The initial state is indicated by the default icon  and is also shown in the details pane of the state.

Edit state

5

Steps

1.   Go to Administration > Device life cycle management > Device life cycles > click a device life cycle name > click States in the menu on the left.

2.   Click the actions icon  and select Edit.

3.   Change the state parameters.

4.   Click Save.

Remove state

5

Steps

1.   Go to Administration > Device life cycle management > Device life cycles > click a device life cycle name > click States in the menu on the left.

2.   Click the actions icon  and select Remove.

3.   Click Remove.
