Configure security on device type

Understand security on a device type

On the device type you create the security accessors. When you add a new security accessor you need to specify the purpose of that security accessor. Will it be used for storing a key or a certificate and of what type. Here you also need to select which levels of users that will be able to view the attributes and edit them.

Understand privileges

In Connexo Admin  the system administrator adds privileges to different user roles. These define what a user is allowed to do within the different parts of the applications. These privileges are further divided in levels. With these levels you can group user roles. For example level 1 can be used for people working with electricity meters while level 2 is used for people working with gas meters.

In MultiSense  you can then link these privileges to the security settings configured on the device type. This will define what different users can do on a device. The following scenarios are possible:

  • No privileges:

  • A user that has no privileges will not see the different security attributes.

  • All action buttons are disabled.

  • Only view privileges:

  • The names of the different security attributes are shown. Values are encrypted and shown as ********, if the value is not empty.

  • All action buttons are disabled.

  • Only edit privileges:

  • The names of the different security attributes are shown. Values are encrypted and shown as ********, if the value is not empty.

  • All action buttons are enabled and it is possible to edit values.

  • View and edit privileges:

  • The names of the different security attributes are shown. Values are encrypted and shown as ********, if the value is not empty.

  • All action buttons are enabled and it is possible to edit values and show or hide the value.

Add security accessors

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Specifications > Security accessors.

2.   Click Add security accessor and complete the following fields:

Parameter

Description

Name

Give a unique name to the security accessor.

Description

Give an optional description.

Accessor type

Select the type of accessor: key or certificate. This will define the other parameters that need to be specified.

If you select accessor type key

Key type

Select one of the types available in Connexo.

Storage method

Select one of the available storage methods to store the keys securely.

Validity period

Only available for some key type. Define how long the key is valid.

If you select accessor type certificate

Certificate type

Select one of the types available in Connexo.

Trust store

Shows all trust stores available in Admin . Select one of the trust stores.

Connexo will use this trust store to validate the trust chain of the certificate.

Storage method

Only available if the selected certificate type is a client certificate type. Select one of the available storage methods.

3.   Click Add.

If the accessor type is a key then this will get level 1 and 2 view and edit privileges by default. To alter these see “Change key security accessor privileges” for more information.

Change key security accessor privileges

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Specifications > Security accessors.

2.   Click the actions icon  next to the key security accessor and select Change privileges.

3.   Select the correct privileges.

4.   Click Save.

You can see which user roles have privileges by clicking the information icon  next to the privileges in the details pane of the selected security accessor.

Edit security accessors

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Specifications > Security accessors.

2.   Click the actions icon  next to the security accessor and select Edit.

3.   Change the security accessor parameters. Only Name, Description and Validity period (if this is available) can be altered.

4.   Click Save.

Remove security accessors

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Specifications > Security accessors.

2.   Click the actions icon  next to the security accessor and select Remove.

3.   Confirm by clicking Remove.

If the device type has active configurations then the security accessor cannot be removed.
