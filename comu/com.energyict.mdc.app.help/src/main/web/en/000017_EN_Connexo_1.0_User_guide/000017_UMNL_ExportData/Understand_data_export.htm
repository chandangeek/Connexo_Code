Understand data export

Data export in Connexo

MultiSense is Connexoâ€™s meter data collection application. Once MultiSense collected data from meters, checked if the data is valid and estimated the data, if needed, you are able to export this data. You also have the option to export unprocessed data. You may need to export data to other parties and systems for further processing, for example to use the data for billing purposes.

In MultiSense different default exporters are available allowing you to export different formats. However there is an option to add extra custom exporters. This will allow you to export any data you need.

How to configure data export

There are five important features when configuring data export tasks: the data selector, the formatter, the destination, the scheduling and the monitoring.

With the data selector you can define which data MultiSense will export by using data types such as reading types, event types, etc. Depending on the selected data selector, optional options may come available, for example:

  • select whether you want valid data or continuous data,

  • select a device group if you only want data from a limited set of devices,

  • select the export period (based on relative periods, you can create relative periods yourself or use these that are available by default).

The formatter puts all information that needs to be exported into a specific format. With the destination you can specify where the file will be exported to. One file can be exported multiple times to the same destination or to different destinations. Examples of destinations are email, ftp, etc.

Finally in MultiSense you can choose to run export tasks manually or add a schedule to them so the tasks are executed at fixed time points. You can monitor all tasks and track their history and logging. This will help you identify and solve errors when data export tasks fail.
