Export on request

Run data export task

5

Steps

1.   Go to Administration > Data exchange > Data export tasks.

2.   Click the actions icon  or the actions button  and select Run.

Click Run in the dialog box to run the export task now. The status in the details pane will change to busy and the start date to now. When the data export was successful the status will change to success and the finish time will be updated. If not, the status will be set to failed and a reason for failure will be displayed.
