View export task history

Steps

1.   Go to Administration > Data export > Data export tasks.

2.   Click the actions icon  and select View history or click the export task name and click History in the menu on the left.

The table shows all instances of the selected export task. Details of every line are given in the details pane below.

Filter data export history

5

Steps

1.   Go to Administration > Data export > Data export tasks.

2.   Click the actions icon  and select View history or click the export task name and click History in the menu on the left.

3.   Select or type criteria in the filter fields to search for export tasks that started or finished in a specific time window.

4.   Click Apply.

Deselect items in the drop down menu to remove single filters or click Clear all to remove all applied filters.

View data export log

5

Steps

1.   Go to Administration > Data export > Data export tasks.

2.   Click the actions icon  and select View history or click the export task name and click History in the menu on the left.

3.   Click the actions icon in a history line or the action button  of the details pane and select View log. Details of the execution of the export task are shown on top of the page. All logs are shown in the table below with timestamp, log level and message.
