Monitor devices

View the meter landing page

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click the device name to go to the overview page of this device.

The device overview page or meter landing page displays the following features:

Parameter

Description

General information

Different parameters unique for the device such as MRID and the serial number but also the used device type and configuration.

Communication topology

Shows the most recently added slave devices in a table and provides a link to meter landing page of these devices.

Open issues

Shows how many open issues are linked to the device together with a link to these issues.

Data validation

Shows the status of the validation, together with the amount of suspects and the last validation date.

Connections

Shows all connection methods that are available on this device together with their status. Click the actions icon  to perform the following actions: Run now: Deactivate or Activate the connection method or View history.

Communication tasks

Shows all communication tasks available on this device. Click the actions icon  to perform the following actions: Run now: Deactivate or Activate the connection method or View history. Activate or deactivate all communication task at once by clicking Activate all or Deactivate all respectively.

 

Figure: Meter landing page.

 

View the gateway landing page

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click the gateway name to go to the overview page of this device.

The overview page or gateway landing page displays the following features:

Parameter

Description

General information

Different parameters unique for the device such as MRID and the serial number but also the used device type and configuration.

Communication topology

Shows the most recently added slave devices (LAN gateway) or all slave devices (HAN gateway) and provides a link to the landing page of the slave devices.

Open issues

Shows how many open issues are linked to the device together with a link to these issues.

Data validation

Shows the status of the validation, together with the amount of suspects and the last validation date.

This information is not shown when there are no registers, load profiles and channels configured.

Connections

Shows all connection methods that are available on this device together with their status. Click the actions icon  to perform the following actions: Run now: Deactivate or Activate the connection method or View history.

Communication tasks

Shows all communication tasks available on this device. Click the actions icon  to perform the following actions: Run now: Deactivate or Activate the connection method or View history. Activate or deactivate all communication task at once by clicking Activate all or Deactivate all respectively.
Figure: Communication topology overview on the LAN gateway landing page

 

Flag devices on the meter/gateway landing page

In Connexo MultiSense you can flag devices so that you can easily access them from The dashboard without having to search for the device.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   On the meter/gateway landing page of the device click the star  in the upper right corner.

3.   Add a comment and click Flag device. When you flag a device the star becomes filled .

4.   Click the grey star to remove the device from your favourite devices.
