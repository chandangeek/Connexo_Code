View register data

1

Important

Add registers to devices via device configurations, see “Configure register configurations”.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Registers in the menu on the left.

3.   Click the actions icon  next to the register or click the actions button  in the details pane and click View data.

Or click the register name > click Register data in the menu on the left.

The overview table shows all values of that register together with the measurement time. On the bottom of the page more details of the selected register value are provided. This is general information over the measurement together with validation details.
