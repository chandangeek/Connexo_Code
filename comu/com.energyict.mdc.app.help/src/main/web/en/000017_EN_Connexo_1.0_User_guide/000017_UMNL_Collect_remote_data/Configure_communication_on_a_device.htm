Configure communication on a device

View communication schedules on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication planning in the menu on the left.

The overview shows the following information:

Table

Description

Recurring communication

Communication schedules for tasks that need to be completed at fixed time points.

 l Shared communication   

       schedules

Communication schedules that are used by multiple devices.

 l Individual communication

        schedule

Communication schedule only used by this device.

Non recurring communication

Communication task that need to be performed ad hoc.

Add shared communication schedule to a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication planning in the menu on the left.

3.   Click Add shared communication schedule.

4.   Select one or more communication schedules.

5.   Click Add.

6.   Click the actions icon to remove a shared communication schedule.

3

Note

Only communication schedules containing tasks that are also available in the device configuration are shown. Communication tasks can only be used by one communication schedule and all tasks in a schedule need to have the same connection method, protocol dialect, security set and urgency.

Add frequency to a communication task in the communication planning

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication planning in the menu on the left.

3.   Click the actions icon  in the Non recurring communication table.

4.   Click Add frequency.

5.   Define the schedule and click Add.

The communication task is now added to the Individual communication schedules table.

Change frequency of a communication task in the communication planning

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication planning in the menu on the left.

3.   Click the actions icon in the individual communication schedules table.

4.   Click Change frequency.

5.   Change the schedule and click Add.
