Configure communication port pools

Add/remove outbound communication port pools

5

Steps

1.   Go to Administration > Device communication.

2.   Click Communication port pools.

3.   Edit or remove port pools via the actions icon .

4.   Click Add communication port pool > Outbound. Complete the following fields:

Parameter

Description

Name

A unique name

Type

Select a communication type.

5.   Click Add.

3

Note

Once a communication port pool has been configured and saved, the parameter Type can no longer be edited. This is to prevent potential harmful effects on devices that use this port pool.
