Configure load profile types

Add/remove load profile types

5

Steps

1.   Go to Administration < Device management.

2.   Click Load profile types.

3.   Click the actions icon  to edit or remove load profile types.

4.   Click Add load profile type. Complete the following fields:

Parameter

Description

Name

A unique name.

Interval

Defines the interval of all channels belonging to this load profile.

OBIS code

Object Identification System, provides standard identifiers for all data originating from the metering devices.

Register types

Click Add register type to select one or more register types and click Add.

5.   Click Add

3

Note

Only the name of the load profile type can be edited when a load profile type is used by a device type. This is to prevent any harmful effects on the data collection process.
