Usage points

Understand usage points

In the CIM model, usage points represent the point of delivery of a single commodity to a customer. At the usage point, it is possible to view the customer's usage of the commodity over time regardless of the specific meter or device installed at any one point in time.

View usage point details

5

Steps

1.   Search for a usage point as described in “Search for usage points”.

2.   Click the usage point name.

The usage point details page shows the metrology configuration of the usage point together with its general attributes.

The metrology configuration shows which devices are linked together with the history.

Link device to usage point via the device attributes

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Select device attributes in the menu on the left.

3.   Click Edit.

4.   Select a usage point from the drop down menu.

5.   Click Save. The device is now linked to the selected usage point. A link to the usage point is available on the device landing page.
