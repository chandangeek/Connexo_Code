Communication history

View communication history

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

3.   Click the actions icon  or the actions button  in the details pane and select View history.

This page shows the history of the communication tasks. For every communication the duration, status, result and used connection are shown.

4.   Click a communication task in the table. Below the table more information is shown.

A details pane shows further details of the communication task. Under the details pane there is a table showing the details of the connection method that was used by the communication task.

View communication log

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

3.   Click the actions icon  or the actions button  and select View history.

4.   Click the actions icon or the actions button in the details pane of one of the communications and select View communication log.

The log page shows the details of the selected communication on top of the page. Below a table is shown containing all communication logs with some details. More details are given in the details pane below the table.

Filter communication log

5

Steps

1.   View the connection logs as described in “View communication log”.

2.   Select the filters in the filter pane on the right of the page.

3.   Click Apply. The applied filters are shown on top of the page below the connection details.

4.    Click the remove icon 8 next to applied filter to remove the filter.

5.   Click Clear all to remove all filters.
