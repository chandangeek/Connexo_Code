Configure device commands on the device configuration

View commands on device configuration

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Configurations > select a configuration.

2.   Click Commands in the menu on the left.

This page show all the command categories and commands that are available on the device configuration.

3.   Click a command category in the top table to see which commands are included together with their status and privileges in the bottom table.

Change privileges of commands

There are two options to change the privileges of commands, you can change the privileges of all commands in a category at once or only change the privileges of certain commands in the category.

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Configurations > select a configuration.

2.   Click Commands in the menu on the left.

3.   To change the privileges of all commands in a category at once click the actions icon  next to the command category in the top table or the actions button  above the bottom table and select Change privileges for all.

4.   Select the privileges and click Save. Note that the changes will only apply to the commands that are active.

5.   To change the privilege of a single command click the actions icon  next to the command in the bottom table and select Change privileges.

6.   Select the privileges and click Save.

(De)Activate commands

There are two options to (de)activate commands, you can (de)activate all commands in a category at once or only (de)activate of certain commands in the category.

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Configurations > select a configuration.

2.   Click Commands in the menu on the left.

3.   To (de)activate all commands in a category at once click the actions icon  next to the command category in the top table or the actions button  above the bottom table and select Activate all or Deactivate all.

4.   When you activate commands you first have to define the privileges of the commands that will be activated. Click Save.

5.   To (de)activate a single command click the actions icon  next to the command in the bottom table and select Activate or Deactivate.

6.   Define the privileges of the command and click Save.
