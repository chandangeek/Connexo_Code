View load profile details

1

Important

Add load profiles to devices via device configurations, see “Configure load profile configurations”.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Load profiles in the menu on the left.

3.   Select a load profile, details are shown in the details pane below.

You can also see a load profile’s specifications when you click the load profile name and select the Specifications tab.

View load profile data in a table

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Load profiles in the menu on the left.

3.   Click the load profile name.

4.   Click Table view in the upper right corner of the Readings tab.

All calculated values of the channels are displayed in the columns which allows to see the values of each interval of the different channel on the load profile next to each other.

On the bottom of the page more details of the selected value are provided. Three tabs show different information.

a. General: shows information of the measured value.

b. Reading values: shows the values of the channels of this load profile.

c. Reading quality: important observation concerning the value, for example a power outage occurred.

The following parameters are displayed:

Parameter

Description

Interval

The consumption interval that was measured.

Last update

Shows when the reading was last handled. This can be due to validation, estimation or editing.

Data validated

Indicates whether the values has been validated or not.

Calculated value

The effective consumption measured in the defined interval together with the result of the validation of every channel.

Collected value

The integral consumption measured on the device by the end of the defined interval together with the result of the validation of every channel.

Reading quality

Indicates that there is something remarkable about this value. Hover over the information icon for more details.

View load profile data in a graph

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Load profiles in the menu on the left.

3.   Click the load profile name.

4.   Click Graph view in the upper right corner of the Readings tab.

Channels of the load profile are displayed under each other, indicated by A in the screenshot below. The unit of the channel is displayed on the left side of the graph. Details of all channels for one interval are shown when hovering over one value.

You can zoom in or out by selecting a standard interval of 4 hours, 1 day, 2days or 1 week (B in the screenshot) or give in a time window in the field indicated by C in the screenshot.

On the bottom of the page (D in the screenshot) there is a bar that shows where the interval that is displayed positions in a larger time frame. The displayed interval can be shifted by sliding the bar to the left or the right. You can also make the interval larger or smaller by making the scroll bar wider or smaller respectively.

Figure: Graphical representation of load profile data

 

Change next reading block start on a load profile

Change the start timestamp of the reading block that will be collected during the next collection phase of a load profile. This allows you to re-read the channel readings in a load profile, e.g. if data is exceptionally different than usual. Or your may want to avoid collecting readings from the past that you are not interested in, e.g. because you know that the device was broken or behaving suspiciously.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Load profiles in the menu on the left.

3.   On the line of the desired load profile, click the actions icon  and select Change next reading block start.

4.   In the pop-up dialogue that appears, pick the next reading block start date.

5.   Click Save.

3

Note

The action Change next reading block start is also available from the Actions button on the    load profile details pane and on its details page.
