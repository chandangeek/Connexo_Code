Configure protocol dialects

Understand protocol dialects

Some tasks require specific properties such as the ability to download a large amount of data as it is often the case for firmware upgrades. Such specific properties can be acquired by configuring protocol dialects.

Which protocol dialects are available is determined by the protocol from which the dialect originates.

Edit protocol dialects on a device configuration

1

Important

Whether there are protocol dialects available is determined by the protocol itself.

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration.

2.   Click Protocol dialects in the menu on the left.

3.   Click the actions icon  to edit.

Which parameters can be edited is determined by the protocol used.

4.   Click Restore all defaults to undo all changes.

5.   Click Save.

3

Note

Which parameters can be changed is determined by the communication protocol. Protocol dialects cannot be removed.

Edit protocol dialects on a device

1

Important

Whether there are protocol dialects available is determined by the protocol itself.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Protocol dialects in menu on the left.

3.   Click the actions icon . and select Edit.

4.   Complete the necessary fields.

5.   Click Restore to default to undo all changes and return to the default settings.

6.   Click Save.

3

Note

Which parameters can be changed is determined by the communication protocol. Protocol dialects cannot be removed.
