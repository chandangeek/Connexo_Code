Install devices

Install device via file import

1

Important

To be able to install devices, you need a CSV file, the first line of this file has to be the header and fields are separated using  "," or ";". The following information has to be included in this file:.

Columns

1. mRID device

2. mRID usage point

3. Service category:

•    Electricity

•    Gas

•    Water

•    Time

•    Heat

•    Refuse

•    Sewerage

•    Rates

•    TVlicense

•    Internet

•    Others

4. Start date of the meter activation (dd/MM/yyyy)

5

Steps

1.   Go to Devices > Device lifecycle management > Install devices.

2.   Click the browse button and select the installation file.

3.   Click Install.
