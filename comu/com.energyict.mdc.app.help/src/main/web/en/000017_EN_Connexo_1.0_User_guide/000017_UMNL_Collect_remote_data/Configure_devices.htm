Understand device configuration

To collect data from a physical device, you have to create a device in Connexo MultiSense. A device uses one device type with a certain configuration. This device type defines the physical capabilities of the device, which data can be read and what communication protocol has to be used.

A device type doesn't necessarily map to a single physical device model. It may represent several models from various manufacturers or different sub-models as long as they have similar behaviour and characteristics. This is achieved through the use of device configurations.

A device type can have multiple device configurations. Device configurations are used to customise or limit some of the functionality of the device type, but also configure additional parameters to interact with the physical device. The device configuration will determine the actual register specifications together with the number of channels. It also contains information on security levels, connection methods, etc.

By using device configurations a more accurate representation of the physical device and its enabled functionality can be created. Configurations can be reused and facilitate the process of creating a large amount of similar devices.
