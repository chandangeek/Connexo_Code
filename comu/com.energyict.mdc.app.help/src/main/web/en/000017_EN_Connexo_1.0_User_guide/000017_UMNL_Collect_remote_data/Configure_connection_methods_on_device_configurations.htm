Configure connection methods on device configurations

Understand connection methods

The connection method allows you to define how to communicate with the device. Some devices can support multiple types of connection methods (GRPS, optical, SMS, etc.). You can configure which is the preferred type. Using connection methods you can minimize your cost via the connection strategy. You can select to minimize the connections or connect as soon as possible.

If you choose to minimize the connections then MultiSense will only connect to a device when the connection method allows it. A communication task will then only be executed at the time the connection is scheduled. If you choose to connect a soon as possible, then a connection will be set up as soon as a communication task needs a connection.

Configure an outbound connection method

1

Important

Before you add a connection method make sure that the device is Directly addressable. If not, then no connection methods can be added. This can be selected on the device configuration edit page. See “Configure device configurations”.

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration.

2.   Click Connection methods in the menu on the left.

3.   Edit or remove connection methods via the actions icon .

4.   Click Add connection method > Add outbound connection method. Complete the following fields:

Parameter

Description

Name

A unique name.

Connection type

Select a connection types. Which type is selected determines which connection details need to be completed.

Communication port pool

Select a communication port pools

Connection strategy

As soon as possible: connect whenever a communication task needs a connection.

Minimize connections: only connect with the device at the scheduled time.

Connection schedule

Only available when “Minimize connections” is selected. The schedule when to set up a connection.

Connection window

Is the time window during which it is allowed to set up a connection and communication tasks can be executed.

Retry delay

The period of delay before the connection will be retried.

Allow simultaneous connection

Allows multiple connections at the same time.

5.   Click Add.

6.   Click the actions icon and select Set as default to make the connection method the default method.
