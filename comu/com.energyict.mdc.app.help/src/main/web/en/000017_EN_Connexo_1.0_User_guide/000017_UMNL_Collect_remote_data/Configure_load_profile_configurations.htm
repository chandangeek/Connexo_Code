Configure load profile configurations

Add/remove load profile configurations

1

Important

Add load profile types to device types to be able to add and configure load profile configurations.

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration.

2.   Click Load profile configurations in the menu on the left.

3.   Edit or remove configurations via the actions icon .

4.   Click Add load profile configuration. Select one of the load profile types that were added previously to the device type.

5.   The OBIS code is automatically generated based on the load profile type you selected. You can edit this value.

6.   Click Add

3

Note

Load profile configurations can only be added when the device configuration is inactive. This is to prevent any possible effects on the data collection process.

On the overview page of the load profile the applied validation rules are shown. From here you can go to the validation rules page, for more information see “Validate data”.

Configure channels

5

Steps

1.   Click a load profile configuration.

2.   Click Add channel configuration. Complete the following fields:

Parameter

Description

Register type

Select the data of which you want to see interval readings. Only register types that have not been used for other channels are available

Overruled OBIS code

Overrule the OBIS code of the register type.

Overflow value

When the arithmetic operation attempts to create a numeric value that is too large to be represented within the available storage space, virtual space is created to accurately calculate usage.

Number of fraction digits

Amount of digits after the decimal

3.   Click Add.
