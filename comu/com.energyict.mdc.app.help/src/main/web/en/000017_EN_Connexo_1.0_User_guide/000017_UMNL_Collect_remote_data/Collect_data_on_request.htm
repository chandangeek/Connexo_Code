Change urgency of a communication task

1

Important

Only the urgency of scheduled communication tasks can be changed.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

3.   Click the actions icon  and select Change urgency. If the communication task has a shared communication schedule assigned to it, the urgency of all communication tasks using this schedule will be altered.

4.   Give in a new value and click Save.

Run a communication task on request from the communication task overview

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

3.   Click the actions icon  and select Run or Run now.

  • Select Run now if you want the task to be performed immediately, only possible when the connection strategy of the connection method is As soon as possible. In this case a new connection will be set up.

  • Select Run when the task can be performed the next time a connection is set up according to the connection strategy and schedule.

Run a communication task on request using the communication planning

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication planning in the menu on the left.

3.   Click the actions icon  next to the communication task. Click Run or Run now.

View communication task history

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication tasks in the menu on the left.

3.   Click the actions icon  and select View history.

the history page shows an overview of all executed instances of the communication task together with the result. On the bottom of the page you can see more details of the communication task and a detailed description of the connection method used.

4.   Click the actions icon  or action button to see the communication or connection log. See “Monitor connections” and “Monitor communication” for more information.
