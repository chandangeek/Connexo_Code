Import devices

Import devices using a CSV file

1

Important

To be able to import devices, you need a CSV file, the first line of this file has to be the header and fields are separated using  "," or ";". The following information has to be included in this file:.

Columns

1. Device arrival date (Format: dd/MM/yyyy)

2. MRID

3. Serial number

4. Year of certification (Format: dd/MM/yyyy)

5. Batch

5

Steps

1.   Go to Devices > Device lifecycle management

2.   Click Import devices.

3.   In step 1 of the wizard upload the CSV file with all device information.

4.   Click Next

5.   In step 2 of the wizard specify the device type and configuration.

6.   Click Import.

7.   In step 3 of the wizard the import status is shown.

8.   Click Devices to go back to the Devices page.

When the process is finished...

3

Note

If no device configurations have been configured yet or they’re inactive, no configurations will be shown. To configure device configuration see “Configure device configurations”.

Click Cancel to stop the import process in step 1 or step 2 of the wizard. No devices will be added.
