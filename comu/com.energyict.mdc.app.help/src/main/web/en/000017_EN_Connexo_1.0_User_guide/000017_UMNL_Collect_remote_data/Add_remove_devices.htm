Understand devices

To be able to collect meter data, you need metering devices. These devices are located in the field. Different types of devices are available originating from different vendors. All of these have their own possibilities and characteristics. The ultimate goal is to collect the data coming from the meter and use this data for other purposes such as billing and energy management.

To be able to collect this data, you need a meter data collection system. Connexo MultiSense connects and communicates with these devices. How communication is possible is explained in “Understand communication”. Devices in Connexo MultiSense are configured using device types and device configurations. Finally devices are added (manually or via import) and the correct device type and configuration are assigned. Now only device specific parameters need to be defined such as passwords and connection details.
