Configure logbook types

Add/remove logbook types

5

Steps

1.   Go to Administration < Device management.

2.   Click Logbook types.

3.   Click the actions icon  to edit or remove logbook types.

4.   Click Add logbook type. Complete the following fields:

Parameter

Description

Name

A unique name.

OBIS code

Object Identification System, provides standard identifiers for all data originating from the metering devices.

5.   Click Add.

3

Note

Only the name of the logbook type can be edited when a logbook type is used by a device type. This is to prevent that a device might no longer be functional as consequence of the change.
