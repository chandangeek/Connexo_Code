Configure communication ports

Add inbound communication port(s)

5

Steps

1.   Go to Administration > Device communication > Communication servers >click a communication server name.

2.   Click Communication ports in the menu on the left or click the communication port link provided on the overview page.

3.   Click Add communication port > Inbound. Complete the following fields:

Parameter

Description

Name

Unique name in MultiSense for the port.

Type

Serial, servlet, TCP, UDP.

Simultaneous connections

Number of threads that can run simultaneously on this port. This option is not available for physical ports such as modems.

Inbound communication port pool

Select a communication port pool out of the list of configured inbound communication port pools.

4.   Click Add.

5.   Click the actions icon  and select Activate.

3

Note

Depending on the type extra fields need to be completed.

Communication ports without an assigned communication port pool can be added, but they cannot be activated.

Add outbound communication port(s)

5

Steps

1.   Go to Administration > Device communication > Communication servers > click a communication server name.

2.   Click Communication ports in the menu on the left or click the communication port link provided on the overview page.

3.   Click Add communication port > Outbound. Complete the following fields:

Parameter

Description

Name

A unique name.

Type

Serial, TCP, UDP.

Simultaneous connections

Number of threads that can run simultaneously on this port. This option is not available for physical ports such as modems.

Outbound communication port pools

Click Add communication port pools to select one or more port pools.

Click Manage communication port pools to create a new outbound port pool.

4.   Click Add.

5.   Click the actions icon  and select Activate.

3

Note

Depending on the type of communication port, different extra fields need to be completed.

Communication ports without an assigned communication port pool can be added, but they cannot be activated.

Edit communication ports

5

Steps

1.   Go to Administration > Device communication > Communication servers > click a communication server name.

2.   Click Communication ports in the menu on the left or click the communication port link provided on the overview page.

3.   Click the actions icon  and select Edit.

4.   Change the communication port parameters.

5.   Click Save.

Remove communication ports

5

Steps

1.   Go to Administration > Device communication > Communication servers > click a communication server name.

2.   Click Communication ports in the menu on the left or click the communication port link provided on the overview page.

3.   Click the actions icon  and select Remove.

4.   Click Remove.

Manage communication ports on communication port pools

5

Steps

1.   Go to Administration > Device communication > Communication port pools > click the communication port pool name. 

2.   Click communication port in the left menu or click the communication port link on the overview page.

3.   Remove communication ports using the actions icon .

4.   Click Add communication port.

5.   Select one or more communication ports.

6.   Click Add.
