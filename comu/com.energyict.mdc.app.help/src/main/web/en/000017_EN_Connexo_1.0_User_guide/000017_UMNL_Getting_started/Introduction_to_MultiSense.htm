Introduction to MultiSense

In Connexo MultiSense different data collection features are spread over different pages. A small introduction to the different pages available in MultiSense is given here.

Dashboard

On the dashboard the meter operator can find the latest status of the items that he is responsible for. The overview shows the issues assigned to the operator together with the server status, active connections and communications.

Workspace

The workspace is the central location where users can see the status of data collection and validation as well as monitor tasks and issues. Different options are available for viewing the status of data collection, data validation, data export, as well as viewing tasks and issues linked to device operations.

Note

Depending on the privileges assigned to a specific user role the view of the workspace may vary for different users.

Devices

The Devices page is the place in the application where you can manage all devices. On this page devices can be added, manually or imported using a CSV files. You can also import different device attributes on this page.

The device page is also the location where to create and manage device groups.

Administration

Via the Administration page all device features can be configured. This consist of the complete communication infrastructure (communication servers, communication ports, communication tasks and communication schedules, etc.), the device types and configurations, as well as validation rules and rules to create issues. Configuration is simplified by the use of templates in the form of shared communication schedules and tasks, master data and device types. These have been developed in such a way that they can be used for multiple devices.

The administration page is divided in different panes that group together different parameters that are closely linked. E.g. all components to set up communication are grouped together in Device communication.
