Connection and communication

View communication server status

5

Steps

1.   Go to the Dashboard.

All communication servers are grouped by status, click the information icon  to see which servers have this status.

2.   Click View all to see all configured communication servers and their details.

View connections

5

Steps

1.   Go to the Dashboard.

The dashboard shows a summary of all active connections, showing the amount of successful connections, successful connections but with at least one failed communication task, ongoing connections and failed connections. You can view KPIs of device groups if you filter on the device group.

When you click the name of a group of connections, you will see all connections with the corresponding filter.

2.   Click View connections overview to view the overview page as described in “Connection overview”.

View communication

5

Steps

1.   Go to the dashboard.

The dashboard shows a summary of all active communications, showing the amount of successful communication tasks, ongoing and failed communication tasks. You can view KPIs of device groups if you filter on the device group.

When you click the name of a group of communications, you will see all communications with the corresponding filter.

2.   Click View communications overview to view the overview page as described in “Communication overview”.

Filter on device group

1

Important

Device groups can be added as described in “Manage device groups”.

5

Steps

1.   Go to the Dashboard.

2.   Click the filter button and select a device group.

Now only the data from the devices that are part of the device group, will be shown in the different widgets. For connections and communications the KPIs of the device group are shown in the widgets, if these are configured.
