View issues

5

Steps

1.   Go to the Dashboard.

The dashboard only shows the latest 5 open issues that are assigned to the operator that is logged in. To view all assigned issues click Assigned to me, to view all unassigned issues click Unassigned issues.

Extra information is shown when hovering over the icon next to the issue.

2.   Click an issue to go to the corresponding details page. From this page it is possible to perform different actions on this issue.
