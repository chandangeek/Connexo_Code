Flag devices on the object landing page

In Connexo MultiSense you can flag devices so that you can easily access them from “The dashboard” without having to search for the device.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   On the meter/gateway landing page of the device click the star  next to device name in the page title.

3.   Add a comment - if needed - and click Flag device. When you flag a device the star becomes filled .

4.   Click the filled star to remove the device from your favourite devices.

Other actions from the object landing page

The available actions on the object landing page depend on the type of object and the life cycle of the object.

It is always possible to refresh the information on the page by clicking the Refresh button in the upper right corner.

Other actions are available by clicking the Actions button in the upper right corner.
