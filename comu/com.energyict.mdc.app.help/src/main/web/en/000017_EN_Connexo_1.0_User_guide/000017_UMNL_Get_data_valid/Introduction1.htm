Introduction

Types of editing

In MultiSense  three different “edit” actions are available. After data has been collected it is possible to change the values or remove them. You also have the option to add values manually.

Audit trail

It is mandatory the keep an audit trail of all changes made to the metered data. This means that all changes of every meter reading must be tracked together with who made the change, what was changed and at what point in time.

MultiSense tracks all these changes and stores them in the database.
