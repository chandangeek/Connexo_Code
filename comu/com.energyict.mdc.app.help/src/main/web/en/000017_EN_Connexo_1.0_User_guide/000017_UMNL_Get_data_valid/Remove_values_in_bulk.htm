Remove channel readings in bulk

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channels in de menu on the left.

3.   Click the name of a channel you want to remove data of.

4.   Select one or more values using the CTRL or SHIFT key and click Bulk action and select Remove readings.

5.   Click Save changes. The removed values will be marked by the remove icon . When you hover over this icon the date of removal is shown.

3

Note

Removed values are not removed from the system, but are retained to allow monitoring of changes.
