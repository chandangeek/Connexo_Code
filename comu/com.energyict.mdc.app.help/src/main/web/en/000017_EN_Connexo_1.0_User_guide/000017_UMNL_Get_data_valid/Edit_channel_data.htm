Edit channel readings

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channels in de menu on the left.

3.   Click the name of the channel you want to edit.

4.   Click the actions icon  next to the value you want to edit and select Edit value or click the value.

5.   Press enter. A red tag is displayed, indicating that the value has been edited.

6.   Click Save changes. The edited value will be marked by the edit icon . When you hover over this icon the date of editing is shown.

3

Note

Edited values are not removed from the system, but are retained to allow monitoring of changes.

Confirm one or more suspect channel readings

When you confirm a reading you acknowledge that the reading was incorrect, but it is acceptable. This is why the value of the reading is not altered.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channels in the menu on the left and filter on suspects using the filter on top of the page.

3.   Click the actions icon  next to the reading and select Confirm value or select multiple values using the CTRL or SHIFT key, click Bulk action and select Confirm value. The confirmed readings are indicated by a red tag.

4.   Click Save changes.

The value of the confirmed reading is not changed and is now indicated by a check mark.

Pre-validate channel data

In MultiSense  you have the option to pre-validate readings after you edited them. This allows you to check whether all edited values are valid. This prevents the need to manually validate the values after you edited them.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channels in the menu on the left and filter on suspects using the filter on top of the page.

3.   Click the value(s) you want to edit, type the new value(s) or copy and paste it.

4.   Click Pre-validate.

MultiSense  pre-validates from the first edited value until the last visible one.

a. In case there a no potential suspects, a op-up appears saying there are no potential suspect readings. Click Close to go back to the edit screen or click Save changes to save the edited readings.

b. In case there a potential suspects, a pop-up appears showing the amount of potential suspects and where they are located. Click Close. Potential suspects are marked by an empty red flag .

When you hover over the flag you can see which validation rule is violated.

5.   Click Save changes to confirm the edited values, potential suspect values are not converted into to real suspects. The last validation date is also not altered.

Correct data

When data was copied from a reference, it is possible that the reference data used a multiplier or a different unit. In this case you need to correct the data so that the copied reading type matches the reading type of the current channel.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channels in the menu on the left and filter on suspects using the filter on top of the page.

3.   Click the value(s) you want to correct it.

4.   Click the actions icon of a single value or the Bulk action button and select Correct. The info message shows which column of values will be affected. Complete the following fields:

Parameter

Description

Type

Select the type of correction to apply to the values.

Correction amount

Define the amount to correct the values with.

Limit values

Select if you only want to correct suspect and estimated data.

Estimation comment

Add comment to the value. By default “Keep original comment” is selected. Click the button next to the drop down menu to remove the comment.

5.   Click Correct.
