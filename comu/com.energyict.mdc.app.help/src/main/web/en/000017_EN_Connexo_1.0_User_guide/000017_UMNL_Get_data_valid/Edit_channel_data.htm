Edit channel readings

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channels in de menu on the left.

3.   Click the name of the channel you want to edit.

4.   Click the actions icon  next to the value you want to edit and select Edit value or click the value.

5.   Press enter. A red tag is displayed, indicating that the value has been edited.

6.   Click Save changes. The edited value will be marked by the edit icon . When you hover over this icon the date of editing is shown.

3

Note

Edited values are not removed from the system, but are retained to allow monitoring of changes.

Confirm one or more suspect channel readings

When you confirm a reading you acknowledge that the reading was incorrect, but it is acceptable. This is why the value of the reading is not altered.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Channels in the menu on the left and filter on suspects using the filter on top of the page.

3.   Click the actions icon  next to the reading and select Confirm value or select multiple values using the CTRL or SHIFT key, click Bulk action and select Confirm value. The confirmed readings are indicated by a red tag.

4.   Click Save changes.

The value of the confirmed reading is not changed and is now indicated by a check mark.
