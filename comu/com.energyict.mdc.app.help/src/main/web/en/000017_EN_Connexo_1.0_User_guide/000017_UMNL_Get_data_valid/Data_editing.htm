What would you like to do?

Correct suspect data

In MultiSense you can easily add values when data is missing or edit and remove invalid data. To identify the data that needs to be adjusted you can use validation rules. Perform the following actions to get your data valid:

1. Configure validation rules to detect missing and incorrect values in the collected data.

2. View suspect data.

3. Edit register data by editing, adding or removing values and Edit channel data by ed­iting or removing values.

4. Validate register data on request or Validate channel data on request to see whether the data is now valid.
