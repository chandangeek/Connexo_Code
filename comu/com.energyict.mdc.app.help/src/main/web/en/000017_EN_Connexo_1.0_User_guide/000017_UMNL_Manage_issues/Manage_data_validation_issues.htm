Manage data validation issues

View data validation issues

5

Steps

1.   Go to Workspace > Data validation > Issues or click My open issues to see all open issues assigned to you.

2.   Click a validation issue.

This will take you to the issue’s overview page. This page shows all issue details such as the usage point and device as well as when the issue was created, the due date, the status and who is assigned to this issue.

The middle of the page shows which and how much data generated the issue. Click an item to expand and see all data. To go the data click the actions icon  and select View data. There you can edit or estimate the data as described in “Edit data” and “Manually trigger estimation of one or more values” respectively.

On the overview page you can also perform different actions such as adding a comment or handling the issue by clicking the actions button  and selecting an action. For more information see the topics described in “Manage data collection issues”.
