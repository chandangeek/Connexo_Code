Filter issues

Filter issues using the filter menu

5

Steps

1.   Go to Workspace > Issues > Issues (or Workspace > Data collection > Issues to see only data collection issue or Workspace > Data validation > Issues to see only data validation issues).

2.   By default the issues are sorted by ascending due date.

3.   Filter using the provided criteria in the menu on top of the page. For some criteria you can select more than one option.

4.   Click Apply.

5.   Deselect items in the drop down menu to remove single filters or click Clear all to remove all applied filters

3

Note

If you click My open issues in the Workspace all open issues that are assigned to you will be shown.

Filter issues using the filter icon

5

Steps

1.   Go to Workspace > Issues > Issues (or Workspace > Data collection > Issues to see only data collection issue or Workspace > Data validation > Issues to see only data validation issues).

2.   Click the filter icon  next to a parameter in the details pane of an issue to use as filter.

Group issues

MultiSense  offers the option to group issues based on different parameters. This allows you fix a group of specific issues in bulk.

5

Steps

1.   Go to Workspace > Issues > Issues (or Workspace > Data collection > Issues to see only data collection issue or Workspace > Data validation > Issues to see only data validation issues).

2.   Under the filter pane, select a group in the drop down menu next to the Group parameter. The results are grouped automatically.

Sort filter results

5

Steps

1.   Go to Workspace > Issues > Issues (or Workspace > Data collection > Issues to see only data collection issue or Workspace > Data validation > Issues to see only data validation issues).

2.   Click Add sort under the filter pane and select one of the options. The sorting is done based on the order of the sorters shown on the screen.

3.   Click the remove icon to remove the sorting option.
