Administrate processes

View system processes

5

Steps

1.   Go to Administration > Process management > Processes.

The overview table shows all processes that have been added and configured in Flow. For each process you can see which version it is, its status and with what feature it is associated.

The associated feature designates the feature within MultiSense the process affects. If there is no feature specified then this is a system wide feature. A feature can for example be a device, then the process will only affects devices.

(De)Activate processes

5

Steps

1.   Go to Administration > Process management > Processes.

2.   Click the actions icon  next to the process and select Activate or Deactivate.

Edit processes

5

Steps

1.   Go to Administration > Process management > Processes.

2.   Click the actions icon  next to the process and select Edit process. You can edit the following fields:

Parameter

Description

Start on

Define on what instance the process will run.

Start when in device state

Designates the device state the device has to reside in to be able to start the process. To add device states click Add device states, click the remove icon to remove states.

Start when allowed

Defines the privileges of the users that are allowed to start processes.T o add privileges click Add privileges, click the remove icon to remove privileges.

3.   Click Save.
