Monitor tasks

View all tasks

5

Steps

1.   Go to Workspace > Task management > Tasks. This view shows all tasks that are open or in progress.

2.   Select a task to see more information in the details pane below.

Filter tasks

5

Steps

1.   Go to Workspace > Task management > Tasks or click My open tasks, Overdue tasks or Unassigned tasks to see a pre-filtered view.

2.   Create a custom filter by selecting one or more filter criteria in the filter bar on top.

3.   Click Apply.

4.   Remove single filters by deselecting items in the drop down menu or remove all filters by click Clear all. Don’t forget to click Apply to execute the changes in your filter.

View tasks on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Processes in the menu on the left.

3.   Select a process in the table.

4.   Process details are shown in the details pane below. Among these details a list of all tasks is shown.

5.   You may click the task to perform the task.
