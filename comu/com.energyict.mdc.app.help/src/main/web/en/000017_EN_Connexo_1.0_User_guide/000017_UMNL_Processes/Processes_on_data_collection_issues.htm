Monitor processes on data collection issues

5

Steps

1.   Go to Workspace > Issues.

2.   Select the desired issue.

3.   On the issue scroll down to the section Contextual information and click the tab Processes.

4.   All running and completed processes are listed with the most recent ones on top.

We see the following data.

Parameter

Description

Process status

Filled circle is a completed process. Open circle with black border is a running process.

Started on

Process creation date and time.

Process ID

Click this link to see all active and completed processes for the selected issue including a status overview and the corresponding variables.

Process version

Started by
