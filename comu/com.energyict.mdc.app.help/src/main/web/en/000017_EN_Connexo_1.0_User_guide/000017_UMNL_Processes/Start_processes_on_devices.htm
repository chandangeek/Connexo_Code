Start processes on devices

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   On the landing page click the actions button  and select Start process.

3.   Select one of the processes available in the drop down menu.

4.   Define the process parameters - if applicable.

5.   Click Start.

3

Note

The available processes are configured in Connexo Admin by activating a process in Process management. Refer to the Connexo Admin guide for more information.
