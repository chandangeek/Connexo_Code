Processes on devices

Monitor processes on a device

View running processes on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Processes in the menu on the left.

3.   Select the tab Running processes.

This overview shows all the processes that are currently running on the device. In the details you can see when and who started the process together with the tasks linked to this process.

View process history on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Processes in the menu on the left.

3.   Select the tab History.

This pages shows the history of all executed processes of this device in table. You can see the status and execution details of every process in the table or the details pane below.

4.   Use the filter on top of the screen to only see specific processes, click Apply.

5.   Remove filters by deselecting items in the drop down menu or clicking Clear.
