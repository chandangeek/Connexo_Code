Monitor time of use

Monitor time of use on device type

View time of use calendars that are assigned to a device type

Before you can start using time of use calendars on devices, they must be assigned to a device type.

5

Steps

1.   Go to Administration > Device types > Open a device type > Specifications > Time of use calendars.

2.   On the Time of use calendars page a list of the assigned time of use calendars with their status is shown in the default tab ToU calendars. The status is Final when the calendar has been added by the user and Ghost if it originates from the device and does not exist in Connexo.

3.   On the details pane you see all periods, day types and tariffs used in the selected time of use calendar.

If there are no assigned time of use calendars to show, you have two options:

  • Add time of use calendars: See “Edit time of use specifications”.

  • Activate time of use: This button is only available if there are inactive time of use calenders associated with the current device type. Clicking this button allows you to edit the time of use specifications where you can activate time of use. See “Edit time of use specifications”.

1

Important

For time of use calendars with status Ghost, there is no detail information available. For this reason you will also not be able to preview such a time of use calendar. They can only be removed.

Preview a time of use calendar

You can view the contents of a time of use calendar, i.e. its entire setup with events, tariffs, etc.,  to verify if it fits your needs.

5

Steps

1.   Go to Administration > Device types > Open a device type > Specifications > Time of use calendars.

2.   In the tab ToU calendars, select a time of use calendar.

3.   Click the action icon  in the selected row or the click the Actions button in the details pane. Then click View preview.

4.   In the filter on top of the page, select the start day for the week you want to view. You will see six days of the calendar starting from the day in your filter. The first day of the current week is set by default.

5.   The time of use calendar is graphically represented with the week days on the horizontal axis and the hours of the day on the vertical axis. For each day, you can see the tariff(s) used during that day. Hovering over each day segment shows a tooltip with the name, the start and the end hour of the tariff.

6.   In the detail pane you see the periods, day types and tariffs that are used in the shown week.

3

Note

Because time of use calendars with the status Ghost have no information in Connexo, it is not possible to preview them.
