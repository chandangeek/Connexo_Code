Configure device commands on a device

Add commands to a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Commands in the menu on the left.

3.   Click Add command. Complete the following fields:

Parameter

Description

Command category

Select the category of commands.

Command

Select the command. If the command is not part of the device’s communication tasks, a warning message will be shown.

If the command is part of a communication task that is not scheduled on the device, then also here a warning message will be shown. However the commands can still be added to the device.

Release date

Select the data when the command will become available to be picked up when there is a connection.

Attributes

Some command require extra attributes to be specified such as the message that needs to be sent to the device.

4.   Click Add.

5.   After adding the command, MultiSense asks if you want to trigger the command now. If yes, click Trigger otherwise click Cancel.

Trigger communication for a command

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Commands in the menu on the left.

3.   Click the actions icon  next to the command you want to trigger and select Trigger now. Note that only pending and waiting commands can be triggered.

4.   Click Trigger.

Change the release date of a command

1

Important

Can only be done when the status of the communication task is “Pending”.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Commands in the menu on the left.

3.   Click the actions icon  next to the command you want to change the release date of and select Change release date.

4.   Change the release date en click Save.

Revoke commands

1

Important

Can only be done when the status of the communication task is “Pending”.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Commands in the menu on the left.

3.   Click the actions icon  next to the command you want to revoke and select Revoke.

4.   Click Revoke.
