File management in MultiSense

In MultiSense it is possible to add files to device types so these can be added to devices when needed. The communication protocol used by the device type, determines whether it is possible to manage files. If the communication protocol supports at least one command that requires a specific file, the file management menu will be available on the device type.

If file management is possible, then the menu item File management is available in the navigation menu of the device type. File management has two tabs: Specifications and Files. On the Specifications tab you can edit file management options. On the Files tab you can manage all files and activate file management when inactive.
