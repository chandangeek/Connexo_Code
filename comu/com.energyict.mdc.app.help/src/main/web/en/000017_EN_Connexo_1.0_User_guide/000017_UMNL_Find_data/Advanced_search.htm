Advanced search

Search for devices

5

Steps

1.   Click Search in the upper right corner of the page.

2.   The following criteria are available by default:

Parameter

Description

MRID

Type the complete Master Resource IDentifier (MRID) of a device or use wildcards.

Serial number

Type the complete serial number of a device or use wildcards.

Device type

Select one or more device types or search for a device type by typing the name of the device type.

Device configuration

Select one or more device configurations or search for a device configuration by typing the name. Is only available once you select a device type.

State

Select one or more states from the list or search for a state by typing the name. Is only available once you select a device type.

Device group

Select one or more device groups from the list or search for a device group by typing the name.

3.   Click Add criteria in the upper right corner to add other criteria.

4.   Remove search criteria by selecting the criteria and clicking the remove icon .

5.   Click Clear all to remove all criteria.

6.   Click Search.

7.   Change the displayed columns by clicking the Columns button above the results table. Check the columns you want to see and uncheck the columns you wish to hide.

3

Note

The following wildcards can be used:

Wildcard

Function

*

Matches zero or more characters

?

Matches exactly one character

When all fields are empty and you click Search, then no result will be shown. You need to type “*” in one of the search criteria to see all possible search results.

Search for usage points

5

Steps

1.   Click Search in the upper right corner of the page.

2.   Select Usage point in the Search for field.

3.   Click the MRID search criteria and enter the complete MRID or use wild cards. Other search criteria can be added by clicking Add criteria in the upper right corner.

4.   Remove search criteria by selecting the criteria and clicking the remove icon .

5.   Click Clear all to remove all criteria.

6.   Click Search.

7.   Change the displayed columns by clicking the Columns button above the results table. Check the columns you want to see and uncheck the columns you wish to hide.
