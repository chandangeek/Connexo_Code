Advanced search

Depending on the object that you are searching for a different set of criteria will be available and the data will be presented differently in the search result table. However, there is a lot of functionality that is common.

Common functionality

5

Steps

1.   Click Search in the upper right corner of each page in Connexo MultiSense.

2.   In Criteria, specify the desired default search criteria. Click the button of the criterion and enter your search criteria.

3.   Click Add criteria to add extra criteria.

4.   Remove search criteria by unchecking or deleting the specified criteria or by simply clicking the remove icon . The default criteria buttons cannot be removed. You can only delete their specified values.

5.   Click Search.

6.   To change the default columns in the search result table,

a. Click Columns above the search results table.

b. Check the columns you want to see.

c. Uncheck the columns you want to hide.

d. Click Done to apply your changes.

7.   To get an exact count of the items in the search results, click Count above the search results table. The number of found items will be shown on the button.

The calculation of found items may take some time depending on the size of the data set. In case of very large data sets, a time out may occur.

8.   Click Clear all to remove all criteria.

Important 

Be aware that some criteria are dependant and become available after you have specified one or more other criteria. The required criteria are shown in a tooltip on the dependant criterion.

Note

While specifying the search criteria the following wildcards can be used:

Wildcard

Function

*

Matches zero or more characters

?

Matches exactly one character

In case no search criteria were specified no search results are shown. To show all, type “*” in one of the search criteria to see all values.

 

Search for devices

5

Steps

1.   In Search for, select Device.

The following primary criteria are available by default when searching for devices.

Parameter

Description

Name

Type the name of a device or use wildcards.

Serial number

Type the complete serial number of a device or use wildcards.

Device type

Select one or more device types or search for a device type by typing the name of the device type.

Device configuration

Select one or more device configurations or search for a device configuration by typing the name. Is only available once you select a device type.

State

Select one or more states from the list or search for a state by typing the name. Is only available once you select a device type.

Device group

Select one or more device groups from the list or search for a device group by typing the name.

Location

Select one or more locations from the list or search for a location by typing the name.

Search for usage points

5

Steps

1.   In Search for, select Usage point.

The following primary criteria are available by default when searching for usage points.

Parameter

Description

Name

Type the name of a usage point or use wildcards.

Service category

Select one or more service categories from the list or search for a service category by typing its name.

Metrology configuration

Select one or more metrology configurations from the list or search for a service category by typing its name.

Search on master device

With the search you are able to search for all master devices or specific devices. You can also find all devices that are not a master device.

5

Steps

1.   Go to Search.

2.   Click Add criteria.

3.   Go to Topology and select Has master device.

4.   To find specific devices do one of the following steps:

a. To find all non-slave devices select No.

b. To find specific slave devices select Yes and provide a name or use wild cards.

c. To find all slave devices select Yes and do not provide a name.

5.   Click Search.
