Data logger slaves

Manage data logger slaves

View data logger slaves

You are only ably to see data logger slaves on devices that are configured as data loggers.

5

Steps

1.   Select a data logger device using Search, see “Advanced search”.

2.   Click Data logger slaves in the menu on the left.

3.   When there are no slave devices yet, you have the option to link devices. Click Link data logger slave and see “Link data logger slave to data logger” for more information.

If data logger slaves are available, select a data logger to see the features such as device type, configuration and the date of linking in the details pane below.

4.   Click the slave device name to go to its landing page.

Add a data logger slave device type

5

Steps

1.   Go to Administration > Device management > Device types.

2.   Click Add device type. Complete the following fields:

Parameter

Description

Type

Select Data logger slave device type.

Name

A unique name.

Device life cycle

Select the device life cycle all devices of this device type will use. For more information see “Device life cycle management”.

3.   Click Add.
