Link data loggers and slaves

Link data logger slave to data logger

5

Steps

1.   Select a data logger device using Search, see “Advanced search”.

2.   Click Data logger slaves in the menu on the left. Or click Manage data logger slaves in the data logger slave widget see “Other widgets on the object landing page” for more information.

You can also link data loggers to a slave device if you go to the channel or register of the data logger you want to link and click the actions icon  in the details pane on the bottom of the page.

3.   Click Link data logger slave.

In step 1 you have two options you can link an existing data logger slave or you can create a new data logger slave.

  • Link an existing data logger slave

a.  Select the option Existing data logger slave and start typing the data logger slave MRID.

b. Select a device.

  • Create a new data logger slave

a.  Select the option New data logger slave.

b. Fill in the device attributes, see “Add device” for more information.

4.   Click Next.

5.   In step 2 map the channels of the data logger slave to those of the data logger by selecting one of the channels in the menu. In this menu you can find all channels of the data logger which have the same interval as the data logger slave channel you are viewing.

You need to map the channels before you can proceed to the next step. You can only use each data logger channel once.

6.   In step 3 map the registers of the data logger slave to those of the data logger by selecting one of the registers in the menu.

You need to map the registers before you can proceed to the next step. You can only use each data logger register once.

7.   In step 5 select the linking date.

8.   Click Next.

9.   In step 5 click Confirm if you want to link the slave with the data logger.

10.   Step 6 shows the result of the action. Click Finish.

Unlink data logger slave

5

Steps

1.   Select a data logger device using Search, see “Advanced search”.

2.   Click Data logger slaves in the menu on the left.

3.   Click the actions icon next to slave device you want to unlink.

4.   Specify the unlinking date and click Unlink.
