View channel data on data logger

5

Steps

1.   Select a data logger device using Search, see “Advanced search”.

2.   Go to Data sources > Channels in the menu on the left.

3.   Select the Readings tab.

4.   See “Monitor channel data” for more information on how to view channel data.

If the data logger is linked to a data logger slave then the bars of data in the graphic will be grey. When you hover over the data the pop-up will show the slave devices linked to this data. If you select a value in the table with all the readings, you can find a link to the data logger slave devices on the General tab of the details pane on the bottom of the page.

5

Steps

1.   Select a data logger device using Search, see “Advanced search”.

2.   Go to Data sources > Registers in the menu on the left.

3.   Select the Readings tab.

4.   See “Monitor register data” for more information on how to view register data.

If you select a value in the table with all the reading, you can find a link to the data logger slave devices on the General tab of details pane on the bottom of the page.
