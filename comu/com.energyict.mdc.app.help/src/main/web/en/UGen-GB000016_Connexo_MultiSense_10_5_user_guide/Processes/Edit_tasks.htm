Edit tasks

Assign task to user

5

Steps

1.   Go to Workspace > Task management > Tasks or select one of the other pre-filtered views.

2.   Click the actions icon  and select Edit task.

3.   Make the required edits.

4.   In Assignee, you may select one of the users from the drop down menu to assign the task to another user. Also consider Due date and Priority of the selected task.

5.   Click Save.
