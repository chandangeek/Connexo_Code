Export on request

Run data export task

5

Steps

1.   Go to Administration > Data exchange > Data export tasks.

2.   Click the actions icon  or the actions button  and select Run now.

3.   Click Run in the dialogue box to run the export task now.

The status in the details pane will change to busy and the start date to now. When the data export was successful the status will change to success and the finish time will be updated. If not, the status will be set to failed and a reason for failure will be displayed.

Run data export task with custom parameters

5

Steps

1.   Go to Administration > Data exchange > Data export tasks.

2.   Click the actions icon  or the actions button  and select Run... .

You can customise the following parameters:

Parameter

Description

Start date

Define when to run the next export task.

Export window

Define the start en end date of the export window. If the continuous export strategy is selected, then it will not be possible to define the start date. Automatically the end date of the previous export will be used as start.

Update window

Only available for export tasks where export of updated data is active.

3.   Click Run.

Retry export task occurrence

You can execute a past data export task again to export the same data set once more.

5

Steps

1.   Go to Workspace > Data exchange > Export history.

2.   Apply filters to only see specific occurrences.

3.   Click the actions icon  or the actions button  and select Retry.

The data export task will run with the same parameters as those of the finished export task.
