Monitor validation results on a device

Understand validation results

In Connexo MultiSense it is possible to view all validation results of a single device in one view. It is possible to see the amount of suspect values per validation rule set, version and validation rule in the Configuration view. The other option is to see the amount of suspect values per channel or register in the Data view. From the data view there is an option to go directly to the suspect values.

View validation results from the configuration view

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Validation results in the menu Reading quality.

3.   Click the Configuration view tab.

This view shows the amount of suspects per validation rule set, version and validation rule.

View validation results from the data view

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Validation results in the menu Reading quality.

3.   Click the Data view tab.

In this view you can see the amount of suspects per channel of load profile or per register available on the device.

4.   Click the link of the suspect values to see the suspect in detail. On this page you can then navigate to the other registers or channels with suspect values by clicking the arrows in the upper right corner.
