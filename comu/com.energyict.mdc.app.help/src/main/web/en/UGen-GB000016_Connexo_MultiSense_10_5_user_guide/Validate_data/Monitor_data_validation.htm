Monitor validation status on a device

View validation status of a device in the validation configuration

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Validation configuration in the menu Reading quality in the device menu on the left.

Under Status you can see the current status of the validation on this device. If validation is active you can also see when data is validated. The status will show if data is validated on storage or by a validation task. this setting is configured by the device configuration (see “Add device configurations” for more information) or can be set manually when you activate validation on the device (see “Manage validation status on a device” for more information).

View validation status of a device on the landing page

5

Steps

1.   Select a device using Search, see “Advanced search”.

All details concerning the validation on the device are shown in the Data validation pane. The following parameters are shown:

Parameter

Description

Device validation status

Indicates whether validation is active or not.

All data validated

Shows if all data is validated.

Validation result

Shows the amount of suspect values.

Last validation run

Date when validation was performed last.
