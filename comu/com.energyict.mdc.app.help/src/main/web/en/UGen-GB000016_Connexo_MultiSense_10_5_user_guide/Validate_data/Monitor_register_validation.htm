Monitor register validation

View validation status of registers

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Registers in the menu on the left.

3.   Select a register in the table. The details of this register are shown in the second part of the details pane on the bottom of the page or on the specification page when you click the register name and select the Specifications tab.

The following information is shown:

Parameter

Description

Device validation status

Indicates whether validation on the device is active.

Register validation status

Indicates that there are active validation rules available on the device that can validate these register values (matching reading types).

All data validated

Shows if all data is validated.

Suspects

Shows the amount of suspect values per violated validation rule and parameters.

Lasted checked

Moment when the last interval was checked in the validation process.

View validation status of register data

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Registers in the menu on the left.

3.   Click the actions icon  and select View data to see the validation status of all values.

Suspect values are marked by a red flag, unvalidated values are marked by a black flag and valid values are not marked. Values with an end device reading quality are marked with the following icon: , see “Understand reading quality” for more information.

4.   Select a value in the table, the following validation details are shown in details pane below: validation status of the validation rules, validation status of the data and the result of the validation. If the validated data was found suspect, then you can also see the validation rule and parameters that were violated.
