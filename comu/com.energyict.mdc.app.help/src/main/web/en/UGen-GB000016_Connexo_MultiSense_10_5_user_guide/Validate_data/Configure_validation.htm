View all configured validation rule sets

5

Steps

1.   Go to Administration > Validation > Validation rule sets.

On top of the page you see all configured validation rule sets.

In the middle of the page you can see the different version of the validation rule set and the period for which they are active. For every version the amount of active and inactive validation rules are shown.

Under the validation rule sets versions the validation rules that belong to the validation rule set version that is selected are shown.

Under the validation rules table you can see the details of the validation rule you selected in the table.

2.   Click a validation rule set.

On the validation rule sets page three items are shown in the menu on the left:

Menu item

Description

Overview

Shows the name, description and the active version of this set.

Validation rules

Shows a list of all version of the validation rule set together with the amount of active validation rules included in this version. Per version all validation rules are shown together with the possibility to remove, edit and view the rules.

Device configurations

Shows a list of all device configurations linked to this rule set and the possibility to add device configurations.
Figure: Validation rule sets homepage.

 

View validation rule sets on a device configuration

5

Steps

1.   Go to Administration > Device management > click Device type > select a device type > click Configurations > select a configuration > Reading Quality > Validation rule sets.

These validation rule sets are used to validate all data that is generated on devices with this configuration and the same reading types as the validation rules.

3

Note

When you go to the register and load profile configuration pages the used validation rules are also shown here.

View validation rules on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Validation configuration under Reading quality to see the validation rules on this device.

On top of the page, the status of the validation is shown and can be changed by clicking the button on the upper right corner.

Below the status there is an overview table with all validation rule sets configured on this device. When you select a rule set in this table, you will see the versions of this rule set together with the included validation rules.
