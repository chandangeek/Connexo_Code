Configure security on device type and configuration

Configure security on device type

On the device type you add the security accessors that have been created in Connexo MultiSense Administration. Security accessors can store a particular type of key or certificate. Depending on the key or certificate type, additional attributes become available. Privileges are used to determine who can view and edit the security accessors.

View security accessors on a device type

You can view the security accessors that are available on a device type to investigate the security options for that device type.

5

Steps

1.   Go to Administration > Device management > Device types.

2.   Open a device type by clicking its name.

3.   Click Security accessors in the group Specifications in the navigation bar on the left.

The available security accessors are shown in a table with name, accessor type (key or certificate), key or certificate type and the validity period (if available). Further details are available in the details pane under the table.

Apart from the fields shown in the table, the details pane also shows:

  • Storage method

  • View and edit privileges (only for key accessor type). You can see what roles are associated with each of the privilege levels by clicking the information icon  next to the fields View privileges and Edit privileges.

  • Trust store (only for certificate accessor type).

  • Status of central management of the security accessor (only for certificate accessor type). If yes, then active and passive certificates are shown.

Add security accessors on a device type

You can add multiple security accessors (keys and certificates) for use in a security setting, connection method or as a general attribute on a device type.

5

Steps

1.   Go to Administration > Device management > Device types.

2.   Open a device type by clicking its name.

3.   Click Security accessors in the group Specifications in the navigation bar on the left.

4.   Click the button Add security accessors. The button is located right above the overview table.

5.   On the Add security accessors page, select the needed security accessors.

The table contains all security accessors that haven’t yet been added to the selected device type.

If needed (and if you have sufficient privileges), you can click the link Manage security accessors on top of the table to go directly to Connexo Administration. Here you can add, edit or remove the available security accessors. See “Manage security accessors”.

6.   Click Add.

The selected security accessors are now shown in the overview table on the page Security accessors.

Remove security accessors from a device type

You can remove unused security accessors from a device type that doesn’t have any active device configurations.

5

Steps

1.   Go to Administration > Device management > Device types.

2.   Open a device type by clicking its name.

3.   Click Security accessors in the group Specifications in the navigation bar on the left.

4.   Click the action icon  next to the needed security accessor and select Remove.

5.   Click Remove to confirm removal.
