Multi-element slaves

Manage multi-meter slaves

View multi-element slaves

You are only ably to see multi-element slaves on devices that are configured as multi-element devices.

5

Steps

1.   Select a multi-element device using Search, see “Advanced search”.

2.   Click Multi-element slaves in the menu on the left.

3.   When there are no slave devices yet, you have the option to link devices. Click Link multi-element slave and see “Link multi-element slave to multi-element device” for more information.

In the table you can see the serial number, device type, configuration and the date of linking of each slave.

4.   Click the slave device name to go to its landing page.

Add a multi-element slave device type

5

Steps

1.   Go to Administration > Device management > Device types.

2.   Click Add device type. Complete the following fields:

Parameter

Description

Type

Select Multi-element slave device type.

Name

A unique name.

Device life cycle

Select the device life cycle all devices of this device type will use. For more information see “Device life cycle management”.

3.   Click Add.
