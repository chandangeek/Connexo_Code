Link multi-element devices and slaves

Link multi-element slave to multi-element device

When you link a multi-element device to a slave then you will always create a new device that inherits all attributes from its master device. There is no other way to create new multi-element slave devices in MultiSense.

5

Steps

1.   Select a multi-element device using Search, see “Advanced search”.

2.   Click multi-element slaves in the menu on the left. Or click Manage multi-element slaves in the data logger slave widget on the device its landing page, see “Other widgets on the object landing page” for more information.

3.   Click Link multi-element slave.

4.   In step 1 you have to select the device type and configuration of the slave. A name is automatically generated, but you can change this if needed.

5.   Click Next.

6.   In step 2 map the channels of the slave to those of the multi-element device by selecting one of the channels in the menu. In this menu you can find all channels of the multi-element meter which have the same interval as the slave channel you are viewing.

You need to map the channels before you can proceed to the next step. You can only use each channel once.

7.   Click Next.

8.   In step 3 map the registers of the slave to those of the multi-element device by selecting one of the registers in the menu.

You need to map the registers before you can proceed to the next step. You can only use each register once.

9.   In step 4 select the linking date is automatically set. This equals the first meter activation date of the master.

10.   Click Next.

11.   In step 5 click Confirm if you want to link the slave with the multi-element device.

12.   Step 6 shows the result of the action. Click Finish.

Unlink multi-element slave

5

Steps

1.   Select a data logger device using Search, see “Advanced search”.

2.   Click Multi-element slaves in the menu on the left.

3.   Click the actions icon next to slave device you want to unlink.

4.   Specify the unlinking date and click Unlink.
