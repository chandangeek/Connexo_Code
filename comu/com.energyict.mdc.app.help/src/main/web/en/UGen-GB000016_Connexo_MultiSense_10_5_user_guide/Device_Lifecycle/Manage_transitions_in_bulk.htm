Manage transitions in bulk

How to transition devices in bulk

In MultiSense multiple devices can be transitioned at the same time, this is achieved through import services. Using these import services you upload CSV files containing all relevant information to a certain file location, where this file is picked up and processed by MultiSense.

Import services are configured in Admin. When creating new import services Connexo offers different default file importers that allow the import of different types of data to perform different actions. During configuration you need to specify the location where files that will be imported have to placed. Next to the import folder you will also specify the folder for in progress files and files of successful and failed import. All this information is also visible in MultiSense when you look at the import service details located here: Administration > Data import > Import services.

2.   Once import is finished successfully the original CSV is moved to the success folder. Shipment information is now added to MultiSense.
