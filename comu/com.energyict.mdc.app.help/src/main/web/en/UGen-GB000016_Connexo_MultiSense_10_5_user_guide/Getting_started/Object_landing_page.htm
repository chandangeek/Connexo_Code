Understand the object landing page

The object landing page is the first page you see when you click an objectâ€™s link in - for example - the search results. On this page you can quickly overview the essential information for the object. You can see its capabilities and how it is performing. More detailed information can be accessed from this page by clicking one of the links in the vertical navigation bar on the left. The widgets also contain a lot of links that you can click to view more detailed information.

The object landing page always shows a number of default widgets. The other widgets depend on the type of object that is shown. For example, if a device is not directly addressable, then you will see no connection widget.
