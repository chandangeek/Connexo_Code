Consolidated system task management

Throughout Connexo MultiSense a number of system tasks can be added and monitored. For instance data estimation and data validation tasks. You can use the consolidated system task management page to see and manage all of these system tasks together on one page. Note that data estimation and validation tasks can also be monitored in “View validation tasks” and “View estimation tasks”. However, there are also system tasks that can only be seen in this consolidated task management.

View all system tasks in Connexo MultiSense

You can make an overview of system tasks in Connexo MultiSense that are relevant to you at a particular point in time.

5

Steps

1.   Go to Administration > Task management > Tasks.

2.   In the filter bar on top, enter the necessary criteria to narrow down the shown system tasks to what is relevant for your needs. Don’t forget to click Apply to show the filtered data.

The table shows all system tasks that are currently available in Connexo MultiSense. You can see the name of the queue they are on and when their next execution is scheduled.

3.   Click the name of a task to see its details.

The details page will depend on the type of task that is selected.

View the history of a system task from the consolidated overview

You can see the history of a system task if it is available for that task type. For instance, an estimation task.

5

Steps

1.   Go to Administration > Task management > Tasks.

2.   Select View history from the actions icon in the line of the needed task.

On the history page you see an overview of the instances of the selected task with date, time, duration and status.

3.   Select View log from the actions icon  in the line of the needed task to see what has happened during each instance.

The log page shows all messages that have been logged during the execution of the task instance.

Add a system task from the consolidated overview

In the consolidated overview of Connexo MultiSense system tasks you can add system tasks.

3

Note

You can’t add export tasks on this page. For this type of tasks you still need to go to the export task page. See “Manage data export tasks”.

5

Steps

1.   Go to Administration > Task management > Tasks.

2.   Click the button Add task.

3.   Select the type of task that you want to add.

Depending on the entered task type, the remaining fields are added to the screen.

4.   Complete all needed fields.

5.   If needed, select a follow up task that must be triggered after the selected task has finished. For this purpose, use the field Followed by.

6.   Click Add.

The new system task is now shown in the overview table. You can see the added task also on the corresponding overview page for that particular task.

Edit a system task from the consolidated overview

In the consolidated overview of Connexo MultiSense system tasks you can edit system tasks to correct eventual mistakes.

3

Note

You can’t edit export tasks on this page. For this type of tasks you still need to go to the export task page. See “Manage data export tasks”.

 

5

Steps

1.   Go to Administration > Task management > Tasks.

2.   Select Edit from the actions icon in the line of needed task.

3.   The edit page shows the fields that are relevant for the corresponding task type.

4.   Click Save.

Remove a system task from the consolidated overview

In the consolidated overview of Connexo MultiSense system tasks you can remove system tasks to correct eventual mistakes.

3

Note

You can’t remove export tasks on this page. For this type of tasks you still need to go to the export task page. See “Manage data export tasks”

5

Steps

1.   Go to Administration > Task management > Tasks.

2.   Select Remove from the actions icon in the line of needed task.

3.   Click Remove to confirm the removal.

Run a task from the consolidated overview

You can queue a system task for immediate execution if this is available for the selected task type. For instance, an estimation task.

3

Note

You can’t run export tasks on this page. For this type of tasks you still need to go to the export task page. See “Manage data export tasks”

5

Steps

1.   Go to Administration > Task management > Tasks.

2.   Select Run from the actions icon in the line of needed task.

3.   Click Run to put the selected task on the queue.
