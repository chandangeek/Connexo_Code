Connection history

View connection history

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Connection methods in the menu on the left.

3.   Click the actions icon  or the actions button  in the details pane and select View history.

This page shows the history of the connection method with all its connections. For every connection the duration, status, result and executed communication tasks that used the connection are shown.

4.   Click a connection in the table. Below the table more information is shown.

A details pane shows further details of the connection. It shows the summary of the connection, here you can see which errors and warnings occurred during this connection. Click Show connection details to see the connection configuration details and the details of the connection execution.

Under the details pane there is a table showing all communication tasks that used the connection together with the details of each task.

5.   Click a communication task in the table. Below the table more information is shown.

A details pane shows further details of the communication task. Here you can see the communication summary that contains the errors and warnings generated during the execution of the communication task.

Click Show communication details to see the communication task configuration details and the details of the communication task execution.

View connection log

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Connection methods in the menu on the left.

3.   Click the actions icon  or the actions button  and select View history.

4.   Click the actions icon or the actions button in the details pane of one of the connections and select View connection log.

The log page shows the details of the selected connection on top of the page.

5.   You can filter the logging and log level (Debug or Trace) and choose to see connection or communication related logging or both. Click Apply.

Below the filter you can find the table containing all connections logs with the message. An icon next to the log entry shows whether it is an error  or a warning . More details are given in the details pane below the table.

Filter connection log

5

Steps

1.   View the connection logs as described in “View connection log”.

2.   Select the filters in the filter pane on top of the page.

3.   Click Apply. The applied filters are shown on top of the page below the connection details.

4.   Deselect the item in the drop down menu to remove the filter.

5.   Click Clear all to remove all filters.
