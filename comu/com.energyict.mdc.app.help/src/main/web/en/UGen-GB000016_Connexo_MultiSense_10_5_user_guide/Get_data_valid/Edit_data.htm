Edit register data

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Registers in the menu on the left.

3.   Click the register name of which you want to edit data to or click View suspects to only see suspect values.

4.   Click the actions icon  next to the register value you want to edit and select Edit.

5.   Write the new value and click Save. The edited value will be marked by the edit icon . When you hover over this icon the date of editing is shown.

3

Note

Edited values are not removed from the system, but are retained to allow monitoring of changes.

Confirm one or more suspect register readings

When you confirm a reading you acknowledge that the reading was incorrect, but it is acceptable. This is why the value of the reading is not altered.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Registers in the menu on the left and filter on suspects using the filter on top of the page.

3.   Click the actions icon  next to the reading and select Confirm value.

The value of the confirmed reading is not changed and is now indicated by a check mark.
