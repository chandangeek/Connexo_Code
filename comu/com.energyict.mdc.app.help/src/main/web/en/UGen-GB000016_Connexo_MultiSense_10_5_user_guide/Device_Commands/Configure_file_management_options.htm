Configure file management options

Edit file management specifications on a device type

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Specifications > File management > select the Specifications tab.

2.   Click the actions button and select Edit.

3.   Define whether it is allowed to send files to devices.

4.   Click Save.

When you deactivate file management then all files added to this device type will be removed.

Add files to a device type

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Specifications > File management > select the Files tab.

The table on this page shows all files added to this device type together with the date added.

2.   Click Add file.

3.   A new window opens where you can select the file you wish to upload. Beware that the maximum allowed size of the file is 2MB and the file name should be unique on the device type. Click Open.

Remove files from device type

5

Steps

1.   Go to Administration > Device management > Device type > select a device type > Specifications > File management > select the Files tab.

2.   Click the remove icon next to the file.

3.   Confirm and click Remove.

The file will no longer be available to send to a device and pending commands that use this file will fail to execute.

Send files to a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Commands in the menu on the left.

3.   Click Add command. See “Add commands to a device” for more information.

4.   Select a command that requires a file and select the file in the attributes of the command.

5.   Click Add.
