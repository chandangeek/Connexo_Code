Configure command limitation rules

Add command limitation rule

5

Steps

1.   Go to Administration > Device communication > Command limitation rules.

2.   Click Add command limitation rule and complete the following parameters:

Parameter

Description

Name

Give a unique name.

Day limit

Define the maximum number of instances of all selected commands that can be sent per day. Amount must be lower than the week and month limit.

Week limit

Define the maximum number of instances of all selected commands that can be sent per week. Amount must be lower than the month limit.

Month limit

Define the maximum number of instances of all selected commands that can be sent per month.

Commands

Select the commands on which the rule applies. You can filter on category. Click Add.

3.   Click Add.

Edit command limitation rule

1

Important

Editing a command limitation rule needs to be approved by the command limitation approver(s) before the change is effective.

5

Steps

1.   Go to Administration > Device communication > Command limitation rules.

2.   Click the actions icon  next to the command limitation rule you want to edit and select Edit.

3.   Change the parameter(s) and click Save.

Remove command limitation rule

1

Important

Removing a command limitation rule needs to be approved by the command limitation approver(s) before the change is effective.

5

Steps

1.   Go to Administration > Device communication > Command limitation rules.

2.   Click the actions icon  next to the command limitation rule you want to remove and select Remove.

3.   Click Remove to confirm.

(De)activate command limitation rule

1

Important

(De)activating a command limitation rule needs to be approved by the command limitation approver(s) before the change is effective.

5

Steps

1.   Go to Administration > Device communication > Command limitation rules.

2.   Click the actions icon  next to the command limitation rule you want to (de)activate and select (De)Activate.

3.   Click (De)Activate to confirm.
