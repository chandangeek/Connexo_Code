View commands on a device

See what commands have been added on a particular device.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Commands in the menu on the left.

The overview table shows all the commands that have been added to the device together with the category they belong to and their status and release date. More details are shown in the details pane on the bottom of the page.

Monitor all commands that have been added on a device

Explore all commands that have been added to one or more devices in Connexo MultiSense.

5

Steps

1.   Go to Workspace > Data communication > Commands.

2.   Enter the needed filter criteria in the filter bar on top of the page. Don’t forget to click Apply to execute the filter.

3.   See the corresponding commands in the table. Details are shown in the details pane under the table.
