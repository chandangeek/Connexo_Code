Monitor data estimation

View estimation task history

5

Steps

1.   Go to Administration > Data estimation > Estimation tasks.

2.   Click the estimation task name and select History in the menu on the left or click the actions icon  next to the task and select View history.

The history view shows every executed estimation task together with its status and duration. The details pane below the overview table shows more details of the estimation task such as when it started and finished.

3.   Use the filter on top of the page to only see the history from a specific time window.

4.   Click the actions icon  and select View log to see a detailed logging of the executed task.
