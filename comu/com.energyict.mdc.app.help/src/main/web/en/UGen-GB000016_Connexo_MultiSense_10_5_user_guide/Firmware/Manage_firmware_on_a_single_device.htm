Manage firmware on a single device

Upload firmware to a device and activate later

Use this option if you want to manually activate or use a command to activate the firmware at a later time.

1

Important

For the upload to proceed successfully the firmware communication task needs to be activated on the device type.

5

Steps

1.   Open the needed device using Search, see “Advanced search”.

2.   In the group Configuration in the left-hand column, click Firmware.

3.   Click the Actions button and select Upload firmware and activate later.

4.   Complete the following fields:

Parameter

Description

Firmware version

Select one of the firmware version added to the device configuration and available on this device. Only test and final versions are shown.

Upload file

Select when the firmware needs to be uploaded: now or on a specific date.

5.   Click Confirm.

3

Note

It is only possible to upload firmware to a device if this is specified by the communication protocol and the device type in the firmware management options. See “Understand firmware management options” for more information.

Activate firmware on a device

Activate firmware on a device to start using a newer version of the firmware on the device.

5

Steps

1.   Open a device using Search, see “Advanced search”.

2.   In the group Configuration in the left-hand column, click Firmware.

3.   If firmware has been uploaded but is not yet activated a message is shown. Click Activate.

4.   Specify the activation date, click Activate.

Check firmware version on a device

Re-read the firmware version on a device in order to make sure that the present firmware version information is shown on the device landing page.

5

Steps

1.   Open a device using Search. See “Advanced search”.

2.   In the group Configuration in the left-hand column, click Firmware.

3.   Click the Actions button and select Check firmware version now.

Once the firmware is checked, the Last checked date will be updated.

See the firmware upgrade history

See what firmware versions have been installed on a device over time and the end result of each these firmware upgrades.

5

Steps

1.   Open a device using Search. See “Advanced search”.

2.   In the group Configuration in the left-hand column, click Firmware.

3.   Click the tab History.

The shown table lists all firmware upgrades for the selected device with their version name, the upload and activation dates, the user who triggered the firmware upgrade and the end result.

4.   Select a firmware version and select View log from the actions icon .

You are now brought to the history page of the Firmware management communication task. Communication and connection details are available under the table. For each line in the history table you can see a log of the corresponding communication and connection by selecting View communication log or View connection log from the actions icon . See “View communication log” and “View connection log”.
