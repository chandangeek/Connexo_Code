Monitor register data

View registers on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Registers in de menu on the left.

This page shows all registers available on this device together with the date until which the register data was last collected and the latest value.

3.   Filter registers on Register groups and Registers. Only register groups that contain registers that are available on that device are available. Click Apply.

4.   Click the Register readings tab on top of the page to see all register readings from the registers specified by the filter. Filters specified on the Register tab are maintained on the Register readings tab.

This page shows all calculated (if these are available) registers readings of the selected registers sorted by measurement time together with the delta value, if available. The measurement time is the time when that data was collected.

View register data

1

Important

Add registers to devices via device configurations, see “Configure register configurations”.

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Registers in the menu on the left.

3.   Click the register name.

On the Readings tab the overview table shows all values of that register together with the measurement time. On the bottom of the page more details of the selected register value are provided. Three tabs show different information.

a. General: shows information of the measured value.

b. Validation: shows validation information.

c. Reading quality: important observation concerning the value, for example a power outage occurred.

The following fields are shown:

Parameter

Description

Measurement time

The timestamp when the register data was measured by the meter.

Reading time

The timestamp when the data was collected from the meter by for example the data collection system.

Collected

The value collected by the device.

Calculated

Shows the calculated value when the register uses a multiplier. If there is no multiplier then this column is not available

Delta value

The difference between the current value and the previous value (only for cumulative values).

Last update

Shows when the reading was last handled. This can be due to validation, estimation or editing.

Validation status

Status of the validation rules (active or inactive).

Data validated

Indicates whether the values has been validated or not.

Validation result

Indicates if value is suspect or not and for suspect values the violated rule is shown.

Reading quality

Indicates that there is something remarkable about this value. Hover over the information icon for more details.

View register history

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Got to Data sources > Registers in de menu on the left.

3.   Click a channel name.

4.   Click the actions icon  next to the value you want to see the history of and select View history.

This page shows a table containing all edits made to this value together with the date of the change and who changed the value. A new history item is created when a value is suspect, edited, estimated or confirmed.

When you select a value you can see all details in the tables on the bottom of the page divided over three tabs: general info, reading value and reading quality.

5

Steps

2.   Got to Data sources > Registers in de menu on the left.

3.   Click a channel name.

4.   The Bulk action button and select View history.

The history page shows all the history of all values that comply with the defined filters, which are the same as on the previous page.

5.   Use the filters on top to see specific values. You can define the time period you want to visualise. Select Changed data to only show values that have been changed over time.
