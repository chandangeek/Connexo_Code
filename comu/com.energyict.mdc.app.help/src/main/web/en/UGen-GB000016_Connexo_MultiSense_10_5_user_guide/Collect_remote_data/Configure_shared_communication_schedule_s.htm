Configure shared communication schedule(s)

Add shared communication schedules

5

Steps

1.   Go to Administration > Device communication > Shared communication schedules.

2.   Click Add shared communication schedule. Complete the following fields:

Parameter

Description

Name

A unique name.

MRID

Unique identifier to link the schedule to one or more devices using the API.

Communication task

Task defined in the previous step, a schedule can have more than one communication task.

Schedule

How many times does it have to be requested.

Starts from

When do the communication tasks need to be performed.

Summary

Summarises the parameters specified above.

Preview

Shows a preview of when the communication task will be executed.

3.   Click Add.

Edit shared communication schedules

5

Steps

1.   Go to Administration > Device communication > Shared communication schedules.

2.   Click the actions icon  and select Edit.

3.   Change the shared communication schedule parameters.

4.   Click Save.

Clone shared communication schedule

Clone a existing shared communication schedule to create a communication schedule that has very similar parameters.

5

Steps

1.   Go to Administration > Device communication > Shared communication schedules.

2.   Click the actions icon  and select Clone.

3.   Change the communication schedule parameters as needed.

4.   Click Clone.

Remove shared communication schedules

5

Steps

1.   Go to Administration > Device communication > Shared communication schedules.

2.   Click the actions icon  and select Remove.

3.   Click Remove.
