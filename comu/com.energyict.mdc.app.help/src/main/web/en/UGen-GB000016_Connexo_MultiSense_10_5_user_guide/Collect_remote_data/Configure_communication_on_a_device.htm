Configure communication on a device

View communication schedules on a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication planning in the menu on the left.

The overview shows the following information:

Parameter

Description

Communication task

Communication task available on this device.

Schedule

What is the schedule of this communication task. Can be specific for this device or part of a shared communication schedule.

Shared communication schedule

Shared communication schedule assigned to this communication task.

Planned date

When the communication task will be executed next.

Add shared communication schedule to a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication planning in the menu on the left.

3.   Click Add shared communication schedule.

This list only contains shared communication schedules that do not contain communication tasks that are already scheduled using another shared communication schedule on this device.

4.   Select one or more communication schedules.

5.   Click Add.

6.   Click the actions icon to remove a shared communication schedule.

3

Note

All tasks in a schedule need to have the same connection method, protocol dialect, security set and urgency.

Remove shared communication schedule from a device

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication planning in the menu on the left.

3.   Click Remove shared communication schedule.

4.   Select one or more shared communication schedules.

5.   Click Remove.

Add schedule to a communication task in the communication planning

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication planning in the menu on the left.

3.   Click the actions icon next to the communication task.

4.   Click Add schedule.

5.   Define the schedule and click Add.

Change schedule of a communication task in the communication planning

5

Steps

1.   Select a device using Search, see “Advanced search”.

2.   Click Communication planning in the menu on the left.

3.   Click the actions icon next to the communication schedule.

4.   Click Change schedule.

5.   Change the schedule and click Add.

Remove schedule of a communication task in the communication task

This action allows you to remove a single schedule that was assigned to a specific communication task on a device.

5

Steps

1.   elect a device using Search, see “Advanced search”.

2.   Click Communication planning in the menu on the left.

3.   Click the actions icon next to the communication schedule.

4.   Click Remove schedule.

5.   Confirm by clicking Remove.
